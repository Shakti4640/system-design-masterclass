🎯 HYPOTHETICAL PROBLEM

You are building a cloud object storage service like Amazon S3,
Google Cloud Storage, or Azure Blob Storage

The situation:
→ 100 MILLION users storing files
→ Total storage: 5 PETABYTES (5,000 TB)
→ 500 BILLION objects (files) stored
→ Objects range from 1 byte to 5 TB
→ Upload rate: 100,000 objects per second
→ Download rate: 500,000 objects per second
→ Users expect:
    PUT /bucket/photo.jpg → upload succeeds, data is SAFE forever
    GET /bucket/photo.jpg → download within 100ms (first byte)
    DELETE /bucket/photo.jpg → gone, storage reclaimed
    LIST /bucket/photos/ → list all objects with prefix

DURABILITY REQUIREMENT — THE HARDEST PART:
→ 99.999999999% annual durability (eleven 9s)
→ Meaning: if you store 10 BILLION objects for a year,
  you might lose AT MOST 1 object
→ For comparison:
    99.9% durability = lose 10 million objects per year (unacceptable)
    99.99% = lose 1 million (still terrible)
    99.999% = lose 100,000 (bad)
    99.9999999% = lose 10 (concerning)
    99.999999999% = lose 0.1 (less than 1 object per year) ← OUR TARGET

WHAT GOES WRONG without proper design:

  Scenario A — SINGLE DISK FAILURE:
    → Store each file on one disk
    → Enterprise disk annual failure rate: ~2%
    → 100,000 disks in cluster → 2,000 disk failures per year
    → Each disk holds 100,000 files
    → 2,000 × 100,000 = 200 MILLION files lost per year
    → Durability: ~96% — CATASTROPHIC

  Scenario B — SIMPLE REPLICATION (3 copies):
    → Store 3 copies of every file on different disks
    → File lost only if ALL 3 disks fail before repair
    → P(lose file) = P(disk1 fails) × P(disk2 fails before repair) × P(disk3)
    → Much better, but 3× storage cost
    → 5 PB of data × 3 = 15 PB of storage needed
    → Very expensive at scale

  Scenario C — METADATA LOSS:
    → You stored the file perfectly across 3 disks
    → But you lost the RECORD of which disks it's on
    → File exists but is unfindable — effectively lost
    → Metadata is the MAP to your data
    → Losing the map = losing everything

  Scenario D — NAMESPACE EXPLOSION:
    → 500 billion objects, each with a path: /bucket/path/to/file.jpg
    → Simple database table for all paths → single point of failure
    → Listing "all files in /bucket/photos/" → scan millions of rows
    → Prefix queries on 500B rows = impossible on single machine

  Scenario E — UPLOAD CORRUPTION:
    → User uploads 2GB video over flaky network
    → Network drops at 1.8GB
    → Server has partial file — is it complete? Corrupted?
    → Without checksums and multipart: data integrity unknown

THE REAL CHALLENGE:
→ Store 5 PETABYTES across thousands of machines
→ With eleven-9s durability (lose < 1 file per year)
→ At lower cost than 3× full replication
→ With a metadata system that handles 500B objects
→ With upload integrity guarantees
→ While thousands of disks fail every year
→ And the system automatically heals itself

SOLUTION CORE 1: ERASURE CODING (The Durability Secret)
═══════════════════════════════════════════════════════════

PROBLEM WITH SIMPLE REPLICATION:
  3 copies × 5 PB = 15 PB of storage
  Storage overhead: 200% (store 3x the actual data)
  Expensive!

ERASURE CODING — THE BETTER WAY:
  Instead of copying the entire file 3 times,
  ENCODE it into N fragments where ANY K fragments
  can reconstruct the original.

  EXAMPLE: Reed-Solomon (10, 4) encoding
    → Split file into 10 DATA fragments
    → Compute 4 PARITY fragments (mathematical redundancy)
    → Total: 14 fragments stored on 14 different disks
    → ANY 10 of 14 fragments can reconstruct the file
    → Can survive 4 simultaneous disk failures!

  STORAGE OVERHEAD:
    Replication (3 copies): 200% overhead (3× storage)
    Erasure Coding (10,4):  40% overhead (1.4× storage)
    
    For 5 PB of data:
      Replication: 15 PB needed
      Erasure Coding: 7 PB needed
      SAVINGS: 8 PB = millions of dollars in hardware!

  DURABILITY MATH:
    Disk annual failure rate (AFR): 2%
    Mean time to repair (MTTR): 4 hours
    
    Replication (3 copies):
      P(lose all 3 before repair) ≈ (AFR × MTTR/year)² 
      ≈ (0.02 × 4/8760)² ≈ 8.3 × 10⁻¹⁰
      Durability: ~99.9999999% (nine 9s)
    
    Erasure Coding (10,4):
      Must lose 5+ of 14 fragments simultaneously
      P(lose 5 before repair) ≈ C(14,5) × (AFR × MTTR/year)⁴
      ≈ extremely small
      Durability: >99.999999999% (eleven 9s) ✅
    
    ELEVEN 9s with only 40% overhead vs NINE 9s with 200% overhead!

  HOW ERASURE CODING WORKS (Simplified):
    Original data: [A, B, C, D] (4 data chunks)
    
    Parity computation (XOR-like, but over Galois Fields):
      P1 = A ⊕ B ⊕ C ⊕ D
      P2 = 2A ⊕ 3B ⊕ C ⊕ 4D
    
    Stored: [A, B, C, D, P1, P2] on 6 different disks
    
    If disk with B fails:
      B = A ⊕ C ⊕ D ⊕ P1  (reconstruct from remaining)
    
    If disks with B AND C fail:
      Solve system of equations using P1 and P2
      Both B and C can be recovered!
    
    In production: Reed-Solomon codes over GF(2^8)
    Implementation: use a library (liberasurecode, Intel ISA-L)

SOLUTION CORE 3: UPLOAD PIPELINE
═══════════════════════════════════════════════════════════

  SMALL FILE UPLOAD (< 64 MB):
    Client ──PUT─→ API Gateway ──→ Data Service
    
    Data Service:
      1. Receive complete file into memory/temp disk
      2. Compute checksum (SHA-256)
      3. Verify checksum matches client-provided checksum
      4. Split into K data chunks
      5. Compute N-K parity chunks (erasure coding)
      6. Write each chunk to a different storage node
      7. Wait for all writes to acknowledge
      8. Write metadata record (atomic)
      9. Return success to client

  LARGE FILE UPLOAD (> 64 MB — Multipart):
    Reference: Project 25 (Chunked Upload)
    
    Step 1: Client → InitiateMultipartUpload
      → Server creates upload_id, allocates temp space
    
    Step 2: Client → UploadPart(upload_id, part_number, data)
      → Each part: 64MB-512MB
      → Client uploads parts in parallel
      → Each part checksummed independently
      → Server stores parts temporarily
    
    Step 3: Client → CompleteMultipartUpload(upload_id, part_list)
      → Server verifies all parts received
      → Assembles into final object
      → Erasure-encodes the complete object
      → Writes fragments to storage nodes
      → Writes metadata (atomic)
      → Cleans up temporary parts
    
    Step 4 (failure): Client → AbortMultipartUpload(upload_id)
      → Server cleans up temporary parts
      → No fragments stored permanently
      → No metadata created

  ATOMIC COMMIT — THE CRITICAL PROPERTY:
    → Object is either FULLY stored and findable, or NOT AT ALL
    → Never: "file partially uploaded and visible to other users"
    → Implementation: write all fragments first, write metadata LAST
    → If crash before metadata write: fragments are orphaned → GC cleans up
    → If crash after metadata write: object is complete ✅

📌 WHAT COULD BE DONE

OPTION A: TRIPLE REPLICATION (Simple, Expensive)
  → Store 3 full copies on different nodes
  → Read from any copy (fast reads)
  → 200% storage overhead
  → Nine 9s durability
  → Used by: HDFS default, GFS, early S3

OPTION B: ERASURE CODING (Our primary approach)
  → Reed-Solomon (10,4) or similar
  → 40% storage overhead
  → Eleven 9s durability
  → Read requires contacting 10+ nodes (higher latency)
  → Used by: Modern S3, Azure, Google Colossus

OPTION C: HYBRID (Production approach)
  → Hot data (frequently accessed): triple replication
    → Fast reads (any single copy suffices)
  → Warm/cold data (rarely accessed): erasure coding
    → Cheaper storage, slightly slower reads
  → Automatic tiering based on access patterns
  → Used by: S3 Intelligent-Tiering

OPTION D: LOCAL REDUNDANCY ONLY (Budget)
  → RAID-6 on each storage node (survives 2 disk failures)
  → No cross-node redundancy
  → Node failure = data loss
  → Only acceptable for reproducible/cacheable data

OUR DESIGN: OPTION B (Erasure Coding) + OPTION C (Hot tier replication)
  → Default: Reed-Solomon (10,4) erasure coding
  → Hot objects: additional replication for read performance
  → Automatic re-encoding when fragments are lost
  → Background data integrity verification (scrubbing)

✅ WHAT SHOULD DO

1. CHECKSUM EVERYTHING AT EVERY LAYER
   → Client computes SHA-256 before upload
   → Server verifies checksum on receipt
   → Each fragment has its own checksum
   → Storage node verifies on write AND on read
   → Background scrubber verifies periodically
   → ANY mismatch = corruption detected = trigger repair

2. SPREAD FRAGMENTS ACROSS FAILURE DOMAINS
   → Different disks (survive disk failure)
   → Different nodes (survive machine failure)
   → Different racks (survive rack failure)
   → Different zones (survive zone failure)
   → NEVER put 2 fragments of same object on same rack

3. METADATA IS MORE CRITICAL THAN DATA — PROTECT IT MORE
   → 5× replication for metadata (vs 1.4× for data)
   → Metadata on SSDs (faster, more reliable than HDD)
   → Metadata backed up to separate system
   → If metadata service is down: entire storage is inaccessible
   → Treat metadata service as the BRAIN of the system

4. IMPLEMENT WRITE-AHEAD LOG FOR METADATA
   → Before writing metadata: log the intent
   → If crash during write: replay log on recovery
   → Ensures atomic metadata updates
   → Same concept as database WAL (Project 3)

5. BACKGROUND DATA SCRUBBING
   → Continuously read and verify fragments
   → Detect "bit rot" (silent data corruption on disk)
   → Disks can corrupt data without reporting errors
   → Scrub every fragment at least once per month
   → On corruption: re-encode from healthy fragments

6. AUTO-REPAIR ON FAILURE DETECTION
   → Storage node goes offline → detect within seconds
   → Immediately begin re-encoding affected fragments
   → Use remaining healthy fragments to reconstruct lost ones
   → Place new fragments on healthy nodes
   → Goal: restore full redundancy within hours

7. SUPPORT VERSIONING
   → Overwrite = create new version, keep old
   → Protects against accidental overwrites
   → Enables "undo delete" within retention period
   → Old versions eventually garbage collected

8. USE CONTENT-ADDRESSABLE STORAGE FOR DEDUP
   → Hash of content = storage address
   → If two users upload identical file → stored once
   → Reference: Project 25 (deduplication concept)
   → Saves significant storage for common files

❌ WHAT SHOULD NOT DO

1. NEVER STORE ALL FRAGMENTS ON THE SAME RACK
   → Rack power failure = lose all fragments = lose file
   → Defeats the entire purpose of erasure coding
   → ALWAYS enforce rack-awareness in placement

2. NEVER TRUST DISK HARDWARE BLINDLY
   → Disks can return corrupted data without error
   → "Silent data corruption" / "bit rot"
   → ALWAYS verify checksums on every read
   → ALWAYS run background scrubbing

3. NEVER MAKE METADATA A SINGLE POINT OF FAILURE
   → Single metadata server = if it dies, entire system is blind
   → ALWAYS replicate metadata across multiple nodes/zones
   → ALWAYS have hot standby for metadata service

4. NEVER EXPOSE INTERNAL STORAGE LAYOUT TO CLIENTS
   → Client sees: /bucket/photo.jpg
   → Internally: 14 fragments across 14 nodes
   → Client NEVER knows or cares about internal layout
   → This allows transparent data movement and rebalancing

5. NEVER DELETE DATA IMMEDIATELY ON DELETE REQUEST
   → Soft-delete first (mark as deleted in metadata)
   → Grace period (24-72 hours) before physical deletion
   → Protects against accidental deletes and malicious actors
   → Background GC reclaims storage after grace period

6. NEVER SKIP THE UPLOAD CHECKSUM VERIFICATION
   → Client sends checksum with upload
   → Server MUST verify before acknowledging success
   → Without verification: network corruption goes undetected
   → User thinks file is safe, but it's corrupted

7. NEVER ALLOW PARTIAL UPLOADS TO BE VISIBLE
   → Multipart upload: parts are temporary until committed
   → Other users must NOT see partial objects
   → Atomic visibility: object appears only after full commit

8. NEVER REBALANCE ALL DATA AT ONCE
   → New node added → move data to it gradually
   → Moving 50TB at once: saturates network, degrades service
   → Throttle rebalancing to 10-20% of available bandwidth
   → Spread over hours/days, not minutes

═══════════════════════════════════════════════════════════
MECHANICS 1: ERASURE CODING — STEP BY STEP
═══════════════════════════════════════════════════════════

  ENCODING (Reed-Solomon simplified):

  Original file: "HELLO WORLD!!!" (14 bytes)
  Configuration: RS(4, 2) → 4 data fragments + 2 parity

  Step 1: SPLIT into data fragments
    Fragment D0: "HEL"
    Fragment D1: "LO "
    Fragment D2: "WOR"
    Fragment D3: "LD!"

  Step 2: COMPUTE parity fragments
    Using Galois Field arithmetic (GF(2^8)):
    
    P0 = D0 ⊕ D1 ⊕ D2 ⊕ D3
       = "HEL" ⊕ "LO " ⊕ "WOR" ⊕ "LD!"
       = [computed bytes]
    
    P1 = 2·D0 ⊕ 3·D1 ⊕ D2 ⊕ 4·D3
       = [computed bytes]
    
    (Actual math uses Vandermonde/Cauchy matrices over GF(2^8))

  Step 3: STORE 6 fragments on 6 different nodes
    Node A: D0 "HEL"
    Node B: D1 "LO "
    Node C: D2 "WOR"
    Node D: D3 "LD!"
    Node E: P0 [parity]
    Node F: P1 [parity]

  DECODING (Normal — all fragments available):
    Read D0, D1, D2, D3 → concatenate → "HELLO WORLD!!!"
    Parity fragments not needed (but verified for integrity)

  DECODING (Degraded — Node B and Node D failed):
    Available: D0, D2, P0, P1
    Missing: D1, D3
    
    System of equations:
      P0 = D0 ⊕ D1 ⊕ D2 ⊕ D3
      P1 = 2·D0 ⊕ 3·D1 ⊕ D2 ⊕ 4·D3
    
    Known: D0, D2, P0, P1
    Solve for: D1, D3
    
    D1 = P0 ⊕ D0 ⊕ D2 ⊕ D3  → but D3 is also unknown
    Use both equations simultaneously (matrix inversion):
    
    [P0 - D0 - D2]   [1  1] [D1]
    [P1 - 2D0 - D2] = [3  4] [D3]
    
    Invert the 2×2 matrix over GF(2^8) → solve for D1, D3
    
    Result: D1 = "LO ", D3 = "LD!" — RECOVERED! ✅

MECHANICS 2: OBJECT UPLOAD — COMPLETE FLOW
═══════════════════════════════════════════════════════════

  Client: PUT /my-bucket/photos/vacation/beach.jpg (5 MB)

  T+0ms: REQUEST ARRIVES AT API GATEWAY
    → Authenticate user (Project 8)
    → Check bucket permissions
    → Rate limit check (Project 9)
    → Route to data service

  T+5ms: DATA SERVICE RECEIVES FILE
    → Buffer entire file in memory (5 MB — small enough)
    → Compute SHA-256 checksum
    → Verify against client-provided checksum (if any)

  T+10ms: ERASURE ENCODE
    → Split 5 MB into 10 data fragments (512 KB each)
    → Compute 4 parity fragments (512 KB each)
    → 14 fragments total, each 512 KB

  T+15ms: SELECT PLACEMENT NODES
    → Consult placement service
    → Get 14 storage nodes across different racks/zones
    → Placement rule: no 2 fragments on same rack
    → Weight by available space, disk health, network proximity

  T+20ms: WRITE FRAGMENTS (parallel)
    → Send each fragment to its assigned storage node
    → Each storage node:
      • Receives fragment
      • Verifies checksum
      • Writes to disk
      • Updates local index
      • Acknowledges write

class ErasureCodingEngine:
    """
    Simulates Reed-Solomon erasure coding.
    
    In production: use Intel ISA-L or liberasurecode for GF(2^8) math.
    Here we simulate the behavior to demonstrate the concept.
    
    RS(data_fragments, parity_fragments):
    → Split data into data_fragments pieces
    → Generate parity_fragments additional pieces
    → Any data_fragments of total can reconstruct original
    """

    def __init__(self, data_fragments: int = 10, parity_fragments: int = 4):
        self.k = data_fragments       # Minimum fragments needed
        self.m = parity_fragments     # Extra parity fragments
        self.n = data_fragments + parity_fragments  # Total fragments
        self.encoding_id = f"RS({self.k},{self.m})"

    def encode(self, data: bytes) -> List[dict]:
        """
        Encode data into N fragments.
        Returns list of fragment dicts with metadata.
        """
        # Pad data to be divisible by k
        fragment_size = math.ceil(len(data) / self.k)
        padded_data = data + b'\x00' * (fragment_size * self.k - len(data))

        fragments = []

        # Create data fragments (actual data chunks)
        for i in range(self.k):
            start = i * fragment_size
            end = start + fragment_size
            chunk = padded_data[start:end]
            checksum = hashlib.sha256(chunk).hexdigest()

            fragments.append({
                "fragment_id": i,
                "fragment_type": "data",
                "data": chunk,
                "size": len(chunk),
                "checksum": checksum,
                "encoding": self.encoding_id,
            })

        # Create parity fragments (computed from data)
        for i in range(self.m):
            parity = self._compute_parity(padded_data, fragment_size, i)
            checksum = hashlib.sha256(parity).hexdigest()

            fragments.append({
                "fragment_id": self.k + i,
                "fragment_type": "parity",
                "data": parity,
                "size": len(parity),
                "checksum": checksum,
                "encoding": self.encoding_id,
            })

        return fragments

    def _compute_parity(
        self, padded_data: bytes, fragment_size: int, parity_index: int
    ) -> bytes:
        """
        Compute parity fragment using XOR-based simulation.
        
        Real Reed-Solomon: uses Galois Field arithmetic with
        Vandermonde or Cauchy matrices. Much more sophisticated.
        
        Our simulation: XOR with byte-level rotation for different parities.
        Demonstrates the CONCEPT even if not mathematically identical.
        """
        parity = bytearray(fragment_size)

        for i in range(self.k):
            start = i * fragment_size
            chunk = padded_data[start:start + fragment_size]

            for j in range(fragment_size):
                # Different parity computations for each parity fragment
                if parity_index == 0:
                    parity[j] ^= chunk[j]
                else:
                    # Rotate bytes by different amounts for different parities
                    rotated_idx = (j + (parity_index * (i + 1))) % fragment_size
                    parity[j] ^= chunk[rotated_idx]

        return bytes(parity)

    def decode(
        self, fragments: List[dict], original_size: int
    ) -> Optional[bytes]:
        """
        Reconstruct original data from available fragments.
        Need at least K fragments to reconstruct.
        """
        # Sort by fragment_id
        available = sorted(fragments, key=lambda f: f["fragment_id"])

        # Verify checksums
        for frag in available:
            computed = hashlib.sha256(frag["data"]).hexdigest()
            if computed != frag["checksum"]:
                return None  # Corrupted fragment

        if len(available) < self.k:
            return None  # Not enough fragments

        # Get data fragments
        data_frags = [f for f in available if f["fragment_type"] == "data"]

        if len(data_frags) >= self.k:
            # All data fragments available — simple concatenation
            data_frags.sort(key=lambda f: f["fragment_id"])
            reconstructed = b"".join(f["data"] for f in data_frags[:self.k])
            return reconstructed[:original_size]

        # Degraded mode: need parity fragments to reconstruct
        # In real RS: solve system of equations over GF(2^8)
        # Simulation: use XOR reconstruction for first parity case
        parity_frags = [f for f in available if f["fragment_type"] == "parity"]

        if not parity_frags:
            return None

        # Simplified reconstruction using first parity (XOR)
        # Real implementation would handle any combination
        fragment_size = data_frags[0]["size"] if data_frags else parity_frags[0]["size"]
        missing_ids = set(range(self.k)) - {f["fragment_id"] for f in data_frags}

        if len(missing_ids) == 1 and parity_frags:
            # Can reconstruct one missing fragment from XOR parity
            missing_id = missing_ids.pop()
            xor_parity = parity_frags[0]

            reconstructed_chunk = bytearray(xor_parity["data"])
            for frag in data_frags:
                for j in range(fragment_size):
                    reconstructed_chunk[j] ^= frag["data"][j]

            # Rebuild complete data
            all_chunks = {}
            for frag in data_frags:
                all_chunks[frag["fragment_id"]] = frag["data"]
            all_chunks[missing_id] = bytes(reconstructed_chunk)

            result = b""
            for i in range(self.k):
                result += all_chunks[i]

            return result[:original_size]

        # Multiple missing — would need full RS decoding
        # For demo: indicate that we need more fragments
        return None

    def get_durability(
        self,
        disk_afr: float = 0.02,
        mttr_hours: float = 4.0,
        num_disks: int = 14,
    ) -> dict:
        """
        Calculate durability based on erasure coding parameters.
        
        Durability = probability that we DON'T lose enough fragments
        to make the object unrecoverable in a year.
        """
        # Probability of a single disk failing during repair window
        p_fail_during_repair = disk_afr * (mttr_hours / 8760)

        # Must lose more than m fragments to lose data
        # P(lose data) = P(m+1 or more fragments fail simultaneously)
        # Approximation using binomial distribution
        max_tolerable = self.m  # Can tolerate up to m failures
        need_to_fail = max_tolerable + 1  # This many must fail to lose data

        # Simplified: P(k failures) ≈ C(n,k) × p^k
        from math import comb
        p_lose = 0
        for k in range(need_to_fail, self.n + 1):
            p_lose += comb(self.n, k) * (p_fail_during_repair ** k) * \
                      ((1 - p_fail_during_repair) ** (self.n - k))

        durability = 1 - p_lose
        nines = -math.log10(max(1 - durability, 1e-15))

        return {
            "encoding": self.encoding_id,
            "data_fragments": self.k,
            "parity_fragments": self.m,
            "total_fragments": self.n,
            "storage_overhead": f"{(self.m / self.k) * 100:.0f}%",
            "max_failures_tolerated": self.m,
            "durability": f"{durability:.15f}",
            "nines_of_durability": f"{nines:.1f}",
            "p_data_loss_per_year": f"{p_lose:.2e}",
        }

class StorageNodeStatus(Enum):
    HEALTHY = "HEALTHY"
    SUSPECT = "SUSPECT"
    DEAD = "DEAD"
    DECOMMISSIONING = "DECOMMISSIONING"


@dataclass
class DiskInfo:
    disk_id: str
    capacity_bytes: int
    used_bytes: int = 0
    fragment_count: int = 0
    error_count: int = 0
    healthy: bool = True


class StorageNode:
    """
    Simulates a storage machine with multiple disks.
    Stores and retrieves erasure-coded fragments.
    """

    def __init__(
        self, node_id: str, rack_id: str, zone_id: str,
        num_disks: int = 12, disk_capacity_gb: int = 8000,
    ):
        self.node_id = node_id
        self.rack_id = rack_id
        self.zone_id = zone_id
        self.status = StorageNodeStatus.HEALTHY
        self.last_heartbeat = time.time()

        # Initialize disks
        self.disks = {}
        for i in range(num_disks):
            disk_id = f"{node_id}_disk_{i}"
            self.disks[disk_id] = DiskInfo(
                disk_id=disk_id,
                capacity_bytes=disk_capacity_gb * 1024 * 1024 * 1024,
            )

        # Fragment storage (in-memory simulation)
        self.fragments: Dict[str, dict] = {}
        self._lock = threading.Lock()

        # Metrics
        self.reads = 0
        self.writes = 0
        self.errors = 0

    def write_fragment(self, fragment_key: str, fragment_data: dict) -> bool:
        """Store a fragment on this node"""
        with self._lock:
            if self.status != StorageNodeStatus.HEALTHY:
                self.errors += 1
                return False

            # Select disk with most free space
            best_disk = None
            best_free = -1
            for disk in self.disks.values():
                if not disk.healthy:
                    continue
                free = disk.capacity_bytes - disk.used_bytes
                if free > best_free:
                    best_free = free
                    best_disk = disk

            if not best_disk or best_free < fragment_data["size"]:
                return False

            # Store fragment
            self.fragments[fragment_key] = {
                **fragment_data,
                "disk_id": best_disk.disk_id,
                "stored_at": time.time(),
            }

            best_disk.used_bytes += fragment_data["size"]
            best_disk.fragment_count += 1
            self.writes += 1

            return True

    def read_fragment(self, fragment_key: str) -> Optional[dict]:
        """Read a fragment from this node"""
        with self._lock:
            if self.status == StorageNodeStatus.DEAD:
                self.errors += 1
                return None

            fragment = self.fragments.get(fragment_key)
            if fragment:
                # Verify checksum on read
                computed = hashlib.sha256(fragment["data"]).hexdigest()
                if computed != fragment["checksum"]:
                    self.errors += 1
                    return None  # Corruption detected!

                self.reads += 1
                return fragment

            return None

    def delete_fragment(self, fragment_key: str) -> bool:
        """Delete a fragment from this node"""
        with self._lock:
            if fragment_key in self.fragments:
                frag = self.fragments.pop(fragment_key)
                disk = None
                for d in self.disks.values():
                    if d.disk_id == frag.get("disk_id"):
                        disk = d
                        break
                if disk:
                    disk.used_bytes -= frag["size"]
                    disk.fragment_count -= 1
                return True
            return False

    def get_stats(self) -> dict:
        with self._lock:
            total_capacity = sum(d.capacity_bytes for d in self.disks.values())
            total_used = sum(d.used_bytes for d in self.disks.values())
            return {
                "node_id": self.node_id,
                "rack_id": self.rack_id,
                "zone_id": self.zone_id,
                "status": self.status.value,
                "num_disks": len(self.disks),
                "healthy_disks": sum(1 for d in self.disks.values() if d.healthy),
                "total_capacity_gb": round(total_capacity / (1024**3)),
                "used_gb": round(total_used / (1024**3), 2),
                "utilization": f"{total_used / max(total_capacity, 1) * 100:.1f}%",
                "fragment_count": len(self.fragments),
                "reads": self.reads,
                "writes": self.writes,
                "errors": self.errors,
            }

    def heartbeat(self):
        self.last_heartbeat = time.time()

    def simulate_failure(self):
        """Simulate node failure for testing"""
        self.status = StorageNodeStatus.DEAD

# SECTION 3: PLACEMENT SERVICE
# Decides where to store fragments (rack/zone awareness)
# ================================================================

class PlacementService:
    """
    Decides which storage nodes should hold each fragment.
    
    Rules:
    1. No two fragments of same object on same node
    2. No two fragments of same object on same rack (if possible)
    3. Spread across zones for maximum durability
    4. Prefer nodes with more free space
    5. Prefer nodes with fewer errors
    """

    def __init__(self):
        self.nodes: Dict[str, StorageNode] = {}
        self.racks: Dict[str, Set[str]] = defaultdict(set)   # rack → node_ids
        self.zones: Dict[str, Set[str]] = defaultdict(set)   # zone → node_ids

    def register_node(self, node: StorageNode):
        self.nodes[node.node_id] = node
        self.racks[node.rack_id].add(node.node_id)
        self.zones[node.zone_id].add(node.node_id)

    def select_nodes(
        self, num_fragments: int, exclude_nodes: Set[str] = None
    ) -> List[StorageNode]:
        """
        Select nodes for fragment placement.
        Maximizes failure domain diversity.
        """
        exclude = exclude_nodes or set()
        selected = []
        used_racks = set()
        used_zones = set()

        # Get healthy nodes sorted by available space (most free first)
        healthy_nodes = [
            n for n in self.nodes.values()
            if n.status == StorageNodeStatus.HEALTHY
            and n.node_id not in exclude
        ]

        # Score each node based on diversity and capacity
        def node_score(node):
            rack_penalty = 100 if node.rack_id in used_racks else 0
            zone_penalty = 50 if node.zone_id in used_zones else 0
            error_penalty = node.errors * 10

            total_capacity = sum(d.capacity_bytes for d in node.disks.values())
            total_used = sum(d.used_bytes for d in node.disks.values())
            free_ratio = (total_capacity - total_used) / max(total_capacity, 1)

            # Higher score = better placement candidate
            return free_ratio * 100 - rack_penalty - zone_penalty - error_penalty

        for _ in range(num_fragments):
            # Re-score with updated used_racks/used_zones
            candidates = [
                n for n in healthy_nodes
                if n.node_id not in {s.node_id for s in selected}
            ]

            if not candidates:
                break

            candidates.sort(key=node_score, reverse=True)
            chosen = candidates[0]
            selected.append(chosen)
            used_racks.add(chosen.rack_id)
            used_zones.add(chosen.zone_id)

        return selected

    def get_placement_stats(self) -> dict:
        total_nodes = len(self.nodes)
        healthy = sum(
            1 for n in self.nodes.values()
            if n.status == StorageNodeStatus.HEALTHY
        )
        return {
            "total_nodes": total_nodes,
            "healthy_nodes": healthy,
            "dead_nodes": total_nodes - healthy,
            "num_racks": len(self.racks),
            "num_zones": len(self.zones),
            "nodes_per_rack": {
                rack: len(nodes) for rack, nodes in self.racks.items()
            },
        }

# SECTION 4: METADATA SERVICE
# Maps object paths to fragment locations
# Reference: Project 18 (Sharding)
# ================================================================

class ObjectVersion:
    def __init__(self):
        self.version_id = str(uuid.uuid4().hex[:12])
        self.created_at = time.time()


@dataclass
class ObjectMetadata:
    bucket: str
    key: str
    size: int
    checksum_sha256: str
    content_type: str
    encoding: str
    fragment_locations: List[dict]   # [{fragment_id, node_id, fragment_key}]
    version_id: str = ""
    created_at: float = 0.0
    deleted: bool = False
    deleted_at: float = 0.0
    storage_class: str = "STANDARD"


class MetadataService:
    """
    Metadata service managing object namespace.
    
    Sharded by hash(bucket + key) for write distribution.
    Supports prefix listing within buckets.
    Replicated for high availability.
    """

    def __init__(self, num_shards: int = 4):
        self.num_shards = num_shards
        # Each shard: {bucket/key → ObjectMetadata}
        self.shards: List[Dict[str, ObjectMetadata]] = [
            {} for _ in range(num_shards)
        ]
        self._lock = threading.Lock()
        self.total_objects = 0
        self.total_bytes = 0
        self.operations = {"put": 0, "get": 0, "delete": 0, "list": 0}

    def _get_shard(self, bucket: str, key: str) -> int:
        full_path = f"{bucket}/{key}"
        return hash(full_path) % self.num_shards

    def _full_key(self, bucket: str, key: str) -> str:
        return f"{bucket}/{key}"

    def put_metadata(self, metadata: ObjectMetadata) -> bool:
        """Store or update object metadata (atomic commit point)"""
        with self._lock:
            shard_id = self._get_shard(metadata.bucket, metadata.key)
            full_key = self._full_key(metadata.bucket, metadata.key)

            # Check if overwriting existing object
            existing = self.shards[shard_id].get(full_key)
            if existing and not existing.deleted:
                self.total_bytes -= existing.size

            metadata.created_at = time.time()
            metadata.version_id = str(uuid.uuid4().hex[:12])
            self.shards[shard_id][full_key] = metadata

            self.total_objects += 1 if not existing else 0
            self.total_bytes += metadata.size
            self.operations["put"] += 1
            return True

    def get_metadata(self, bucket: str, key: str) -> Optional[ObjectMetadata]:
        """Retrieve object metadata"""
        with self._lock:
            shard_id = self._get_shard(bucket, key)
            full_key = self._full_key(bucket, key)
            self.operations["get"] += 1

            meta = self.shards[shard_id].get(full_key)
            if meta and not meta.deleted:
                return meta
            return None

    def delete_metadata(self, bucket: str, key: str) -> bool:
        """Soft-delete object (mark as deleted, don't remove immediately)"""
        with self._lock:
            shard_id = self._get_shard(bucket, key)
            full_key = self._full_key(bucket, key)
            self.operations["delete"] += 1

            meta = self.shards[shard_id].get(full_key)
            if meta and not meta.deleted:
                meta.deleted = True
                meta.deleted_at = time.time()
                self.total_bytes -= meta.size
                return True
            return False

    def list_objects(
        self, bucket: str, prefix: str = "", max_keys: int = 1000
    ) -> List[ObjectMetadata]:
        """List objects in bucket with optional prefix filter"""
        with self._lock:
            self.operations["list"] += 1
            results = []
            target_prefix = self._full_key(bucket, prefix)

            for shard in self.shards:
                for full_key, meta in shard.items():
                    if (full_key.startswith(target_prefix)
                            and not meta.deleted):
                        results.append(meta)
                        if len(results) >= max_keys:
                            break
                if len(results) >= max_keys:
                    break

            # Sort by key for consistent listing
            results.sort(key=lambda m: m.key)
            return results[:max_keys]

    def get_objects_on_node(self, node_id: str) -> List[ObjectMetadata]:
        """Find all objects with fragments on a specific node"""
        with self._lock:
            affected = []
            for shard in self.shards:
                for meta in shard.values():
                    if meta.deleted:
                        continue
                    for frag in meta.fragment_locations:
                        if frag["node_id"] == node_id:
                            affected.append(meta)
                            break
            return affected

    def update_fragment_location(
        self, bucket: str, key: str,
        old_node_id: str, new_node_id: str,
        new_fragment_key: str,
    ) -> bool:
        """Update fragment location after repair/rebalance"""
        with self._lock:
            shard_id = self._get_shard(bucket, key)
            full_key = self._full_key(bucket, key)
            meta = self.shards[shard_id].get(full_key)

            if not meta:
                return False

            for frag in meta.fragment_locations:
                if frag["node_id"] == old_node_id:
                    frag["node_id"] = new_node_id
                    frag["fragment_key"] = new_fragment_key
                    return True
            return False

    def get_stats(self) -> dict:
        with self._lock:
            shard_sizes = [len(s) for s in self.shards]
            deleted_count = sum(
                1 for s in self.shards
                for m in s.values() if m.deleted
            )
            return {
                "total_objects": self.total_objects,
                "deleted_pending_gc": deleted_count,
                "total_bytes": self.total_bytes,
                "total_bytes_human": self._human_size(self.total_bytes),
                "num_shards": self.num_shards,
                "shard_sizes": shard_sizes,
                "operations": dict(self.operations),
            }

    @staticmethod
    def _human_size(size_bytes: int) -> str:
        for unit in ['B', 'KB', 'MB', 'GB', 'TB', 'PB']:
            if size_bytes < 1024:
                return f"{size_bytes:.1f} {unit}"
            size_bytes /= 1024
        return f"{size_bytes:.1f} EB"

# SECTION 5: REPAIR SERVICE
# Detects failures and reconstructs lost fragments
# ================================================================

class RepairService:
    """
    Background service that:
    1. Detects node failures via heartbeat monitoring
    2. Finds objects affected by failures
    3. Reconstructs lost fragments from surviving ones
    4. Places new fragments on healthy nodes
    5. Updates metadata
    """

    def __init__(
        self,
        placement: PlacementService,
        metadata: MetadataService,
        erasure_engine: ErasureCodingEngine,
    ):
        self.placement = placement
        self.metadata = metadata
        self.erasure_engine = erasure_engine
        self.repair_log = []
        self.fragments_repaired = 0
        self.objects_repaired = 0
        self.repair_failures = 0

    def detect_dead_nodes(self, timeout_seconds: float = 300) -> List[str]:
        """Find nodes that haven't sent heartbeat recently"""
        now = time.time()
        dead_nodes = []

        for node_id, node in self.placement.nodes.items():
            if node.status == StorageNodeStatus.DEAD:
                dead_nodes.append(node_id)
            elif now - node.last_heartbeat > timeout_seconds:
                node.status = StorageNodeStatus.SUSPECT
                if now - node.last_heartbeat > timeout_seconds * 2:
                    node.status = StorageNodeStatus.DEAD
                    dead_nodes.append(node_id)

        return dead_nodes

    def repair_node_failure(self, dead_node_id: str) -> dict:
        """Repair all objects affected by a node failure"""
        repair_start = time.time()

        # Find all objects with fragments on the dead node
        affected_objects = self.metadata.get_objects_on_node(dead_node_id)

        repaired = 0
        failed = 0

        for obj_meta in affected_objects:
            success = self._repair_object(obj_meta, dead_node_id)
            if success:
                repaired += 1
                self.objects_repaired += 1
            else:
                failed += 1
                self.repair_failures += 1

        elapsed = time.time() - repair_start

        result = {
            "dead_node": dead_node_id,
            "affected_objects": len(affected_objects),
            "repaired": repaired,
            "failed": failed,
            "elapsed_seconds": round(elapsed, 3),
        }

        self.repair_log.append(result)
        return result

    def _repair_object(self, obj_meta: ObjectMetadata, dead_node_id: str) -> bool:
        """Repair a single object by reconstructing lost fragment"""
        # Collect surviving fragments
        surviving_fragments = []
        lost_fragment_info = None

        for frag_loc in obj_meta.fragment_locations:
            if frag_loc["node_id"] == dead_node_id:
                lost_fragment_info = frag_loc
                continue

            node = self.placement.nodes.get(frag_loc["node_id"])
            if node and node.status == StorageNodeStatus.HEALTHY:
                fragment = node.read_fragment(frag_loc["fragment_key"])
                if fragment:
                    surviving_fragments.append(fragment)

        if not lost_fragment_info:
            return True  # No fragment on dead node

        # Check if we have enough fragments to reconstruct
        if len(surviving_fragments) < self.erasure_engine.k:
            return False  # Not enough fragments — DATA LOSS!

        # Reconstruct the original data
        original_data = self.erasure_engine.decode(
            surviving_fragments, obj_meta.size
        )

        if original_data is None:
            return False

        # Re-encode to get the missing fragment
        all_fragments = self.erasure_engine.encode(original_data)
        lost_frag_id = lost_fragment_info.get("fragment_id", 0)

        # Find the specific fragment we need
        replacement_fragment = None
        for frag in all_fragments:
            if frag["fragment_id"] == lost_frag_id:
                replacement_fragment = frag
                break

        if not replacement_fragment:
            return False

        # Place on a new healthy node
        exclude = {
            fl["node_id"] for fl in obj_meta.fragment_locations
            if fl["node_id"] != dead_node_id
        }
        new_nodes = self.placement.select_nodes(1, exclude_nodes=exclude)

        if not new_nodes:
            return False

        new_node = new_nodes[0]
        new_fragment_key = f"frag_{uuid.uuid4().hex[:16]}"

        success = new_node.write_fragment(new_fragment_key, replacement_fragment)
        if not success:
            return False

        # Update metadata
        self.metadata.update_fragment_location(
            obj_meta.bucket, obj_meta.key,
            dead_node_id, new_node.node_id, new_fragment_key,
        )

        self.fragments_repaired += 1
        return True

    def get_stats(self) -> dict:
        return {
            "objects_repaired": self.objects_repaired,
            "fragments_repaired": self.fragments_repaired,
            "repair_failures": self.repair_failures,
            "repair_log_entries": len(self.repair_log),
        }

# SECTION 6: DATA SCRUBBER
# Background integrity verification
# ================================================================

class DataScrubber:
    """
    Continuously reads and verifies fragment integrity.
    Detects bit rot (silent corruption) before it's needed.
    """

    def __init__(
        self, placement: PlacementService, metadata: MetadataService
    ):
        self.placement = placement
        self.metadata = metadata
        self.fragments_scanned = 0
        self.corruptions_found = 0
        self.scan_log = []

    def scrub_node(self, node_id: str) -> dict:
        """Verify all fragments on a specific node"""
        node = self.placement.nodes.get(node_id)
        if not node or node.status == StorageNodeStatus.DEAD:
            return {"node_id": node_id, "status": "skipped", "reason": "dead"}

        scanned = 0
        corrupted = 0
        healthy = 0

        for frag_key, fragment in list(node.fragments.items()):
            scanned += 1
            self.fragments_scanned += 1

            # Verify checksum
            computed = hashlib.sha256(fragment["data"]).hexdigest()
            if computed != fragment["checksum"]:
                corrupted += 1
                self.corruptions_found += 1
            else:
                healthy += 1

        result = {
            "node_id": node_id,
            "fragments_scanned": scanned,
            "healthy": healthy,
            "corrupted": corrupted,
        }
        self.scan_log.append(result)
        return result

    def full_scrub(self) -> dict:
        """Scrub all nodes in the cluster"""
        results = []
        total_corrupted = 0

        for node_id in self.placement.nodes:
            result = self.scrub_node(node_id)
            results.append(result)
            total_corrupted += result.get("corrupted", 0)

        return {
            "nodes_scanned": len(results),
            "total_corruptions": total_corrupted,
            "details": results,
        }

# SECTION 7: OBJECT STORAGE SERVICE (The Complete System)
# Orchestrates all components
# ================================================================

class ObjectStorageService:
    """
    Complete object storage service orchestrating:
    → Erasure coding
    → Fragment placement
    → Metadata management
    → Upload/download pipeline
    → Repair and scrubbing
    """

    def __init__(
        self,
        data_fragments: int = 4,
        parity_fragments: int = 2,
        metadata_shards: int = 4,
    ):
        self.erasure_engine = ErasureCodingEngine(data_fragments, parity_fragments)
        self.placement = PlacementService()
        self.metadata = MetadataService(num_shards=metadata_shards)
        self.repair_service = RepairService(
            self.placement, self.metadata, self.erasure_engine
        )
        self.scrubber = DataScrubber(self.placement, self.metadata)

        # Metrics
        self.uploads = 0
        self.downloads = 0
        self.upload_bytes = 0
        self.download_bytes = 0
        self.failed_uploads = 0
        self.failed_downloads = 0
        self.degraded_reads = 0

    def setup_cluster(
        self, num_nodes: int = 12, racks_per_zone: int = 2,
        zones: int = 3, disks_per_node: int = 4,
    ):
        """Initialize storage cluster with nodes across racks and zones"""
        node_idx = 0
        for zone in range(zones):
            zone_id = f"zone_{zone}"
            for rack in range(racks_per_zone):
                rack_id = f"rack_{zone}_{rack}"
                nodes_per_rack = num_nodes // (zones * racks_per_zone)
                for n in range(max(nodes_per_rack, 1)):
                    node_id = f"node_{node_idx:03d}"
                    node = StorageNode(
                        node_id=node_id,
                        rack_id=rack_id,
                        zone_id=zone_id,
                        num_disks=disks_per_node,
                        disk_capacity_gb=100,
                    )
                    self.placement.register_node(node)
                    node_idx += 1

    def put_object(
        self, bucket: str, key: str, data: bytes,
        content_type: str = "application/octet-stream",
    ) -> dict:
        """
        Upload an object to the storage system.
        
        Pipeline:
        1. Compute checksum
        2. Erasure encode
        3. Place fragments on selected nodes
        4. Write metadata (atomic commit)
        """
        start_time = time.time()

        # Step 1: Compute checksum
        checksum = hashlib.sha256(data).hexdigest()

        # Step 2: Erasure encode
        fragments = self.erasure_engine.encode(data)
        num_fragments = len(fragments)

        # Step 3: Select placement nodes
        nodes = self.placement.select_nodes(num_fragments)

        if len(nodes) < num_fragments:
            self.failed_uploads += 1
            return {
                "success": False,
                "error": f"Not enough healthy nodes. Need {num_fragments}, have {len(nodes)}",
            }

        # Step 4: Write fragments to nodes
        fragment_locations = []
        write_failures = 0

        for i, (fragment, node) in enumerate(zip(fragments, nodes)):
            fragment_key = f"frag_{uuid.uuid4().hex[:16]}"
            success = node.write_fragment(fragment_key, fragment)

            if success:
                fragment_locations.append({
                    "fragment_id": fragment["fragment_id"],
                    "fragment_type": fragment["fragment_type"],
                    "node_id": node.node_id,
                    "rack_id": node.rack_id,
                    "zone_id": node.zone_id,
                    "fragment_key": fragment_key,
                })
            else:
                write_failures += 1

        if write_failures > self.erasure_engine.m:
            self.failed_uploads += 1
            return {
                "success": False,
                "error": f"Too many write failures: {write_failures}",
            }

        # Step 5: Write metadata (ATOMIC COMMIT POINT)
        obj_metadata = ObjectMetadata(
            bucket=bucket,
            key=key,
            size=len(data),
            checksum_sha256=checksum,
            content_type=content_type,
            encoding=self.erasure_engine.encoding_id,
            fragment_locations=fragment_locations,
            storage_class="STANDARD",
        )

        self.metadata.put_metadata(obj_metadata)
        elapsed = (time.time() - start_time) * 1000

        self.uploads += 1
        self.upload_bytes += len(data)

        # Analyze placement diversity
        racks_used = len(set(fl["rack_id"] for fl in fragment_locations))
        zones_used = len(set(fl["zone_id"] for fl in fragment_locations))

        return {
            "success": True,
            "bucket": bucket,
            "key": key,
            "size": len(data),
            "checksum": checksum,
            "version_id": obj_metadata.version_id,
            "encoding": self.erasure_engine.encoding_id,
            "fragments": num_fragments,
            "racks_used": racks_used,
            "zones_used": zones_used,
            "time_ms": round(elapsed, 3),
        }

    def get_object(self, bucket: str, key: str) -> dict:
        """
        Download an object from the storage system.
        
        Pipeline:
        1. Look up metadata
        2. Fetch data fragments (parallel)
        3. If degraded: fetch parity + reconstruct
        4. Verify checksum
        5. Return data
        """
        start_time = time.time()

        # Step 1: Metadata lookup
        meta = self.metadata.get_metadata(bucket, key)
        if not meta:
            self.failed_downloads += 1
            return {"success": False, "error": "Object not found"}

        # Step 2: Fetch fragments
        fetched_fragments = []
        failed_fetches = 0
        degraded = False

        # Try data fragments first
        data_locations = [
            fl for fl in meta.fragment_locations
            if fl.get("fragment_type", "data") == "data"
        ]
        parity_locations = [
            fl for fl in meta.fragment_locations
            if fl.get("fragment_type") == "parity"
        ]

        for fl in data_locations:
            node = self.placement.nodes.get(fl["node_id"])
            if node and node.status == StorageNodeStatus.HEALTHY:
                fragment = node.read_fragment(fl["fragment_key"])
                if fragment:
                    fetched_fragments.append(fragment)
                    continue
            failed_fetches += 1

        # If not enough data fragments, try parity
        if len(fetched_fragments) < self.erasure_engine.k:
            degraded = True
            for fl in parity_locations:
                if len(fetched_fragments) >= self.erasure_engine.k:
                    break
                node = self.placement.nodes.get(fl["node_id"])
                if node and node.status == StorageNodeStatus.HEALTHY:
                    fragment = node.read_fragment(fl["fragment_key"])
                    if fragment:
                        fetched_fragments.append(fragment)

        if len(fetched_fragments) < self.erasure_engine.k:
            self.failed_downloads += 1
            return {
                "success": False,
                "error": f"Not enough fragments. Need {self.erasure_engine.k}, "
                         f"have {len(fetched_fragments)}",
                "fragments_available": len(fetched_fragments),
            }

        # Step 3: Decode
        data = self.erasure_engine.decode(fetched_fragments, meta.size)

        if data is None:
            self.failed_downloads += 1
            return {"success": False, "error": "Decode failed (corruption?)"}

        # Step 4: Verify checksum
        computed_checksum = hashlib.sha256(data).hexdigest()
        checksum_valid = computed_checksum == meta.checksum_sha256

        elapsed = (time.time() - start_time) * 1000

        self.downloads += 1
        self.download_bytes += len(data)
        if degraded:
            self.degraded_reads += 1

        return {
            "success": True,
            "bucket": bucket,
            "key": key,
            "data": data,
            "size": len(data),
            "checksum_valid": checksum_valid,
            "degraded_read": degraded,
            "fragments_fetched": len(fetched_fragments),
            "fragments_failed": failed_fetches,
            "content_type": meta.content_type,
            "time_ms": round(elapsed, 3),
        }

    def delete_object(self, bucket: str, key: str) -> dict:
        """Soft-delete an object"""
        success = self.metadata.delete_metadata(bucket, key)
        return {
            "success": success,
            "bucket": bucket,
            "key": key,
            "message": "Marked for deletion" if success else "Object not found",
        }

    def list_objects(
        self, bucket: str, prefix: str = "", max_keys: int = 100
    ) -> dict:
        """List objects in a bucket"""
        objects = self.metadata.list_objects(bucket, prefix, max_keys)
        return {
            "bucket": bucket,
            "prefix": prefix,
            "count": len(objects),
            "objects": [
                {
                    "key": o.key,
                    "size": o.size,
                    "content_type": o.content_type,
                    "created_at": o.created_at,
                    "checksum": o.checksum_sha256[:16] + "...",
                }
                for o in objects
            ],
        }

    def simulate_node_failure(self, node_id: str) -> dict:
        """Simulate a storage node failure and trigger repair"""
        node = self.placement.nodes.get(node_id)
        if not node:
            return {"error": f"Node {node_id} not found"}

        # Kill the node
        node.simulate_failure()

        # Run repair
        repair_result = self.repair_service.repair_node_failure(node_id)
        return {
            "failed_node": node_id,
            "rack": node.rack_id,
            "zone": node.zone_id,
            "fragments_on_node": len(node.fragments),
            "repair_result": repair_result,
        }

    def run_scrub(self) -> dict:
        """Run full data integrity scrub"""
        return self.scrubber.full_scrub()

    def get_cluster_stats(self) -> dict:
        meta_stats = self.metadata.get_stats()
        placement_stats = self.placement.get_placement_stats()
        repair_stats = self.repair_service.get_stats()
        durability = self.erasure_engine.get_durability()

        return {
