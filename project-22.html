🎯 HYPOTHETICAL PROBLEM

You are the on-call engineer at a ride-sharing platform
Your system has 12 microservices running 47 instances total

WEDNESDAY 11:23 PM — Customer complaint arrives:

  "I requested a ride 10 minutes ago. The app showed 'Finding driver...'
   for 3 minutes, then said 'Ride confirmed', but then the driver
   never arrived and the app showed 'Error: Something went wrong'.
   I was charged $23.50 but never got a ride."

YOUR JOB: Figure out what happened to THIS specific request

THE NIGHTMARE:
  This ONE customer request touched:
    1. API Gateway (Project 19)       → received the HTTP request
    2. Auth Service                    → validated the user's token
    3. Ride Service                    → created the ride request
    4. Location Service                → found nearby drivers
    5. Matching Service                → matched with a driver
    6. Notification Service            → notified the driver
    7. Payment Service                 → charged $23.50
    8. Driver Service                  → driver accepted
    9. ETA Service                     → calculated arrival time
    10. Ride Service (again)           → updated ride status

  EACH service runs on MULTIPLE instances (Project 20)
  EACH instance writes logs to its OWN local file
  THAT'S 47 different log files across 12 different machines

  TO DEBUG THIS YOU NEED TO:
    → SSH into machine 1 → grep logs for user-id → find timestamp
    → SSH into machine 2 → grep logs around that timestamp
    → SSH into machine 3 → grep logs around that timestamp
    → ... repeat for all 12 services
    → Manually correlate events across different time zones
    → Hope the clocks are synchronized (they're not)
    → Hope the log formats are consistent (they're not)
    → Reconstruct the ENTIRE journey manually

  THIS TAKES: 2-4 hours for ONE request
  YOU HAVE: 50,000 requests per minute
  CUSTOMER IS WAITING: RIGHT NOW on the phone

  THE QUESTIONS YOU NEED ANSWERED INSTANTLY:
    → What was the COMPLETE journey of this request?
    → WHERE exactly did it fail?
    → HOW LONG did each step take?
    → WHAT data was passed at each step?
    → WHY did Payment charge if the ride failed?
    → WHO else was affected by the same issue?

THIS is the Distributed Observability problem

🔍 HOW TO APPROACH IT

Think of it like a DETECTIVE investigating a crime:

WITHOUT investigation tools:
  → Crime happened somewhere in a city of 1 million people
  → No security cameras
  → No fingerprints
  → No witnesses
  → No timeline
  → Case unsolvable

WITH investigation tools:
  → Every intersection has a camera (LOGGING)
  → Every person has a unique fingerprint (CORRELATION ID)
  → Every camera timestamps footage (STRUCTURED LOGS)
  → All cameras feed into ONE control room (LOG AGGREGATION)
  → You can follow ONE person across ALL cameras (TRACING)
  → Each camera shows time spent at that location (SPANS)
  → You can reconstruct the EXACT journey in minutes

MAP TO MICROSERVICES:

  Security Camera     = Log statement in code
  Fingerprint         = Correlation ID / Trace ID
  Camera Timestamp    = Structured log with ISO timestamp
  Control Room        = Centralized log aggregation (ELK, Datadog)
  Following a person  = Distributed trace
  Time at location    = Span (timing of one operation)
  Crime scene         = The failed request

KEY CONCEPTS:

  1. TRACE ID — One unique ID for the entire request journey
     → Generated at the entry point (API Gateway)
     → Passed to every service in headers
     → Every log line includes this ID
     → Query by trace ID → see EVERYTHING

  2. SPAN — One operation within the trace
     → "Payment Service processed payment" = one span
     → Has: start_time, end_time, duration, status, tags
     → Spans form a TREE (parent-child relationships)

  3. SPAN CONTEXT — Propagated between services
     → trace_id + span_id + parent_span_id
     → Carried in HTTP headers or message metadata
     → Each service extracts context, creates child span

📌 WHAT COULD BE DONE

OPTION A: Centralized Logging Only (No Tracing)
  → Ship all logs to one place (ELK Stack)
  → Search by keywords, timestamps
  → PROBLEM: Can't follow one request across services
  → No timing information, no parent-child relationships
  → USE: Small systems, basic debugging

OPTION B: Correlation ID Only (Simple Tracing)
  → Generate one ID per request at the gateway
  → Pass it through all services in headers
  → Every log line includes this ID
  → PROBLEM: No timing, no spans, no tree structure
  → Just "grep all logs by this ID"
  → USE: Medium complexity, good starting point

OPTION C: Full Distributed Tracing (OpenTelemetry Style)
  → Trace ID + Span ID + Parent Span ID
  → Each service creates spans with timing
  → Spans form a tree showing the complete journey
  → Centralized collector aggregates all spans
  → Query by trace ID → see full waterfall view
  → USE: Production microservices — THIS is the standard

OPTION D: Service Mesh Observability
  → Sidecar proxy (Envoy/Istio) captures all traffic
  → Automatic tracing without code changes
  → Application is unaware of tracing
  → USE: Kubernetes environments with service mesh

FOR THIS PROJECT → We build Option C
  Complete distributed tracing system with:
  → Trace context propagation
  → Span creation and timing
  → Centralized collection and storage
  → Query and visualization
  → Integration with logging

✅ WHAT SHOULD DO

1. GENERATE trace ID at the ENTRY POINT
   → API Gateway (Project 19) creates the trace ID
   → If incoming request already has trace ID → use it
   → If not → generate new UUID
   → This ONE ID follows the request EVERYWHERE

2. PROPAGATE context in HEADERS
   → Standard headers (W3C Trace Context):
     → traceparent: 00-{trace_id}-{span_id}-{flags}
     → tracestate: vendor-specific data
   → Or custom headers:
     → X-Trace-ID: abc-123
     → X-Span-ID: def-456
     → X-Parent-Span-ID: ghi-789
   → EVERY service-to-service call MUST carry these

3. USE structured logging (JSON)
   → NOT: "Payment failed for user 123"
   → YES: {"level":"ERROR","service":"payment","trace_id":"abc-123",
           "span_id":"def-456","user_id":"123","action":"charge",
           "error":"bank_timeout","duration_ms":5023}
   → Structured logs are SEARCHABLE and PARSEABLE
   → Every log line MUST include trace_id and span_id

4. RECORD timing for every span
   → start_time (high-precision)
   → end_time (high-precision)
   → duration_ms (calculated)
   → This is HOW you find bottlenecks
   → "Why did this request take 5 seconds?"
   → → "Payment span took 4.8 seconds (Bank API slow)"

5. ADD meaningful tags/attributes to spans
   → service.name, service.version, instance.id
   → http.method, http.url, http.status_code
   → user.id, order.id (business context)
   → error.type, error.message (when failed)
   → Tags make spans QUERYABLE

6. SAMPLE traces in production
   → 100% tracing = massive data volume
   → Sample: trace 1% of requests
   → Always trace: errors, slow requests, flagged users
   → Adaptive sampling: increase rate during incidents

❌ WHAT SHOULD NOT DO

1. NEVER log sensitive data in traces
   → NO: credit card numbers, passwords, tokens
   → NO: full request bodies with PII
   → YES: redacted versions, IDs, hashes
   → Traces are stored centrally → accessible to many people

2. NEVER generate trace IDs inside inner services
   → If Service B creates a NEW trace ID instead of using the
     one from Service A → the chain is BROKEN
   → You get two separate traces instead of one connected trace
   → ALWAYS extract and propagate the incoming trace context

3. NEVER use unstructured log formats
   → NOT: "2025-01-01 ERROR: something went wrong in payment"
   → This is impossible to parse, search, or correlate
   → ALWAYS use JSON or structured format
   → ALWAYS include trace_id in every log line

4. NEVER ignore clock synchronization
   → Service A clock says 14:00:00
   → Service B clock says 14:00:05 (5 seconds ahead)
   → Span ordering becomes nonsensical
   → Child span appears to START before parent
   → USE NTP (Network Time Protocol) on all servers
   → Or use logical ordering (parent_span_id)

5. NEVER trace synchronously on the hot path
   → Sending trace data should be ASYNC
   → Don't block the request to ship a span
   → Buffer locally → batch send to collector
   → Tracing should add <1ms overhead

6. NEVER create too many spans
   → One span per SERVICE call = good
   → One span per DATABASE query = sometimes useful
   → One span per FOR LOOP iteration = too much
   → Hundreds of spans per trace = unreadable, expensive
   → Be strategic about what you instrument

7. NEVER rely on traces as your ONLY debugging tool
   → Traces show the JOURNEY
   → Logs show the DETAILS
   → Metrics show the TRENDS
   → You need ALL THREE for complete observability

🚫 WHEN NOT TO USE

1. Single monolith application
   → All code in one process, one log file
   → Standard application logging is sufficient
   → No service-to-service calls to trace
   → Tracing adds complexity without value

2. Batch jobs with no user-facing impact
   → Nightly data processing pipeline
   → Simple job logging with timestamps is enough
   → No request journey to follow

3. When cost exceeds value
   → Full tracing generates MASSIVE data
   → Storage costs can be significant
   → For non-critical systems, sampling at 0.1% might suffice
   → Or skip tracing entirely for internal tooling

4. Extremely high-throughput, low-value transactions
   → Ad impression logging (millions/second)
   → Each transaction is low-value
   → Aggregate metrics matter more than individual traces
   → Sample extremely aggressively (0.01%)

5. When all services are stateless and idempotent
   → Every request is independent
   → Failures are trivially retryable
   → No need to understand the journey
   → Simple error counts suffice

COMPONENT 4: TRACE ASSEMBLY — BUILDING THE TREE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Spans arrive INDEPENDENTLY from different services
  They might arrive OUT OF ORDER
  The collector must ASSEMBLE them into a tree

  Raw spans received (unordered):
    {trace: abc, span: C, parent: B, service: bank-api}
    {trace: abc, span: A, parent: null, service: gateway}
    {trace: abc, span: D, parent: A, service: notification}
    {trace: abc, span: B, parent: A, service: payment}

  Assembly algorithm:
    1. Group all spans by trace_id
    2. Find root span (parent_span_id = null)
    3. For each span, find its children (where parent = this span_id)
    4. Build tree recursively
    5. Calculate total trace duration from root span

  Assembled tree:
    A (gateway, root, 850ms)
    ├── B (payment, 210ms)
    │   └── C (bank-api, 185ms)
    └── D (notification, 35ms) [ERROR]

  LATE-ARRIVING SPANS:
    → Span C might arrive 5 seconds after span A
    → Collector buffers incomplete traces
    → After timeout (e.g., 60s), marks trace as "partial"
    → Late span arrives → updates the trace

COMPONENT 5: SAMPLING — CONTROLLING DATA VOLUME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problem: 50,000 requests/minute × 10 spans each = 500,000 spans/minute
  That's 720 MILLION spans per day
  Storage cost: enormous

  SAMPLING STRATEGIES:

  A) HEAD-BASED SAMPLING (decide at entry point)
     → Gateway decides: "This request WILL be traced" (1% chance)
     → Sets flags=01 in trace context
     → ALL downstream services honor this flag
     → Consistent: either trace ALL spans or NONE
     → Simple but misses interesting requests

  B) TAIL-BASED SAMPLING (decide after trace completes)
     → Collect ALL spans temporarily
     → After trace completes, examine it:
       → Has errors? → KEEP
       → Duration > 5s? → KEEP
       → Normal? → DROP (99%)
     → Catches all interesting traces
     → More complex, requires buffering

  C) ADAPTIVE SAMPLING

class TraceContext:
    """
    Carries trace identity across service boundaries.
    Generated at entry point, propagated via headers.
    """

    def __init__(self, trace_id=None, span_id=None,
                 parent_span_id=None, sampled=True, baggage=None):
        self.trace_id = trace_id or self._generate_id(32)
        self.span_id = span_id or self._generate_id(16)
        self.parent_span_id = parent_span_id
        self.sampled = sampled
        # Baggage: key-value pairs that travel with the trace
        # Useful for passing business context (user_id, etc.)
        self.baggage = baggage or {}

    @staticmethod
    def _generate_id(length):
        return uuid.uuid4().hex[:length]

    def create_child_context(self):
        """Create a new context for a child span."""
        return TraceContext(
            trace_id=self.trace_id,
            span_id=self._generate_id(16),
            parent_span_id=self.span_id,
            sampled=self.sampled,
            baggage=dict(self.baggage)
        )

    def to_headers(self):
        """Serialize context to HTTP headers for propagation."""
        headers = {
            "X-Trace-ID": self.trace_id,
            "X-Span-ID": self.span_id,
            "X-Sampled": "1" if self.sampled else "0"
        }
        if self.parent_span_id:
            headers["X-Parent-Span-ID"] = self.parent_span_id
        # W3C traceparent format
        flags = "01" if self.sampled else "00"
        headers["traceparent"] = (
            f"00-{self.trace_id}-{self.span_id}-{flags}"
        )
        # Baggage as JSON header
        if self.baggage:
            headers["X-Trace-Baggage"] = json.dumps(self.baggage)
        return headers

    @classmethod
    def from_headers(cls, headers):
        """Deserialize context from incoming HTTP headers."""
        trace_id = headers.get("X-Trace-ID")
        span_id = headers.get("X-Span-ID")
        parent_span_id = headers.get("X-Parent-Span-ID")
        sampled_str = headers.get("X-Sampled", "1")
        sampled = sampled_str == "1"

        # Try W3C traceparent if custom headers not present
        if not trace_id and "traceparent" in headers:
            parts = headers["traceparent"].split("-")
            if len(parts) == 4:
                trace_id = parts[1]
                span_id = parts[2]
                sampled = parts[3] == "01"

        baggage = {}
        if "X-Trace-Baggage" in headers:
            try:
                baggage = json.loads(headers["X-Trace-Baggage"])
            except json.JSONDecodeError:
                pass

        if trace_id:
            return cls(
                trace_id=trace_id,
                span_id=span_id,
                parent_span_id=parent_span_id,
                sampled=sampled,
                baggage=baggage
            )
        return None

    def __repr__(self):
        return (f"TraceContext(trace={self.trace_id[:8]}..., "
                f"span={self.span_id[:8]}..., "
                f"parent={self.parent_span_id[:8] + '...' if self.parent_span_id else 'None'})")

class SpanStatus(Enum):

    ERROR = "ERROR"
    UNSET = "UNSET"

# SPAN — One operation in the trace
# ============================================================

class Span:
    """
    Represents one unit of work within a trace.
    Created when entering a service/operation.
    Finished when the operation completes.
    """

    def __init__(self, context, operation_name, service_name,
                 instance_id=None):
        self.context = context
        self.operation_name = operation_name
        self.service_name = service_name
        self.instance_id = instance_id or "default"

        # Timing
        self.start_time = None
        self.end_time = None
        self.duration_ms = None

        # Status
        self.status = SpanStatus.UNSET
        self.error_message = None

        # Tags: searchable key-value metadata
        self.tags = {
            "service.name": service_name,
            "service.instance": self.instance_id
        }

        # Events: timestamped log entries within the span
        self.events = []

        # State
        self._finished = False

    def start(self):
        self.start_time = time.time()
        self.start_time_iso = datetime.now(timezone.utc).isoformat()
        return self

    def finish(self, status=None, error=None):
        if self._finished:
            return
        self._finished = True
        self.end_time = time.time()
        self.end_time_iso = datetime.now(timezone.utc).isoformat()
        self.duration_ms = round(
            (self.end_time - self.start_time) * 1000, 3
        )
        if error:
            self.status = SpanStatus.ERROR
            self.error_message = str(error)
            self.set_tag("error", True)
            self.set_tag("error.message", str(error))
        elif status:
            self.status = status
        else:
            self.status = SpanStatus.OK

    def set_tag(self, key, value):
        self.tags[key] = value
        return self

    def add_event(self, name, attributes=None):
        self.events.append({
            "timestamp": datetime.now(timezone.utc).isoformat(),
            "name": name,
            "attributes": attributes or {}
        })
        return self

    def to_dict(self):
        return {
            "trace_id": self.context.trace_id,
            "span_id": self.context.span_id,
            "parent_span_id": self.context.parent_span_id,
            "operation_name": self.operation_name,
            "service_name": self.service_name,
            "instance_id": self.instance_id,
            "start_time": self.start_time_iso if self.start_time else None,
            "end_time": self.end_time_iso if self.end_time else None,
            "duration_ms": self.duration_ms,
            "status": self.status.value,
            "error_message": self.error_message,
            "tags": self.tags,
            "events": self.events,
            "baggage": self.context.baggage
        }

class TracedLogger:
    """
    Structured JSON logger that ALWAYS includes trace context.
    Every log line can be correlated to a specific trace and span.
    """

    def __init__(self, service_name, instance_id="default",
                 log_store=None):
        self.service_name = service_name
        self.instance_id = instance_id
        self.log_store = log_store
        self.lock = threading.Lock()

    def _log(self, level, message, trace_context=None,
             span_id=None, extra=None):
        log_entry = {
            "timestamp": datetime.now(timezone.utc).isoformat(),
            "level": level,
            "service": self.service_name,
            "instance": self.instance_id,
            "message": message,
            "trace_id": (trace_context.trace_id
                         if trace_context else None),
            "span_id": span_id or (
                trace_context.span_id if trace_context else None
            )
        }
        if extra:
            log_entry.update(extra)

        # Print structured log
        with self.lock:
            level_icons = {
                "DEBUG": "🔍", "INFO": "ℹ️ ",
                "WARN": "⚠️ ", "ERROR": "❌"
            }
            icon = level_icons.get(level, "  ")
            trace_short = (log_entry["trace_id"][:8]
                           if log_entry["trace_id"] else "--------")
            print(f"    {icon} [{trace_short}] "
                  f"{self.service_name:>20} | "
                  f"{level:<5} | {message}")

        # Store for centralized querying
        if self.log_store is not None:
            self.log_store.append(log_entry)

        return log_entry

    def debug(self, msg, ctx=None, **kw):
        return self._log("DEBUG", msg, ctx, extra=kw)

    def info(self, msg, ctx=None, **kw):
        return self._log("INFO", msg, ctx, extra=kw)

    def warn(self, msg, ctx=None, **kw):
        return self._log("WARN", msg, ctx, extra=kw)

    def error(self, msg, ctx=None, **kw):
        return self._log("ERROR", msg, ctx, extra=kw)

class TraceCollector:
    """
    Centralized collector that receives spans from all services.
    Assembles spans into complete traces.
    Provides query interface for debugging.
    """

    def __init__(self):
        # Storage: trace_id → list of span dicts
        self.traces = defaultdict(list)
        self.lock = threading.RLock()
        # Centralized log storage
        self.logs = []
        self.log_lock = threading.Lock()
        # Statistics
        self.stats = {
            "total_spans_received": 0,
            "total_traces": 0,
            "error_traces": 0,
            "total_logs": 0
        }
        # Sampling configuration
        self.sample_rate = 1.0  # 100% by default
        self.always_sample_errors = True

    def receive_span(self, span_dict):
        """Receive a single span from a service."""
        with self.lock:
            trace_id = span_dict["trace_id"]
            self.traces[trace_id].append(span_dict)
            self.stats["total_spans_received"] += 1
            if span_dict.get("parent_span_id") is None:
                self.stats["total_traces"] += 1
            if span_dict.get("status") == "ERROR":
                self.stats["error_traces"] += 1

    def receive_log(self, log_entry):
        """Receive a log entry from a service."""
        with self.log_lock:
            self.logs.append(log_entry)
            self.stats["total_logs"] += 1
            # Retention: keep last 10,000 logs
            if len(self.logs) > 10000:
                self.logs = self.logs[-10000:]

    def get_trace(self, trace_id):
        """Get all spans for a trace, assembled into a tree."""
        with self.lock:
            spans = self.traces.get(trace_id, [])
        if not spans:
            return None
        return self._assemble_trace(trace_id, spans)

    def _assemble_trace(self, trace_id, spans):
        """Assemble flat span list into a tree structure."""
        # Index spans by span_id
        span_map = {}
        root_span = None

        for span in spans:
            sid = span["span_id"]
            span_map[sid] = {**span, "children": []}
            if span.get("parent_span_id") is None:
                root_span = span_map[sid]

        # Build tree by linking children to parents
        for sid, span in span_map.items():
            parent_id = span.get("parent_span_id")
            if parent_id and parent_id in span_map:
                span_map[parent_id]["children"].append(span)

        # Calculate trace-level metrics
        has_error = any(
            s.get("status") == "ERROR" for s in spans
        )
        total_duration = (
            root_span["duration_ms"] if root_span else None
        )
        services_involved = list(set(
            s.get("service_name", "unknown") for s in spans
        ))

        return {
            "trace_id": trace_id,
            "root_span": root_span,
            "total_spans": len(spans),
            "total_duration_ms": total_duration,
            "has_error": has_error,
            "services": services_involved,
            "spans_flat": spans
        }

    def query_traces(self, service_name=None, has_error=None,
                     min_duration_ms=None, tag_filter=None,
                     limit=20):
        """
        Query traces with filters.
        This is what the debugging dashboard uses.
        """
        results = []

        with self.lock:
            for trace_id, spans in self.traces.items():
                trace = self._assemble_trace(trace_id, spans)
                if not trace:
                    continue

                # Filter by service
                if (service_name and
                        service_name not in trace["services"]):
                    continue

                # Filter by error
                if has_error is not None:
                    if trace["has_error"] != has_error:
                        continue

                # Filter by minimum duration
                if min_duration_ms is not None:
                    if (trace["total_duration_ms"] and
                            trace["total_duration_ms"] < min_duration_ms):
                        continue

                # Filter by tag
                if tag_filter:
                    tag_key, tag_value = tag_filter
                    found = False
                    for span in spans:
                        if span.get("tags", {}).get(tag_key) == tag_value:
                            found = True
                            break
                    if not found:
                        continue

                results.append(trace)

                if len(results) >= limit:
                    break

        return results

    def get_logs_for_trace(self, trace_id):
        """Get all logs associated with a specific trace."""
        with self.log_lock:
            return [
                log for log in self.logs
                if log.get("trace_id") == trace_id
            ]

    def get_service_stats(self):
        """Aggregate statistics per service from trace data."""
        service_stats = defaultdict(lambda: {
            "total_spans": 0,
            "error_spans": 0,
            "total_duration_ms": 0,
            "durations": []
        })

        with self.lock:
            for trace_id, spans in self.traces.items():
                for span in spans:
                    svc = span.get("service_name", "unknown")
                    stats = service_stats[svc]
                    stats["total_spans"] += 1
                    if span.get("status") == "ERROR":
                        stats["error_spans"] += 1
                    dur = span.get("duration_ms", 0) or 0
                    stats["total_duration_ms"] += dur
                    stats["durations"].append(dur)

        # Calculate percentiles
        result = {}
        for svc, stats in service_stats.items():
            durations = sorted(stats["durations"])
            n = len(durations)
            result[svc] = {
                "total_spans": stats["total_spans"],
                "error_spans": stats["error_spans"],
                "error_rate": (f"{stats['error_spans']/stats['total_spans']:.1%}"
                               if stats['total_spans'] > 0 else "0%"),
                "avg_duration_ms": (
                    round(stats["total_duration_ms"] / n, 2)
                    if n > 0 else 0
                ),
                "p50_ms": durations[n // 2] if n > 0 else 0,
                "p95_ms": (durations[int(n * 0.95)]
                           if n > 0 else 0),
                "p99_ms": (durations[int(n * 0.99)]
                           if n > 0 else 0)
            }

        return result

    def visualize_trace(self, trace_id):
        """Print a waterfall visualization of a trace."""
        trace = self.get_trace(trace_id)
        if not trace:
            print(f"  Trace {trace_id} not found")
            return

        print(f"\n  {'═' * 75}")
        print(f"  TRACE: {trace_id}")
        print(f"  Duration: {trace['total_duration_ms']}ms | "
              f"Spans: {trace['total_spans']} | "
              f"Services: {', '.join(trace['services'])}")
        print(f"  Error: {'YES ❌' if trace['has_error'] else 'NO ✅'}")
        print(f"  {'═' * 75}")

        if not trace["root_span"]:
            print(f"  ⚠️  No root span found (partial trace)")
            for span in trace["spans_flat"]:
                self._print_span(span, indent=2)
            return

        # Print tree from root
        self._print_span_tree(trace["root_span"],
                              trace["total_duration_ms"],
                              indent=2)

        # Print associated logs
        logs = self.get_logs_for_trace(trace_id)
        if logs:
            print(f"\n  {'─' * 75}")
            print(f"  ASSOCIATED LOGS ({len(logs)} entries):")
            print(f"  {'─' * 75}")
            for log in logs[:20]:
                level = log.get("level", "?")
                svc = log.get("service", "?")
                msg = log.get("message", "?")
                ts = log.get("timestamp", "?")[:19]
                icon = {"ERROR": "❌", "WARN": "⚠️ ",
                        "INFO": "ℹ️ ", "DEBUG": "🔍"}.get(level, "  ")
                print(f"    {icon} [{ts}] {svc:<20} {level:<5} {msg}")

        print(f"  {'═' * 75}\n")

    def _print_span_tree(self, span, total_ms, indent=2, depth=0):
        self._print_span(span, total_ms, indent, depth)
        for child in span.get("children", []):
            self._print_span_tree(child, total_ms, indent, depth + 1)

    def _print_span(self, span, total_ms=None, indent=2, depth=0):
        prefix = " " * indent + "│  " * depth + "├─ "
        svc = span.get("service_name", "?")
        op = span.get("operation_name", "?")
        dur = span.get("duration_ms", 0) or 0
        status = span.get("status", "?")

        # Status icon
        if status == "ERROR":
            icon = "❌"
        elif status == "OK":
            icon = "✅"
        else:
            icon = "⬜"

        # Duration bar (proportional)
        if total_ms and total_ms > 0:
            bar_len = max(1, int((dur / total_ms) * 30))
        else:
            bar_len = 1
        bar = "█" * bar_len

        print(f"{prefix}{icon} {svc}::{op} "
              f"[{dur:.1f}ms] {bar}")

        # Show error if present
        if span.get("error_message"):
            error_prefix = " " * indent + "│  " * (depth + 1)
            print(f"{error_prefix}  💥 ERROR: {span['error_message']}")

        # Show events
        for event in span.get("events", []):
            event_prefix = " " * indent + "│  " * (depth + 1)
            print(f"{event_prefix}  📌 {event['name']}")

        self.instance_id = instance_id
        self.collector = collector
        # Thread-local storage for current span context
        self._local = threading.local()
        # Logger integrated with tracing
        self.logger = TracedLogger(
            service_name, instance_id,
            log_store=None
        )

    @property
    def current_context(self):
        return getattr(self._local, 'context', None)

    @current_context.setter
    def current_context(self, ctx):
        self._local.context = ctx

    def start_span(self, operation_name, parent_context=None,
                   tags=None):
        """Start a new span."""
        if parent_context:
            context = parent_context.create_child_context()
        elif self.current_context:
            context = self.current_context.create_child_context()
        else:
            context = TraceContext()

        span = Span(
            context=context,
            operation_name=operation_name,
            service_name=self.service_name,
            instance_id=self.instance_id
        )

        if tags:
            for k, v in tags.items():
                span.set_tag(k, v)

        span.start()
        self.current_context = context
        return span

    def finish_span(self, span, error=None):
        """Finish a span and report to collector."""
        span.finish(error=error)

        if self.collector and span.context.sampled:
            self.collector.receive_span(span.to_dict())

        # Restore parent context
        if span.context.parent_span_id:
            self.current_context = TraceContext(
                trace_id=span.context.trace_id,
                span_id=span.context.parent_span_id,
                sampled=span.context.sampled,
                baggage=span.context.baggage
            )
        else:
            self.current_context = None

    @contextmanager
    def trace(self, operation_name, parent_context=None,
              tags=None):
        """
        Context manager for easy span management.
        Usage:
            with tracer.trace("payment.charge") as span:
                span.set_tag("amount", 23.50)
                result = charge_payment()
        """
        span = self.start_span(operation_name, parent_context, tags)
        try:
            yield span
            self.finish_span(span)
        except Exception as e:
            self.finish_span(span, error=e)
            raise

    def extract_context(self, headers):
        """Extract trace context from incoming HTTP headers."""
        return TraceContext.from_headers(headers)

    def inject_context(self, context=None):
        """Inject current trace context into outgoing headers."""
        ctx = context or self.current_context
        if ctx:
            return ctx.to_headers()
        return {}

    def log(self, level, message, extra=None):
        """Log with automatic trace context inclusion."""
        ctx = self.current_context
        log_entry = self.logger._log(
            level, message, ctx, extra=extra
        )
        if self.collector:
            self.collector.receive_log(log_entry)
        return log_entry

THE STORAGE PROBLEM TRACING REVEALS:

  → Our trace collector stores spans in Python dictionaries
  → In production, you need FAST, SCALABLE key-value storage
  → Look up trace by trace_id → needs O(1) access
  → Store span data → needs efficient serialization
  → Set TTL on old traces → needs expiration support
  → Cache frequently accessed data → needs in-memory speed

  → The SAME pattern appears everywhere:
    → Caching (Project 5): store cached data by key
    → Rate Limiting (Project 9): track request counts by key
    → Session Storage (Project 8): store user sessions by token
    → Service Discovery (Project 20): store instance data by service name
    → Circuit Breaker (Project 21): store breaker state by name
    → Trace Storage (Project 22): store spans by trace_id

  → ALL of these need the SAME thing:
    → Fast key-value storage
    → In-memory for speed
    → Optional persistence for durability
    → TTL for automatic expiration
    → This is Redis. This is Memcached. This is fundamental.
