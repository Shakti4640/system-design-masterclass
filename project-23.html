🔍 HOW TO APPROACH IT

Think of it like a FILING CABINET vs a DESK:

  DATABASE (PostgreSQL) = Filing Cabinet
    → Massive storage capacity
    → Well-organized drawers (tables, indexes)
    → Takes time to open drawer, find folder, read document
    → Survives office fire (durability)
    → Great for: complex queries, relationships, large datasets

  KEY-VALUE STORE (Redis) = Your Desk
    → Limited space (fits on desk = fits in RAM)
    → Everything labeled with a sticky note (key)
    → Instant access: see the label, grab the item
    → If office burns, desk items lost (unless you backup)
    → Great for: frequently accessed, simple lookups, speed

MAP TO COMPUTER SCIENCE:

  Filing Cabinet = Disk-based B-Tree
    → O(log n) lookup
    → Disk I/O: ~10ms per read
    → Terabytes of storage

  Desk = In-Memory Hash Table
    → O(1) average lookup
    → Memory access: ~0.0001ms per read
    → Gigabytes of storage (limited by RAM)

  Two strategies for expiring keys:

  A) LAZY EXPIRATION (on access)
     → Check TTL when key is READ
     → If expired → delete and return "not found"
     → Pro: Zero background CPU cost
     → Con: Expired keys linger in memory until accessed

  B) ACTIVE EXPIRATION (background sweep)
     → Background thread periodically scans keys
     → Deletes expired ones
     → Pro: Memory freed promptly
     → Con: CPU cost for scanning

  REDIS USES BOTH → and so will we

  In-memory = data lost on restart

  Two persistence strategies:

  A) RDB (Snapshot)
     → Periodically dump entire memory to disk
     → Binary file: compact, fast to load
     → "Save everything every 5 minutes"
     → Pro: Fast recovery, small file
     → Con: Lose up to 5 minutes of data

  B) AOF (Append-Only File)
     → Log every WRITE command to a file
     → On restart: replay all commands
     → Pro: Minimal data loss (every command saved)
     → Con: Larger file, slower recovery

  REDIS OFFERS BOTH → we implement both

✅ WHAT SHOULD DO

1. SINGLE-THREADED command processing
   → Redis is single-threaded for commands
   → No locks needed for data operations
   → Simplifies atomicity guarantees
   → I/O multiplexing handles many connections
   → We use a command queue + single processor thread

2. LAZY + ACTIVE expiration combined
   → Lazy: check on every GET (catches most)
   → Active: background sweep every 100ms
   → Randomly sample 20 keys with TTL
   → If >25% are expired → repeat immediately
   → Balances memory reclaim vs CPU usage

3. ATOMIC operations
   → INCR must be atomic (read + increment + write in one step)
   → SETNX must be atomic (check + set if not exists)
   → Single-threaded command processing guarantees this

4. MEMORY tracking
   → Track approximate memory usage
   → When memory limit reached → evict keys (Project 5 LRU/LFU)
   → maxmemory policy: allkeys-lru, volatile-lru, noeviction

5. PERSISTENCE with fsync control
   → AOF fsync options: always, everysec, no
   → "always": every write → slow but safe
   → "everysec": buffer 1 second → good balance
   → "no": OS decides → fastest but risky

6. KEY naming conventions
   → Use colons as separators: "user:123:profile"
   → Namespace by function: "cache:", "session:", "lock:"
   → Keep keys short (memory matters)

❌ WHAT SHOULD NOT DO

1. NEVER use a key-value store as primary database
   → No complex queries (JOIN, GROUP BY, WHERE)
   → No ACID transactions across multiple keys
   → No relational integrity
   → USE: Cache layer IN FRONT of your database
   → Database is source of truth, KV store is speed layer

2. NEVER store more data than fits in RAM
   → Key-value store = ALL data in memory
   → 64 GB RAM = 64 GB max data (roughly)
   → If dataset exceeds RAM → use disk-based DB
   → Or partition across multiple nodes (Project 14/18)

3. NEVER rely on TTL alone for data cleanup
   → TTL is best-effort, not guaranteed exact
   → Lazy expiration means key might linger
   → For critical cleanup: explicit DELETE

4. NEVER store large blobs (>1MB per value)
   → Key-value optimized for small, frequent reads
   → Large values slow down everything
   → For large objects: use object storage (Project 25)
   → Store only the reference/URL in KV store

5. NEVER skip persistence in production
   → "It's just a cache, we don't need persistence"
   → Server restarts → cold cache → database gets slammed
   → Called "thundering herd" or "cache stampede"
   → Persistence lets you restore cache on restart

6. NEVER ignore memory limits
   → Without maxmemory → process grows until OOM kill
   → OS kills the Redis process → all data lost
   → ALWAYS set maxmemory + eviction policy

   → Need ACID, durability, complex queries
   → Use PostgreSQL/MySQL as primary
   → Use KV store as cache layer only

2. Data with complex relationships
   → "Find all friends of friends who like pizza"
   → KV store has no JOIN, no graph traversal
   → Use relational DB or graph DB (Project 40)

3. Full-text search
   → "Find all articles containing 'distributed systems'"
   → KV store can only look up by exact key
   → Use Elasticsearch / search engine (Project 41)

   → "Average order value by region by month"
   → KV store has no GROUP BY, no SUM

⚙️ HOW IT ONLY WORKS (INTERNAL MECHANICS)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
COMPONENT 1: HASH TABLE — THE CORE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  HOW A HASH TABLE WORKS:

  Step 1: Key comes in → "user:123"
  Step 2: Hash function → hash("user:123") = 7392847
  Step 3: Bucket index → 7392847 % num_buckets = 47
  Step 4: Go to bucket 47 → find the entry
  Step 5: Return the value

  Memory Layout:
    Bucket 0:  → empty
    Bucket 1:  → ("session:abc", {data...}) → null
    Bucket 2:  → empty
    ...
    Bucket 47: → ("user:123", {data...}) → ("rate:456", 12) → null
    ...                                      ↑ collision chain

  COLLISION HANDLING:
    → Two different keys hash to same bucket
    → Chain them in a linked list
    → O(1) average, O(n) worst case
    → Redis uses two hash tables for incremental resizing

  RESIZING:
    → When load factor (entries/buckets) > 1.0
    → Create new hash table with 2x buckets
    → Incrementally rehash: move 1 bucket per operation
    → Both tables active during rehash
    → Avoids big pause for full rehash

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
COMPONENT 2: TTL ENGINE — EXPIRATION MECHANICS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Separate dictionary tracks expiry times:

  data_store = {
      "session:abc": "user-data-here",
      "cache:profile:123": "{name: Alice}",
      "temp:key": "value"
  }

  expiry_store = {
      "session:abc": 1704110400.0,      # Unix timestamp when it expires
      "cache:profile:123": 1704110700.0,
      "temp:key": 1704110100.0
  }

  LAZY EXPIRATION (on read):
    def get(key):
        if key in expiry_store:
            if time.time() > expiry_store[key]:
                # Expired! Delete it
                del data_store[key]
                del expiry_store[key]
                return None
        return data_store.get(key)

  ACTIVE EXPIRATION (background):
    Every 100ms:
      1. Get all keys that have TTL
      2. Randomly sample 20 of them
      3. Delete the ones that are expired
      4. If more than 25% were expired → repeat immediately
      5. Otherwise → wait for next cycle

    WHY random sampling?
      → Scanning ALL keys is O(n) — too expensive
      → Random sample catches most expired keys
      → Multiple rounds clean up quickly during burst expiration
      → Redis processes ~1 million key expirations/second this way

COMPONENT 3: PERSISTENCE — SURVIVING RESTARTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  A) RDB SNAPSHOT:

  Trigger: Every N seconds if M keys changed
    → "save 300 10" = snapshot every 300s if 10+ keys changed

  Process:
    1. Fork the main process (copy-on-write)
    2. Child process writes ALL data to temp file
    3. When done, rename temp file to dump.rdb
    4. Child exits
    → Main process continues serving (no blocking!)
    → Copy-on-write means child sees a FROZEN snapshot

  RDB File Format:
    [MAGIC: REDIS] [VERSION: 0009]
    [DB-SELECTOR: 0]
    [KEY-VALUE-PAIR: type=string, key="name", value="Alice", ttl=...]
    [KEY-VALUE-PAIR: type=list, key="queue", values=["a","b","c"]]
    ...
    [EOF] [CHECKSUM]

  Recovery: Load entire file into memory on startup

  B) AOF (Append-Only File):

  Process:
    1. Every WRITE command → append to file
    2. File grows over time

  AOF File Content:
    SET session:abc "user-data" EX 1800
    SET cache:profile:123 "{name: Alice}" EX 300
    INCR rate:user-789
    INCR rate:user-789
    INCR rate:user-789
    DEL temp:key
    SET session:abc "updated-data" EX 1800  ← overwrites earlier SET
    ...

  AOF REWRITE (compaction):
    → File grows forever → needs compaction
    → Rewrite: scan current memory, write minimal commands
    → Before: 10,000 INCRs for counter → After: SET counter 10000
    → Fork + write new AOF → replace old file
    → Same copy-on-write trick as RDB

  Recovery: Replay all commands from AOF on startup

  COMPARISON:
  ┌──────────────┬─────────────────┬─────────────────┐
  │              │ RDB (Snapshot)  │ AOF (Append Log) │
  ├──────────────┼─────────────────┼─────────────────┤
  │ Data loss    │ Up to 5 min     │ Up to 1 second   │
  │ File size    │ Compact         │ Larger            │
  │ Recovery     │ Fast (load)     │ Slower (replay)   │
  │ Write impact │ Periodic fork   │ Every write       │
  │ Best for     │ Backups, DR     │ Minimal data loss │
  └──────────────┴─────────────────┴─────────────────┘

COMPONENT 4: MEMORY EVICTION — WHEN RAM IS FULL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  When maxmemory is reached, new writes must evict old data

  EVICTION POLICIES (same concepts from Project 5):

  → noeviction:    Reject writes, return error
  → allkeys-lru:   Evict least recently used key (any key)
  → volatile-lru:  Evict least recently used key WITH TTL only
  → allkeys-lfu:   Evict least frequently used key
  → volatile-lfu:  Evict least frequently used key WITH TTL
  → allkeys-random: Evict random key
  → volatile-random: Evict random key WITH TTL
  → volatile-ttl:  Evict key with shortest remaining TTL

  HOW REDIS DOES LRU (approximation):
    → True LRU needs a linked list of ALL keys → expensive
    → Redis uses APPROXIMATED LRU:
      1. When eviction needed, randomly sample 5 keys
      2. Among the 5, evict the one with oldest access time
      3. Repeat until enough memory freed
    → Surprisingly close to true LRU with much less overhead
    → Each key stores last access time (24-bit timestamp, ~194 days)

  Internal structure: Skip List + Hash Table

  Skip List (for sorted order):
    Level 3: HEAD ──────────────────────────────→ 200(Bob)
    Level 2: HEAD ────────→ 100(Alice) ────────→ 200(Bob)
    Level 1: HEAD → 50(Eve) → 100(Alice) → 150(Dan) → 200(Bob)

  Hash Table (for O(1) member lookup):
    {"Alice": 100, "Bob": 200, "Eve": 50, "Dan": 150}

  Operations:
    ZADD:  O(log n) — insert into skip list
    ZRANK: O(log n) — find position in skip list
    ZRANGE: O(log n + k) — find start + walk k elements
    ZSCORE: O(1) — hash table lookup

  WHY skip list instead of balanced tree?
    → Simpler implementation
    → Similar O(log n) performance
    → Better for range queries (just walk forward)
    → Easier concurrent access (not needed in single-threaded Redis)

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: Hot Key Problem
  → One key accessed millions of times/second
  → "celebrity:profile:taylorswift" → massive traffic
  → Single-threaded: all requests queue up
  → FIX: Replicate hot keys across read replicas
  → FIX: Application-level caching for hottest keys
  → FIX: Key splitting: "celebrity:profile:ts:{shard_id}"

PITFALL 2: Big Key Problem
  → One list with 10 million elements
  → DEL on this key → blocks server for seconds
  → All other operations wait
  → FIX: UNLINK instead of DEL (async deletion)
  → FIX: Don't let collections grow unbounded
  → FIX: Split big keys into smaller ones

PITFALL 3: Cache Stampede / Thundering Herd
  → Popular key expires
  → 10,000 concurrent requests all miss cache
  → All 10,000 hit the database simultaneously
  → Database collapses
  → FIX: Lock-based recomputation (only one request rebuilds)
  → FIX: Staggered TTLs (add random jitter to TTL)
  → FIX: "Stale-while-revalidate" (serve stale, refresh in background)

PITFALL 4: Memory Fragmentation
  → Frequent SET/DELETE of different-sized values
  → Memory fragments → actual usage > logical usage
  → 10 GB data but 15 GB RSS (process memory)
  → FIX: Use jemalloc (Redis default memory allocator)
  → FIX: Periodic restart to defragment
  → FIX: Active defragmentation (Redis 4.0+)

PITFALL 5: Forgetting About Persistence Costs
  → AOF with fsync=always → every write goes to disk
  → Performance drops from 100k ops/s to 10k ops/s
  → FIX: Use fsync=everysec (lose at most 1 second)
  → FIX: Use RDB snapshots for less write-heavy workloads

PITFALL 6: Using KEYS * in Production
  → KEYS * scans ALL keys → O(n) → blocks server
  → With 10 million keys → blocks for seconds
  → FIX: NEVER use KEYS in production
  → FIX: Use SCAN (cursor-based, non-blocking iteration)

PITFALL 7: Not Setting maxmemory
  → Redis grows until all RAM consumed
  → OS kills Redis process (OOM killer)
  → ALL cached data lost instantly
  → FIX: ALWAYS set maxmemory
  → FIX: ALWAYS set maxmemory-policy
  → FIX: Monitor memory usage, alert at 80%

    # STRING OPERATIONS
    # ============================================================

    def set(self, key, value, ex=None, px=None, nx=False, xx=False):
        """
        SET key value [EX seconds] [PX milliseconds] [NX|XX]
        NX: Only set if key does NOT exist
        XX: Only set if key DOES exist
        """
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_writes"] += 1

            # Check NX/XX conditions
            exists = key in self.store and not self._is_expired(key)
            if nx and exists:
                return None  # Key exists, NX says don't set
            if xx and not exists:
                return None  # Key doesn't exist, XX says don't set

            # Evict if memory limit reached
            if self.maxmemory:
                self._check_memory_and_evict()

            # Calculate memory delta
            old_size = self._estimate_size(key) if key in self.store else 0
            new_size = self._estimate_entry_size(key, value)

            # Store the value
            self.store[key] = {
                "type": "string",
                "value": str(value),
                "last_access": time.time(),
                "created_at": time.time()
            }

            # Set expiry
            if ex:
                self.expiry[key] = time.time() + ex
            elif px:
                self.expiry[key] = time.time() + (px / 1000.0)
            elif key in self.expiry and not (ex or px):
                # Overwrite clears TTL unless new TTL specified
                pass  # Keep existing TTL on overwrite (Redis behavior)

            # Update memory tracking
            self.memory_used += (new_size - old_size)

            # AOF logging
            self._aof_log("SET", key, value, ex=ex)

            return "OK"

    def get(self, key):
        """GET key — Returns value or None."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_reads"] += 1

            # Lazy expiration check
            if self._is_expired(key):
                self._delete_key(key)
                self.stats["keyspace_misses"] += 1
                return None

            if key not in self.store:
                self.stats["keyspace_misses"] += 1
                return None

            entry = self.store[key]
            if entry["type"] != "string":
                return f"WRONGTYPE: Key holds {entry['type']}, not string"

            entry["last_access"] = time.time()
            self.stats["keyspace_hits"] += 1
            return entry["value"]

    def mget(self, *keys):
        """MGET key1 key2 ... — Get multiple keys at once."""
        return [self.get(k) for k in keys]

    def mset(self, mapping):
        """MSET key1 val1 key2 val2 ... — Set multiple keys."""
        for key, value in mapping.items():
            self.set(key, value)
        return "OK"

    def incr(self, key, amount=1):
        """INCR key — Atomically increment integer value."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_writes"] += 1

            if self._is_expired(key):
                self._delete_key(key)

            if key not in self.store:
                self.store[key] = {
                    "type": "string",
                    "value": "0",
                    "last_access": time.time(),
                    "created_at": time.time()
                }

            entry = self.store[key]
            try:
                current = int(entry["value"])
                new_val = current + amount
                entry["value"] = str(new_val)
                entry["last_access"] = time.time()
                self._aof_log("INCRBY", key, amount)
                return new_val
            except ValueError:
                return "ERR: value is not an integer"

    def decr(self, key, amount=1):
        """DECR key — Atomically decrement."""
        return self.incr(key, -amount)

    def setnx(self, key, value, ex=None):
        """SETNX key value — Set only if Not eXists (for locks)."""
        return self.set(key, value, nx=True, ex=ex)

    def append(self, key, value):
        """APPEND key value — Append to existing string."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if key not in self.store or self._is_expired(key):
                self.store[key] = {
                    "type": "string", "value": str(value),
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            else:
                self.store[key]["value"] += str(value)
                self.store[key]["last_access"] = time.time()
            return len(self.store[key]["value"])

    def strlen(self, key):
        """STRLEN key — Length of string value."""
        val = self.get(key)
        return len(val) if val and not val.startswith("WRONGTYPE") else 0

    def lpush(self, key, *values):
        """LPUSH key val1 val2 ... — Push to left (head)."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_writes"] += 1

            if self._is_expired(key):
                self._delete_key(key)

            if key not in self.store:
                self.store[key] = {
                    "type": "list", "value": [],
                    "last_access": time.time(),
                    "created_at": time.time()
                }

            entry = self.store[key]
            if entry["type"] != "list":
                return f"WRONGTYPE: Key holds {entry['type']}, not list"

            for v in values:
                entry["value"].insert(0, str(v))
            entry["last_access"] = time.time()

            self._aof_log("LPUSH", key, *values)
            return len(entry["value"])

    def rpush(self, key, *values):
        """RPUSH key val1 val2 ... — Push to right (tail)."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_writes"] += 1

            if self._is_expired(key):
                self._delete_key(key)

            if key not in self.store:
                self.store[key] = {
                    "type": "list", "value": [],
                    "last_access": time.time(),
                    "created_at": time.time()
                }

            entry = self.store[key]
            if entry["type"] != "list":
                return f"WRONGTYPE: Key holds {entry['type']}, not list"

            for v in values:
                entry["value"].append(str(v))
            entry["last_access"] = time.time()

            self._aof_log("RPUSH", key, *values)
            return len(entry["value"])

    def lpop(self, key):
        """LPOP key — Remove and return from left."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key):
                self._delete_key(key)
                return None
            if key not in self.store:
                return None
            entry = self.store[key]
            if entry["type"] != "list" or not entry["value"]:
                return None
            val = entry["value"].pop(0)
            entry["last_access"] = time.time()
            if not entry["value"]:
                self._delete_key(key)
            self._aof_log("LPOP", key)
            return val

    def rpop(self, key):
        """RPOP key — Remove and return from right."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key):
                self._delete_key(key)
                return None
            if key not in self.store:
                return None
            entry = self.store[key]
            if entry["type"] != "list" or not entry["value"]:
                return None
            val = entry["value"].pop()
            entry["last_access"] = time.time()
            if not entry["value"]:
                self._delete_key(key)
            self._aof_log("RPOP", key)
            return val

    def lrange(self, key, start, stop):
        """LRANGE key start stop — Get range of list elements."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key) or key not in self.store:
                return []
            entry = self.store[key]
            if entry["type"] != "list":
                return []
            lst = entry["value"]
            if stop == -1:
                stop = len(lst)
            else:
                stop += 1
            entry["last_access"] = time.time()
            return lst[start:stop]

    def llen(self, key):
        """LLEN key — Length of list."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return 0
            entry = self.store[key]
            return len(entry["value"]) if entry["type"] == "list" else 0

    def hset(self, key, field, value):

        with self.cmd_lock:

            self.stats["total_writes"] += 1

            if self._is_expired(key):

    def hget(self, key, field):
        """HGET key field — Get hash field value."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key) or key not in self.store:
                return None
            entry = self.store[key]
            if entry["type"] != "hash":
                return None
            entry["last_access"] = time.time()
            return entry["value"].get(field)

    def hgetall(self, key):
        """HGETALL key — Get all fields and values."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key) or key not in self.store:

            entry = self.store[key]
            if entry["type"] != "hash":
                return {}
            entry["last_access"] = time.time()
            return dict(entry["value"])

    def hdel(self, key, *fields):
        """HDEL key field1 field2 ... — Delete hash fields."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key) or key not in self.store:
                return 0
            entry = self.store[key]
            if entry["type"] != "hash":
                return 0
            removed = 0
            for f in fields:
                if f in entry["value"]:
                    del entry["value"][f]
                    removed += 1
            if not entry["value"]:
                self._delete_key(key)
            return removed

    def hincrby(self, key, field, amount=1):

        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if key not in self.store:
                self.store[key] = {
                    "type": "hash", "value": {},
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            entry = self.store[key]
            current = int(entry["value"].get(field, "0"))
            new_val = current + amount
            entry["value"][field] = str(new_val)
            entry["last_access"] = time.time()
            return new_val

    def sadd(self, key, *members):
        """SADD key member1 member2 ... — Add to set."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_writes"] += 1

            if self._is_expired(key):
                self._delete_key(key)

            if key not in self.store:
                self.store[key] = {
                    "type": "set", "value": set(),
                    "last_access": time.time(),
                    "created_at": time.time()
                }

            entry = self.store[key]
            if entry["type"] != "set":
                return f"WRONGTYPE: Key holds {entry['type']}, not set"

            added = 0
            for m in members:
                if str(m) not in entry["value"]:
                    entry["value"].add(str(m))
                    added += 1
            entry["last_access"] = time.time()
            self._aof_log("SADD", key, *members)
            return added

    def smembers(self, key):
        """SMEMBERS key — Get all members of set."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key) or key not in self.store:
                return set()
            entry = self.store[key]
            if entry["type"] != "set":
                return set()
            entry["last_access"] = time.time()
            return set(entry["value"])

    def sismember(self, key, member):
        """SISMEMBER key member — Check if member exists in set."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return False
            entry = self.store[key]
            return str(member) in entry["value"] if entry["type"] == "set" else False

    def scard(self, key):
        """SCARD key — Get set cardinality (size)."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return 0
            entry = self.store[key]
            return len(entry["value"]) if entry["type"] == "set" else 0

    # SORTED SET OPERATIONS (leaderboard)
    # ============================================================

    def zadd(self, key, score, member):
        """ZADD key score member — Add to sorted set."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            self.stats["total_writes"] += 1

            if self._is_expired(key):
                self._delete_key(key)

            if key not in self.store:
                self.store[key] = {
                    "type": "zset",
                    "value": {
                        "members": {},     # member → score
                        "sorted": []       # [(score, member), ...] sorted
                    },
                    "last_access": time.time(),
                    "created_at": time.time()
                }

            entry = self.store[key]
            if entry["type"] != "zset":
                return f"WRONGTYPE"

            zdata = entry["value"]
            member_str = str(member)
            score = float(score)

            is_new = member_str not in zdata["members"]

            # Remove old entry if updating score
            if not is_new:
                old_score = zdata["members"][member_str]
                zdata["sorted"].remove((old_score, member_str))

            # Insert new entry
            zdata["members"][member_str] = score
            bisect.insort(zdata["sorted"], (score, member_str))
            entry["last_access"] = time.time()

            self._aof_log("ZADD", key, score, member)
            return 1 if is_new else 0

    def zscore(self, key, member):
        """ZSCORE key member — Get score of member."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return None
            entry = self.store[key]
            if entry["type"] != "zset":
                return None
            return entry["value"]["members"].get(str(member))

    def zrank(self, key, member):
        """ZRANK key member — Get rank (0-based, lowest score = 0)."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return None
            entry = self.store[key]
            if entry["type"] != "zset":
                return None
            member_str = str(member)
            if member_str not in entry["value"]["members"]:
                return None
            score = entry["value"]["members"][member_str]
            for i, (s, m) in enumerate(entry["value"]["sorted"]):
                if m == member_str and s == score:
                    return i
            return None

    def zrange(self, key, start, stop, with_scores=False):
        """ZRANGE key start stop — Get range by rank."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return []
            entry = self.store[key]
            if entry["type"] != "zset":
                return []
            sorted_list = entry["value"]["sorted"]
            if stop == -1:
                stop = len(sorted_list)
            else:
                stop += 1
            slice_result = sorted_list[start:stop]
            if with_scores:
                return [(m, s) for s, m in slice_result]
            return [m for s, m in slice_result]

    def zrevrange(self, key, start, stop, with_scores=False):
        """ZREVRANGE key start stop — Get range by rank (highest first)."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return []
            entry = self.store[key]
            if entry["type"] != "zset":
                return []
            sorted_list = list(reversed(entry["value"]["sorted"]))
            if stop == -1:
                stop = len(sorted_list)
            else:
                stop += 1
            slice_result = sorted_list[start:stop]
            if with_scores:
                return [(m, s) for s, m in slice_result]
            return [m for s, m in slice_result]

    def zcard(self, key):
        """ZCARD key — Get sorted set size."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return 0
            entry = self.store[key]
            if entry["type"] != "zset":
                return 0
            return len(entry["value"]["members"])

    # ============================================================
    # KEY OPERATIONS
    # ============================================================

    def delete(self, *keys):
        """DEL key1 key2 ... — Delete keys."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            removed = 0
            for key in keys:
                if key in self.store:
                    self._delete_key(key)
                    removed += 1
            self._aof_log("DEL", *keys)
            return removed

    def exists(self, key):
        """EXISTS key — Check if key exists."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if self._is_expired(key):
                self._delete_key(key)
                return False
            return key in self.store

    def expire(self, key, seconds):
        """EXPIRE key seconds — Set TTL on key."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if key not in self.store or self._is_expired(key):
                return 0
            self.expiry[key] = time.time() + seconds
            self._aof_log("EXPIRE", key, seconds)
            return 1

    def ttl(self, key):
        """TTL key — Get remaining TTL in seconds."""
        with self.cmd_lock:
            if key not in self.store:
                return -2  # Key doesn't exist
            if key not in self.expiry:
                return -1  # Key exists but no TTL
            remaining = self.expiry[key] - time.time()
            if remaining <= 0:
                self._delete_key(key)
                return -2
            return int(remaining)

    def persist(self, key):
        """PERSIST key — Remove TTL from key."""
        with self.cmd_lock:
            if key in self.expiry:
                del self.expiry[key]
                return 1
            return 0

    def keys(self, pattern="*"):
        """KEYS pattern — Find keys matching pattern."""
        with self.cmd_lock:
            self.stats["total_commands"] += 1
            if pattern == "*":
                return [k for k in self.store
                        if not self._is_expired(k)]
            # Simple glob matching
            result = []
            for k in self.store:
                if not self._is_expired(k):
                    if self._match_pattern(k, pattern):
                        result.append(k)
            return result

    def scan(self, cursor=0, match="*", count=10):
        """SCAN cursor [MATCH pattern] [COUNT count] — Non-blocking iteration."""
        with self.cmd_lock:
            all_keys = [k for k in self.store
                        if not self._is_expired(k)]
            if match != "*":
                all_keys = [k for k in all_keys
                            if self._match_pattern(k, match)]
            end = min(cursor + count, len(all_keys))
            batch = all_keys[cursor:end]
            next_cursor = end if end < len(all_keys) else 0
            return next_cursor, batch

    def type_of(self, key):
        """TYPE key — Get the type of key."""
        with self.cmd_lock:
            if self._is_expired(key) or key not in self.store:
                return "none"
            return self.store[key]["type"]

    def dbsize(self):
        """DBSIZE — Total number of keys."""
        with self.cmd_lock:
            return sum(1 for k in self.store
                       if not self._is_expired(k))

    def flushdb(self):
        """FLUSHDB — Delete all keys."""
        with self.cmd_lock:
            self.store.clear()
            self.expiry.clear()
            self.memory_used = 0
            return "OK"

    # INTERNAL: Expiration
    # ============================================================

    def _is_expired(self, key):
        if key in self.expiry:
            if time.time() > self.expiry[key]:
                return True
        return False

    def _delete_key(self, key):
        if key in self.store:
            self.memory_used -= self._estimate_size(key)
            del self.store[key]
        if key in self.expiry:
            del self.expiry[key]
        self.stats["expired_keys"] += 1

    def _start_expiration_thread(self):
        def _expire_loop():
            while True:
                time.sleep(0.1)  # Every 100ms
                self._active_expire_cycle()

        t = threading.Thread(target=_expire_loop, daemon=True)
        t.start()

    def _active_expire_cycle(self):
        """Redis-style active expiration: sample and delete."""
        with self.cmd_lock:
            if not self.expiry:
                return

            keys_with_ttl = list(self.expiry.keys())
            sample_size = min(20, len(keys_with_ttl))
            sample = random.sample(keys_with_ttl, sample_size)

            expired_count = 0
            for key in sample:
                if self._is_expired(key):
                    self._delete_key(key)
                    expired_count += 1

            # If >25% of sample was expired → run again immediately
            if sample_size > 0 and (expired_count / sample_size) > 0.25:
                # Recursive but limited by sample size shrinking
                if len(self.expiry) > 0:
                    self._active_expire_cycle()

    # ============================================================
    # INTERNAL: Memory Estimation
    # ============================================================

    def _estimate_entry_size(self, key, value):
        """Rough memory estimate for a key-value entry."""
        key_size = len(str(key)) * 2  # Unicode chars
        if isinstance(value, str):
            val_size = len(value) * 2
        elif isinstance(value, (list, set)):
            val_size = sum(len(str(v)) * 2 for v in value) + 64
        elif isinstance(value, dict):
            val_size = sum(
                len(str(k)) * 2 + len(str(v)) * 2
                for k, v in value.items()
            ) + 64
        else:
            val_size = 64
        overhead = 128  # Dict entry + metadata overhead
        return key_size + val_size + overhead

    def _estimate_size(self, key):
        """Estimate current memory for a key."""
        if key not in self.store:
            return 0
        entry = self.store[key]
        return self._estimate_entry_size(key, entry["value"])

    # ============================================================
    # INTERNAL: Memory Eviction
    # ============================================================

    def _check_memory_and_evict(self):
        """Evict keys when memory limit is reached."""
        if not self.maxmemory:
            return
        if self.memory_used <= self.maxmemory:
            return

        if self.eviction_policy == "noeviction":
            return  # Will reject writes in the SET method

        evicted = 0
        max_evictions = 10  # Evict up to 10 keys per check

        while (self.memory_used > self.maxmemory
               and evicted < max_evictions
               and self.store):

            victim_key = self._select_eviction_victim()
            if victim_key is None:
                break

            self.memory_used -= self._estimate_size(victim_key)
            del self.store[victim_key]
            if victim_key in self.expiry:
                del self.expiry[victim_key]
            self.stats["evicted_keys"] += 1
            evicted += 1

    def _select_eviction_victim(self):
        """
        Select a key to evict based on eviction policy.
        Uses Redis-style approximated LRU: sample 5 keys,
        evict the one with oldest last_access time.
        """
        if not self.store:
            return None

        policy = self.eviction_policy

        if policy == "allkeys-random":
            return random.choice(list(self.store.keys()))

        elif policy == "volatile-random":
            volatile_keys = list(self.expiry.keys())
            if volatile_keys:
                return random.choice(volatile_keys)
            return None

        elif policy == "volatile-ttl":
            # Evict key with shortest remaining TTL
            if not self.expiry:
                return None
            return min(self.expiry, key=self.expiry.get)

        elif policy in ("allkeys-lru", "volatile-lru"):
            # Approximated LRU: sample 5 keys, pick oldest access
            if policy == "volatile-lru":
                candidates = list(self.expiry.keys())
            else:
                candidates = list(self.store.keys())

            if not candidates:
                return None

            sample_size = min(5, len(candidates))
            sample = random.sample(candidates, sample_size)

            oldest_key = None
            oldest_time = float('inf')
            for key in sample:
                if key in self.store:
                    access_time = self.store[key].get(
                        "last_access", 0
                    )
                    if access_time < oldest_time:
                        oldest_time = access_time
                        oldest_key = key
            return oldest_key

        elif policy in ("allkeys-lfu", "volatile-lfu"):
            # Approximated LFU: sample 5 keys, pick least accessed
            # We track via last_access as a simplified proxy
            if policy == "volatile-lfu":
                candidates = list(self.expiry.keys())
            else:
                candidates = list(self.store.keys())

            if not candidates:
                return None

            sample_size = min(5, len(candidates))
            sample = random.sample(candidates, sample_size)

            # Use creation time distance from last_access as frequency proxy
            least_key = None
            least_score = float('inf')
            for key in sample:
                if key in self.store:
                    entry = self.store[key]
                    created = entry.get("created_at", 0)
                    accessed = entry.get("last_access", 0)
                    # More time between accesses = less frequent
                    score = accessed - created if accessed > created else 0
                    if score < least_score:
                        least_score = score
                        least_key = key
            return least_key

        return None

    # ============================================================
    # INTERNAL: Pattern Matching
    # ============================================================

    def _match_pattern(self, key, pattern):
        """Simple glob pattern matching (supports * and ?)."""
        import fnmatch
        return fnmatch.fnmatch(key, pattern)

    def enable_aof(self, filepath="miniredis.aof"):
        """Enable AOF persistence."""
        self.aof_enabled = True
        self.aof_filepath = filepath
        self.aof_file = open(filepath, "a")

    def _aof_log(self, command, *args):
        """Append a command to AOF file."""
        if not self.aof_enabled or not self.aof_file:
            return
        # Serialize command
        parts = [str(command)] + [str(a) for a in args]
        line = json.dumps(parts)
        try:
            self.aof_file.write(line + "\n")
            self.aof_file.flush()
        except Exception as e:

    def replay_aof(self, filepath="miniredis.aof"):
        """Replay AOF file to restore state."""
        if not os.path.exists(filepath):
            print(f"  [AOF] No file found: {filepath}")
            return 0

        commands_replayed = 0
        # Temporarily disable AOF to avoid re-logging during replay
        was_enabled = self.aof_enabled
        self.aof_enabled = False

        with open(filepath, "r") as f:
            for line in f:
                line = line.strip()
                if not line:
                    continue
                try:
                    parts = json.loads(line)
                    cmd = parts[0].upper()
                    args = parts[1:]
                    self._execute_aof_command(cmd, args)
                    commands_replayed += 1
                except Exception as e:
                    print(f"  [AOF REPLAY ERROR] {line}: {e}")

        self.aof_enabled = was_enabled
        print(f"  [AOF] Replayed {commands_replayed} commands "
              f"from {filepath}")
        return commands_replayed

    def _execute_aof_command(self, cmd, args):
        """Execute a single command during AOF replay."""
        if cmd == "SET":
            key, value = args[0], args[1]
            ex = None
            if len(args) > 2 and args[2] == "ex":
                ex = int(args[3])
            # Use internal set without lock (already in replay context)
            self.store[key] = {
                "type": "string",
                "value": str(value),
                "last_access": time.time(),
                "created_at": time.time()
            }
            if ex:
                self.expiry[key] = time.time() + ex
        elif cmd == "DEL":
            for key in args:
                self.store.pop(key, None)
                self.expiry.pop(key, None)
        elif cmd == "INCRBY":
            key = args[0]
            amount = int(args[1])
            if key not in self.store:
                self.store[key] = {
                    "type": "string", "value": "0",
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            current = int(self.store[key]["value"])
            self.store[key]["value"] = str(current + amount)
        elif cmd == "LPUSH":
            key = args[0]
            values = args[1:]
            if key not in self.store:
                self.store[key] = {
                    "type": "list", "value": [],
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            for v in values:
                self.store[key]["value"].insert(0, str(v))
        elif cmd == "RPUSH":
            key = args[0]
            values = args[1:]
            if key not in self.store:
                self.store[key] = {
                    "type": "list", "value": [],
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            for v in values:
                self.store[key]["value"].append(str(v))
        elif cmd == "HSET":
            key, field, value = args[0], args[1], args[2]
            if key not in self.store:
                self.store[key] = {
                    "type": "hash", "value": {},
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            self.store[key]["value"][field] = str(value)
        elif cmd == "SADD":
            key = args[0]
            members = args[1:]
            if key not in self.store:
                self.store[key] = {
                    "type": "set", "value": set(),
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            for m in members:
                self.store[key]["value"].add(str(m))
        elif cmd == "ZADD":
            key, score, member = args[0], float(args[1]), args[2]
            if key not in self.store:
                self.store[key] = {
                    "type": "zset",
                    "value": {"members": {}, "sorted": []},
                    "last_access": time.time(),
                    "created_at": time.time()
                }
            zdata = self.store[key]["value"]
            member_str = str(member)
            if member_str in zdata["members"]:
                old_score = zdata["members"][member_str]
                zdata["sorted"].remove((old_score, member_str))
            zdata["members"][member_str] = score
            bisect.insort(zdata["sorted"], (score, member_str))
        elif cmd == "EXPIRE":
            key, seconds = args[0], int(args[1])
            if key in self.store:
                self.expiry[key] = time.time() + seconds

    def aof_rewrite(self, filepath=None):
        """Compact AOF by writing current state as minimal commands."""
        filepath = filepath or getattr(
            self, 'aof_filepath', 'miniredis.aof'
        )
        temp_path = filepath + ".rewrite"
        commands_written = 0

        with self.cmd_lock:
            with open(temp_path, "w") as f:
                for key, entry in self.store.items():
                    if self._is_expired(key):
                        continue

                    entry_type = entry["type"]

                    if entry_type == "string":
                        cmd = ["SET", key, entry["value"]]
                        f.write(json.dumps(cmd) + "\n")
                    elif entry_type == "list":
                        if entry["value"]:
                            cmd = ["RPUSH", key] + entry["value"]
                            f.write(json.dumps(cmd) + "\n")
                    elif entry_type == "hash":
                        for field, val in entry["value"].items():
                            cmd = ["HSET", key, field, val]
                            f.write(json.dumps(cmd) + "\n")
                    elif entry_type == "set":
                        if entry["value"]:
                            cmd = ["SADD", key] + list(entry["value"])
                            f.write(json.dumps(cmd) + "\n")
                    elif entry_type == "zset":
                        for score, member in entry["value"]["sorted"]:
                            cmd = ["ZADD", key, score, member]
                            f.write(json.dumps(cmd) + "\n")

                    # Persist TTL
                    if key in self.expiry:
                        remaining = self.expiry[key] - time.time()
                        if remaining > 0:
                            cmd = ["EXPIRE", key, int(remaining)]
                            f.write(json.dumps(cmd) + "\n")

                    commands_written += 1

        # Atomically replace old AOF
        if os.path.exists(filepath):
            os.remove(filepath)
        os.rename(temp_path, filepath)

        # Reopen file handle
        if self.aof_enabled:
            if self.aof_file:
                self.aof_file.close()
            self.aof_file = open(filepath, "a")

        print(f"  [AOF REWRITE] Compacted to {commands_written} "
              f"commands")
        return commands_written

    def save_rdb(self, filepath="miniredis.rdb"):
        """Save current state as an RDB snapshot."""
        with self.cmd_lock:
            snapshot = {}
            for key, entry in self.store.items():
                if self._is_expired(key):
                    continue

                snap_entry = {
                    "type": entry["type"],
                    "created_at": entry["created_at"]
                }

                if entry["type"] == "set":
                    snap_entry["value"] = list(entry["value"])
                elif entry["type"] == "zset":
                    snap_entry["value"] = {
                        "members": entry["value"]["members"],
                        "sorted": entry["value"]["sorted"]
                    }
                else:
                    snap_entry["value"] = entry["value"]

                if key in self.expiry:
                    snap_entry["expiry"] = self.expiry[key]

                snapshot[key] = snap_entry

        with open(filepath, "w") as f:
            json.dump({
                "magic": "MINIREDIS",
                "version": "1.0",
                "timestamp": datetime.utcnow().isoformat(),
                "keys": len(snapshot),
                "data": snapshot
            }, f, indent=2)

        print(f"  [RDB] Snapshot saved: {len(snapshot)} keys → {filepath}")
        return len(snapshot)

    def load_rdb(self, filepath="miniredis.rdb"):
        """Load state from RDB snapshot."""
        if not os.path.exists(filepath):
            print(f"  [RDB] No file found: {filepath}")
            return 0

        with open(filepath, "r") as f:
            snapshot = json.load(f)

        if snapshot.get("magic") != "MINIREDIS":
            print(f"  [RDB] Invalid file format")
            return 0

        loaded = 0
        with self.cmd_lock:
            self.store.clear()
            self.expiry.clear()

            for key, snap_entry in snapshot.get("data", {}).items():
                entry = {
                    "type": snap_entry["type"],
                    "last_access": time.time(),
                    "created_at": snap_entry.get(
                        "created_at", time.time()
                    )
                }

                if snap_entry["type"] == "set":
                    entry["value"] = set(snap_entry["value"])
                elif snap_entry["type"] == "zset":
                    entry["value"] = {
                        "members": snap_entry["value"]["members"],
                        "sorted": [
                            tuple(item) for item
                            in snap_entry["value"]["sorted"]
                        ]
                    }
                else:
                    entry["value"] = snap_entry["value"]

                self.store[key] = entry

                if "expiry" in snap_entry:
                    if snap_entry["expiry"] > time.time():
                        self.expiry[key] = snap_entry["expiry"]
                    else:
                        # Already expired, skip
                        del self.store[key]
                        continue

                loaded += 1

        print(f"  [RDB] Loaded {loaded} keys from {filepath}")
        return loaded

import socket


class MiniRedisServer:
    """
    TCP server for MiniRedis.
    Accepts text-based commands over TCP.
    Single command processor thread (like real Redis).
    """

    def __init__(self, host="127.0.0.1", port=6380,
                 maxmemory=None, eviction_policy="allkeys-lru"):
        self.host = host
        self.port = port
        self.redis = MiniRedisCore(
            maxmemory_bytes=maxmemory,
            eviction_policy=eviction_policy
        )
        self.running = False

    def start(self):
        """Start the server."""
        self.running = True
        server_socket = socket.socket(
            socket.AF_INET, socket.SOCK_STREAM
        )
        server_socket.setsockopt(
            socket.SOL_SOCKET, socket.SO_REUSEADDR, 1
        )
        server_socket.bind((self.host, self.port))
        server_socket.listen(128)
        server_socket.settimeout(1.0)

        print(f"  {'=' * 55}")
        print(f"  🔴 MINIREDIS SERVER v1.0")
        print(f"  Listening on {self.host}:{self.port}")
        print(f"  {'=' * 55}\n")

        while self.running:
            try:
                client_socket, addr = server_socket.accept()
                thread = threading.Thread(
                    target=self._handle_client,
                    args=(client_socket, addr),
                    daemon=True
                )
                thread.start()
            except socket.timeout:
                continue
            except Exception as e:
                if self.running:
                    print(f"  [SERVER ERROR] {e}")

        server_socket.close()

    def stop(self):
        self.running = False

    def _handle_client(self, client_socket, addr):
        """Handle a single client connection."""
        try:
            while self.running:
                data = client_socket.recv(4096)
                if not data:
                    break

                command_str = data.decode("utf-8").strip()
                if not command_str:
                    continue

                # Parse and execute command
                response = self._execute_command(command_str)

                # Send response
                response_str = json.dumps(response) + "\n"
                client_socket.send(response_str.encode("utf-8"))

        except ConnectionResetError:
            pass
        except Exception as e:
            print(f"  [CLIENT ERROR] {addr}: {e}")
        finally:
            client_socket.close()

    def _execute_command(self, command_str):
        """Parse and execute a Redis-like command."""
        parts = self._parse_command(command_str)
        if not parts:
            return {"error": "Empty command"}

        cmd = parts[0].upper()
        args = parts[1:]

        try:
            # ---- String commands ----
            if cmd == "SET":
                if len(args) < 2:
                    return {"error": "SET requires key and value"}
                key, value = args[0], args[1]
                ex = None
                nx = False
                xx = False
                i = 2

    def _parse_command(self, command_str):
        """Parse command string into parts (handle quoted strings)."""
        parts = []
        current = ""
        in_quotes = False
        quote_char = None

        for char in command_str:
            if in_quotes:
                if char == quote_char:
                    in_quotes = False
                else:
                    current += char
            elif char in ('"', "'"):
                in_quotes = True
                quote_char = char
            elif char == " ":
                if current:
                    parts.append(current)
                    current = ""
            else:
                current += char

        if current:
            parts.append(current)

        return parts

    session_redis = MiniRedisCore()

    def create_session(user_id, ttl_seconds=1800):
        """Create a new session (like login)."""
        import hashlib
        token = hashlib.sha256(
            f"{user_id}:{time.time()}:{random.random()}".encode()
        ).hexdigest()[:32]

        session_key = f"session:{token}"
        session_redis.hset(session_key, "user_id", user_id)
        session_redis.hset(session_key, "created_at",
                           datetime.utcnow().isoformat())
        session_redis.hset(session_key, "last_active",
                           datetime.utcnow().isoformat())
        session_redis.expire(session_key, ttl_seconds)
        return token

    def get_session(token):
        """Validate and return session data."""
        session_key = f"session:{token}"
        data = session_redis.hgetall(session_key)
        if not data:
            return None  # Session expired or invalid
        # Refresh TTL on activity (sliding expiration)
        session_redis.expire(session_key, 1800)
        session_redis.hset(session_key, "last_active",
                           datetime.utcnow().isoformat())
        return data

    def destroy_session(token):
        """Logout — destroy session."""
        session_redis.delete(f"session:{token}")

    # Simulate session lifecycle
    print(f"  Creating sessions for 3 users...")
    tokens = {}
    for user in ["alice", "bob", "charlie"]:
        token = create_session(user, ttl_seconds=5)

✅ TTL/Expiration uses dual strategy:
   → Lazy: check on read (zero background cost)
   → Active: sample and sweep (reclaim memory)
   → Combined: efficient and thorough

✅ Persistence has two approaches:
   → RDB snapshots: periodic dump, fast recovery, some data loss
   → AOF: log every write, minimal loss, slower recovery
   → Production uses both for optimal tradeoffs

✅ Memory eviction prevents OOM:
   → Approximated LRU: sample 5 keys, evict oldest access
   → Multiple policies: allkeys-lru, volatile-ttl, etc.
   → ALWAYS set maxmemory in production
