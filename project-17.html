📋 PHASE 3 (Medium) — Professional Techniques | Project 2 of Phase 3

🎯 HYPOTHETICAL PROBLEM

Your engineering team has a problem:
→ Developers share code snippets via Slack messages
→ Slack reformats the code (breaks indentation, adds smart quotes)
→ Log files get truncated in email threads
→ Config files shared in Google Docs lose their formatting
→ Debugging sessions need sharing 50KB stack traces
→ Some snippets are sensitive (API keys) and should auto-delete

They ask you to build an INTERNAL Pastebin:

  INPUT:  A block of text (code, logs, config, anything)
          + optional title, syntax language, expiration
  
  OUTPUT: A unique URL like https://paste.internal.co/Xk9mP2
          Anyone with the link can view the content
          Content renders with syntax highlighting
          Optionally auto-expires after N hours/days

Requirements from the team:
  → 500K new pastes per month
  → 2M views per month (4:1 read-to-write ratio)
  → Paste sizes: 1 byte to 10MB (average ~5KB)
  → Support syntax highlighting for 50+ languages
  → Paste expiration: 10 minutes, 1 hour, 1 day, 1 week, never
  → "Burn after read" option (delete after first view)
  → Optional password protection
  → Raw text endpoint (for curl/wget access)
  → System must be available 99.9%

Let's do the math:
  → 500K writes/month ≈ 0.2 writes/second (low write volume)
  → 2M reads/month ≈ 0.8 reads/second (low-moderate)
  → Peak: 10x average = ~8 reads/second (still manageable)
  → Storage per paste: avg 5KB content + 500 bytes metadata = 5.5KB
  → 500K × 5.5KB = 2.75GB per month
  → 1 year: 33GB, 5 years: 165GB (moderate — fits single DB initially)
  → BUT: some pastes are 10MB → 10% large pastes = significant storage
  → Large content should NOT live in the database rows

KEY DIFFERENCE FROM PROJECT 16 (URL Shortener):
  → Project 16: Stored ~200 bytes per URL (tiny)
  → Project 17: Stores 1B to 10MB per paste (1000x-50000x larger)
  → This changes WHERE and HOW we store data fundamentally
  → Introduces: Object Storage vs Database storage decision

THE KEY DECISION: Where to Store Content?

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
OPTION A: Everything in Database (Simple)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  TABLE pastes:
    id, short_code, content (TEXT/BLOB), title, language, ...
  
  Pros:
    → Simplest implementation
    → Single system to manage
    → ACID transactions for everything
    → Easy backups (just backup the DB)
  
  Cons:
    → Large content bloats DB size (10MB rows!)
    → DB backups become huge
    → Database not optimized for large blob storage
    → Replication (Project 13) copies ALL content = slow
    → Cannot leverage CDN for content delivery
  
  Best for: < 100K pastes, all small (< 100KB)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
OPTION B: Metadata in DB + Content in Object Store  ← BEST
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  DATABASE (small, fast):
    paste_id, content_hash, content_size, title, language,
    created_at, expires_at, is_burned, password_hash
  
  OBJECT STORE (large, cheap):
    key = content_hash → value = raw text content
  
  Pros:
    → DB stays small and fast (only metadata)
    → Object store optimized for large files
    → Natural deduplication (same hash = same content = store once)
    → Can serve content from CDN/edge
    → Independent scaling (more storage ≠ bigger DB)
  
  Cons:
    → Two systems to manage
    → Slightly more complex read path
    → Must handle consistency (what if DB write succeeds but object fails?)
  
  Best for: Production systems at any scale

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
OPTION C: Hybrid (Small in DB, Large in Object Store)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  IF content_size < 64KB:
    → Store directly in DB column (inline)
    → One less hop on read
  
  IF content_size >= 64KB:
    → Store in object store
    → DB stores only the hash/pointer
  
  Pros:
    → Best of both worlds
    → Most pastes (< 64KB) served from single DB query
    → Large pastes don't bloat DB
  
  Cons:
    → More complex logic (two code paths)
    → Must handle both storage modes consistently
  
  Best for: Optimized production systems

FOR THIS PROJECT: We implement OPTION C (Hybrid)
  → Small content inline in DB (fast single-query reads)
  → Large content in filesystem-based object store
  → Deduplication via content hashing for both

✅ WHAT SHOULD DO

1. USE content-addressable storage for deduplication
   → Hash the content (SHA256)
   → If same hash exists → reuse stored content
   → Two people paste the same log file → stored ONCE
   → Saves significant storage over time

2. SEPARATE metadata from content
   → Already covered in Project 16: keep DB lean
   → Metadata: small, structured, indexed, queried
   → Content: large, unstructured, read-only after creation

3. COMPRESS large content before storage
   → Text compresses extremely well (60-90% reduction)
   → 10MB log file → 1-2MB compressed
   → Decompress on read (fast with modern algorithms)
   → Use gzip or zstd

4. IMPLEMENT proper expiration cleanup
   → Don't delete on every read (check expiry, but batch delete)
   → Background job scans for expired pastes
   → Delete metadata first, then content
   → Content deletion must check: is anyone else using this hash?
     (dedup means multiple pastes can reference same content)

5. CACHE popular pastes aggressively
   → From Project 5: cache-aside pattern
   → But ONLY cache metadata + small content (< 64KB)
   → Don't cache 10MB pastes in Redis (waste of memory)
   → Hot pastes (shared on social media) benefit most

6. PROVIDE raw content endpoint
   → GET /raw/Xk9mP2 → returns plain text (Content-Type: text/plain)
   → Useful for: curl, wget, scripting, API consumption
   → No HTML wrapping, no syntax highlighting
   → Different from GET /p/Xk9mP2 which returns formatted view

7. IMPLEMENT burn-after-read atomically
   → Read and mark-as-burned must be ONE atomic operation
   → Otherwise two simultaneous readers both see the content
   → Use database UPDATE ... RETURNING or SELECT FOR UPDATE

❌ WHAT SHOULD NOT DO

1. NEVER store large content directly in database rows
   → Already explained above
   → 10MB × 100K pastes = 1TB database
   → Backups take hours, replication lags badly
   → DB is for metadata, object store is for content

2. NEVER skip content size limits
   → Without limits: user uploads 1GB "paste"
   → Server memory exhausted reading the request
   → Disk fills up quickly
   → ALWAYS enforce: max 10MB per paste, reject larger

3. NEVER delete content without checking reference count
   → Dedup means content_hash "abc123" might be referenced by 5 pastes
   → Deleting paste #1 should NOT delete the content
   → Only delete content when ZERO pastes reference that hash
   → Reference counting or garbage collection pattern

4. NEVER store passwords in plaintext
   → Password-protected pastes need hashed passwords
   → Use bcrypt or argon2 (not MD5, not SHA256 for passwords)
   → Same principle as user authentication

5. NEVER return expired content "one last time"
   → If paste expired at 3:00 PM and user requests at 3:01 PM → 410 Gone
   → Don't be lenient — expiration is a security feature
   → Burn-after-read especially: once burned, GONE

6. NEVER cache burn-after-read pastes
   → Cache would serve the content after it's "burned"
   → Defeats the entire purpose
   → Mark burn-after-read pastes as non-cacheable

7. NEVER trust client-reported content type
   → Client says "this is Python code"
   → Could actually be malicious HTML/JavaScript
   → ALWAYS treat stored content as plain text
   → Never render paste content as HTML (XSS risk)

🚫 WHEN NOT TO USE

1. DON'T use Pastebin architecture for binary files
   → Binary files (images, PDFs, executables) need different handling
   → Use a file upload service (Project 25: CloudVault)
   → Pastebin is for TEXT content specifically

2. DON'T use content-addressable storage when content is MUTABLE
   → Pastebin content is immutable (create once, read many)
   → If content changes → hash changes → it's a different object
   → For mutable documents: use versioning (Project 39: DocSync)

3. DON'T use database for object storage at scale
   → Already covered: DB is for metadata, not blobs
   → Exception: very small scale (< 10K items, all < 100KB)

4. DON'T use Pastebin for real-time collaboration
   → Pastebin is fire-and-forget: create → share → view
   → For real-time editing: use CRDT/OT (Project 39)
   → Different architecture entirely

5. DON'T use deduplication when content is sensitive
   → If user A and user B paste the same API key
   → Dedup means they share storage
   → If user A deletes → content still exists (user B's reference)
   → For highly sensitive content: skip dedup, isolate storage

MECHANIC 1: CONTENT-ADDRESSABLE STORAGE (CAS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  CONCEPT:
    → Instead of "I'll put this content at location X"
    → We say "The content IS its address"
    → Hash of content = storage key
  
  FLOW:
    content = "def hello():\n    print('world')"
    hash = SHA256(content) = "a1b2c3d4e5f6..."
    
    Store: object_store[hash] = content
    
    Later: content = object_store[hash]
  
  WHY THIS ENABLES DEDUP:
    User A pastes: "def hello():\n    print('world')"
    → hash = "a1b2c3d4e5f6..."
    → stored at objects/a1/b2/c3d4e5f6...
    
    User B pastes the EXACT same text:
    → hash = "a1b2c3d4e5f6..." (same!)
    → object already exists → skip storage
    → just create new metadata pointing to same hash
    
    Storage used: 1x instead of 2x ✓
  
  DIRECTORY STRUCTURE (filesystem object store):
    objects/
    ├── a1/
    │   ├── b2c3d4e5f6...7890.gz    ← content file
    │   └── f8e7d6c5b4...1234.gz
    ├── b3/
    │   └── 9a8b7c6d5e...5678.gz
    └── ...
    
    First 2 chars of hash = directory (prevents too many files in one dir)
    Rest of hash = filename
    .gz = compressed

  REFERENCE COUNTING:
    ┌─────────────────────────────────────────┐
    │ content_refs table                       │
    │                                          │
    │ content_hash    │ ref_count              │
    │ ────────────────┼────────────            │
    │ a1b2c3d4...     │ 3  (3 pastes use this) │
    │ x9y8z7w6...     │ 1  (1 paste uses this) │
    └─────────────────────────────────────────┘
    
    When paste created → ref_count += 1
    When paste deleted → ref_count -= 1
    When ref_count = 0 → safe to delete content file

MECHANIC 3: BURN-AFTER-READ (Atomic Operation)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  THE PROBLEM:
    User A and User B click the link at the EXACT same time
    Both read "is_burned = False"
    Both get the content
    Both try to set "is_burned = True"
    → Content was seen TWICE (violates burn-after-read)
  
  THE SOLUTION: Atomic Update + Check
  
    SQL (atomic):
      UPDATE pastes 
      SET is_burned = TRUE, burned_at = NOW()
      WHERE paste_id = 'Xk9mP2' 
        AND is_burned = FALSE
      RETURNING content_hash, storage_type, ...
    
    IF rows_affected = 1:
      → WE got it. Fetch content and return.
    IF rows_affected = 0:
      → Someone else already burned it.
      → Return 410 Gone.
    
    This is ONE atomic database operation.
    Only ONE reader can ever "win" the burn.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
MECHANIC 4: EXPIRATION CLEANUP STRATEGY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  TWO-LAYER APPROACH:
  
  Layer 1: LAZY EXPIRATION (on read)
    → When someone requests a paste
    → Check expires_at < NOW
    → If expired → return 410 Gone
    → DON'T delete yet (just reject the read)
    → Fast, no extra work on reads
  
  Layer 2: BATCH CLEANUP (background job)
    → Runs every hour (or configurable)
    → Step A: Find all expired pastes
        SELECT paste_id, content_hash, storage_type
        FROM pastes
        WHERE expires_at < NOW() AND is_active = TRUE
        LIMIT 1000  (batch to avoid long transactions)
    
    → Step B: Mark as inactive
        UPDATE pastes SET is_active = FALSE
        WHERE paste_id IN (...)

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: Content Deletion with Active References
  → Paste A and Paste B both reference hash "xyz"
  → Paste A expires → cleanup deletes content "xyz"
  → Paste B now points to non-existent content!
  → FIX: Reference counting (ALWAYS check ref_count before delete)

PITFALL 2: Race Condition on Dedup Check
  → Two simultaneous requests paste identical content
  → Both check: "hash exists?" → both get NO
  → Both try to store → one fails or content stored twice
  → FIX: Use INSERT ON CONFLICT for content_refs
         Let filesystem handle duplicate writes (idempotent)

PITFALL 3: Memory Exhaustion on Large Pastes
  → User uploads 10MB paste
  → Server reads entire 10MB into memory
  → 100 concurrent 10MB uploads = 1GB memory
  → FIX: Stream large content to disk, don't buffer in memory
         Set request body size limits at LB level

PITFALL 4: Burn-After-Read Cache Leak
  → Burn paste viewed → marked as burned in DB
  → But cached copy still exists in Redis
  → Second viewer gets cached (non-burned) copy
  → FIX: NEVER cache burn-after-read pastes
         Skip cache entirely for is_burned=True pastes

PITFALL 5: Compression Bomb
  → Attacker uploads 1KB that decompresses to 10GB
  → Server tries to decompress → runs out of memory
  → FIX: Compress STORED content, not RECEIVED content
         We compress what the user sends, then store compressed
         We DON'T accept pre-compressed content and decompress it

PITFALL 6: Hash Collision (Theoretical)
  → SHA256 collision: two different contents produce same hash
  → Probability: ~1 in 2^128 (effectively impossible)
  → But if paranoid: store content_size alongside hash
  → If hash matches but size differs → NOT a duplicate

PITFALL 7: Expired Content Still in Cache
  → Same as Project 16 pitfall
  → FIX: Set cache TTL = min(default_ttl, time_until_expiry)
         Always check expiration AFTER cache retrieval

import os
import gzip
import hashlib
import shutil
import json
import threading
from datetime import datetime


class FilesystemObjectStore:
    """
    Content-addressable object store using local filesystem.
    
    Key design decisions:
      → Content addressed by SHA256 hash (dedup-friendly)
      → First 2 chars of hash = subdirectory (avoid huge dirs)
      → Optional gzip compression for text content
      → Thread-safe operations
    
    In production: Replace with S3/MinIO (same interface)
    """
    
    def __init__(self, base_path="./paste_objects", compress=True,
                 compress_threshold=1024):
        self.base_path = base_path
        self.compress = compress
        self.compress_threshold = compress_threshold  # Bytes
        self.lock = threading.Lock()
        
        # Create base directory
        os.makedirs(base_path, exist_ok=True)
        
        # Metrics
        self.total_stored = 0
        self.total_retrieved = 0
        self.total_deleted = 0
        self.bytes_saved_by_dedup = 0
    
    def _get_object_path(self, content_hash):
        """
        Convert hash to filesystem path.
        
        Hash: a1b2c3d4e5f6...
        Path: ./paste_objects/a1/b2c3d4e5f6...
        
        First 2 chars as directory prevents having millions
        of files in a single directory (filesystem performance).
        """
        prefix = content_hash[:2]
        filename = content_hash[2:]
        dir_path = os.path.join(self.base_path, prefix)
        return dir_path, os.path.join(dir_path, filename)
    
    def compute_hash(self, content):
        """Compute SHA256 hash of content"""
        if isinstance(content, str):
            content = content.encode('utf-8')
        return hashlib.sha256(content).hexdigest()
    
    def exists(self, content_hash):
        """Check if content with this hash already exists"""
        _, file_path = self._get_object_path(content_hash)
        # Check both compressed and uncompressed
        return os.path.exists(file_path) or os.path.exists(file_path + ".gz")
    
    def store(self, content, content_hash=None):
        """
        Store content. Returns (content_hash, stored_size, was_duplicate).
        
        If content with same hash exists → skip storage (dedup!)
        If new → optionally compress and write to disk.
        """
        if isinstance(content, str):
            content_bytes = content.encode('utf-8')
        else:
            content_bytes = content
        
        # Compute hash if not provided
        if not content_hash:
            content_hash = self.compute_hash(content_bytes)
        
        original_size = len(content_bytes)
        
        # Check for duplicate
        if self.exists(content_hash):
            self.bytes_saved_by_dedup += original_size
            return {
                "content_hash": content_hash,
                "original_size": original_size,
                "stored_size": 0,
                "is_compressed": False,
                "was_duplicate": True
            }
        
        # Determine if we should compress
        should_compress = (
            self.compress and 
            original_size >= self.compress_threshold
        )
        
        dir_path, file_path = self._get_object_path(content_hash)
        
        with self.lock:
            # Create directory if needed
            os.makedirs(dir_path, exist_ok=True)
            
            if should_compress:
                compressed = gzip.compress(content_bytes, compresslevel=6)
                final_path = file_path + ".gz"
                
                with open(final_path, 'wb') as f:
                    f.write(compressed)
                
                stored_size = len(compressed)
            else:
                with open(file_path, 'wb') as f:
                    f.write(content_bytes)
                
                stored_size = original_size
        
        self.total_stored += 1
        
        return {
            "content_hash": content_hash,
            "original_size": original_size,
            "stored_size": stored_size,
            "is_compressed": should_compress,
            "was_duplicate": False,
            "compression_ratio": round(
                (1 - stored_size / original_size) * 100, 1
            ) if should_compress and original_size > 0 else 0
        }
    
    def retrieve(self, content_hash):
        """
        Retrieve content by hash.
        Returns content as string, or None if not found.
        Handles both compressed and uncompressed files.
        """
        _, file_path = self._get_object_path(content_hash)
        
        try:
            # Try compressed first (more common)
            if os.path.exists(file_path + ".gz"):
                with open(file_path + ".gz", 'rb') as f:
                    compressed = f.read()
                content_bytes = gzip.decompress(compressed)
                self.total_retrieved += 1
                return content_bytes.decode('utf-8')
            
            # Try uncompressed
            if os.path.exists(file_path):
                with open(file_path, 'rb') as f:
                    content_bytes = f.read()
                self.total_retrieved += 1
                return content_bytes.decode('utf-8')
            
            return None
            
        except Exception as e:
            print(f"[OBJECT STORE ERROR] Retrieve {content_hash}: {e}")
            return None
    
    def delete(self, content_hash):
        """
        Delete content by hash.
        ONLY call this when reference count reaches 0.
        """
        _, file_path = self._get_object_path(content_hash)
        
        deleted = False
        
        with self.lock:
            if os.path.exists(file_path + ".gz"):
                os.remove(file_path + ".gz")
                deleted = True
            elif os.path.exists(file_path):
                os.remove(file_path)
                deleted = True
        
        if deleted:
            self.total_deleted += 1
            
            # Clean up empty directory
            dir_path = os.path.dirname(file_path)
            try:
                if not os.listdir(dir_path):
                    os.rmdir(dir_path)
            except OSError:
                pass
        
        return deleted
    
    def get_stats(self):
        """Get storage statistics"""
        total_files = 0
        total_bytes = 0
        
        for root, dirs, files in os.walk(self.base_path):
            for f in files:
                total_files += 1
                total_bytes += os.path.getsize(os.path.join(root, f))
        
        return {
            "total_objects": total_files,
            "total_bytes_on_disk": total_bytes,
            "total_mb_on_disk": round(total_bytes / 1024 / 1024, 2),
            "total_stored_operations": self.total_stored,
            "total_retrieved_operations": self.total_retrieved,
            "total_deleted_operations": self.total_deleted,
            "bytes_saved_by_dedup": self.bytes_saved_by_dedup,
            "mb_saved_by_dedup": round(
                self.bytes_saved_by_dedup / 1024 / 1024, 2
            )
        }
    
    def cleanup(self):
        """Remove entire object store (for testing)"""
        if os.path.exists(self.base_path):
            shutil.rmtree(self.base_path)

class Base62:
    ALPHABET = "0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ"
    BASE = 62
    
    @staticmethod
    def encode(number):
        if number == 0:
            return Base62.ALPHABET[0]
        chars = []
        while number > 0:
            chars.append(Base62.ALPHABET[number % Base62.BASE])
            number //= Base62.BASE
        return ''.join(reversed(chars))
    
    @staticmethod
    def encode_with_padding(number, min_length=6):
        encoded = Base62.encode(number)
        while len(encoded) < min_length:
            encoded = Base62.ALPHABET[0] + encoded
        return encoded


class LRUCache:
    def __init__(self, capacity=10000, default_ttl=3600):
        self.capacity = capacity
        self.default_ttl = default_ttl
        self.cache = OrderedDict()
        self.lock = threading.Lock()
        self.hits = 0
        self.misses = 0
    
    def get(self, key):
        with self.lock:
            if key not in self.cache:
                self.misses += 1
                return None
            value, expires_at = self.cache[key]
            if expires_at and time.time() > expires_at:
                del self.cache[key]
                self.misses += 1
                return None
            self.cache.move_to_end(key)
            self.hits += 1
            return value
    
    def put(self, key, value, ttl=None):
        ttl = ttl or self.default_ttl
        expires_at = time.time() + ttl if ttl else None
        with self.lock:
            if key in self.cache:
                self.cache.move_to_end(key)
            self.cache[key] = (value, expires_at)
            while len(self.cache) > self.capacity:
                self.cache.popitem(last=False)
    
    def delete(self, key):
        with self.lock:
            self.cache.pop(key, None)
    
    def get_stats(self):
        total = self.hits + self.misses
        hit_rate = (self.hits / total * 100) if total > 0 else 0
        return {
            "size": len(self.cache),
            "capacity": self.capacity,
            "hits": self.hits,
            "misses": self.misses,
            "hit_rate_pct": round(hit_rate, 2)
        }

class PasswordHelper:
    @staticmethod
    def hash_password(password):
        salt = os.urandom(16).hex()
        hashed = hashlib.sha256(
            (salt + password).encode('utf-8')
        ).hexdigest()
        return f"{salt}:{hashed}"
    
    @staticmethod
    def verify_password(password, stored_hash):
        if not stored_hash:
            return True  # No password set
        salt, expected = stored_hash.split(':')
        actual = hashlib.sha256(
            (salt + password).encode('utf-8')
        ).hexdigest()
        return actual == expected

class ExpirationPolicy:
    NEVER = "never"
    TEN_MINUTES = "10m"
    ONE_HOUR = "1h"
    ONE_DAY = "1d"
    ONE_WEEK = "1w"
    ONE_MONTH = "1M"
    BURN_AFTER_READ = "burn"
    
    DURATIONS = {
        "10m": timedelta(minutes=10),
        "1h": timedelta(hours=1),
        "1d": timedelta(days=1),
        "1w": timedelta(weeks=1),
        "1M": timedelta(days=30),
    }
    
    @staticmethod
    def compute_expiry(policy):
        """Returns expires_at datetime or None"""
        if policy == ExpirationPolicy.NEVER:
            return None
        if policy == ExpirationPolicy.BURN_AFTER_READ:
            return None  # Burn is handled differently
        
        duration = ExpirationPolicy.DURATIONS.get(policy)
        if duration:
            return datetime.now() + duration
        return None
    
    @staticmethod

class PasteDropEngine:

      → Content-addressable deduplication
      → Compression for storage efficiency
      → Burn-after-read (atomic)
      → Password protection
      → Expiration policies
    """

    # CORE OPERATION 1: CREATE PASTE
    # ================================================================
    
    def create_paste(self, content, title="", language="text",
                     expiration="never", password=None,
                     creator_ip=None):
        """
        Create a new paste.
        
        Flow:
          1. Validate input
          2. Hash content (for dedup)
          3. Decide storage type (inline vs object)
          4. Store content (skip if duplicate)
          5. Generate paste ID
          6. Store metadata in DB
          7. Update reference count
          8. Populate cache
          9. Return paste info
        """
        
        # ── Step 1: Validate ──
        validation = self._validate_input(content, title, language, expiration)
        if not validation["valid"]:
            return {"success": False, "error": validation["error"]}
        
        content = content if isinstance(content, str) else content.decode('utf-8')
        content_size = len(content.encode('utf-8'))
        
        # ── Step 2: Hash content ──
        content_hash = self.object_store.compute_hash(content)
        
        # ── Step 3: Decide storage type ──
        # Small content → inline in database
        # Large content → external object store
        if content_size < self.INLINE_THRESHOLD:
            storage_type = "inline"
        else:
            storage_type = "object"
        
        # ── Step 4: Store content ──
        was_duplicate = False
        stored_size = content_size
        is_compressed = False
        
        if storage_type == "object":
            store_result = self.object_store.store(content, content_hash)
            was_duplicate = store_result["was_duplicate"]
            stored_size = store_result["stored_size"]
            is_compressed = store_result["is_compressed"]
        else:
            # For inline, check if we already have this hash
            conn = sqlite3.connect(self.db_path)
            cursor = conn.cursor()
            cursor.execute(
                "SELECT ref_count FROM content_refs WHERE content_hash = ?",
                (content_hash,)
            )
            existing = cursor.fetchone()
            was_duplicate = existing is not None
            conn.close()
        
        if was_duplicate:
            self.dedup_count += 1
        
        # ── Step 5: Compute expiration ──
        is_burn = expiration == "burn"
        expires_at = ExpirationPolicy.compute_expiry(expiration)
        expires_at_str = expires_at.isoformat() if expires_at else None
        
        # ── Step 6: Hash password if provided ──
        password_hash = None
        if password:
            password_hash = PasswordHelper.hash_password(password)
        
        # ── Step 7: Generate paste ID and store metadata ──
        now = datetime.now().isoformat()
        
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        try:
            # Insert with temporary paste_id
            cursor.execute(
                """INSERT INTO pastes 
                   (paste_id, title, language, content_hash, content_size,
                    storage_type, inline_content, is_compressed,
                    created_at, expires_at, expiration_policy,
                    is_burn, password_hash, creator_ip)
                   VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)""",
                ("__temp__", title, language, content_hash, content_size,
                 storage_type,
                 content if storage_type == "inline" else None,
                 int(is_compressed),
                 now, expires_at_str, expiration,
                 int(is_burn), password_hash, creator_ip)
            )
            
            row_id = cursor.lastrowid
            paste_id = Base62.encode_with_padding(
                row_id + self.id_offset, 6
            )
            
            cursor.execute(
                "UPDATE pastes SET paste_id = ? WHERE id = ?",
                (paste_id, row_id)
            )
            
            # ── Step 8: Update reference count ──
            cursor.execute(
                """INSERT INTO content_refs 
                   (content_hash, ref_count, content_size, compressed_size, first_seen)
                   VALUES (?, 1, ?, ?, ?)
                   ON CONFLICT(content_hash) DO UPDATE 
                   SET ref_count = ref_count + 1""",
                (content_hash, content_size,
                 stored_size if not was_duplicate else 0, now)
            )
            
            conn.commit()
            
            # ── Step 9: Populate cache (skip burn-after-read) ──
            if not is_burn:
                cache_data = {
                    "paste_id": paste_id,
                    "title": title,
                    "language": language,
                    "content_hash": content_hash,
                    "content_size": content_size,
                    "storage_type": storage_type,
                    "created_at": now,
                    "expires_at": expires_at_str,
                    "is_burn": is_burn,
                    "has_password": password is not None,
                    "view_count": 0
                }
                
                # Cache content too if small enough
                if storage_type == "inline" and content_size < 32768:
                    cache_data["content"] = content
                
                cache_ttl = 3600
                if expires_at:
                    remaining = (expires_at - datetime.now()).total_seconds()
                    cache_ttl = max(60, min(cache_ttl, remaining))
                
                self.cache.put(paste_id, cache_data, ttl=cache_ttl)
            
            self.total_created += 1
            conn.close()
            
            return {
                "success": True,
                "paste_id": paste_id,
                "paste_url": f"{self.base_url}/p/{paste_id}",
                "raw_url": f"{self.base_url}/raw/{paste_id}",
                "title": title,
                "language": language,
                "content_size": content_size,
                "content_hash": content_hash[:16] + "...",
                "storage_type": storage_type,
                "was_duplicate": was_duplicate,
                "is_compressed": is_compressed,
                "expiration_policy": expiration,
                "expires_at": expires_at_str,
                "is_burn_after_read": is_burn,
                "has_password": password is not None,
                "created_at": now
            }
            
        except Exception as e:
            conn.rollback()
            conn.close()
            return {"success": False, "error": f"Failed to create paste: {str(e)}"}
    
    def _validate_input(self, content, title, language, expiration):
        """Validate paste input"""
        if not content:
            return {"valid": False, "error": "Content is required"}
        
        if isinstance(content, str):
            content_bytes = content.encode('utf-8')
        else:
            content_bytes = content
        
        if len(content_bytes) > self.MAX_PASTE_SIZE:
            return {
                "valid": False,
                "error": f"Content exceeds maximum size of "
                         f"{self.MAX_PASTE_SIZE // (1024*1024)}MB"
            }
        
        if len(content_bytes) == 0:
            return {"valid": False, "error": "Content cannot be empty"}
        
        if title and len(title) > 255:
            return {"valid": False, "error": "Title must be under 255 characters"}
        
        if language and language not in SUPPORTED_LANGUAGES:
            return {
                "valid": False,
                "error": f"Unsupported language: '{language}'",
                "supported": SUPPORTED_LANGUAGES[:20]
            }
        
        valid_expirations = list(ExpirationPolicy.DURATIONS.keys()) + ["never", "burn"]
        if expiration not in valid_expirations:
            return {
                "valid": False,
                "error": f"Invalid expiration: '{expiration}'",
                "valid_options": valid_expirations
            }
        
        return {"valid": True}

    def view_paste(self, paste_id, password=None, ip_address=None):
        """
        Retrieve a paste by ID.
        
        Flow:
          1. Check cache for metadata
          2. Cache miss → load from DB
          3. Check expiration
          4. Check password
          5. Handle burn-after-read (atomic)
          6. Load content (inline or object store)
          7. Update view count (async-style)
          8. Return content + metadata
        """
        
        # ── Step 1: Cache lookup ──
        cached = self.cache.get(paste_id)
        source = "cache"
        metadata = None
        cached_content = None
        
        if cached:
            metadata = cached
            cached_content = cached.get("content")
        else:
            # ── Step 2: Database lookup ──
            source = "database"
            metadata = self._load_metadata_from_db(paste_id)
            
            if not metadata:
                return {
                    "success": False,
                    "error": "Paste not found",
                    "status_code": 404
                }
        
        # ── Step 3: Check active status ──
        if not metadata.get("is_active", True):
            return {
                "success": False,
                "error": "This paste has been deleted",
                "status_code": 410
            }
        
        # ── Step 4: Check expiration ──
        expires_at = metadata.get("expires_at")
        if expires_at:
            try:
                expires_dt = datetime.fromisoformat(expires_at)
                if datetime.now() > expires_dt:
                    self.cache.delete(paste_id)
                    self.total_expired += 1
                    return {
                        "success": False,
                        "error": "This paste has expired",
                        "expired_at": expires_at,
                        "status_code": 410
                    }
            except Exception:
                pass
        
        # ── Step 5: Check password ──
        if metadata.get("has_password") or metadata.get("password_hash"):
            pw_hash = metadata.get("password_hash")
            if not pw_hash:
                # Need to load from DB
                conn = sqlite3.connect(self.db_path)
                cursor = conn.cursor()
                cursor.execute(
                    "SELECT password_hash FROM pastes WHERE paste_id = ?",
                    (paste_id,)
                )
                row = cursor.fetchone()
                conn.close()
                pw_hash = row[0] if row else None
            
            if pw_hash:
                if not password:
                    return {
                        "success": False,
                        "error": "This paste is password-protected",
                        "requires_password": True,
                        "status_code": 401
                    }
                
                if not PasswordHelper.verify_password(password, pw_hash):
                    return {
                        "success": False,
                        "error": "Incorrect password",
                        "status_code": 403
                    }
        
        # ── Step 6: Handle burn-after-read (ATOMIC) ──
        if metadata.get("is_burn"):
            burn_result = self._atomic_burn(paste_id)
            
            if not burn_result["success"]:
                return {
                    "success": False,
                    "error": "This paste has already been viewed and burned",
                    "status_code": 410
                }
            
            # Remove from cache permanently
            self.cache.delete(paste_id)
            self.total_burned += 1
            
            # Content was loaded during burn
            content = burn_result.get("content")
            
            if not content:
                content = self._load_content(metadata)
            
            return {
                "success": True,
                "paste_id": paste_id,
                "title": metadata.get("title", ""),
                "language": metadata.get("language", "text"),
                "content": content,
                "content_size": metadata.get("content_size", 0),
                "created_at": metadata.get("created_at"),
                "is_burned": True,
                "burn_notice": "This paste has been permanently deleted after viewing",
                "view_count": 1,
                "source": "database",
                "status_code": 200
            }
        
        # ── Step 7: Load content (non-burn path) ──
        content = cached_content
        
        if not content:
            content = self._load_content(metadata)
            
            if content is None:
                return {
                    "success": False,
                    "error": "Content could not be retrieved",
                    "status_code": 500
                }
            
            # Update cache with content if small
            if (metadata.get("content_size", 0) < 32768 and
                    not metadata.get("is_burn")):
                metadata["content"] = content
                cache_ttl = 3600
                if expires_at:
                    try:
                        remaining = (
                            datetime.fromisoformat(expires_at) - datetime.now()
                        ).total_seconds()
                        cache_ttl = max(60, min(cache_ttl, remaining))
                    except Exception:
                        pass
                self.cache.put(paste_id, metadata, ttl=cache_ttl)
        
        # ── Step 8: Update view count (fire-and-forget style) ──
        threading.Thread(
            target=self._increment_view_count,
            args=(paste_id,),
            daemon=True
        ).start()
        
        self.total_views += 1
        view_count = metadata.get("view_count", 0) + 1
        
        return {
            "success": True,
            "paste_id": paste_id,
            "paste_url": f"{self.base_url}/p/{paste_id}",
            "raw_url": f"{self.base_url}/raw/{paste_id}",
            "title": metadata.get("title", ""),
            "language": metadata.get("language", "text"),
            "content": content,
            "content_size": metadata.get("content_size", 0),
            "created_at": metadata.get("created_at"),
            "expires_at": metadata.get("expires_at"),
            "view_count": view_count,
            "source": source,
            "status_code": 200
        }

    def _atomic_burn(self, paste_id):
        """
        Atomically burn a paste.
        
        This is the KEY mechanism for burn-after-read:
          → UPDATE ... WHERE is_burned = 0
          → Only ONE reader can "win" this update
          → Others get rows_affected = 0 → paste already burned
        """
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        try:
            now = datetime.now().isoformat()
            
            cursor.execute(
                """UPDATE pastes 
                   SET is_burned = 1, burned_at = ?, view_count = 1
                   WHERE paste_id = ? AND is_burned = 0""",
                (now, paste_id)
            )
            
            affected = cursor.rowcount
            
            if affected == 0:
                conn.close()
                return {"success": False, "reason": "Already burned"}
            
            # Load content before we potentially clean up
            cursor.execute(
                """SELECT inline_content, content_hash, storage_type
                   FROM pastes WHERE paste_id = ?""",
                (paste_id,)
            )
            row = cursor.fetchone()
            conn.commit()
            conn.close()
            
            content = None
            if row:
                if row[2] == "inline":
                    content = row[0]
                else:
                    content = self.object_store.retrieve(row[1])
            
            return {"success": True, "content": content}
            
        except Exception as e:
            conn.close()
            return {"success": False, "reason": str(e)}

    def delete_paste(self, paste_id):
        """Soft-delete a paste and update reference counts"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        # Get content hash before deactivating
        cursor.execute(
            "SELECT content_hash, is_active FROM pastes WHERE paste_id = ?",
            (paste_id,)
        )
        row = cursor.fetchone()
        
        if not row:
            conn.close()
            return {"success": False, "error": "Paste not found"}
        
        if not row[1]:
            conn.close()
            return {"success": False, "error": "Paste already deleted"}
        
        content_hash = row[0]
        
        # Deactivate paste
        cursor.execute(
            "UPDATE pastes SET is_active = 0 WHERE paste_id = ?",
            (paste_id,)
        )
        
        # Decrement reference count
        cursor.execute(
            "UPDATE content_refs SET ref_count = ref_count - 1 "
            "WHERE content_hash = ?",
            (content_hash,)
        )
        
        # Check if content is now orphaned
        cursor.execute(
            "SELECT ref_count FROM content_refs WHERE content_hash = ?",
            (content_hash,)
        )
        ref_row = cursor.fetchone()
        
        conn.commit()
        conn.close()
        
        # Delete orphaned content from object store
        if ref_row and ref_row[0] <= 0:
            self.object_store.delete(content_hash)
        
        # Remove from cache
        self.cache.delete(paste_id)
        
        return {"success": True, "message": f"Paste {paste_id} deleted"}

    # OPERATION 4: CLEANUP EXPIRED PASTES
    # ================================================================
    
    def cleanup_expired(self, batch_size=500):
        """
        Background job: find and deactivate expired pastes.
        
        Two-layer approach (from Internal Mechanics):
          Layer 1: Lazy check on read (already in view_paste)
          Layer 2: This batch cleanup job
        """
        now = datetime.now().isoformat()
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        # Find expired pastes
        cursor.execute(
            """SELECT paste_id, content_hash FROM pastes
               WHERE expires_at IS NOT NULL
               AND expires_at < ?
               AND is_active = 1
               LIMIT ?""",
            (now, batch_size)
        )
        expired = cursor.fetchall()
        
        if not expired:
            conn.close()
            return {"cleaned_up": 0}
        
        paste_ids = [row[0] for row in expired]
        content_hashes = [row[1] for row in expired]
        
        # Deactivate expired pastes
        placeholders = ','.join('?' * len(paste_ids))
        cursor.execute(
            f"UPDATE pastes SET is_active = 0 WHERE paste_id IN ({placeholders})",
            paste_ids
        )
        
        # Decrement reference counts
        hash_counts = {}
        for h in content_hashes:
            hash_counts[h] = hash_counts.get(h, 0) + 1
        
        for h, count in hash_counts.items():
            cursor.execute(
                "UPDATE content_refs SET ref_count = ref_count - ? "
                "WHERE content_hash = ?",
                (count, h)
            )
        
        # Find orphaned content
        cursor.execute(
            "SELECT content_hash FROM content_refs WHERE ref_count <= 0"
        )
        orphaned = [row[0] for row in cursor.fetchall()]
        
        # Delete orphaned content refs
        if orphaned:
            placeholders = ','.join('?' * len(orphaned))
            cursor.execute(
                f"DELETE FROM content_refs WHERE content_hash IN ({placeholders})",
                orphaned
            )
        
        conn.commit()
        conn.close()
        
        # Delete orphaned content files
        for h in orphaned:
            self.object_store.delete(h)
        
        # Clear cache for expired pastes
        for pid in paste_ids:
            self.cache.delete(pid)
        
        self.total_expired += len(paste_ids)
        
        return {
            "cleaned_up": len(paste_ids),
            "content_orphans_removed": len(orphaned)
        }

                    "paste_url": f"{self.base_url}/p/{r[0]}",
                    "title": r[1] or "(untitled)",
                    "language": r[2],
                    "content_size": r[3],
                    "content_size_human": self._human_size(r[3]),
                    "created_at": r[4],
                    "expires_at": r[5],
                    "view_count": r[6]
                }
                for r in rows
            ]
        }
    
    # ================================================================
    # OPERATION 6: SYSTEM STATS
    # ================================================================
    
    def get_system_stats(self):
        """Comprehensive system statistics"""
        conn = sqlite3.connect(self.db_path)
        cursor = conn.cursor()
        
        cursor.execute("SELECT COUNT(*) FROM pastes WHERE is_active = 1")
        active_pastes = cursor.fetchone()[0]
        
        cursor.execute("SELECT COUNT(*) FROM pastes")
        total_pastes = cursor.fetchone()[0]
        
        cursor.execute(
            "SELECT SUM(content_size) FROM pastes WHERE is_active = 1"
        )
        total_content_size = cursor.fetchone()[0] or 0
        
        cursor.execute("SELECT SUM(view_count) FROM pastes")
        total_views_db = cursor.fetchone()[0] or 0
        
        cursor.execute("SELECT SUM(ref_count) FROM content_refs")
        total_refs = cursor.fetchone()[0] or 0
        
        cursor.execute("SELECT COUNT(*) FROM content_refs")
        unique_contents = cursor.fetchone()[0]
        
        # Language distribution
        cursor.execute(
            """SELECT language, COUNT(*) as cnt FROM pastes
               WHERE is_active = 1
               GROUP BY language ORDER BY cnt DESC LIMIT 10"""
        )
        language_dist = [
            {"language": r[0], "count": r[1]} for r in cursor.fetchall()
        ]
        
        # Size distribution
        cursor.execute(
            """SELECT 
                 CASE 
                   WHEN content_size < 1024 THEN '< 1KB'
                   WHEN content_size < 10240 THEN '1-10KB'
                   WHEN content_size < 65536 THEN '10-64KB'
                   WHEN content_size < 1048576 THEN '64KB-1MB'
                   ELSE '> 1MB'
                 END as size_bucket,
                 COUNT(*) as cnt
               FROM pastes WHERE is_active = 1
               GROUP BY size_bucket ORDER BY cnt DESC"""
        )
        size_dist = [
            {"bucket": r[0], "count": r[1]} for r in cursor.fetchall()
        ]
        
        conn.close()
        
        # Dedup stats
        dedup_ratio = 0
        if total_refs > 0 and unique_contents > 0:
            dedup_ratio = round(
                (1 - unique_contents / total_refs) * 100, 1
            )
        
        return {
            "total_pastes": total_pastes,
            "active_pastes": active_pastes,
            "total_views": total_views_db,
            "total_content_bytes": total_content_size,
            "total_content_human": self._human_size(total_content_size),
            "unique_content_objects": unique_contents,
            "dedup_ratio_pct": dedup_ratio,
            "engine_metrics": {
                "created": self.total_created,
                "views": self.total_views,
                "burned": self.total_burned,
                "expired_cleaned": self.total_expired,
                "dedup_saves": self.dedup_count
            },
            "cache": self.cache.get_stats(),
            "language_distribution": language_dist,
            "size_distribution": size_dist
        }
    
    @staticmethod
    def _human_size(size_bytes):
        """Convert bytes to human readable"""
        for unit in ['B', 'KB', 'MB', 'GB']:
            if size_bytes < 1024:
                return f"{size_bytes:.1f} {unit}"
            size_bytes /= 1024
        return f"{size_bytes:.1f} TB"
    
    def shutdown(self):
        """Graceful shutdown"""
        pass
    
    def cleanup_all(self):
        """Remove all data (for testing)"""
        self.object_store.cleanup_all()
        if os.path.exists(self.db_path):
            os.remove(self.db_path)

class PasteDropHTTPHandler(BaseHTTPRequestHandler):
    """
    Endpoints:
      POST   /api/paste              → Create paste
      GET    /p/{paste_id}           → View paste (formatted JSON)
      GET    /raw/{paste_id}         → Raw content (text/plain)
      GET    /api/paste/{paste_id}   → Paste info + metadata
      DELETE /api/paste/{paste_id}   → Delete paste
      GET    /api/recent             → Recent public pastes
      GET    /api/stats              → System statistics
      GET    /api/languages          → Supported languages
      GET    /api/expirations        → Expiration options
      GET    /healthz                → Liveness (Project 15)
      GET    /ready                  → Readiness (Project 15)
    """
    
    def send_json(self, status_code, data):
        self.send_response(status_code)
        self.send_header('Content-Type', 'application/json')
        self.send_header('Access-Control-Allow-Origin', '*')
        body = json.dumps(data, indent=2)
        self.send_header('Content-Length', str(len(body)))
        self.end_headers()
        self.wfile.write(body.encode('utf-8'))
    
    def send_text(self, status_code, text):
        self.send_response(status_code)
        self.send_header('Content-Type', 'text/plain; charset=utf-8')
        self.send_header('Access-Control-Allow-Origin', '*')
        body = text.encode('utf-8')
        self.send_header('Content-Length', str(len(body)))
        self.end_headers()
        self.wfile.write(body)
    
    def read_body(self):
        content_length = int(self.headers.get('Content-Length', 0))
        if content_length > 0:
            body = self.rfile.read(content_length)
            return json.loads(body.decode('utf-8'))
        return {}
    
    def read_raw_body(self):
        content_length = int(self.headers.get('Content-Length', 0))
        if content_length > 0:
            return self.rfile.read(content_length).decode('utf-8')
        return ""
    
    def get_client_ip(self):
        forwarded = self.headers.get('X-Forwarded-For')
        if forwarded:
            return forwarded.split(',')[0].strip()
        return self.client_address[0]

                    "ready": True,
                    "active_pastes": stats["active_pastes"]
                })
            except Exception as e:
                self.send_json(503, {"ready": False, "error": str(e)})
            return
        
        # ── View paste (JSON format) ──
        if path.startswith('/p/'):
            paste_id = path[3:]
            password = query.get('password', [None])[0]
            
            result = engine.view_paste(
                paste_id,
                password=password,
                ip_address=self.get_client_ip()
            )
            
            status = result.get("status_code", 200)

    # POST HANDLERS
    # ================================================================
    
    def do_POST(self):
        path = self.path
        
        if path == '/api/paste':
            content_type = self.headers.get('Content-Type', '')
            
            # Support both JSON and plain text submission
            if 'application/json' in content_type:
                try:
                    body = self.read_body()
                except Exception:
                    self.send_json(400, {"error": "Invalid JSON"})
                    return
                
                content = body.get("content", "")
                title = body.get("title", "")
                language = body.get("language", "text")
                expiration = body.get("expiration", "never")
                password = body.get("password")
            
            elif 'text/plain' in content_type:
                # Raw text submission (curl friendly)
                content = self.read_raw_body()
                parsed = urlparse(self.path)
                query = parse_qs(parsed.query)
                title = query.get('title', [''])[0]
                language = query.get('language', ['text'])[0]
                expiration = query.get('expiration', ['never'])[0]
                password = query.get('password', [None])[0]
            
            else:
                # Default: try JSON
                try:
                    body = self.read_body()
                    content = body.get("content", "")
                    title = body.get("title", "")
                    language = body.get("language", "text")
                    expiration = body.get("expiration", "never")
                    password = body.get("password")
                except Exception:
                    self.send_json(400, {
                        "error": "Provide JSON body with 'content' field "
                                 "or send text/plain"
                    })
                    return
            
            if not content:
                self.send_json(400, {"error": "Content is required"})
                return
            
            result = engine.create_paste(
                content=content,
                title=title,
                language=language,
                expiration=expiration,
                password=password,
                creator_ip=self.get_client_ip()
            )
            
            if result["success"]:
                self.send_json(201, result)
            else:
                self.send_json(400, result)
            return
        
        self.send_json(404, {"error": f"Cannot POST to '{path}'"})

def run_cleanup_job(engine_instance, interval=300):
    """
    Periodically clean up expired pastes.
    
    From Internal Mechanics:
      Layer 1: Lazy check on read (already in view_paste)
      Layer 2: This batch cleanup (runs every N seconds)
    
    Why both?
      → Lazy: instant response for expired paste requests
      → Batch: reclaims storage, removes orphaned content
    """
    while True:
        time.sleep(interval)
        try:
            result = engine_instance.cleanup_expired()
            if result["cleaned_up"] > 0:
                print(f"  [CLEANUP] Removed {result['cleaned_up']} expired pastes, "
                      f"{result['content_orphans_removed']} orphaned objects")
        except Exception as e:
            print(f"  [CLEANUP ERROR] {e}")

def api_request(method, path, body=None, content_type="application/json"):
    """Helper to make HTTP requests to PasteDrop"""
    url = f"{BASE_URL}{path}"
    
    if body and content_type == "application/json":
        data = json.dumps(body).encode('utf-8')
    elif body and content_type == "text/plain":
        data = body.encode('utf-8')
    else:
        data = None
    
    req = urllib.request.Request(url, data=data, method=method)
    if data:
        req.add_header('Content-Type', content_type)
    
    try:
        resp = urllib.request.urlopen(req, timeout=10)
        ct = resp.headers.get('Content-Type', '')
        if 'json' in ct:
            return {"status": resp.status, "body": json.loads(resp.read().decode('utf-8'))}
        else:
            return {"status": resp.status, "text": resp.read().decode('utf-8')}
    except urllib.error.HTTPError as e:
        ct = e.headers.get('Content-Type', '')
        raw = e.read().decode('utf-8')
        if 'json' in ct:
            return {"status": e.code, "body": json.loads(raw)}
        else:
            return {"status": e.code, "text": raw}
    except Exception as e:
        return {"status": 0, "body": {"error": str(e)}}

def run_concurrent_test(paste_id):
    """Test concurrent reads to verify thread safety"""
    
    num_threads = 10
    requests_per_thread = 15
    results = {"success": 0, "errors": 0}
    results_lock = threading.Lock()
    
    def worker(worker_id):
        local_ok = 0
        local_err = 0
        for _ in range(requests_per_thread):
            result = api_request("GET", f"/p/{paste_id}")
            if result["status"] == 200:
                local_ok += 1
            else:
                local_err += 1
        with results_lock:
            results["success"] += local_ok
            results["errors"] += local_err
    
    total = num_threads * requests_per_thread
    print(f"  Starting {num_threads} threads × {requests_per_thread} requests = {total} total")
    
    start = time.time()
    threads = []
    for i in range(num_threads):
        t = threading.Thread(target=worker, args=(i,))
        threads.append(t)
        t.start()
    for t in threads:
        t.join()
    elapsed = time.time() - start
    
    throughput = total / elapsed if elapsed > 0 else 0
    
    print(f"  ✅ Success: {results['success']}/{total}")
    print(f"  ❌ Errors:  {results['errors']}/{total}")
    print(f"  ⏱️  Time:    {elapsed:.2f}s ({throughput:.1f} req/sec)")

def test_burn_race_condition():

    print(f"  Created burn paste: {burn_id}")
    
    # Launch 10 threads to read simultaneously
    num_threads = 10
    thread_results = []
    results_lock = threading.Lock()
    barrier = threading.Barrier(num_threads)

        # Wait for all threads to be ready
        barrier.wait()
        # All fire at the same time
        result = api_request("GET", f"/p/{burn_id}")
        with results_lock:

                "success": result.get("body", {}).get("success", False),
                "got_content": "content" in result.get("body", {})

        t.start()
    
    for t in threads:

━━━━━━━━━━━━━━━
