🎯 HYPOTHETICAL PROBLEM

You are building a video streaming platform — like YouTube, Netflix, or Twitch

→ 1 billion users worldwide
→ 500 million daily active viewers
→ 500 hours of video uploaded EVERY MINUTE
→ Average watch time: 40 minutes per day per user
→ Videos range from 30 seconds to 12 hours long

A user in Mumbai opens the app:
  → Taps on a 4K cooking tutorial
  → Video MUST start playing within 2 seconds
  → Her WiFi is fast → serves 1080p
  → She walks to the kitchen → switches to mobile data
  → Bandwidth drops → video seamlessly drops to 480p
  → No buffering, no interruption
  → She scrubs to minute 45 → video jumps there INSTANTLY

Meanwhile:
  → A creator in São Paulo uploads a 2-hour documentary
  → The platform must convert it into 6 different quality levels
  → Each quality level must be split into thousands of small chunks
  → All within 30 minutes of upload
  → Available to viewers worldwide

AND:
  → 100 million people are watching simultaneously
  → A viral video has 50 million concurrent viewers
  → Viewers are spread across 6 continents
  → Each viewer gets the CLOSEST copy of video chunks
  → Storage: 1 EXABYTE (1,000,000 TB) of total video content

HOW do you architect a system that:
  → Ingests video uploads at massive scale
  → Transcodes into multiple quality levels
  → Splits into streamable chunks
  → Stores petabytes efficiently
  → Delivers globally with sub-second latency
  → Adapts quality to each viewer's bandwidth in real-time
  → Handles 100M concurrent streams without breaking

🔍 HOW TO APPROACH IT

DECOMPOSE INTO SUB-SYSTEMS:

Sub-System 1: UPLOAD PIPELINE
  → Accept raw video file from creator
  → Validate format, size, content
  → Store original in durable object storage
  → Trigger transcoding pipeline

Sub-System 2: TRANSCODING PIPELINE
  → Convert raw video into multiple quality levels (renditions)
  → Split each rendition into small chunks (2-10 seconds each)
  → Generate manifest files describing all available chunks
  → Async, parallelizable, compute-heavy

Sub-System 3: STORAGE LAYER
  → Object storage for video chunks (petabyte-scale)
  → Metadata database for video info, user data, analytics
  → CacheGrid (Project 33) for hot content caching

Sub-System 4: DELIVERY NETWORK
  → CDN edge servers worldwide (from Project 12)
  → Adaptive bitrate streaming protocol (HLS / DASH)
  → Cache hot chunks at edge, pull cold from origin

Sub-System 5: PLAYBACK SERVICE
  → Client requests manifest → learns available qualities
  → Client monitors bandwidth → selects appropriate quality
  → Client requests chunks sequentially → plays continuously
  → Seek/scrub → jump to specific chunk instantly

Sub-System 6: RECOMMENDATION + METADATA
  → Video catalog, search, recommendations
  → View counts, likes, comments
  → Not our focus here — we focus on the STREAMING infrastructure

PART A: ADAPTIVE BITRATE STREAMING (ABR)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

THE CORE IDEA:
  → Don't stream one continuous file
  → Split video into small CHUNKS (segments)
  → Each chunk = 2-10 seconds of video
  → Encode each chunk at MULTIPLE quality levels
  → Client picks quality PER CHUNK based on bandwidth

  One 10-minute video becomes:

  ┌──────────────────────────────────────────────────┐
  │ Quality    │ Chunk 1 │ Chunk 2 │ ... │ Chunk 60  │
  ├────────────┼─────────┼─────────┼─────┼───────────┤
  │ 1080p 5Mbps│ C1_1080 │ C2_1080 │ ... │ C60_1080  │
  │ 720p  2Mbps│ C1_720  │ C2_720  │ ... │ C60_720   │
  │ 480p  1Mbps│ C1_480  │ C2_480  │ ... │ C60_480   │
  │ 360p 500Kbps│ C1_360 │ C2_360  │ ... │ C60_360   │
  │ 240p 250Kbps│ C1_240 │ C2_240  │ ... │ C60_240   │
  │ Audio 128Kbps│ A1     │ A2      │ ... │ A60       │
  └──────────────┴────────┴─────────┴─────┴───────────┘

  Total chunks: 60 × 6 = 360 files for one 10-min video

  Playback scenario:
    Chunk 1: Good WiFi → request 1080p
    Chunk 2: Still good → 1080p
    Chunk 3: Bandwidth drops → switch to 720p
    Chunk 4: Bandwidth recovers → back to 1080p
    → User sees continuous video with varying quality
    → NO buffering, NO interruption


TWO MAIN PROTOCOLS:

  HLS (HTTP Live Streaming) — Apple's protocol:
    → Master playlist (.m3u8) lists all quality levels
    → Each quality has its own playlist listing chunk URLs
    → Chunks are .ts (MPEG-2 Transport Stream) files
    → Most widely supported

  DASH (Dynamic Adaptive Streaming over HTTP):
    → MPD (Media Presentation Description) XML manifest
    → Similar concept, more flexible
    → Open standard (ISO)

  WE IMPLEMENT: HLS-style approach (simpler, dominant)

  THREE TIERS:

  Tier 1: HOT STORAGE (CacheGrid — Project 33)
    → Manifests (.m3u8 files) — small, read extremely frequently
    → First few chunks of popular videos (instant start)
    → Video metadata (title, duration, thumbnail URL)
    → Distributed cache cluster, sub-millisecond reads

  Tier 2: WARM STORAGE (CDN Edge — Project 12)
    → Chunks of currently-being-watched videos
    → Cached at edge servers closest to viewers
    → TTL-based: popular chunks stay, unpopular evicted
    → Origin pull when edge doesn't have the chunk

  Tier 3: COLD STORAGE (Object Storage — Project 17/25)
    → ALL chunks, ALL qualities, ALL videos
    → Petabyte-scale blob storage (S3, GCS, Azure Blob)
    → The "source of truth" for all video content
    → Accessed when CDN cache misses
    → Cheaper per GB, higher latency

  Storage path for a single video chunk:

    Object Storage Key:
      videos/{video_id}/chunks/{quality}/{chunk_number}.ts

📌 WHAT COULD BE DONE

TRANSCODING ARCHITECTURE OPTIONS:

Option 1: Monolithic Transcoder
  → One big server transcodes entire video sequentially
  → Simple but slow — 2-hour video takes 4+ hours
  → Single point of failure

Option 2: Distributed Chunked Transcoding (OUR APPROACH)
  → Split raw video into segments first
  → Each segment transcoded independently in parallel
  → Workers from a pool (auto-scaling)
  → Message queue distributes tasks (from Project 11)
  → 2-hour video done in 15-30 minutes with enough workers

Option 3: Serverless Transcoding
  → Each chunk = one Lambda/Cloud Function invocation
  → Auto-scales to thousands of concurrent workers
  → Pay per millisecond of compute
  → Good for bursty upload patterns

WE USE: Option 2 with queue-based worker pool

BANDWIDTH ESTIMATION OPTIONS:

Option 1: Download Speed Measurement
  → Measure how fast each chunk downloads
  → Estimate bandwidth = chunk_size / download_time
  → Simple, reactive

Option 2: EWMA (Exponentially Weighted Moving Average)
  → Smooth out bandwidth spikes
  → bandwidth = α × latest_measurement + (1-α) × previous_estimate
  → Prevents oscillation between qualities

Option 3: Buffer-Based Selection
  → If buffer is full → upgrade quality
  → If buffer is low → downgrade quality
  → If buffer is empty → emergency lowest quality
  → More stable than pure bandwidth measurement

WE USE: Combined EWMA bandwidth + buffer-based (industry standard)

✅ WHAT SHOULD DO

1. CHUNK DURATION: 2-10 SECONDS
   → Too short (1s): Too many HTTP requests, overhead
   → Too long (30s): Slow quality switching, large files
   → Sweet spot: 6-10 seconds for VOD, 2-4 for live
   → Each chunk starts with a keyframe (IDR frame)
   → Keyframe = complete image, not dependent on previous frames

2. BUFFER AHEAD (PREFETCH)
   → While playing chunk N, download chunk N+1 and N+2
   → Maintains a buffer of 20-30 seconds ahead
   → Absorbs network fluctuations
   → Don't prefetch TOO far — wastes bandwidth if user leaves

3. START WITH LOWEST QUALITY
   → First chunk = 240p or 360p (smallest file)
   → Starts playing in < 1 second
   → Upgrade quality on subsequent chunks as bandwidth measured
   → Users perceive "instant start" even on slow connections
   → This is called "fast start" or "quick start"

4. USE BYTE-RANGE REQUESTS FOR SEEKING
   → Chunk files can support HTTP Range headers
   → Seek within a chunk without downloading entire chunk
   → CDN supports byte-range caching natively

5. SEPARATE AUDIO AND VIDEO TRACKS
   → Audio changes quality less noticeably
   → Keep audio at consistent quality (128-192 Kbps)
   → Only switch VIDEO quality based on bandwidth
   → Reduces total number of chunks needed

6. GENERATE THUMBNAILS AT INTERVALS
   → Every 10 seconds, extract a thumbnail image
   → Used for seek preview (hover over timeline)
   → Small sprite sheets: 160×90px thumbnails
   → Loaded once, cached forever

7. IMPLEMENT CHUNK PRELOADING FOR POPULAR VIDEOS
   → Viral video detected → pre-push chunks to CDN edges
   → Don't wait for first cache miss
   → From Project 33: hot key detection → proactive caching

❌ WHAT SHOULD NOT DO

1. NEVER STREAM RAW UPLOADED VIDEO DIRECTLY
   → Uploaded in MOV, AVI, MKV — not web-playable
   → Huge file sizes, no adaptive quality
   → ALWAYS transcode into HLS/DASH format first

2. NEVER USE A SINGLE QUALITY LEVEL
   → User on 3G connection can't play 4K
   → User on fiber doesn't want 240p
   → Multiple renditions are NON-NEGOTIABLE for any real service

3. NEVER MAKE CHUNKS TOO LARGE
   → 60-second chunks = 60 seconds before quality can switch
   → User buffering for 60 seconds on bandwidth drop
   → Keep chunks ≤ 10 seconds

4. NEVER TRANSCODE SYNCHRONOUSLY IN UPLOAD REQUEST
   → "Upload and wait until all qualities ready" → timeout
   → Upload returns immediately → transcoding runs async
   → From Project 11: decouple via message queue

5. NEVER STORE ALL CHUNKS IN A SINGLE STORAGE BUCKET
   → Billions of chunk files in one directory → listing is slow
   → Partition by: video_id / quality / chunk_number
   → Object storage key = structured path

6. NEVER SKIP CDN FOR VIDEO DELIVERY
   → Serving video chunks from origin = 200ms+ latency per chunk
   → CDN edge = 10-50ms latency
   → Video needs many sequential requests → latency compounds
   → CDN is MANDATORY, not optional

7. NEVER CACHE MANIFESTS FOREVER
   → For VOD: manifests are static → long TTL (hours/days)
   → For LIVE streaming: manifest updates every few seconds
   → Stale manifest = missing new chunks = video freezes
   → Short TTL or no-cache for live manifests

8. NEVER IGNORE DRM (Digital Rights Management)
   → Paid content must be encrypted
   → Each chunk encrypted with a content key
   → Key delivered via separate license server
   → Without DRM: anyone can download and redistribute
   → Not implemented in demo but critical in production

🚫 WHEN NOT TO USE

1. DO NOT USE ABR STREAMING FOR:
   → Static images or GIFs (just serve the file)
   → Audio-only podcasts < 5 minutes (progressive download fine)
   → Videos < 15 seconds (progressive download is simpler)
   → Internal/private videos with < 10 viewers (over-engineering)

2. DO NOT USE HLS WHEN:
   → You need < 1 second latency (use WebRTC — Project 47)
   → Target is only Android (DASH might be more native)
   → HLS has inherent 3-segment delay for live (~6-30 seconds)

3. DO NOT BUILD YOUR OWN TRANSCODING ENGINE WHEN:
   → FFmpeg exists and handles 99% of use cases
   → Cloud services (AWS MediaConvert, GCP Transcoder) exist
   → Build the orchestration, not the codec

4. DO NOT USE CDN PUSH FOR ALL CONTENT:
   → Only 10% of videos get 90% of views
   → Pushing ALL content to ALL edges = enormous waste
   → Use origin pull: CDN fetches on first request
   → Pre-push only for known viral content

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: QUALITY OSCILLATION
  → Bandwidth fluctuates rapidly
  → Client keeps switching: 1080p → 480p → 1080p → 480p
  → Visually jarring, annoying
  → FIX: Hysteresis — require bandwidth to be CONSISTENTLY
         above threshold for N chunks before upgrading
         Only downgrade immediately, upgrade cautiously
         EWMA smoothing reduces oscillation

PITFALL 2: CHUNK BOUNDARY ALIGNMENT
  → Video chunks must start on keyframes (IDR frames)
  → If chunk starts on a non-keyframe → decoder can't start
  → Black screen or corruption for first few frames
  → FIX: Force keyframes at exact chunk boundaries during transcoding
         FFmpeg: -force_key_frames "expr:gte(t,n_forced*10)"

PITFALL 3: CDN THUNDERING HERD ON NEW VIRAL VIDEO
  → Video goes viral → millions of requests simultaneously
  → CDN edge doesn't have chunks yet → ALL miss → ALL hit origin
  → Origin overwhelmed
  → FIX: Request coalescing at CDN edge (one origin fetch per chunk)
         Pre-warm CDN when video trending detected
         Shield layer between CDN and origin (CDN shield / mid-tier cache)

PITFALL 4: TRANSCODING QUEUE BACKLOG
  → 500 hours uploaded per minute
  → If transcoding workers can't keep up → growing delay
  → Creator waits hours for video to be available
  → FIX: Auto-scale transcoding workers based on queue depth
         Priority queue: popular creators get faster transcoding
         Progressive availability: 480p ready first, others follow

PITFALL 5: STORAGE COST EXPLOSION
  → Each video × 6 qualities × thousands of chunks
  → Plus raw original kept for potential re-transcoding
  → Storage grows linearly forever
  → FIX: Delete raw after transcoding (keep chunks only)
         Lifecycle policies: delete 240p chunks after 1 year if never accessed
         For very old, unwatched videos: keep only 480p
         Compress/archive cold storage tier

PITFALL 6: MANIFEST STALENESS IN LIVE STREAMING
  → Live stream manifest updates every 2-4 seconds
  → CDN caches manifest with TTL → viewer gets stale manifest
  → Viewer stuck 30 seconds behind live
  → FIX: Very short TTL for live manifests (1-2 seconds)
         Or: Cache-Control: no-cache for live manifests
         LL-HLS uses partial segments for lower latency

PITFALL 7: AUDIO/VIDEO SYNC DRIFT
  → Audio and video processed independently
  → Slight timing differences accumulate
  → Lips don't match speech
  → FIX: Precise PTS (Presentation Time Stamps) in chunks
         Mux audio+video with correct timestamps
         Client synchronizes based on PTS

PITFALL 8: SEEK INTO UNCACHED REGION
  → User seeks to random point in 3-hour movie
  → That chunk not cached anywhere nearby
  → Takes 500ms+ to fetch from origin
  → User sees spinner
  → FIX: Pre-cache first chunk of every 5-minute segment

class ObjectStorage:
    """Simulates S3/GCS blob storage for video chunks and manifests"""

    def __init__(self):
        self.objects: Dict[str, bytes] = {}
        self.metadata: Dict[str, dict] = {}
        self.lock = threading.Lock()
        self.total_bytes = 0
        self.reads = 0
        self.writes = 0

    def put(self, key: str, data: bytes, metadata: dict = None):
        with self.lock:
            self.objects[key] = data
            self.metadata[key] = metadata or {}
            self.total_bytes += len(data)
            self.writes += 1

    def get(self, key: str) -> Optional[bytes]:
        with self.lock:
            self.reads += 1
            return self.objects.get(key)

    def exists(self, key: str) -> bool:
        return key in self.objects

    def list_keys(self, prefix: str) -> List[str]:
        return [k for k in self.objects.keys() if k.startswith(prefix)]

    def get_stats(self) -> dict:
        return {
            "total_objects": len(self.objects),
            "total_bytes": self.total_bytes,
            "total_mb": round(self.total_bytes / 1024 / 1024, 2),
            "reads": self.reads,
            "writes": self.writes
        }

class CDNEdgeCache:
    """Simulates a CDN edge node with LRU caching"""

    def __init__(self, edge_id: str, max_size_bytes: int = 100 * 1024 * 1024,
                 latency_ms: float = 5.0):
        self.edge_id = edge_id
        self.max_size = max_size_bytes
        self.current_size = 0
        self.latency_ms = latency_ms
        self.cache: OrderedDict[str, bytes] = OrderedDict()
        self.hits = 0
        self.misses = 0
        self.lock = threading.Lock()

    def get(self, key: str) -> Optional[bytes]:
        with self.lock:
            if key in self.cache:
                self.cache.move_to_end(key)
                self.hits += 1
                return self.cache[key]
            self.misses += 1
            return None

    def put(self, key: str, data: bytes):
        with self.lock:
            size = len(data)
            # Evict if necessary
            while self.current_size + size > self.max_size and self.cache:
                evicted_key, evicted_data = self.cache.popitem(last=False)
                self.current_size -= len(evicted_data)
            self.cache[key] = data
            self.current_size += size

    @property
    def hit_rate(self) -> float:
        total = self.hits + self.misses
        return (self.hits / total * 100) if total > 0 else 0

    def get_stats(self) -> dict:
        return {
            "edge_id": self.edge_id,
            "cached_objects": len(self.cache),
            "size_mb": round(self.current_size / 1024 / 1024, 2),
            "max_mb": round(self.max_size / 1024 / 1024, 2),
            "hits": self.hits,
            "misses": self.misses,
            "hit_rate_pct": round(self.hit_rate, 1)
        }


class CDNNetwork:
    """Network of CDN edge caches"""

    def __init__(self, origin: ObjectStorage):
        self.origin = origin
        self.edges: Dict[str, CDNEdgeCache] = {}
        self.origin_pulls = 0

    def add_edge(self, edge_id: str, latency_ms: float = 5.0,
                 max_size_bytes: int = 100 * 1024 * 1024):
        self.edges[edge_id] = CDNEdgeCache(
            edge_id, max_size_bytes, latency_ms
        )

    def fetch(self, key: str, edge_id: str) -> Tuple[Optional[bytes], dict]:
        """Fetch content — try edge first, then origin"""
        edge = self.edges.get(edge_id)
        if not edge:
            return None, {"source": "error", "latency_ms": 0}

        # Try edge cache
        data = edge.get(key)
        if data:
            return data, {
                "source": "cdn_edge",
                "edge_id": edge_id,
                "latency_ms": edge.latency_ms
            }

        # Cache miss — pull from origin
        data = self.origin.get(key)
        if data:
            self.origin_pulls += 1
            edge.put(key, data)  # Cache for future
            return data, {
                "source": "origin",
                "edge_id": edge_id,
                "latency_ms": 100.0  # Origin latency
            }

        return None, {"source": "not_found", "latency_ms": 0}

    def get_stats(self) -> dict:
        return {
            "total_edges": len(self.edges),
            "origin_pulls": self.origin_pulls,
            "edges": {
                eid: e.get_stats() for eid, e in self.edges.items()
            }
        }

class TranscodeQueue:
    """Priority queue for transcoding tasks (from Project 11)"""

    def __init__(self):
        self.queue: List[TranscodeTask] = []
        self.lock = threading.Lock()

    def push(self, task: TranscodeTask):
        with self.lock:
            self.queue.append(task)
            self.queue.sort(key=lambda t: t.priority)

    def pop(self) -> Optional[TranscodeTask]:
        with self.lock:
            if self.queue:
                return self.queue.pop(0)
            return None

    def size(self) -> int:
        return len(self.queue)


class TranscodingPipeline:
    """
    Distributed transcoding service
    Splits video into chunks, transcodes each at multiple qualities
    """

    def __init__(self, object_storage: ObjectStorage,
                 num_workers: int = 4):
        self.storage = object_storage
        self.num_workers = num_workers
        self.task_queue = TranscodeQueue()
        self.completed_tasks: Dict[str, List[TranscodeTask]] = defaultdict(list)
        self.lock = threading.Lock()
        # Metrics
        self.total_tasks_created = 0
        self.total_tasks_completed = 0
        self.total_transcode_time_ms = 0

    def create_transcode_job(self, video: VideoMetadata) -> int:
        """Create all transcoding tasks for a video"""
        chunk_count = video.total_chunks

        # Determine which qualities to generate
        source_height = int(video.original_resolution.split("x")[1])
        qualities_to_generate = []
        for label in TRANSCODE_PRIORITY:
            profile = QUALITY_PROFILES[label]
            if profile.height <= source_height or label == "audio":
                qualities_to_generate.append(label)

        total_tasks = 0
        for quality in qualities_to_generate:
            priority = TRANSCODE_PRIORITY.index(quality)
            for chunk_idx in range(chunk_count):
                task = TranscodeTask(
                    task_id=f"task_{uuid.uuid4().hex[:8]}",
                    video_id=video.video_id,
                    chunk_index=chunk_idx,
                    quality=quality,
                    time_start_sec=chunk_idx * video.chunk_duration_sec,
                    time_end_sec=min(
                        (chunk_idx + 1) * video.chunk_duration_sec,
                        video.duration_sec
                    ),
                    raw_storage_key=(
                        f"raw/{video.video_id}/original"
                    ),
                    output_storage_key=(
                        f"videos/{video.video_id}/chunks/"
                        f"{quality}/chunk_{chunk_idx:04d}.ts"
                    ),
                    priority=priority
                )
                self.task_queue.push(task)
                total_tasks += 1

        self.total_tasks_created += total_tasks
        return total_tasks

    def process_tasks(self, max_tasks: int = None) -> dict:
        """Process transcoding tasks (simulated)"""
        processed = 0
        limit = max_tasks or self.task_queue.size()

        while processed < limit:
            task = self.task_queue.pop()
            if task is None:
                break

            # Simulate transcoding work
            start = time.time()
            chunk_data = self._simulate_transcode(task)
            elapsed_ms = (time.time() - start) * 1000

            # Store the transcoded chunk
            self.storage.put(
                task.output_storage_key,
                chunk_data,
                {
                    "video_id": task.video_id,
                    "quality": task.quality,
                    "chunk": task.chunk_index,
                    "duration": task.time_end_sec - task.time_start_sec
                }
            )

            task.status = TranscodeTaskStatus.COMPLETED
            with self.lock:
                self.completed_tasks[task.video_id].append(task)
            self.total_tasks_completed += 1
            self.total_transcode_time_ms += elapsed_ms
            processed += 1

        return {
            "processed": processed,
            "remaining": self.task_queue.size()
        }

    def _simulate_transcode(self, task: TranscodeTask) -> bytes:
        """Simulate transcoding — generate fake chunk data"""
        profile = QUALITY_PROFILES[task.quality]
        duration = task.time_end_sec - task.time_start_sec
        # Calculate simulated chunk size
        size_bytes = int(profile.bitrate_kbps * 1000 / 8 * duration)
        # Generate deterministic "data" based on task params
        seed = f"{task.video_id}:{task.quality}:{task.chunk_index}"
        chunk_hash = hashlib.md5(seed.encode()).hexdigest()
        # Create simulated binary data (small for demo)
        simulated_size = min(size_bytes, 1024)  # Cap for memory
        return (chunk_hash.encode() * (simulated_size // 32 + 1))[:simulated_size]

    def get_video_progress(self, video_id: str,
                           total_tasks: int) -> Dict[str, float]:
        """Get per-quality completion percentage"""
        completed = self.completed_tasks.get(video_id, [])
        quality_counts = defaultdict(int)
        quality_totals = defaultdict(int)

        for task in completed:
            quality_counts[task.quality] += 1

        # Estimate totals from what we've seen
        progress = {}
        for quality in QUALITY_PROFILES:
            done = quality_counts.get(quality, 0)
            if done > 0 or quality in [t.quality for t in completed]:
                progress[quality] = min(done * 100.0 / max(total_tasks // len(QUALITY_PROFILES), 1), 100.0)

        return progress

    def get_stats(self) -> dict:
        avg_time = (
            self.total_transcode_time_ms / self.total_tasks_completed
            if self.total_tasks_completed > 0 else 0
        )
        return {
            "total_tasks_created": self.total_tasks_created,
            "total_tasks_completed": self.total_tasks_completed,
            "queue_depth": self.task_queue.size(),
            "avg_transcode_time_ms": round(avg_time, 2),
            "workers": self.num_workers
        }

class HLSManifestGenerator:
    """Generates HLS .m3u8 manifest files"""

    @staticmethod
    def generate_master_manifest(
        video_id: str,
        available_qualities: List[str]
    ) -> str:
        """Generate master playlist listing all quality levels"""
        lines = ["#EXTM3U", "#EXT-X-VERSION:3", ""]

        for quality in available_qualities:
            if quality == "audio":
                continue
            profile = QUALITY_PROFILES[quality]
            lines.append(
                f"#EXT-X-STREAM-INF:BANDWIDTH={profile.bitrate_bps},"
                f"RESOLUTION={profile.resolution_str},"
                f"NAME=\"{profile.label}\""
            )
            lines.append(
                f"/videos/{video_id}/manifests/{quality}.m3u8"
            )
            lines.append("")

        return "\n".join(lines)

    @staticmethod
    def generate_quality_manifest(
        video_id: str,
        quality: str,
        total_chunks: int,
        chunk_duration_sec: float,
        total_duration_sec: float
    ) -> str:
        """Generate quality-level playlist listing all chunks"""
        lines = [
            "#EXTM3U",
            "#EXT-X-VERSION:3",
            f"#EXT-X-TARGETDURATION:{int(chunk_duration_sec)}",
            "#EXT-X-MEDIA-SEQUENCE:0",
            ""
        ]

        for i in range(total_chunks):
            if i == total_chunks - 1:
                remaining = total_duration_sec - (i * chunk_duration_sec)
                duration = min(chunk_duration_sec, remaining)
            else:
                duration = chunk_duration_sec

            lines.append(f"#EXTINF:{duration:.1f},")
            lines.append(
                f"/videos/{video_id}/chunks/{quality}/chunk_{i:04d}.ts"
            )

        lines.append("")
        lines.append("#EXT-X-ENDLIST")
        return "\n".join(lines)

# SECTION 6: BANDWIDTH ESTIMATOR (Client-Side)
# ============================================================

class BandwidthEstimator:
    """
    EWMA-based bandwidth estimation
    Smooths out fluctuations to prevent quality oscillation
    """

    def __init__(self, initial_bandwidth_kbps: float = 1000.0,
                 alpha: float = 0.3):
        self.alpha = alpha  # Smoothing factor (0-1)
        self.estimated_bw_kbps = initial_bandwidth_kbps
        self.measurements: List[float] = []
        self.raw_history: List[float] = []

    def record_download(self, size_bytes: int, duration_sec: float):
        """Record a chunk download and update bandwidth estimate"""
        if duration_sec <= 0:
            return

        measured_kbps = (size_bytes * 8) / (duration_sec * 1000)
        self.raw_history.append(measured_kbps)

        # EWMA smoothing
        self.estimated_bw_kbps = (
            self.alpha * measured_kbps +
            (1 - self.alpha) * self.estimated_bw_kbps
        )
        self.measurements.append(self.estimated_bw_kbps)

    def get_estimate_kbps(self) -> float:
        return self.estimated_bw_kbps

    def get_safe_estimate_kbps(self, safety_factor: float = 0.8) -> float:
        """Conservative estimate — don't use full bandwidth"""
        return self.estimated_bw_kbps * safety_factor

# SECTION 7: ADAPTIVE BITRATE SELECTOR (Client-Side)
# ============================================================

class AdaptiveBitrateSelector:
    """
    Selects optimal quality based on bandwidth and buffer level
    Combines EWMA bandwidth estimation with buffer-based rules
    """

    # Buffer thresholds (seconds)
    BUFFER_EMERGENCY = 5.0    # Below this → lowest quality
    BUFFER_LOW = 15.0         # Below this → conservative
    BUFFER_TARGET = 30.0      # Ideal buffer level
    BUFFER_HIGH = 45.0        # Above this → can try upgrading

    # Upgrade requires sustained bandwidth for N chunks
    UPGRADE_STABILITY_COUNT = 3

    def __init__(self, available_qualities: List[str]):
        # Sort by bitrate ascending (lowest to highest)
        self.qualities = sorted(
            available_qualities,
            key=lambda q: QUALITY_PROFILES[q].bitrate_kbps
        )
        # Filter out audio-only
        self.qualities = [q for q in self.qualities if q != "audio"]
        self.current_quality_idx = 0  # Start with lowest
        self.consecutive_upgrade_eligible = 0

    def select_quality(self, bandwidth_kbps: float,
                       buffer_sec: float) -> str:
        """Select best quality for next chunk"""
        current_idx = self.current_quality_idx

        # EMERGENCY: Buffer critically low
        if buffer_sec < self.BUFFER_EMERGENCY:
            self.current_quality_idx = 0  # Lowest quality
            self.consecutive_upgrade_eligible = 0
            return self.qualities[0]

        # CONSERVATIVE: Buffer low
        if buffer_sec < self.BUFFER_LOW:
            safe_bw = bandwidth_kbps * 0.7
            target_idx = self._find_quality_for_bandwidth(safe_bw)
            # Only downgrade, don't upgrade in this state
            self.current_quality_idx = min(current_idx, target_idx)
            self.consecutive_upgrade_eligible = 0
            return self.qualities[self.current_quality_idx]

        # STEADY: Normal buffer level
        if buffer_sec < self.BUFFER_HIGH:
            safe_bw = bandwidth_kbps * 0.85
            target_idx = self._find_quality_for_bandwidth(safe_bw)

            if target_idx > current_idx:
                # Want to upgrade — check stability
                self.consecutive_upgrade_eligible += 1
                if self.consecutive_upgrade_eligible >= self.UPGRADE_STABILITY_COUNT:
                    self.current_quality_idx = target_idx
                    self.consecutive_upgrade_eligible = 0
                # Don't upgrade yet — stay current
            elif target_idx < current_idx:
                # Downgrade immediately
                self.current_quality_idx = target_idx
                self.consecutive_upgrade_eligible = 0
            else:
                self.consecutive_upgrade_eligible = 0

            return self.qualities[self.current_quality_idx]

        # AGGRESSIVE: Buffer very healthy
        safe_bw = bandwidth_kbps * 0.95
        target_idx = self._find_quality_for_bandwidth(safe_bw)
        if target_idx > current_idx:
            # Upgrade immediately when buffer is very healthy
            self.current_quality_idx = target_idx
        elif target_idx < current_idx:
            self.current_quality_idx = target_idx

        self.consecutive_upgrade_eligible = 0
        return self.qualities[self.current_quality_idx]

    def _find_quality_for_bandwidth(self, bandwidth_kbps: float) -> int:
        """Find highest quality that fits within bandwidth"""
        best_idx = 0
        for i, quality in enumerate(self.qualities):
            profile = QUALITY_PROFILES[quality]
            if profile.bitrate_kbps <= bandwidth_kbps:
                best_idx = i
        return best_idx

    def get_current_quality(self) -> str:
        return self.qualities[self.current_quality_idx]

# SECTION 8: NETWORK SIMULATOR
# ============================================================

class NetworkSimulator:
    """
    Simulates varying network conditions for realistic testing
    Models bandwidth fluctuation, latency, and packet loss
    """

    def __init__(self, base_bandwidth_kbps: float = 3000.0,
                 volatility: float = 0.3):
        self.base_bw = base_bandwidth_kbps
        self.current_bw = base_bandwidth_kbps
        self.volatility = volatility
        self.events: List[Tuple[int, float]] = []  # (chunk_num, new_bw)
        self.history: List[Tuple[int, float]] = []

    def add_event(self, at_chunk: int, new_bandwidth_kbps: float):
        """Schedule a bandwidth change at a specific chunk"""
        self.events.append((at_chunk, new_bandwidth_kbps))
        self.events.sort(key=lambda x: x[0])

    def get_bandwidth_for_chunk(self, chunk_num: int) -> float:
        """Get current bandwidth with natural fluctuation"""
        # Apply scheduled events
        for event_chunk, event_bw in self.events:
            if chunk_num >= event_chunk:
                self.base_bw = event_bw

        # Natural fluctuation (random walk)
        fluctuation = random.gauss(0, self.volatility * self.base_bw)
        self.current_bw = max(100, self.base_bw + fluctuation)

        self.history.append((chunk_num, self.current_bw))
        return self.current_bw

    def simulate_download(self, chunk_size_bytes: int,
                          chunk_num: int) -> float:
        """Simulate downloading a chunk — returns download time in seconds"""
        bw_kbps = self.get_bandwidth_for_chunk(chunk_num)
        bw_bytes_per_sec = bw_kbps * 1000 / 8

        # Base download time
        download_time = chunk_size_bytes / bw_bytes_per_sec

        # Add some jitter (5-15%)
        jitter = random.uniform(0.95, 1.15)
        download_time *= jitter

        return max(download_time, 0.01)

class StreamingPlatform:
    """
    Main platform — ties together upload, transcode, storage, CDN, playback
    """

    CHUNK_DURATION_SEC = 10.0

    def __init__(self):
        # Infrastructure
        self.object_storage = ObjectStorage()
        self.cdn = CDNNetwork(self.object_storage)
        self.transcoder = TranscodingPipeline(
            self.object_storage, num_workers=8
        )
        self.manifest_gen = HLSManifestGenerator()

        # Data
        self.videos: Dict[str, VideoMetadata] = {}
        self.active_sessions: Dict[str, PlaybackSession] = {}

        # Setup CDN edges
        self.cdn.add_edge("us-east", latency_ms=5.0,
                          max_size_bytes=50 * 1024 * 1024)
        self.cdn.add_edge("eu-west", latency_ms=8.0,
                          max_size_bytes=50 * 1024 * 1024)
        self.cdn.add_edge("asia-pac", latency_ms=12.0,
                          max_size_bytes=50 * 1024 * 1024)

        # Metrics
        self.total_uploads = 0
        self.total_playback_sessions = 0

    # ============================================================
    # UPLOAD + TRANSCODING
    # ============================================================

    def upload_video(self, title: str, creator_id: str,
                     duration_sec: float,
                     original_resolution: str = "1920x1080",
                     file_size_mb: float = 500.0) -> VideoMetadata:
        """Upload a video and start transcoding pipeline"""
        self.total_uploads += 1
        video_id = f"vid_{uuid.uuid4().hex[:8]}"

        total_chunks = math.ceil(duration_sec / self.CHUNK_DURATION_SEC)

        video = VideoMetadata(
            video_id=video_id,
            title=title,
            creator_id=creator_id,
            duration_sec=duration_sec,
            original_resolution=original_resolution,
            chunk_duration_sec=self.CHUNK_DURATION_SEC,
            total_chunks=total_chunks,
            status=VideoStatus.UPLOADING,
            created_at=time.time(),
            file_size_bytes=int(file_size_mb * 1024 * 1024)
        )

        # Simulate storing raw upload
        raw_key = f"raw/{video_id}/original"
        raw_data = b"RAW_VIDEO_DATA_PLACEHOLDER"
        self.object_storage.put(raw_key, raw_data, {
            "title": title,
            "duration": duration_sec
        })

        video.status = VideoStatus.PROCESSING
        self.videos[video_id] = video

        # Create transcoding tasks
        video.status = VideoStatus.TRANSCODING
        num_tasks = self.transcoder.create_transcode_job(video)

        return video

    def run_transcoding(self, video_id: str = None) -> dict:
        """Process transcoding tasks (simulates worker pool)"""
        result = self.transcoder.process_tasks()

        # Check for completed videos
        for vid, video in self.videos.items():
            if video.status != VideoStatus.TRANSCODING:
                continue

            if video_id and vid != video_id:
                continue

            completed = self.transcoder.completed_tasks.get(vid, [])
            completed_qualities = set()

            # Check which qualities are fully done
            quality_chunk_counts = defaultdict(int)
            for task in completed:
                quality_chunk_counts[task.quality] += 1

            for quality, count in quality_chunk_counts.items():
                if count >= video.total_chunks:
                    completed_qualities.add(quality)

            video.available_qualities = sorted(
                list(completed_qualities),
                key=lambda q: QUALITY_PROFILES[q].bitrate_kbps,
                reverse=True
            )

            # Generate manifests if at least one quality is done
            if completed_qualities:
                self._generate_manifests(video)

            # Check if all expected qualities are done
            source_height = int(video.original_resolution.split("x")[1])
            expected_qualities = set()
            for label in TRANSCODE_PRIORITY:
                profile = QUALITY_PROFILES[label]
                if profile.height <= source_height or label == "audio":
                    expected_qualities.add(label)

            if completed_qualities >= expected_qualities:
                video.status = VideoStatus.READY
                video.ready_at = time.time()

        return result

    def _generate_manifests(self, video: VideoMetadata):
        """Generate and store HLS manifests"""
        # Master manifest
        master = self.manifest_gen.generate_master_manifest(
            video.video_id, video.available_qualities
        )
        master_key = f"videos/{video.video_id}/manifests/master.m3u8"
        self.object_storage.put(master_key, master.encode('utf-8'))

        # Quality-level manifests
        for quality in video.available_qualities:
            if quality == "audio":
                continue
            manifest = self.manifest_gen.generate_quality_manifest(
                video.video_id,
                quality,
                video.total_chunks,
                video.chunk_duration_sec,
                video.duration_sec
            )
            quality_key = (
                f"videos/{video.video_id}/manifests/{quality}.m3u8"
            )
            self.object_storage.put(quality_key, manifest.encode('utf-8'))

    # ============================================================
    # PLAYBACK
    # ============================================================

    def start_playback(self, video_id: str, viewer_id: str,
                       edge_id: str = "us-east") -> Optional[PlaybackSession]:
        """Initialize a playback session"""
        video = self.videos.get(video_id)
        if not video or video.status != VideoStatus.READY:
            return None

        self.total_playback_sessions += 1
        session_id = f"sess_{uuid.uuid4().hex[:8]}"

        session = PlaybackSession(
            session_id=session_id,
            video_id=video_id,
            viewer_id=viewer_id,
            current_chunk=0,
            current_quality="240p",  # Start low for fast start
            buffer_sec=0.0,
            start_time=time.time()
        )

        self.active_sessions[session_id] = session
        return session

    def fetch_manifest(self, video_id: str,
                       edge_id: str) -> Tuple[Optional[str], dict]:
        """Fetch master manifest via CDN"""
        key = f"videos/{video_id}/manifests/master.m3u8"
        data, meta = self.cdn.fetch(key, edge_id)
        if data:
            return data.decode('utf-8'), meta
        return None, meta

    def fetch_chunk(self, video_id: str, quality: str,
                    chunk_index: int,
                    edge_id: str) -> Tuple[Optional[bytes], dict]:
        """Fetch a video chunk via CDN"""
        key = (
            f"videos/{video_id}/chunks/{quality}/"
            f"chunk_{chunk_index:04d}.ts"
        )
        return self.cdn.fetch(key, edge_id)

    def get_chunk_size(self, quality: str,
                       duration_sec: float) -> int:
        """Calculate expected chunk size in bytes"""
        profile = QUALITY_PROFILES[quality]
        return int(profile.bitrate_kbps * 1000 / 8 * duration_sec)

    # ============================================================
    # SEEK OPERATION
    # ============================================================

    def seek_to_time(self, session: PlaybackSession,
                     target_time_sec: float) -> dict:
        """Handle seek/scrub to a specific time"""
        video = self.videos[session.video_id]

        # Calculate target chunk
        target_chunk = int(target_time_sec / video.chunk_duration_sec)
        target_chunk = max(0, min(target_chunk, video.total_chunks - 1))

        old_chunk = session.current_chunk
        session.current_chunk = target_chunk
        session.buffer_sec = 0.0  # Flush buffer on seek

        return {
            "action": "seek",
            "from_chunk": old_chunk,
            "to_chunk": target_chunk,
            "target_time": target_time_sec,
            "buffer_flushed": True
        }

    # ============================================================
    # METRICS
    # ============================================================

    def get_platform_metrics(self) -> dict:
        return {
            "videos": {
                "total_uploaded": self.total_uploads,
                "ready": sum(
                    1 for v in self.videos.values()
                    if v.status == VideoStatus.READY
                ),
                "processing": sum(
                    1 for v in self.videos.values()
                    if v.status == VideoStatus.TRANSCODING
                )
            },
            "storage": self.object_storage.get_stats(),
            "cdn": self.cdn.get_stats(),
            "transcoding": self.transcoder.get_stats(),
            "playback": {
                "total_sessions": self.total_playback_sessions,
                "active_sessions": len(self.active_sessions)
            }
        }

# SECTION 10: PLAYBACK SESSION SIMULATOR
# ============================================================

class PlaybackSimulator:
    """
    Simulates a complete video watching experience
    Including bandwidth changes, quality adaptation, seeking
    """

    def __init__(self, platform: StreamingPlatform):
        self.platform = platform

    def simulate_full_playback(
        self,
        video_id: str,
        viewer_id: str,
        edge_id: str = "us-east",
        network: NetworkSimulator = None,
        max_chunks: int = None,
        seek_events: List[Tuple[int, float]] = None
    ) -> dict:
        """
        Simulate watching a video from start to end (or max_chunks)
        Returns detailed playback report
        """
        video = self.platform.videos.get(video_id)
        if not video or video.status != VideoStatus.READY:
            return {"error": "Video not ready"}

        # Setup
        if network is None:
            network = NetworkSimulator(base_bandwidth_kbps=3000, volatility=0.2)

        session = self.platform.start_playback(video_id, viewer_id, edge_id)
        bw_estimator = BandwidthEstimator(initial_bandwidth_kbps=1000)
        abr_selector = AdaptiveBitrateSelector(video.available_qualities)

        chunks_to_play = min(
            max_chunks or video.total_chunks,
            video.total_chunks
        )

        seek_map = {}
        if seek_events:
            for at_chunk, target_time in seek_events:
                seek_map[at_chunk] = target_time

        # Playback log
        chunk_log = []
        total_download_time = 0
        total_play_time = 0
        rebuffer_events = 0
        quality_switches = 0
        previous_quality = None
        cdn_hits = 0
        cdn_misses = 0
        time_to_first_frame = None

        # Simulate chunk-by-chunk playback
        chunk_idx = 0
        chunks_played = 0

        while chunks_played < chunks_to_play and chunk_idx < video.total_chunks:

            # Check for seek events
            if chunks_played in seek_map:
                target_time = seek_map[chunks_played]
                seek_result = self.platform.seek_to_time(session, target_time)
                chunk_idx = seek_result["to_chunk"]
                session.buffer_sec = 0.0
                chunk_log.append({
                    "event": "SEEK",
                    "chunk": chunks_played,
                    "target_time": target_time,
                    "target_chunk": chunk_idx
                })

            # Select quality based on bandwidth and buffer
            quality = abr_selector.select_quality(
                bw_estimator.get_safe_estimate_kbps(),
                session.buffer_sec
            )

            if previous_quality and quality != previous_quality:
                quality_switches += 1
                session.quality_switches += 1

            # Calculate chunk size
            chunk_duration = min(
                video.chunk_duration_sec,
                video.duration_sec - chunk_idx * video.chunk_duration_sec
            )
            chunk_size = self.platform.get_chunk_size(quality, chunk_duration)

            # Simulate download through network
            download_time = network.simulate_download(chunk_size, chunk_idx)
            total_download_time += download_time

            # Record bandwidth measurement
            bw_estimator.record_download(chunk_size, download_time)

            # Fetch from CDN (simulated)
            chunk_data, fetch_meta = self.platform.fetch_chunk(
                video_id, quality, chunk_idx, edge_id
            )

            if fetch_meta["source"] == "cdn_edge":
                cdn_hits += 1
            else:
                cdn_misses += 1

            # Time to first frame (for first chunk)
            if time_to_first_frame is None:
                time_to_first_frame = download_time

            # Buffer management
            session.buffer_sec += chunk_duration
            session.buffer_sec -= chunk_duration  # Consumed during playback

            # Check for rebuffering
            if download_time > chunk_duration and session.buffer_sec <= 0:
                rebuffer_events += 1
                session.rebuffer_events += 1
                rebuffer_duration = download_time - chunk_duration
            else:
                rebuffer_duration = 0

            # Prefetch buffer simulation
            if download_time < chunk_duration:
                session.buffer_sec += (chunk_duration - download_time)
                # Cap buffer
                session.buffer_sec = min(session.buffer_sec, 60.0)

            total_play_time += chunk_duration

            # Log this chunk
            actual_bw = network.history[-1][1] if network.history else 0
            chunk_log.append({
                "chunk": chunk_idx,
                "quality": quality,
                "bitrate_kbps": QUALITY_PROFILES[quality].bitrate_kbps,
                "chunk_size_kb": round(chunk_size / 1024, 1),
                "download_time_sec": round(download_time, 3),
                "actual_bw_kbps": round(actual_bw, 0),
                "estimated_bw_kbps": round(
                    bw_estimator.get_estimate_kbps(), 0
                ),
                "buffer_sec": round(session.buffer_sec, 1),
                "cdn_source": fetch_meta["source"],
                "rebuffered": rebuffer_duration > 0
            })

            session.quality_history.append(quality)
            session.bandwidth_history.append(actual_bw)
            previous_quality = quality
            chunk_idx += 1
            chunks_played += 1
            session.current_chunk = chunk_idx
            session.total_watched_sec = total_play_time

        # Calculate final metrics
        quality_distribution = defaultdict(int)
        for entry in chunk_log:
            if isinstance(entry, dict) and "quality" in entry:
                quality_distribution[entry["quality"]] += 1

        avg_bitrate = 0
        bitrate_entries = [
            e["bitrate_kbps"] for e in chunk_log
            if isinstance(e, dict) and "bitrate_kbps" in e
        ]
        if bitrate_entries:
            avg_bitrate = sum(bitrate_entries) / len(bitrate_entries)

        cdn_total = cdn_hits + cdn_misses
        cdn_hit_rate = (cdn_hits / cdn_total * 100) if cdn_total > 0 else 0

        return {
            "session_id": session.session_id,
            "video_id": video_id,
            "viewer_id": viewer_id,
            "edge": edge_id,
            "chunks_played": chunks_played,
            "total_duration_sec": round(total_play_time, 1),
            "time_to_first_frame_sec": round(time_to_first_frame, 3),
            "quality_switches": quality_switches,
            "rebuffer_events": rebuffer_events,
            "rebuffer_rate_pct": round(
                rebuffer_events / max(chunks_played, 1) * 100, 1
            ),
            "avg_bitrate_kbps": round(avg_bitrate, 0),
            "quality_distribution": dict(quality_distribution),
            "cdn_hit_rate_pct": round(cdn_hit_rate, 1),
            "cdn_hits": cdn_hits,
            "cdn_misses": cdn_misses,
            "chunk_log": chunk_log
        }

        """Visual timeline of chunk playback"""
        lines = []
        lines.append("\n  📈 CHUNK PLAYBACK TIMELINE:")
        lines.append(
            f"  {'Chunk':>6} │ {'Quality':<7} │ {'BW(kbps)':>9} │ "
            f"{'Est BW':>9} │ {'Buffer':>7} │ {'CDN':>8} │ Status"
        )
        lines.append(f"  {'─' * 72}")

        display_log = [
            e for e in chunk_log if isinstance(e, dict) and "quality" in e
        ]

        for entry in display_log[:max_display]:
            if "event" in entry and entry.get("event") == "SEEK":
                lines.append(
                    f"  {'>>>':>6} │ {'SEEK':<7} │ "
                    f"{'→ t=' + str(entry['target_time']) + 's':>9} │ "
                    f"{'':>9} │ {'':>7} │ {'':>8} │ 🔀"
                )
                continue

            rebuf = "🔴 REBUF" if entry.get("rebuffered") else "✅"
            cdn_icon = "⚡" if entry["cdn_source"] == "cdn_edge" else "🌐"

            lines.append(
                f"  {entry['chunk']:>6} │ {entry['quality']:<7} │ "
                f"{entry['actual_bw_kbps']:>8.0f}k │ "
                f"{entry['estimated_bw_kbps']:>8.0f}k │ "
                f"{entry['buffer_sec']:>6.1f}s │ "
                f"{cdn_icon:>8} │ {rebuf}"
            )

        if len(display_log) > max_display:
            lines.append(
                f"  {'...':>6} │ ({len(display_log) - max_display} more chunks)"
            )

        return "\n".join(lines)

   → Hot (cache): manifests, metadata — sub-millisecond
   → Warm (CDN): popular chunks — 5-50ms
   → Cold (object storage): everything — 100-200ms

✅ EWMA bandwidth estimation prevents quality oscillation
   → Raw measurements are noisy
   → Smoothing gives stable quality selection
   → Buffer level provides secondary signal

✅ Start low, upgrade fast
   → First chunk at lowest quality → instant start
   → Upgrade on subsequent chunks as bandwidth confirmed
   → Users perceive "instant" playback
