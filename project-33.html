PROJECT 33: CACHEGRID — DESIGN A DISTRIBUTED CACHE CLUSTER

📋 PHASE 4 (Hard) — Advanced Patterns

🎯 HYPOTHETICAL PROBLEM

You are the infrastructure lead at a social media company

Your FeedEngine (Project 32) works beautifully on one machine
→ Redis holds 100M user feeds
→ Each feed = sorted set of 500 post IDs
→ Total data: ~800GB of cache

THEN the CEO says:
  "We just hit 500 million users. Scale it."

Your single Redis instance:
  → Has 64GB RAM → can hold only 8% of the data
  → Handles 100K ops/sec → but you need 2M ops/sec
  → Dies once → ALL feeds gone → 500M users see empty screens

YOU NEED:
  → 20+ cache nodes working together as ONE logical cache
  → Data distributed evenly across all nodes
  → If one node dies → data still available on another node
  → If traffic spikes → add more nodes without downtime
  → Clients shouldn't know or care how many nodes exist
  → The whole cluster looks like ONE giant cache from outside

BUT:
  → How do you decide WHICH node holds WHICH data?
  → What happens when a node crashes mid-write?
  → How do nodes DISCOVER each other?
  → How do you add a node without reshuffling ALL data?
  → How do you keep replicas in sync?
  → What if two clients write the SAME key to different nodes?

This is the Distributed Cache problem
→ The backbone of every large-scale internet system
→ Memcached, Redis Cluster, Hazelcast, Apache Ignite — all solve this

🔍 HOW TO APPROACH IT

DECOMPOSE INTO LAYERS:

Layer 1: PARTITIONING — Which node owns which keys?
  → Consistent hashing (from Project 14)
  → Maps key → node deterministically
  → Adding/removing nodes moves minimal data

Layer 2: REPLICATION — How to survive node failure?
  → Each key stored on N nodes (typically 3)
  → Primary + replicas
  → If primary dies → replica takes over

Layer 3: DISCOVERY — How do nodes find each other?
  → Gossip protocol — nodes share membership info
  → No central coordinator needed
  → Self-healing cluster membership

Layer 4: ROUTING — How does client find the right node?
  → Client-side routing: client knows the hash ring
  → OR proxy-based routing: proxy routes to correct node
  → OR any-node routing: any node forwards to correct one

Layer 5: CONSISTENCY — What guarantees do we provide?
  → Strong consistency: all replicas agree before responding
  → Eventual consistency: write to primary, async replicate
  → Tunable consistency: choose per-operation (quorum reads/writes)

Layer 6: FAILURE DETECTION — How to know a node is dead?
  → Heartbeat (from Project 15)
  → Gossip protocol propagates failure info
  → Timeout-based detection

PART A: PARTITIONING WITH CONSISTENT HASHING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

From Project 14 — but now applied to real cache distribution:

  Hash Ring (0 to 2^32):

  Position:  0 ──── 1B ──── 2B ──── 3B ──── 4.29B
             │      │       │       │        │
             N0     N1      N2      N0'      N1'
             (real) (real)  (real)  (virtual) (virtual)

  Key "user_001:feed" → hash = 1.5B → lands between N1 and N2
                       → Owned by N2 (next node clockwise)

  Virtual nodes (from Project 14):
    → Each physical node gets 100-200 positions on ring
    → Ensures even distribution
    → Prevents hotspots when nodes have different capacities

  Token ranges:
    → Each node "owns" a range of hash values
    → Node knows exactly which keys it's responsible for
    → Client computes hash → knows exactly which node to contact


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
PART B: REPLICATION FOR FAULT TOLERANCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Replication Factor (RF) = 3 (industry standard)

  For key "user_001:feed" → hash lands on Node 2:
    → Primary:  Node 2 (owns the token range)
    → Replica 1: Node 3 (next node clockwise on ring)
    → Replica 2: Node 4 (next-next node clockwise)

  Ring with RF=3:
                    ┌──── Primary: N2
  Key hash ────────→├──── Replica: N3
                    └──── Replica: N4

  Replication modes:
    → Synchronous: Write to ALL 3 before confirming (slow, strong)
    → Asynchronous: Write to primary, async copy to replicas (fast, eventual)
    → Quorum: Write to W nodes, read from R nodes where W+R > N

PART C: GOSSIP PROTOCOL FOR CLUSTER MEMBERSHIP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Every T seconds, each node:
    1. Picks a RANDOM other node
    2. Sends its membership table (who it knows, their status)
    3. Receives the other node's membership table
    4. MERGES both tables (union of knowledge)
    5. Updates local view of cluster

  Epidemic spread:
    → Information reaches ALL nodes in O(log N) rounds
    → 100 nodes → ~7 rounds → everyone knows everything
    → No single point of failure (no coordinator)
    → Tolerates network partitions gracefully

  Membership table entry:
    {
      node_id: "node_005",
      address: "10.0.1.5:7000",
      status: "ALIVE",           // ALIVE, SUSPECT, DEAD
      heartbeat_counter: 14523,  // Monotonically increasing
      last_updated: 1706140800
    }

  Failure detection via gossip:
    → Node A hasn't incremented its heartbeat in 30 seconds
    → Other nodes mark it as SUSPECT
    → After 60 more seconds without heartbeat → mark as DEAD
    → Gossip spreads DEAD status to all nodes
    → Hash ring updated → dead node's keys redistributed

PART D: QUORUM-BASED CONSISTENCY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  N = total replicas (3)
  W = write quorum (how many must confirm a write)
  R = read quorum (how many must respond to a read)

  RULE: W + R > N → guarantees read sees latest write

  Configuration options:
  ┌────────────┬───┬───┬───┬───────────────────────────────┐
  │ PROFILE    │ N │ W │ R │ BEHAVIOR                      │
  ├────────────┼───┼───┼───┼───────────────────────────────┤
  │ Strong     │ 3 │ 3 │ 1 │ All must write, any can read  │
  │ Balanced   │ 3 │ 2 │ 2 │ Majority write + majority read│
  │ Fast Write │ 3 │ 1 │ 3 │ Any can write, all must read  │
  │ Fast Read  │ 3 │ 3 │ 1 │ All must write, any can read  │
  │ Eventual   │ 3 │ 1 │ 1 │ No overlap → may read stale   │
  └────────────┴───┴───┴───┴───────────────────────────────┘

  For FeedEngine cache: "Balanced" (W=2, R=2)
    → Survives 1 node failure for both reads and writes
    → Reasonable latency (wait for 2 of 3)
    → Strong enough for feed data (not financial)

📌 WHAT COULD BE DONE

ROUTING ARCHITECTURE OPTIONS:

Option 1: CLIENT-SIDE ROUTING
  → Client has a copy of the hash ring
  → Client computes hash(key) → finds correct node
  → Client sends request directly to that node
  → PRO: Lowest latency (no proxy hop)
  → CON: Client must track cluster membership changes
  → USED BY: Redis Cluster, Cassandra drivers

Option 2: PROXY-BASED ROUTING
  → All clients talk to a proxy layer
  → Proxy knows the hash ring
  → Proxy routes request to correct node
  → PRO: Clients are simple (no cluster awareness)
  → CON: Extra network hop, proxy can be bottleneck
  → USED BY: Twemproxy, Codis

Option 3: ANY-NODE ROUTING (Coordinator Pattern)
  → Client sends to ANY node
  → That node becomes "coordinator" for this request
  → Coordinator forwards to correct node(s)
  → Coordinator gathers responses and replies to client
  → PRO: Simple client, any node is entry point
  → CON: Extra internal hop if wrong node chosen
  → USED BY: Cassandra, DynamoDB

WE IMPLEMENT: Option 1 (client-side) + Option 3 (any-node fallback)
  → Client tries direct routing first
  → If client's ring view is stale → any node can forward

✅ WHAT SHOULD DO

1. USE VIRTUAL NODES — ALWAYS
   → Without virtual nodes: 5 physical nodes = 5 points on ring
   → Data distribution wildly uneven
   → With 150 virtual nodes per physical: smooth distribution
   → Also enables heterogeneous hardware
     (powerful machine gets 200 vnodes, weak gets 100)

2. REPLICATE TO DISTINCT PHYSICAL NODES
   → RF=3 means 3 DIFFERENT physical machines
   → Not 3 virtual nodes on the SAME machine
   → Walk clockwise on ring, skip vnodes of same physical node
   → Otherwise: one machine failure loses all 3 copies

3. USE VECTOR CLOCKS OR VERSION NUMBERS FOR CONFLICT DETECTION
   → Two clients write same key to different replicas simultaneously
   → Without versioning → last-write-wins → data loss
   → Vector clocks detect conflicts → application resolves
   → For cache data: last-write-wins is usually acceptable

4. IMPLEMENT READ REPAIR
   → During quorum read: if replicas disagree on value
   → Coordinator sends latest value back to stale replicas
   → Self-healing mechanism — consistency improves over time

5. IMPLEMENT HINTED HANDOFF
   → Node A should receive a write but is temporarily down
   → Neighbor Node B stores the write with a "hint"
   → When Node A comes back → Node B sends the hinted data
   → Ensures writes aren't lost during brief outages

6. BOUND MEMORY WITH EVICTION POLICIES
   → From Project 5: LRU, LFU, TTL-based eviction
   → Each node has a max memory budget
   → When full → evict least recently used entries
   → Critical: evict from local node, NOT from replicas
   → Replicas manage their own eviction independently

7. USE PIPELINING FOR BATCH OPERATIONS
   → FeedEngine fan-out writes to 1000 feeds in a burst
   → Pipeline: batch multiple writes into one network round-trip
   → 10x-50x throughput improvement for bulk operations

8. SEPARATE HOT AND COLD DATA
   → Frequently accessed feeds → kept in memory
   → Rarely accessed feeds → can be evicted, rebuilt on demand
   → Track access frequency per key (LFU metadata)

❌ WHAT SHOULD NOT DO

1. NEVER USE MODULO HASHING FOR DISTRIBUTION
   → hash(key) % N → changing N reshuffles EVERYTHING
   → Add 1 node to 10-node cluster → ~90% of keys move
   → Consistent hashing → only ~10% of keys move
   → Already covered in Project 14 — this is WHY it matters

2. NEVER HAVE A SINGLE COORDINATOR NODE
   → Coordinator = single point of failure
   → Coordinator dies → entire cluster unusable
   → Use gossip → every node is equal
   → Any node can coordinate any request

3. NEVER REPLICATE ALL DATA TO ALL NODES
   → That's not partitioning, that's full replication
   → Doesn't scale storage (every node holds everything)
   → Write amplification = N (every write goes everywhere)
   → RF=3 means 3 copies, not N copies

4. NEVER IGNORE NETWORK PARTITIONS
   → Two groups of nodes can't talk to each other
   → Both groups think the other is dead
   → Both accept writes → divergent data (split brain)
   → FIX: Quorum requires majority → minority partition rejects writes
   → From Project 26: leader election prevents split brain

5. NEVER BLOCK READS WAITING FOR FULL REPLICATION
   → Synchronous replication to all replicas → high latency
   → One slow replica blocks everything
   → Use quorum: W=2 of 3 → tolerate 1 slow replica
   → Remaining replica catches up asynchronously

6. NEVER SKIP FAILURE DETECTION TUNING
   → Too aggressive timeout → healthy node marked dead → unnecessary rebalancing
   → Too lenient timeout → dead node not detected → requests fail
   → Typical: suspect after 15s, dead after 60s
   → Must be tuned for your network characteristics

7. NEVER REDISTRIBUTE DATA DURING FAILURE
   → Node dies → don't immediately rebuild all its data on other nodes
   → It might come back in 5 minutes (reboot, network blip)
   → Wait for configurable period before permanent redistribution
   → Hinted handoff covers the interim

🚫 WHEN NOT TO USE

1. DO NOT USE DISTRIBUTED CACHE WHEN:
   → Data fits in one machine's memory
   → Complexity of distributed system outweighs benefit
   → Your database is fast enough without caching
   → You have < 1000 requests/second (DB handles it fine)

2. DO NOT USE GOSSIP PROTOCOL WHEN:
   → Cluster has < 5 nodes (just use a shared config)
   → You need INSTANT failure detection (gossip has delay)
   → Strong consistency required for membership
     (use ZooKeeper/etcd instead — from Project 26)

3. DO NOT USE QUORUM READS/WRITES WHEN:
   → Cache is purely ephemeral (loss is acceptable)
   → Performance matters more than consistency
   → W=1, R=1 with TTL is sufficient for your use case

4. DO NOT USE CLIENT-SIDE ROUTING WHEN:
   → Clients are in untrusted environments (browsers)
   → Cluster topology changes frequently
   → You have hundreds of different client applications
   → Proxy routing is simpler to manage

5. DO NOT BUILD YOUR OWN WHEN:
   → Redis Cluster exists and handles your scale
   → Memcached with consistent hashing solves your problem
   → Build only when you need custom behavior
     (custom eviction, custom replication, custom consistency)

COMPLETE LIFECYCLE: KEY WRITE → REPLICATION → READ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WRITE PATH:

  Client wants to SET key="user_001:feed" value=[post_ids...]

  STEP 1: Client computes hash
    hash("user_001:feed") → 2,147,483,648 (example)

  STEP 2: Client looks up hash ring
    Ring positions: N0@500M, N1@1.5B, N2@2.5B, N3@3.5B
    2.14B falls between N1 and N2
    → Primary owner: N2

  STEP 3: Client identifies replica set
    RF=3 → walk clockwise: N2 (primary), N3 (replica1), N0 (replica2)
    Skip virtual nodes of same physical machine

  STEP 4: Client sends write to coordinator (N2)
    Request: {op: SET, key: "user_001:feed", value: [...], version: 42}

  STEP 5: Coordinator writes locally
    N2 stores key in its local hash table
    N2 assigns a version/timestamp

  STEP 6: Coordinator replicates (quorum W=2)
    N2 sends write to N3 and N0 simultaneously
    Waits for at least ONE to confirm (2 total including self)

    ┌──────┐          ┌──────┐          ┌──────┐
    │  N2  │──WRITE──→│  N3  │          │  N0  │
    │(coord│          │(repl)│          │(repl)│
    │+prim)│──WRITE──→│      │──────────│      │
    └──┬───┘          └──┬───┘          └──┬───┘
       │                 │                 │
       │    ACK ←────────┘                 │
       │    (W=2 achieved: self + N3)      │
       │                                   │
       │    ACK ←──────────────────────────┘
       │    (async — already confirmed to client)
       ▼
    CONFIRM to client: "Write successful"

  STEP 7: If a replica is DOWN (hinted handoff)
    N0 is unreachable
    N2 stores the write locally with a HINT:
      {key: "user_001:feed", value: [...], hint_for: "N0"}
    When N0 comes back → N2 sends hinted data → N0 catches up


READ PATH:

  Client wants to GET key="user_001:feed"

  STEP 1-3: Same hash computation and node lookup

  STEP 4: Client sends read to coordinator (N2)

  STEP 5: Coordinator performs quorum read (R=2)
    Reads from self (N2) and one replica (N3)
    Waits for both responses

    ┌──────┐          ┌──────┐
    │  N2  │──READ───→│  N3  │
    │(coord│          │(repl)│
    └──┬───┘          └──┬───┘
       │                 │
       │  {v:42, data}   │
       │  {v:42, data} ←─┘
       │
       ▼
    Both agree on version 42 → return data to client

  STEP 6: READ REPAIR (if versions disagree)
    N2 has version 42, N3 has version 41

    → Coordinator returns version 42 (latest) to client
    → Coordinator sends version 42 to N3
    → N3 updates to version 42
    → Self-healing ✅

GOSSIP PROTOCOL INTERNALS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Every node maintains a MEMBERSHIP TABLE:

  Node N2's table:
  ┌─────────┬────────────────┬────────┬───────────┬──────────┐
  │ Node ID │ Address        │ Status │ Heartbeat │ Updated  │
  ├─────────┼────────────────┼────────┼───────────┼──────────┤
  │ N0      │ 10.0.1.0:7000  │ ALIVE  │ 14523     │ 12:00:01 │
  │ N1      │ 10.0.1.1:7000  │ ALIVE  │ 14520     │ 12:00:00 │
  │ N2      │ 10.0.1.2:7000  │ ALIVE  │ 14525     │ 12:00:02 │
  │ N3      │ 10.0.1.3:7000  │ SUSPECT│ 14400     │ 11:59:30 │
  │ N4      │ 10.0.1.4:7000  │ ALIVE  │ 14521     │ 12:00:01 │
  └─────────┴────────────────┴────────┴───────────┴──────────┘

  Gossip round (every 1 second):
    1. N2 increments own heartbeat: 14525 → 14526
    2. N2 picks random node → N4
    3. N2 sends its table to N4
    4. N4 sends its table to N2
    5. Both MERGE: for each node, keep HIGHER heartbeat

  Failure detection:
    N3's heartbeat hasn't increased in 30 seconds
    → N2 marks N3 as SUSPECT
    → Gossips SUSPECT status to others
    → If N3 doesn't recover in 60s → mark DEAD
    → All nodes update hash ring → N3's keys redistributed

  Node JOINING:
    1. New node N5 contacts ANY existing node (seed node)
    2. N5 sends JOIN message with its address
    3. Seed node adds N5 to membership table
    4. Gossip spreads N5's existence to all nodes
    5. Hash ring updated → N5 gets assigned token ranges
    6. Data migration: keys in N5's range copied from current owners
    7. Once migration complete → N5 is fully active

DATA MIGRATION ON NODE JOIN/LEAVE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  BEFORE: 4 nodes, ring divided into 4 ranges

  Range:  [0, 25%)    [25%, 50%)   [50%, 75%)   [75%, 100%)
  Owner:    N0           N1           N2           N3

  NEW NODE N4 joins at position 37.5%:

  AFTER:
  Range:  [0, 25%)  [25%, 37.5%)  [37.5%, 50%)  [50%, 75%)  [75%, 100%)
  Owner:    N0         N1             N4            N2           N3

  What moves:
    → Keys in [37.5%, 50%) move from N1 to N4
    → ONLY 12.5% of total data moves
    → All other keys stay put
    → This is the power of consistent hashing

  Migration process:
    1. N4 announces it's taking range [37.5%, 50%)
    2. N1 streams keys in that range to N4
    3. During migration: N1 still serves reads for those keys
    4. Once complete: hash ring updated → N4 serves those keys
    5. N1 deletes the migrated keys (after confirmation)

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: UNEVEN DATA DISTRIBUTION
  → Few virtual nodes → some nodes hold 3x more data
  → FIX: Use 150+ virtual nodes per physical node
  → Monitor: track key count and memory per node
  → Rebalance if skew exceeds 20%

PITFALL 2: CASCADING FAILURE
  → Node dies → its load shifts to neighbors
  → Neighbors now handle 1.5x normal load
  → If they were at 80% → now at 120% → they die too
  → Domino effect takes down entire cluster
  → FIX: Keep nodes at < 60% capacity
         Auto-scale before hitting limits
         Circuit breakers (Project 21) on overloaded nodes

PITFALL 3: SPLIT BRAIN DURING NETWORK PARTITION
  → Nodes in datacenter A can't talk to datacenter B
  → Both sides think the other is dead
  → Both sides accept writes → divergent data
  → FIX: Quorum requires majority (3 of 5 nodes)
         Minority partition rejects writes → availability sacrifice
         Sloppy quorum: allow writes to any available nodes
         (acceptable for cache — data is ephemeral)

PITFALL 4: THUNDERING HERD ON NODE FAILURE
  → Node 3 dies → its keys are now cache misses
  → All clients for those keys hit the database simultaneously
  → Database overwhelmed
  → FIX: Replicas serve reads immediately
         If no replica available → use request coalescing
         (only one request to DB per key, others wait for result)
         From Project 5: cache stampede prevention

PITFALL 5: GOSSIP PROTOCOL CONVERGENCE DELAY
  → New node joins → takes O(log N) gossip rounds to propagate
  → During propagation: some clients don't know about new node
  → Route to old node → redirect → extra latency
  → FIX: Broadcast JOIN to all known nodes immediately
         Use gossip only for ongoing health monitoring
         Client ring refresh on redirect (learning)

PITFALL 6: HOT KEYS
  → One key accessed 100x more than others (e.g., celebrity feed)
  → Single node handling all traffic for that key
  → FIX: Hot key detection → replicate to more nodes
         Client-side caching of hot keys (local LRU)
         Read from any replica (not just primary)

PITFALL 7: VERSION CONFLICT ON CONCURRENT WRITES
  → Client A writes key=X version=5 to Node 1
  → Client B writes key=X version=5 to Node 2 (network partition)
  → Both succeed locally → version 6 on both with different values
  → FIX: Vector clocks detect concurrent writes
         Last-write-wins (for cache: acceptable)
         Or: application-level conflict resolution

PITFALL 8: MEMORY FRAGMENTATION
  → Frequent SET/DELETE of variable-size values
  → Memory becomes fragmented → usable memory < allocated
  → Node appears full but has wasted space
  → FIX: Use fixed-size slabs (memcached approach)
         Periodic compaction
         Monitor fragmentation ratio

# SECTION 2: LRU CACHE STORE (Per-Node Storage Engine)
# Builds on Project 5 (caching) + Project 23 (KV store)

class LRUCacheStore:
    """
    Single-node storage engine with LRU eviction and TTL
    Uses OrderedDict for O(1) LRU operations
    """

    def __init__(self, max_memory_bytes: int = 50 * 1024 * 1024):
        # 50MB default per node (small for demo)
        self.max_memory = max_memory_bytes
        self.current_memory = 0
        self.store: OrderedDict[str, CacheEntry] = OrderedDict()
        self.lock = threading.Lock()
        # Metrics
        self.hits = 0
        self.misses = 0
        self.evictions = 0
        self.expired_cleaned = 0

    def get(self, key: str) -> Optional[CacheEntry]:
        with self.lock:
            if key not in self.store:
                self.misses += 1
                return None

            entry = self.store[key]

            # Check TTL expiry
            if entry.is_expired():
                self._remove_entry(key)
                self.expired_cleaned += 1
                self.misses += 1
                return None

            # Move to end (most recently used)
            self.store.move_to_end(key)
            entry.last_accessed = time.time()
            self.hits += 1
            return entry

    def put(self, key: str, value: Any, version: int,
            ttl: Optional[float] = None) -> bool:
        with self.lock:
            entry_size = self._estimate_size(key, value)

            # If key exists, update it
            if key in self.store:
                old_entry = self.store[key]
                # Only accept if version is newer or equal
                if version < old_entry.version:
                    return False  # Stale write rejected
                self.current_memory -= old_entry.size_bytes
                del self.store[key]

            # Evict if necessary to make room
            while (self.current_memory + entry_size > self.max_memory
                   and self.store):
                self._evict_one()

            # Store the entry
            entry = CacheEntry(
                key=key,
                value=value,
                version=version,
                created_at=time.time(),
                last_accessed=time.time(),
                ttl=ttl,
                size_bytes=entry_size
            )
            self.store[key] = entry
            self.current_memory += entry_size
            return True

    def delete(self, key: str) -> bool:
        with self.lock:
            if key in self.store:
                self._remove_entry(key)
                return True
            return False

    def get_all_keys_in_range(self, range_start: int,
                               range_end: int,
                               hash_func) -> Dict[str, CacheEntry]:
        """Get all keys whose hash falls in given range — used for migration"""
        result = {}
        with self.lock:
            for key, entry in self.store.items():
                key_hash = hash_func(key)
                if range_start <= key_hash < range_end:
                    result[key] = entry
                elif range_start > range_end:
                    # Wrapping range (crosses 0 on the ring)
                    if key_hash >= range_start or key_hash < range_end:
                        result[key] = entry
        return result

    def _evict_one(self):
        """Evict the least recently used entry"""
        if not self.store:
            return
        # First: try to evict expired entries
        for key in list(self.store.keys()):
            if self.store[key].is_expired():
                self._remove_entry(key)
                self.expired_cleaned += 1
                return

        # No expired → evict LRU (first item in OrderedDict)
        oldest_key = next(iter(self.store))
        self._remove_entry(oldest_key)
        self.evictions += 1

    def _remove_entry(self, key: str):
        if key in self.store:
            self.current_memory -= self.store[key].size_bytes
            del self.store[key]

    def _estimate_size(self, key: str, value: Any) -> int:
        """Rough size estimation in bytes"""
        return len(str(key)) + len(str(value)) + 64  # overhead

    def get_stats(self) -> dict:
        total_ops = self.hits + self.misses
        hit_rate = (self.hits / total_ops * 100) if total_ops > 0 else 0
        return {
            "entries": len(self.store),
            "memory_used_bytes": self.current_memory,
            "memory_max_bytes": self.max_memory,
            "memory_pct": round(
                self.current_memory / self.max_memory * 100, 1
            ),
            "hits": self.hits,
            "misses": self.misses,
            "hit_rate_pct": round(hit_rate, 1),
            "evictions": self.evictions,
            "expired_cleaned": self.expired_cleaned
        }

class ConsistentHashRing:

    def __init__(self, virtual_nodes_per_physical: int = 150):
        self.vnodes_per_physical = virtual_nodes_per_physical
        # Sorted list of (hash_position, physical_node_id)
        self.ring_positions: List[int] = []
        self.ring_nodes: Dict[int, str] = {}  # position → node_id
        # Physical node tracking
        self.physical_nodes: Set[str] = set()
        self.lock = threading.Lock()

    def _hash(self, key: str) -> int:
        """Consistent hash function — returns position on ring (0 to 2^32)"""
        h = hashlib.md5(key.encode()).hexdigest()
        return int(h[:8], 16)  # Use first 8 hex chars = 32 bits

    def add_node(self, node_id: str):
        """Add a physical node with its virtual nodes to the ring"""
        with self.lock:
            if node_id in self.physical_nodes:
                return
            self.physical_nodes.add(node_id)
            for i in range(self.vnodes_per_physical):
                vnode_key = f"{node_id}:vnode:{i}"
                position = self._hash(vnode_key)
                self.ring_nodes[position] = node_id
                bisect.insort(self.ring_positions, position)

    def remove_node(self, node_id: str):
        """Remove a physical node and all its virtual nodes"""
        with self.lock:
            if node_id not in self.physical_nodes:
                return
            self.physical_nodes.discard(node_id)
            positions_to_remove = [
                pos for pos, nid in self.ring_nodes.items()
                if nid == node_id
            ]
            for pos in positions_to_remove:
                del self.ring_nodes[pos]
                idx = bisect.bisect_left(self.ring_positions, pos)
                if (idx < len(self.ring_positions)
                        and self.ring_positions[idx] == pos):
                    self.ring_positions.pop(idx)

    def get_node(self, key: str) -> Optional[str]:
        """Find the primary node responsible for a key"""
        if not self.ring_positions:
            return None
        position = self._hash(key)
        idx = bisect.bisect_right(self.ring_positions, position)
        if idx >= len(self.ring_positions):
            idx = 0  # Wrap around
        return self.ring_nodes[self.ring_positions[idx]]

    def get_replica_nodes(self, key: str,
                          replication_factor: int = 3) -> List[str]:
        """
        Get N DISTINCT physical nodes for a key
        Walk clockwise, skip virtual nodes of already-seen physical nodes
        """
        if not self.ring_positions:
            return []

        position = self._hash(key)
        idx = bisect.bisect_right(self.ring_positions, position)
        replicas = []
        seen_physical = set()
        checked = 0

        while (len(replicas) < replication_factor
               and checked < len(self.ring_positions)):
            actual_idx = (idx + checked) % len(self.ring_positions)
            ring_pos = self.ring_positions[actual_idx]
            physical_node = self.ring_nodes[ring_pos]

            if physical_node not in seen_physical:
                replicas.append(physical_node)
                seen_physical.add(physical_node)

            checked += 1

        return replicas

    def get_key_hash(self, key: str) -> int:
        return self._hash(key)

    def get_node_count(self) -> int:
        return len(self.physical_nodes)

    def get_distribution(self) -> Dict[str, int]:
        """Count virtual node positions per physical node"""
        dist = defaultdict(int)
        for pos, node_id in self.ring_nodes.items():
            dist[node_id] += 1
        return dict(dist)

# SECTION 4: GOSSIP PROTOCOL
# Decentralized cluster membership management
# ============================================================

class GossipProtocol:

    SUSPECT_TIMEOUT_SEC = 5    # Low for demo; production = 15-30s
    DEAD_TIMEOUT_SEC = 10      # Low for demo; production = 60-120s
    GOSSIP_INTERVAL_SEC = 1    # How often to gossip

    def __init__(self, local_node_id: str):
        self.local_node_id = local_node_id
        self.membership: Dict[str, MembershipEntry] = {}
        self.lock = threading.Lock()
        self.local_heartbeat = 0
        self.gossip_log: List[str] = []
        # Callbacks
        self.on_node_alive = None   # Called when new node detected
        self.on_node_dead = None    # Called when node declared dead

    def initialize(self, local_address: str):
        """Register self in membership table"""
        with self.lock:
            self.membership[self.local_node_id] = MembershipEntry(
                node_id=self.local_node_id,
                address=local_address,
                status=NodeStatus.ALIVE,
                heartbeat=0,
                last_updated=time.time()
            )

    def heartbeat_tick(self):
        """Increment local heartbeat — call periodically"""
        with self.lock:
            self.local_heartbeat += 1
            if self.local_node_id in self.membership:
                self.membership[self.local_node_id].heartbeat = (
                    self.local_heartbeat
                )
                self.membership[self.local_node_id].last_updated = time.time()

    def get_gossip_payload(self) -> Dict[str, MembershipEntry]:
        """Get membership table to send to another node"""
        with self.lock:
            return dict(self.membership)

    def merge_gossip(self, remote_membership: Dict[str, MembershipEntry]):
        """Merge received membership info with local knowledge"""
        with self.lock:
            for node_id, remote_entry in remote_membership.items():
                if node_id == self.local_node_id:
                    continue  # Don't overwrite our own entry

                local_entry = self.membership.get(node_id)

                if local_entry is None:
                    # New node discovered
                    self.membership[node_id] = MembershipEntry(
                        node_id=remote_entry.node_id,
                        address=remote_entry.address,
                        status=remote_entry.status,
                        heartbeat=remote_entry.heartbeat,
                        last_updated=time.time()
                    )
                    self._log(
                        f"Discovered new node: {node_id} "
                        f"at {remote_entry.address}"
                    )
                    if self.on_node_alive:
                        self.on_node_alive(node_id, remote_entry.address)

                elif remote_entry.heartbeat > local_entry.heartbeat:
                    # Remote has fresher info
                    local_entry.heartbeat = remote_entry.heartbeat
                    local_entry.last_updated = time.time()
                    # Revive if was suspect
                    if (local_entry.status == NodeStatus.SUSPECT
                            and remote_entry.status == NodeStatus.ALIVE):
                        local_entry.status = NodeStatus.ALIVE
                        self._log(f"Node {node_id} revived (was SUSPECT)")

    def detect_failures(self) -> List[str]:
        """Check for nodes that haven't heartbeated — call periodically"""
        now = time.time()
        newly_dead = []

        with self.lock:
            for node_id, entry in self.membership.items():
                if node_id == self.local_node_id:
                    continue

                if entry.status == NodeStatus.DEAD:
                    continue

                if entry.status == NodeStatus.LEAVING:
                    continue

                staleness = now - entry.last_updated

                if (staleness > self.DEAD_TIMEOUT_SEC
                        and entry.status == NodeStatus.SUSPECT):
                    entry.status = NodeStatus.DEAD
                    newly_dead.append(node_id)
                    self._log(
                        f"Node {node_id} declared DEAD "
                        f"(no heartbeat for {staleness:.0f}s)"
                    )
                    if self.on_node_dead:
                        self.on_node_dead(node_id)

                elif (staleness > self.SUSPECT_TIMEOUT_SEC
                      and entry.status == NodeStatus.ALIVE):
                    entry.status = NodeStatus.SUSPECT
                    self._log(
                        f"Node {node_id} marked SUSPECT "
                        f"(no heartbeat for {staleness:.0f}s)"
                    )

        return newly_dead

    def pick_random_peer(self) -> Optional[str]:
        """Pick a random alive node to gossip with"""
        with self.lock:
            alive_peers = [
                nid for nid, entry in self.membership.items()
                if nid != self.local_node_id
                and entry.status in (NodeStatus.ALIVE, NodeStatus.SUSPECT)
            ]
        if alive_peers:
            return random.choice(alive_peers)
        return None

    def get_alive_nodes(self) -> List[str]:
        with self.lock:
            return [
                nid for nid, entry in self.membership.items()
                if entry.status == NodeStatus.ALIVE
            ]

    def mark_leaving(self, node_id: str):
        with self.lock:
            if node_id in self.membership:
                self.membership[node_id].status = NodeStatus.LEAVING

    def _log(self, message: str):
        entry = f"[{self.local_node_id}] {message}"
        self.gossip_log.append(entry)

    def get_membership_table(self) -> List[dict]:
        with self.lock:
            return [
                {
                    "node_id": e.node_id,
                    "address": e.address,
                    "status": e.status.value,
                    "heartbeat": e.heartbeat,
                    "staleness_sec": round(time.time() - e.last_updated, 1)
                }
                for e in self.membership.values()
            ]

class CacheNode:

    def __init__(self, node_id: str, address: str,
                 max_memory_bytes: int = 50 * 1024 * 1024):

        self.address = address
        self.is_running = False

        # Storage engine
        self.store = LRUCacheStore(max_memory_bytes=max_memory_bytes)

        # Gossip protocol
        self.gossip = GossipProtocol(local_node_id=node_id)
        self.gossip.initialize(address)

        # Hinted handoff storage
        self.hinted_store: Dict[str, List[HintedEntry]] = defaultdict(list)
        self.hint_lock = threading.Lock()

        # Metrics
        self.requests_handled = 0
        self.writes_handled = 0
        self.reads_handled = 0
        self.hints_stored = 0
        self.hints_delivered = 0

    def start(self):
        self.is_running = True

    def stop(self):
        self.is_running = False

    # ---- Core Operations ----

    def local_get(self, key: str) -> Optional[CacheEntry]:
        """Read from this node's local store"""
        if not self.is_running:
            return None
        self.requests_handled += 1
        self.reads_handled += 1
        return self.store.get(key)

    def local_put(self, key: str, value: Any, version: int,
                  ttl: Optional[float] = None) -> bool:
        """Write to this node's local store"""
        if not self.is_running:
            return False
        self.requests_handled += 1
        self.writes_handled += 1
        return self.store.put(key, value, version, ttl)

    def local_delete(self, key: str) -> bool:
        if not self.is_running:
            return False
        self.requests_handled += 1
        return self.store.delete(key)

    # ---- Hinted Handoff ----

    def store_hint(self, hint: HintedEntry):
        """Store a write intended for a down node"""
        with self.hint_lock:
            self.hinted_store[hint.target_node_id].append(hint)
            self.hints_stored += 1

    def get_hints_for(self, target_node_id: str) -> List[HintedEntry]:
        """Retrieve and clear hints for a node that came back"""
        with self.hint_lock:
            hints = self.hinted_store.pop(target_node_id, [])
            self.hints_delivered += len(hints)
            return hints

    # ---- Gossip ----

    def gossip_tick(self):
        """Perform one gossip round"""
        if not self.is_running:
            return
        self.gossip.heartbeat_tick()

    def receive_gossip(
        self, remote_membership: Dict[str, MembershipEntry]
    ):
        if not self.is_running:
            return
        self.gossip.merge_gossip(remote_membership)

    # ---- Stats ----

    def get_stats(self) -> dict:
        return {
            "node_id": self.node_id,
            "address": self.address,
            "is_running": self.is_running,
            "requests_handled": self.requests_handled,
            "writes": self.writes_handled,
            "reads": self.reads_handled,
            "hints_stored": self.hints_stored,
            "hints_delivered": self.hints_delivered,
            "store": self.store.get_stats()
        }

# SECTION 6: CACHE CLUSTER — THE DISTRIBUTED COORDINATOR
# Ties nodes, hash ring, gossip, replication together
# ============================================================

class CacheCluster:

    def __init__(
        self,
        replication_factor: int = 3,
        write_quorum: int = 2,
        read_quorum: int = 2,
        virtual_nodes: int = 150,
        node_memory_bytes: int = 50 * 1024 * 1024
    ):
        self.rf = replication_factor
        self.write_quorum = write_quorum
        self.read_quorum = read_quorum
        self.node_memory = node_memory_bytes

        # Hash ring
        self.ring = ConsistentHashRing(
            virtual_nodes_per_physical=virtual_nodes
        )

        # Nodes registry
        self.nodes: Dict[str, CacheNode] = {}
        self.nodes_lock = threading.Lock()

        # Gossip management
        self.gossip_rounds = 0

        # Metrics
        self.total_reads = 0
        self.total_writes = 0
        self.total_deletes = 0
        self.read_repairs = 0
        self.hinted_handoffs_used = 0
        self.quorum_failures = 0

    # ============================================================
    # NODE MANAGEMENT
    # ============================================================

    def add_node(self, node_id: str, address: str = None) -> CacheNode:
        """Add a new node to the cluster"""
        if address is None:
            address = f"10.0.1.{len(self.nodes)}:7000"

        node = CacheNode(
            node_id=node_id,
            address=address,
            max_memory_bytes=self.node_memory
        )
        node.start()

        with self.nodes_lock:
            self.nodes[node_id] = node
            self.ring.add_node(node_id)

        # Announce to all existing nodes via gossip
        for existing_id, existing_node in self.nodes.items():
            if existing_id != node_id:
                existing_node.gossip.merge_gossip(
                    node.gossip.get_gossip_payload()
                )
                node.gossip.merge_gossip(
                    existing_node.gossip.get_gossip_payload()
                )

        return node

    def remove_node(self, node_id: str, graceful: bool = True) -> dict:
        """Remove a node from the cluster"""
        if node_id not in self.nodes:
            return {"error": f"Node {node_id} not found"}

        node = self.nodes[node_id]

        if graceful:
            # Mark as LEAVING so gossip propagates
            node.gossip.mark_leaving(node_id)
            for other_id, other_node in self.nodes.items():
                if other_id != node_id:
                    other_node.gossip.membership[node_id] = MembershipEntry(
                        node_id=node_id,
                        address=node.address,
                        status=NodeStatus.LEAVING,
                        heartbeat=0,
                        last_updated=time.time()
                    )

            # Migrate data to new owners BEFORE removing
            migrated = self._migrate_data_from_node(node_id)
        else:
            migrated = 0

        # Stop and remove
        node.stop()
        with self.nodes_lock:
            self.ring.remove_node(node_id)
            del self.nodes[node_id]

        # Deliver any hinted handoffs other nodes hold for remaining nodes
        self._deliver_all_hints()

        return {
            "status": "removed",
            "node_id": node_id,
            "graceful": graceful,
            "keys_migrated": migrated
        }

    def kill_node(self, node_id: str) -> dict:
        """Simulate sudden node failure — no graceful migration"""
        if node_id not in self.nodes:
            return {"error": f"Node {node_id} not found"}

        node = self.nodes[node_id]
        node.stop()  # Node stops responding but stays in registry

        return {
            "status": "killed",
            "node_id": node_id,
            "message": "Node stopped. Gossip will detect failure."
        }

    def revive_node(self, node_id: str) -> dict:
        """Simulate node coming back after failure"""
        if node_id not in self.nodes:
            return {"error": f"Node {node_id} not found"}

        node = self.nodes[node_id]
        node.start()

        # Update gossip status
        for other_id, other_node in self.nodes.items():
            if other_id != node_id and other_node.is_running:
                if node_id in other_node.gossip.membership:
                    other_node.gossip.membership[node_id].status = (
                        NodeStatus.ALIVE
                    )
                    other_node.gossip.membership[node_id].last_updated = (
                        time.time()
                    )

        # Deliver hinted handoffs
        hints_delivered = self._deliver_hints_for_node(node_id)

        return {
            "status": "revived",
            "node_id": node_id,
            "hints_delivered": hints_delivered
        }

    # ============================================================
    # WRITE PATH — QUORUM WRITES WITH REPLICATION
    # ============================================================

    def put(self, key: str, value: Any,
            ttl: Optional[float] = None) -> WriteResult:
        """
        Distributed write with quorum
        1. Find replica set from hash ring
        2. Write to all available replicas
        3. Confirm when W replicas acknowledge
        4. Use hinted handoff for unavailable replicas
        """
        self.total_writes += 1

        # Find which nodes should hold this key
        replica_nodes = self.ring.get_replica_nodes(key, self.rf)

        if not replica_nodes:
            return WriteResult(
                success=False, version=0,
                nodes_written=[], nodes_failed=["no_nodes"]
            )

        # Generate version (timestamp-based for simplicity)
        # In production: vector clock or hybrid logical clock
        version = int(time.time() * 1000000)

        nodes_written = []
        nodes_failed = []
        used_hinted = False

        for target_node_id in replica_nodes:
            node = self.nodes.get(target_node_id)

            if node and node.is_running:
                # Node is alive — write directly
                success = node.local_put(key, value, version, ttl)
                if success:
                    nodes_written.append(target_node_id)
                else:
                    nodes_failed.append(target_node_id)
            else:
                # Node is down — use hinted handoff
                nodes_failed.append(target_node_id)
                hint = HintedEntry(
                    target_node_id=target_node_id,
                    key=key,
                    value=value,
                    version=version,
                    timestamp=time.time(),
                    ttl=ttl
                )
                # Store hint on a live node (any available neighbor)
                hint_stored = self._store_hint_on_neighbor(
                    hint, exclude=[target_node_id]
                )
                if hint_stored:
                    used_hinted = True
                    self.hinted_handoffs_used += 1

        # Check quorum
        quorum_met = len(nodes_written) >= self.write_quorum

        if not quorum_met:
            self.quorum_failures += 1

        return WriteResult(
            success=quorum_met,
            version=version,
            nodes_written=nodes_written,
            nodes_failed=nodes_failed,
            used_hinted_handoff=used_hinted
        )

    # ============================================================
    # READ PATH — QUORUM READS WITH READ REPAIR
    # ============================================================

    def get(self, key: str) -> ReadResult:
        """
        Distributed read with quorum
        1. Find replica set from hash ring
        2. Read from R replicas
        3. Return latest version
        4. If versions disagree → read repair
        """
        self.total_reads += 1

        # Find which nodes hold this key
        replica_nodes = self.ring.get_replica_nodes(key, self.rf)

        if not replica_nodes:
            return ReadResult(
                success=False, key=key, value=None,
                version=0, source_node="none"
            )

        # Read from all available replicas
        responses = []
        nodes_consulted = []

        for target_node_id in replica_nodes:
            node = self.nodes.get(target_node_id)
            if node and node.is_running:
                entry = node.local_get(key)
                nodes_consulted.append(target_node_id)
                if entry:
                    responses.append((target_node_id, entry))

        # Check if we got enough responses
        if len(nodes_consulted) < self.read_quorum:
            self.quorum_failures += 1
            # Fall back: try ANY available node that might have the data
            for nid, node in self.nodes.items():
                if (nid not in nodes_consulted
                        and node.is_running
                        and len(nodes_consulted) < self.read_quorum):
                    entry = node.local_get(key)
                    nodes_consulted.append(nid)
                    if entry:
                        responses.append((nid, entry))

        if not responses:
            return ReadResult(
                success=False, key=key, value=None,
                version=0, source_node="none",
                nodes_consulted=nodes_consulted
            )

        # Find the latest version among responses
        latest_node_id = None
        latest_entry = None
        for node_id, entry in responses:
            if latest_entry is None or entry.version > latest_entry.version:
                latest_entry = entry
                latest_node_id = node_id

        # READ REPAIR: if any replica has stale data, update it
        was_repaired = False
        for node_id, entry in responses:
            if entry.version < latest_entry.version:
                # This replica is stale — repair it
                stale_node = self.nodes.get(node_id)
                if stale_node and stale_node.is_running:
                    stale_node.local_put(
                        key=key,
                        value=latest_entry.value,
                        version=latest_entry.version,
                        ttl=latest_entry.ttl
                    )
                    was_repaired = True
                    self.read_repairs += 1

        return ReadResult(
            success=True,
            key=key,
            value=latest_entry.value,
            version=latest_entry.version,
            source_node=latest_node_id,
            was_read_repaired=was_repaired,
            nodes_consulted=nodes_consulted
        )

    # ============================================================
    # DELETE PATH
    # ============================================================

    def delete(self, key: str) -> dict:
        """Delete from all replicas"""
        self.total_deletes += 1
        replica_nodes = self.ring.get_replica_nodes(key, self.rf)

        deleted_from = []
        failed_on = []

        for target_node_id in replica_nodes:
            node = self.nodes.get(target_node_id)
            if node and node.is_running:
                success = node.local_delete(key)
                if success:
                    deleted_from.append(target_node_id)
                else:
                    failed_on.append(target_node_id)
            else:
                failed_on.append(target_node_id)

        return {
            "success": len(deleted_from) >= self.write_quorum,
            "deleted_from": deleted_from,
            "failed_on": failed_on
        }

    # ============================================================
    # HINTED HANDOFF MECHANICS
    # ============================================================

    def _store_hint_on_neighbor(self, hint: HintedEntry,
                                 exclude: List[str]) -> bool:
        """Find a live node to store the hint temporarily"""
        for node_id, node in self.nodes.items():
            if node_id not in exclude and node.is_running:
                node.store_hint(hint)
                return True
        return False

    def _deliver_hints_for_node(self, target_node_id: str) -> int:
        """Deliver stored hints when a node comes back online"""
        total_delivered = 0
        target_node = self.nodes.get(target_node_id)

        if not target_node or not target_node.is_running:
            return 0

        # Check all nodes for hints targeted at this node
        for node_id, node in self.nodes.items():
            if node_id == target_node_id:
                continue
            hints = node.get_hints_for(target_node_id)
            for hint in hints:
                target_node.local_put(
                    key=hint.key,
                    value=hint.value,
                    version=hint.version,
                    ttl=hint.ttl
                )
                total_delivered += 1

        return total_delivered

    def _deliver_all_hints(self):
        """Attempt to deliver all pending hints to their targets"""
        for node_id in list(self.nodes.keys()):
            node = self.nodes.get(node_id)
            if node and node.is_running:
                self._deliver_hints_for_node(node_id)

    # ============================================================
    # DATA MIGRATION (Node Join/Leave)
    # ============================================================

    def _migrate_data_from_node(self, departing_node_id: str) -> int:
        """Migrate data from a leaving node to new owners"""
        departing_node = self.nodes.get(departing_node_id)
        if not departing_node or not departing_node.is_running:
            return 0

        migrated = 0

        # Iterate all keys on the departing node
        with departing_node.store.lock:
            keys_to_migrate = list(departing_node.store.store.keys())

        for key in keys_to_migrate:
            entry = departing_node.store.get(key)
            if entry is None:
                continue

            # Find new owners (excluding the departing node)
            new_replicas = []
            all_replicas = self.ring.get_replica_nodes(key, self.rf + 1)
            for nid in all_replicas:
                if nid != departing_node_id and len(new_replicas) < self.rf:
                    new_replicas.append(nid)

            # Copy to new owners
            for target_id in new_replicas:
                target_node = self.nodes.get(target_id)
                if target_node and target_node.is_running:
                    target_node.local_put(
                        key=entry.key,
                        value=entry.value,
                        version=entry.version,
                        ttl=entry.ttl
                    )
                    migrated += 1

        return migrated

    # ============================================================
    # GOSSIP SIMULATION
    # ============================================================

    def run_gossip_round(self) -> dict:
        """
        Simulate one round of gossip across the cluster
        Each node picks a random peer and exchanges membership info
        """
        self.gossip_rounds += 1
        exchanges = 0
        failures_detected = []

        alive_node_ids = [
            nid for nid, node in self.nodes.items()
            if node.is_running
        ]

        for node_id in alive_node_ids:
            node = self.nodes[node_id]
            node.gossip_tick()

            # Pick random peer to gossip with
            if len(alive_node_ids) < 2:
                continue

            peer_candidates = [
                nid for nid in alive_node_ids if nid != node_id
            ]
            if not peer_candidates:
                continue

            peer_id = random.choice(peer_candidates)
            peer_node = self.nodes.get(peer_id)

            if peer_node and peer_node.is_running:
                # Exchange membership tables
                node_payload = node.gossip.get_gossip_payload()
                peer_payload = peer_node.gossip.get_gossip_payload()

                node.gossip.merge_gossip(peer_payload)
                peer_node.gossip.merge_gossip(node_payload)
                exchanges += 1

            # Detect failures
            newly_dead = node.gossip.detect_failures()
            failures_detected.extend(newly_dead)

        return {
            "round": self.gossip_rounds,
            "exchanges": exchanges,
            "failures_detected": list(set(failures_detected))
        }

    # ============================================================
    # KEY LOCATION INSPECTOR
    # ============================================================

    def inspect_key(self, key: str) -> dict:
        """Debug tool: show where a key lives and its status on each replica"""
        replica_nodes = self.ring.get_replica_nodes(key, self.rf)
        key_hash = self.ring.get_key_hash(key)

        replicas_info = []
        for node_id in replica_nodes:
            node = self.nodes.get(node_id)
            if node:
                entry = node.local_get(key) if node.is_running else None
                replicas_info.append({
                    "node_id": node_id,
                    "is_running": node.is_running,
                    "has_key": entry is not None,
                    "version": entry.version if entry else None,
                    "value_preview": (
                        str(entry.value)[:50] if entry else None
                    ),
                    "ttl_remaining": (
                        round(
                            (entry.created_at + entry.ttl) - time.time(), 1
                        )
                        if entry and entry.ttl else None
                    )
                })

        return {
            "key": key,
            "hash": key_hash,
            "expected_replicas": replica_nodes,
            "replica_details": replicas_info
        }

    # ============================================================
    # CLUSTER-WIDE METRICS
    # ============================================================

    def get_cluster_metrics(self) -> dict:
        alive = sum(1 for n in self.nodes.values() if n.is_running)
        dead = sum(1 for n in self.nodes.values() if not n.is_running)

        per_node_stats = {}
        total_entries = 0
        total_memory = 0

        for nid, node in self.nodes.items():
            stats = node.get_stats()
            per_node_stats[nid] = stats
            if node.is_running:
                total_entries += stats["store"]["entries"]
                total_memory += stats["store"]["memory_used_bytes"]

        # Calculate distribution balance
        entry_counts = [
            per_node_stats[nid]["store"]["entries"]
            for nid in self.nodes
            if self.nodes[nid].is_running
        ]
        if entry_counts:
            avg_entries = sum(entry_counts) / len(entry_counts)
            max_entries = max(entry_counts)
            min_entries = min(entry_counts)
            balance_ratio = (
                min_entries / max_entries if max_entries > 0 else 1.0
            )
        else:
            avg_entries = max_entries = min_entries = 0
            balance_ratio = 1.0

        return {
            "cluster": {
                "total_nodes": len(self.nodes),
                "alive_nodes": alive,
                "dead_nodes": dead,
                "replication_factor": self.rf,
                "write_quorum": self.write_quorum,
                "read_quorum": self.read_quorum,
                "gossip_rounds": self.gossip_rounds
            },
            "operations": {
                "total_reads": self.total_reads,
                "total_writes": self.total_writes,
                "total_deletes": self.total_deletes,
                "read_repairs": self.read_repairs,
                "hinted_handoffs": self.hinted_handoffs_used,
                "quorum_failures": self.quorum_failures
            },
            "data": {
                "total_entries": total_entries,
                "total_memory_bytes": total_memory,
                "total_memory_mb": round(total_memory / 1024 / 1024, 2),
                "avg_entries_per_node": round(avg_entries, 1),
                "max_entries_on_node": max_entries,
                "min_entries_on_node": min_entries,
                "balance_ratio": round(balance_ratio, 3)
            },
            "per_node": per_node_stats
        }

# SECTION 7: CACHE CLIENT — CLIENT-SIDE ROUTING
# ============================================================

class CacheClient:
    """
    Smart client that knows the hash ring
    Routes directly to the correct node
    Falls back to cluster coordinator if needed
    """

    def __init__(self, cluster: CacheCluster):
        self.cluster = cluster
        self.request_count = 0
        self.latencies: List[float] = []

    def set(self, key: str, value: Any,
            ttl: Optional[float] = None) -> WriteResult:
        """Write a key-value pair to the cluster"""
        start = time.time()
        self.request_count += 1
        result = self.cluster.put(key, value, ttl)
        self.latencies.append((time.time() - start) * 1000)
        return result

    def get(self, key: str) -> ReadResult:
        """Read a key from the cluster"""
        start = time.time()
        self.request_count += 1
        result = self.cluster.get(key)
        self.latencies.append((time.time() - start) * 1000)
        return result

    def delete(self, key: str) -> dict:
        """Delete a key from the cluster"""
        self.request_count += 1
        return self.cluster.delete(key)

    def multi_set(self, items: Dict[str, Any],
                  ttl: Optional[float] = None) -> dict:
        """Batch write — pipeline multiple sets"""
        results = {"success": 0, "failed": 0}
        for key, value in items.items():
            result = self.set(key, value, ttl)
            if result.success:
                results["success"] += 1
            else:
                results["failed"] += 1
        return results

    def multi_get(self, keys: List[str]) -> Dict[str, Any]:
        """Batch read — pipeline multiple gets"""
        results = {}
        for key in keys:
            result = self.get(key)
            if result.success:
                results[key] = result.value
        return results

    def get_client_stats(self) -> dict:
        if not self.latencies:
            return {"requests": 0}
        sorted_lat = sorted(self.latencies)
        return {
            "requests": self.request_count,
            "avg_latency_ms": round(
                sum(sorted_lat) / len(sorted_lat), 3
            ),
            "p50_latency_ms": round(
                sorted_lat[len(sorted_lat) // 2], 3
            ),
            "p99_latency_ms": round(
                sorted_lat[int(len(sorted_lat) * 0.99)], 3
            ),
            "max_latency_ms": round(sorted_lat[-1], 3)
        }

class ClusterDisplayFormatter:

    @staticmethod
    def format_write_result(key: str, result: WriteResult) -> str:
        status = "✅" if result.success else "❌"
        hint = " (+ hinted handoff)" if result.used_hinted_handoff else ""
        return (
            f"  {status} SET '{key}' → "
            f"v{result.version} | "
            f"Written: {result.nodes_written} | "
            f"Failed: {result.nodes_failed}{hint}"
        )

    @staticmethod
    def format_read_result(result: ReadResult) -> str:
        if not result.success:
            return f"  ❌ GET '{result.key}' → NOT FOUND"
        repair = " (read-repaired)" if result.was_read_repaired else ""
        return (
            f"  ✅ GET '{result.key}' → "
            f"v{result.version} | "
            f"Value: {str(result.value)[:60]} | "
            f"From: {result.source_node} | "
            f"Consulted: {result.nodes_consulted}{repair}"
        )

    @staticmethod
    def format_cluster_metrics(metrics: dict) -> str:
        lines = []
        lines.append("")
        lines.append("=" * 70)
        lines.append("  📊 CLUSTER METRICS")
        lines.append("=" * 70)

        for section_name, section_data in metrics.items():
            if section_name == "per_node":
                lines.append(f"\n  ┌─ PER-NODE SUMMARY")
                for nid, nstats in section_data.items():
                    running = "🟢" if nstats["is_running"] else "🔴"
                    store = nstats["store"]
                    lines.append(
                        f"  │  {running} {nid}: "
                        f"{store['entries']} keys | "

                        f"HR: {store['hit_rate_pct']}% | "

                lines.append(f"  └{'─' * 55}")

        lines.append("")
        return "\n".join(lines)

        for r in info["replica_details"]:
            status = "🟢" if r["is_running"] else "🔴"
            has = "✅" if r["has_key"] else "❌"

                f"{has} {ver} | "
                f"TTL: {r['ttl_remaining']}s"

        return "\n".join(lines)

    @staticmethod
    def format_membership_table(node: CacheNode) -> str:
        lines = []
        lines.append(f"\n  📋 MEMBERSHIP TABLE on {node.node_id}:")
        lines.append(
            f"  {'Node':<12} {'Address':<20} "
            f"{'Status':<10} {'HB':<8} {'Stale(s)':<10}"
        )
        lines.append(f"  {'─' * 60}")

                f"  {entry['node_id']:<12} {entry['address']:<20} "
                f"{entry['status']:<10} {entry['heartbeat']:<8} "
                f"{entry['staleness_sec']:<10}"
            )
        return "\n".join(lines)

   → Neighbor stores the write with a hint
   → Delivers to target when it recovers
   → No data loss for transient outages

✅ Read repair self-heals consistency
   → During quorum read: detect stale replicas
   → Push latest version to stale nodes
   → System gets more consistent over time

✅ LRU eviction + TTL keeps memory bounded
   → Each node independently manages its memory
   → Expired entries cleaned lazily on access
