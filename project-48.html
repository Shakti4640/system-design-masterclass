    → If Broker A's order arrives 1μs before Broker B's

    → Given the same sequence of orders

  Correctness:
    → No phantom trades (matching orders that don't exist)
    → No double-fills (filling same order twice)
    → No price violations (buying above ask, selling below bid)
    → Account balances must ALWAYS be consistent

  Availability:
    → Market hours: 9:30 AM - 4:00 PM (6.5 hours)
    → Downtime during market hours = MILLIONS in losses
    → Must handle hardware failures with instant failover
    → Recovery: replay from log → exact same state

DECOMPOSITION:

  Sub-Problem 1: SEQUENCER
    → Assign monotonically increasing sequence number to every order
    → SINGLE point of ordering (no distributed consensus needed)
    → This is the "Raft leader" but without the election overhead
    → Throughput: millions of sequences per second

  Sub-Problem 2: ORDER BOOK
    → Data structure holding all active buy/sell orders per instrument
    → Buy orders sorted by price DESCENDING (highest bid first)
    → Sell orders sorted by price ASCENDING (lowest ask first)
    → Operations: insert, cancel, match — all O(log N) or O(1)

  Sub-Problem 3: MATCHING ENGINE
    → Takes sequenced orders, applies to order book
    → Determines if incoming order matches existing orders
    → Produces trade executions (fills)
    → MUST be deterministic — same input → same output ALWAYS

  Sub-Problem 4: MARKET DATA DISSEMINATION
    → After each match: broadcast price/volume updates
    → To all connected brokers simultaneously
    → Multicast for fairness (everyone gets data at same time)

  Sub-Problem 5: RISK MANAGEMENT
    → Pre-trade risk checks (does broker have funds?)
    → Position limits (max shares per broker)
    → Circuit breakers (halt trading if price moves too fast)
    → Must be fast enough to not bottleneck matching

  Sub-Problem 6: DISASTER RECOVERY
    → Sequenced order log = source of truth
    → Backup system replays log → builds identical state
    → Failover: backup takes over from last sequence number
    → Recovery time: seconds (replay millions of orders)

📌 WHAT COULD BE DONE

ORDER BOOK DATA STRUCTURE OPTIONS:

  Option A: Sorted Array
    → Bids and asks in sorted arrays
    → Insert: O(N) (shift elements)
    → Match: O(1) (top of book)
    → Cancel: O(N) (find and remove)
    → Simple but too slow for high volume

  Option B: Red-Black Tree / TreeMap
    → Self-balancing BST keyed by price
    → Each price level holds a FIFO queue of orders
    → Insert: O(log P) where P = number of price levels
    → Match: O(1) (access min/max)
    → Cancel: O(1) with order-id lookup hash map
    → Used by: many production exchanges

  Option C: Hash Map of Price Levels
    → Price → linked list of orders at that price
    → Track best bid/ask separately
    → Insert: O(1) amortized
    → Match: O(1)
    → Cancel: O(1) with order-id lookup
    → Need to maintain sorted price levels separately
    → Used by: LMAX, some crypto exchanges

  Option D: Array-based with Price Indexing
    → Prices mapped to array indices (fixed tick size)
    → AAPL: $100.00-$200.00, tick=$0.01 → 10,000 slots
    → Insert: O(1) (direct index)
    → Match: O(1) (scan from best price)
    → Cancel: O(1)
    → Fastest but wastes memory, only for tick-based instruments
    → Used by: CME, high-frequency systems

  FOR THIS PROJECT → Option B (Tree + Queue) as primary
                   → Show Option C for comparison
                   → Both used in production

MATCHING ALGORITHM OPTIONS:

  Option A: Price-Time Priority (FIFO)
    → Most common: best price first, then earliest time
    → Fair, simple, deterministic
    → Used by: NYSE, NASDAQ, most stock exchanges

  Option B: Pro-Rata
    → At same price level, fill proportional to order size
    → Large orders get proportionally more fill
    → Used by: some options/futures exchanges (CME for some)

  Option C: Price-Time with Minimum Quantity
    → Like FIFO but orders can specify minimum fill size
    → Prevents "penny-ing" (placing tiny orders for priority)

  FOR THIS PROJECT → Option A (Price-Time FIFO)
    → Industry standard, simplest to implement correctly
    → Most widely understood

✅ WHAT SHOULD DO

1. SINGLE-THREADED MATCHING ENGINE
   → NO locks, NO synchronization on the hot path
   → One thread processes orders sequentially
   → Sequence number guarantees ordering
   → This ELIMINATES all concurrency bugs
   → LMAX Disruptor pattern: single writer, multiple readers

2. DETERMINISTIC PROCESSING
   → No random numbers, no wall-clock timestamps in matching logic
   → Use sequence numbers for ordering, not timestamps
   → Same sequence of orders → same trades → ALWAYS
   → This enables: replay, audit, backup, testing

3. PRE-ALLOCATE EVERYTHING
   → Allocate all memory at startup
   → Object pools for orders, trades, messages
   → No dynamic allocation during market hours
   → GC pauses = latency spikes = unfair matching

4. SEPARATE HOT PATH FROM COLD PATH
   → Hot path: sequencer → matching → execution (microseconds)
   → Cold path: logging, market data, risk reports (milliseconds)
   → Hot path runs on dedicated CPU core (pinned)
   → Cold path runs on separate cores
   → Never let cold path block hot path

5. WRITE-AHEAD LOG EVERYTHING
   → Every sequenced order → written to persistent log
   → Every trade execution → written to trade log
   → Log is the source of truth
   → If system crashes → replay log → exact same state
   → Backup system continuously replays log

6. IMPLEMENT CIRCUIT BREAKERS
   → If price moves >10% in 5 minutes → halt trading
   → If order rate exceeds limit → throttle broker
   → If system latency exceeds threshold → alert
   → Prevent flash crashes and cascading failures

❌ WHAT SHOULD NOT DO

1. NEVER USE DISTRIBUTED CONSENSUS FOR PER-ORDER DECISIONS
   → Raft/Paxos: 1-5ms per commit (Project 45)
   → Exchange needs: <100μs per order
   → Consensus is 10,000× too slow
   → Instead: single sequencer + deterministic replay

2. NEVER USE FLOATING POINT FOR PRICES
   → 0.1 + 0.2 = 0.30000000000000004 in floating point
   → Price of $150.05 might become $150.04999999999
   → Use integers: price in cents (15005) or ticks
   → Fixed-point arithmetic: exact, no rounding errors

3. NEVER PROCESS ORDERS OUT OF SEQUENCE
   → If Seq#1000 arrives before Seq#999 → WAIT for #999
   → Out-of-order processing = non-deterministic = unfair
   → The sequencer guarantees order; the engine must respect it

4. NEVER ALLOCATE MEMORY ON THE HOT PATH
   → malloc/new = system call = context switch = microseconds
   → Pre-allocate pools at startup
   → Reuse objects (ring buffers)
   → In Java: avoid creating objects in matching loop

5. NEVER LOG SYNCHRONOUSLY ON THE HOT PATH
   → Disk write = milliseconds
   → Write to memory-mapped file or ring buffer
   → Background thread flushes to disk
   → If crash before flush → replay from last checkpoint

6. NEVER ALLOW UNBOUNDED ORDER BOOK GROWTH
   → Rogue algorithm could send millions of orders
   → Each order consumes memory in the book
   → Implement: per-broker order limits
   → Implement: order expiry (good-till-cancel with max duration)
   → Implement: price band limits (reject orders far from market)

      → Find or create price level
      → Append to queue at that level
      → Update best bid/ask if needed
      → O(log P) for tree, O(1) for hash

    match(incoming_order):
      → Compare incoming with opposite side's best
      → If prices cross (bid ≥ ask) → execute trade
      → Remove filled quantity from book
      → Repeat until no more matches or order filled

    cancel(order_id):
      → Look up order by ID (hash map)
      → Remove from its price level queue
      → If level empty → remove level
      → Update best bid/ask if needed

MECHANIC 3: MATCHING ALGORITHM — PRICE-TIME PRIORITY
═══════════════════════════════════════════════════════════

  INCOMING: SELL 350 shares AAPL at $149.90 (Seq#200)

      → Trade: Buyer(Seq#100) ↔ Seller(Seq#200), 200 @ $150.00
      → Seq#100 fully filled → remove from book
      → Remaining sell: 150 shares
    
    Match vs Seq#105: min(150, 100) = 100 shares at $150.00
      → Trade: Buyer(Seq#105) ↔ Seller(Seq#200), 100 @ $150.00
      → Seq#105 fully filled → remove from book
      → Remaining sell: 50 shares
    
    Price level $150.00 empty → remove level

  Step 3: Next best bid: $149.95
    $149.95 ≥ $149.90 → YES, continue matching
    
    Queue at $149.95: [Seq#98: 500]
    
    Match vs Seq#98: min(50, 500) = 50 shares at $149.95
      → Trade: Buyer(Seq#98) ↔ Seller(Seq#200), 50 @ $149.95

      → Remaining sell: 0 shares → FULLY FILLED

    3 trades executed:
      Trade 1: 200 shares @ $150.00
      Trade 2: 100 shares @ $150.00
      Trade 3:  50 shares @ $149.95

class PriceLevel:
    """
    A single price level in the order book.
    Contains a FIFO queue of orders at this price.
    """

    __slots__ = ['price', 'orders', 'total_quantity']

    def __init__(self, price: int):
        self.price = price
        self.orders: deque = deque()
        self.total_quantity = 0

    def add_order(self, order: Order):
        self.orders.append(order)
        self.total_quantity += order.remaining_quantity

    def remove_order(self, order_id: str) -> Optional[Order]:
        for i, order in enumerate(self.orders):
            if order.order_id == order_id:
                removed = order
                self.total_quantity -= removed.remaining_quantity
                del self.orders[i]
                return removed
        return None

    def peek_front(self) -> Optional[Order]:
        if self.orders:
            return self.orders[0]
        return None

    def pop_front(self) -> Optional[Order]:
        if self.orders:
            order = self.orders.popleft()
            self.total_quantity -= order.remaining_quantity
            return order
        return None

    def update_quantity(self, delta: int):
        self.total_quantity += delta

    @property
    def is_empty(self) -> bool:
        return len(self.orders) == 0

    @property
    def order_count(self) -> int:
        return len(self.orders)


class OrderBook:
    """
    Order book for a single instrument.
    
    Maintains bid (buy) and ask (sell) sides.
    Bids sorted by price DESCENDING (highest first).
    Asks sorted by price ASCENDING (lowest first).
    
    Uses SortedDict for O(log N) operations on price levels.
    Plus HashMap for O(1) order lookup by ID.
    """

    def __init__(self, symbol: str):
        self.symbol = symbol

        # Bid side: sorted ascending, we peek at LAST (highest)
        self.bids: SortedDict = SortedDict()
        # Ask side: sorted ascending, we peek at FIRST (lowest)
        self.asks: SortedDict = SortedDict()

        # Order ID → Order (for fast cancel)
        self.order_map: Dict[str, Order] = {}

        # Market data
        self.last_trade_price: int = 0
        self.last_trade_qty: int = 0
        self.total_volume: int = 0
        self.trade_count: int = 0
        self.high_price: int = 0
        self.low_price: int = 999999999
        self.open_price: int = 0

    @property
    def best_bid(self) -> Optional[int]:
        try:
            price, _ = self.bids.peekitem(-1)
            return price
        except (KeyError, IndexError):
            return None

    @property
    def best_ask(self) -> Optional[int]:
        try:
            price, _ = self.asks.peekitem(0)
            return price
        except (KeyError, IndexError):
            return None

    @property
    def best_bid_size(self) -> int:
        bb = self.best_bid
        if bb is not None and bb in self.bids:
            return self.bids[bb].total_quantity
        return 0

    @property
    def best_ask_size(self) -> int:
        ba = self.best_ask
        if ba is not None and ba in self.asks:
            return self.asks[ba].total_quantity
        return 0

    @property
    def spread(self) -> int:
        bb = self.best_bid
        ba = self.best_ask
        if bb is not None and ba is not None:
            return ba - bb
        return 0

    def add_order(self, order: Order):
        """Add an order to the appropriate side of the book"""
        self.order_map[order.order_id] = order

        if order.side == Side.BUY:
            if order.price not in self.bids:
                self.bids[order.price] = PriceLevel(order.price)
            self.bids[order.price].add_order(order)
        else:
            if order.price not in self.asks:
                self.asks[order.price] = PriceLevel(order.price)
            self.asks[order.price].add_order(order)

    def remove_order(self, order_id: str) -> Optional[Order]:
        """Remove an order by ID (for cancellation)"""
        order = self.order_map.get(order_id)
        if not order:
            return None

        if order.side == Side.BUY:
            level = self.bids.get(order.price)
            if level:
                level.remove_order(order_id)
                if level.is_empty:
                    del self.bids[order.price]
        else:
            level = self.asks.get(order.price)
            if level:
                level.remove_order(order_id)
                if level.is_empty:
                    del self.asks[order.price]

        del self.order_map[order_id]
        return order

    def record_trade(self, price: int, quantity: int):
        """Update market data after a trade"""
        self.last_trade_price = price
        self.last_trade_qty = quantity
        self.total_volume += quantity
        self.trade_count += 1

        if price > self.high_price:
            self.high_price = price
        if price < self.low_price:
            self.low_price = price
        if self.open_price == 0:
            self.open_price = price

    def get_market_data(self) -> MarketData:
        return MarketData(
            symbol=self.symbol,
            best_bid=self.best_bid or 0,
            best_ask=self.best_ask or 0,
            bid_size=self.best_bid_size,
            ask_size=self.best_ask_size,
            last_trade_price=self.last_trade_price,
            last_trade_qty=self.last_trade_qty,
            total_volume=self.total_volume,
            trade_count=self.trade_count,
            high=self.high_price,
            low=self.low_price if self.low_price < 999999999 else 0,
            open_price=self.open_price,
            spread=self.spread
        )

    def get_depth(self, levels: int = 5) -> dict:
        """Get order book depth (top N levels each side)"""
        bid_levels = []
        ask_levels = []

        bid_keys = list(reversed(list(self.bids.keys())))[:levels]
        for price in bid_keys:
            level = self.bids[price]
            bid_levels.append({
                "price": price / 100,
                "quantity": level.total_quantity,
                "orders": level.order_count
            })

        ask_keys = list(self.asks.keys())[:levels]
        for price in ask_keys:
            level = self.asks[price]
            ask_levels.append({
                "price": price / 100,
                "quantity": level.total_quantity,
                "orders": level.order_count
            })

        return {"bids": bid_levels, "asks": ask_levels}

    @property
    def total_orders(self) -> int:
        return len(self.order_map)

class Sequencer:
    """
    Assigns monotonically increasing sequence numbers to all incoming orders.
    Single-threaded, no locks needed.
    
    This is the TOTAL ORDERING point — the heart of the exchange.
    """

    def __init__(self):
        self.current_sequence: int = 0
        self.log: List[dict] = []
        self._lock = threading.Lock()

    def sequence(self, event: dict) -> int:
        with self._lock:
            self.current_sequence += 1
            event["sequence_num"] = self.current_sequence
            event["sequenced_at_ns"] = time.time_ns()
            self.log.append(event)
            return self.current_sequence

    def get_log_from(self, start_seq: int) -> List[dict]:
        with self._lock:
            result = []
            for entry in self.log:
                if entry["sequence_num"] >= start_seq:
                    result.append(entry)
            return result

    @property
    def last_sequence(self) -> int:
        return self.current_sequence

    @property
    def log_size(self) -> int:
        return len(self.log)

class RiskEngine:
    """
    Pre-trade risk checks.
    All checks must complete in microseconds.
    All state is in memory — no database lookups.
    """

    def __init__(self):
        self.broker_positions: Dict[str, Dict[str, int]] = defaultdict(lambda: defaultdict(int))
        self.broker_buying_power: Dict[str, int] = defaultdict(lambda: 10_000_000_00)
        self.position_limits: Dict[str, int] = defaultdict(lambda: 1_000_000)
        self.order_rate: Dict[str, deque] = defaultdict(lambda: deque(maxlen=1000))
        self.rate_limit_per_sec: int = 500
        self.price_bands: Dict[str, Tuple[int, int]] = {}
        self.halted_symbols: Set[str] = set()
        self.rejection_count: int = 0
        self.check_count: int = 0

    def set_reference_price(self, symbol: str, price: int, band_pct: float = 10.0):
        """Set price band based on reference price"""
        band = int(price * band_pct / 100)
        self.price_bands[symbol] = (price - band, price + band)

    def check_order(self, order: Order) -> Tuple[bool, str]:
        """
        Run all pre-trade risk checks.
        Returns (passed, reason).
        """
        self.check_count += 1

        # Check 1: Symbol halted?
        if order.symbol in self.halted_symbols:
            self.rejection_count += 1
            return False, f"HALTED: {order.symbol} trading is halted"

        # Check 2: Price band (limit orders only)
        if order.order_type == OrderType.LIMIT and order.symbol in self.price_bands:
            low, high = self.price_bands[order.symbol]
            if order.price < low or order.price > high:
                self.rejection_count += 1
                return False, (f"PRICE_BAND: {order.price/100:.2f} outside "
                               f"[{low/100:.2f}, {high/100:.2f}]")

        # Check 3: Position limit
        current_pos = abs(self.broker_positions[order.broker_id].get(order.symbol, 0))
        new_pos = current_pos + order.quantity
        if new_pos > self.position_limits.get(order.symbol, 1_000_000):
            self.rejection_count += 1
            return False, f"POSITION_LIMIT: {new_pos} exceeds limit"

        # Check 4: Buying power (buy orders)
        if order.side == Side.BUY:
            required = order.price * order.quantity
            available = self.broker_buying_power[order.broker_id]
            if required > available:
                self.rejection_count += 1
                return False, f"BUYING_POWER: need {required/100:.2f}, have {available/100:.2f}"

        # Check 5: Order rate limit
        now = time.time()
        rate_queue = self.order_rate[order.broker_id]
        # Remove old entries
        while rate_queue and rate_queue[0] < now - 1.0:
            rate_queue.popleft()
        if len(rate_queue) >= self.rate_limit_per_sec:
            self.rejection_count += 1
            return False, f"RATE_LIMIT: {len(rate_queue)}/sec exceeds {self.rate_limit_per_sec}"
        rate_queue.append(now)

        return True, "PASSED"

    def update_position(self, broker_id: str, symbol: str, quantity: int, side: Side):
        if side == Side.BUY:
            self.broker_positions[broker_id][symbol] += quantity
        else:
            self.broker_positions[broker_id][symbol] -= quantity

    def reserve_buying_power(self, broker_id: str, amount: int):
        self.broker_buying_power[broker_id] -= amount

    def release_buying_power(self, broker_id: str, amount: int):
        self.broker_buying_power[broker_id] += amount

    def halt_symbol(self, symbol: str):
        self.halted_symbols.add(symbol)

    def resume_symbol(self, symbol: str):
        self.halted_symbols.discard(symbol)

    def get_stats(self) -> dict:
        return {
            "checks_performed": self.check_count,
            "rejections": self.rejection_count,
            "rejection_rate_pct": round(
                self.rejection_count / max(self.check_count, 1) * 100, 2
            ),
            "halted_symbols": list(self.halted_symbols)
        }

# SECTION 5: MATCHING ENGINE
# ============================================================

class MatchingEngine:
    """
    The core matching engine.
    SINGLE-THREADED, DETERMINISTIC.
    """

    def __init__(self):
        self.order_books: Dict[str, OrderBook] = {}
        self.risk_engine = RiskEngine()
        self.trade_counter: int = 0
        self.all_trades: List[Trade] = []
        self.all_orders: Dict[str, Order] = {}
        self.execution_log: List[dict] = []
        self.last_processed_seq: int = 0
        self.total_orders_processed: int = 0
        self.total_cancels_processed: int = 0
        self.processing_times_ns: List[int] = []

    def get_or_create_book(self, symbol: str) -> OrderBook:
        if symbol not in self.order_books:
            self.order_books[symbol] = OrderBook(symbol)
        return self.order_books[symbol]

    def process_event(self, event: dict) -> List[dict]:
        """
        Process a single sequenced event.
        Returns list of execution reports (fills, acks, rejects).
        
        THIS IS THE HOT PATH — must be as fast as possible.
        """
        start_ns = time.time_ns()
        results = []

        event_type = event.get("type", "")
        seq_num = event.get("sequence_num", 0)

        # Ensure sequential processing
        if seq_num != self.last_processed_seq + 1:
            if seq_num <= self.last_processed_seq:
                return [{"type": "DUPLICATE", "seq": seq_num}]
            # Gap detected — in production, would wait/halt
            pass

        self.last_processed_seq = seq_num

        if event_type == "NEW_ORDER":
            results = self._process_new_order(event)
        elif event_type == "CANCEL_ORDER":
            results = self._process_cancel(event)
        elif event_type == "MODIFY_ORDER":
            results = self._process_modify(event)

        elapsed_ns = time.time_ns() - start_ns
        self.processing_times_ns.append(elapsed_ns)

        return results

    def _process_new_order(self, event: dict) -> List[dict]:
        """Process a new order: risk check → match → add to book"""
        results = []

        order = Order(
            order_id=event["order_id"],
            sequence_num=event["sequence_num"],
            symbol=event["symbol"],
            side=Side(event["side"]),
            order_type=OrderType(event["order_type"]),
            price=event["price_cents"],
            quantity=event["quantity"],
            broker_id=event["broker_id"],
            timestamp_ns=event.get("sequenced_at_ns", 0)
        )

        self.total_orders_processed += 1

        # ── RISK CHECK ──
        passed, reason = self.risk_engine.check_order(order)
        if not passed:
            order.status = OrderStatus.REJECTED
            results.append({
                "type": "REJECTED",
                "order_id": order.order_id,
                "reason": reason,
                "seq": order.sequence_num
            })
            self.execution_log.append({
                "type": "REJECT",
                "order": order.to_dict(),
                "reason": reason
            })
            return results

        # Reserve buying power for buy orders
        if order.side == Side.BUY:
            self.risk_engine.reserve_buying_power(
                order.broker_id, order.price * order.quantity
            )

        # ── ACK ──
        results.append({
            "type": "ACKNOWLEDGED",
            "order_id": order.order_id,
            "seq": order.sequence_num
        })

        self.all_orders[order.order_id] = order
        book = self.get_or_create_book(order.symbol)

        # ── MATCHING ──
        trades = self._match_order(order, book)

        for trade in trades:
            results.append({
                "type": "FILL",
                "trade": trade.to_dict()
            })

        # ── ADD REMAINDER TO BOOK ──
        if not order.is_filled:
            if order.order_type == OrderType.IOC:
                # Immediate-or-Cancel: cancel unfilled portion
                order.status = OrderStatus.CANCELLED
                if order.side == Side.BUY:
                    self.risk_engine.release_buying_power(
                        order.broker_id,
                        order.price * order.remaining_quantity
                    )
                results.append({
                    "type": "CANCELLED",
                    "order_id": order.order_id,
                    "reason": "IOC_UNFILLED",
                    "filled": order.filled_quantity,
                    "cancelled": order.remaining_quantity
                })
            elif order.order_type == OrderType.FOK:
                # Fill-or-Kill: should have been checked before matching
                # If we get here with unfilled, cancel everything
                # (FOK is handled specially in _match_order)
                pass
            elif order.order_type in (OrderType.LIMIT, OrderType.MARKET):
                # Add remaining to book
                if order.order_type == OrderType.LIMIT:
                    book.add_order(order)
                elif order.order_type == OrderType.MARKET:
                    # Market orders that aren't fully filled are cancelled
                    order.status = OrderStatus.CANCELLED
                    results.append({
                        "type": "CANCELLED",
                        "order_id": order.order_id,
                        "reason": "MARKET_NO_LIQUIDITY"
                    })

        return results

    def _match_order(self, incoming: Order, book: OrderBook) -> List[Trade]:
        """
        Core matching logic: Price-Time Priority.
        
        For BUY orders: match against ASK side (sellers)
        For SELL orders: match against BID side (buyers)
        
        Match condition:
          BUY:  incoming.price >= ask.price (buyer willing to pay ask)
          SELL: incoming.price <= bid.price (seller willing to accept bid)
        """
        trades = []

        if incoming.side == Side.BUY:
            opposite_side = book.asks
            # Match while: asks exist AND best_ask <= buy_price
            while (not incoming.is_filled and opposite_side):
                try:
                    best_price, best_level = opposite_side.peekitem(0)
                except (KeyError, IndexError):
                    break

                # Price check: buyer must be willing to pay the ask price
                if incoming.order_type == OrderType.MARKET or incoming.price >= best_price:
                    trade = self._execute_against_level(
                        incoming, best_level, best_price, book
                    )
                    if trade:
                        trades.extend(trade)
                    # Remove empty level
                    if best_level.is_empty:
                        del opposite_side[best_price]
                else:
                    break  # No more matchable prices

        elif incoming.side == Side.SELL:
            opposite_side = book.bids
            # Match while: bids exist AND best_bid >= sell_price
            while (not incoming.is_filled and opposite_side):
                try:
                    best_price, best_level = opposite_side.peekitem(-1)
                except (KeyError, IndexError):
                    break

                if incoming.order_type == OrderType.MARKET or incoming.price <= best_price:
                    trade = self._execute_against_level(
                        incoming, best_level, best_price, book
                    )
                    if trade:
                        trades.extend(trade)
                    if best_level.is_empty:
                        del opposite_side[best_price]
                else:
                    break

        return trades

    def _execute_against_level(self, incoming: Order, level: PriceLevel,
                                exec_price: int, book: OrderBook) -> List[Trade]:
        """Execute incoming order against orders at a price level (FIFO)"""
        trades = []

        while not incoming.is_filled and not level.is_empty:
            resting = level.peek_front()
            if not resting:
                break

            # Self-trade prevention
            if incoming.broker_id == resting.broker_id:
                level.pop_front()
                resting.status = OrderStatus.CANCELLED
                book.order_map.pop(resting.order_id, None)
                self.execution_log.append({
                    "type": "STP_CANCEL",
                    "order_id": resting.order_id,
                    "reason": "Self-trade prevention"
                })
                continue

            # Calculate fill quantity
            fill_qty = min(incoming.remaining_quantity, resting.remaining_quantity)

            # Generate trade
            self.trade_counter += 1
            trade_id = f"T{self.trade_counter:08d}"

            # Determine buyer/seller
            if incoming.side == Side.BUY:
                buyer_order = incoming
                seller_order = resting
            else:
                buyer_order = resting
                seller_order = incoming

            trade = Trade(
                trade_id=trade_id,
                symbol=incoming.symbol,
                price=exec_price,
                quantity=fill_qty,
                buyer_order_id=buyer_order.order_id,
                seller_order_id=seller_order.order_id,
                buyer_broker=buyer_order.broker_id,
                seller_broker=seller_order.broker_id,
                sequence_num=incoming.sequence_num,
                timestamp_ns=time.time_ns()
            )

            # Update orders
            incoming.fill(fill_qty, exec_price, resting.order_id, trade_id)
            resting.fill(fill_qty, exec_price, incoming.order_id, trade_id)

            # Update level quantity
            level.update_quantity(-fill_qty)

            # Update risk engine
            self.risk_engine.update_position(
                buyer_order.broker_id, incoming.symbol, fill_qty, Side.BUY
            )
            self.risk_engine.update_position(
                seller_order.broker_id, incoming.symbol, fill_qty, Side.SELL
            )

            # Release excess buying power
            if buyer_order == incoming and incoming.price > exec_price:
                savings = (incoming.price - exec_price) * fill_qty
                self.risk_engine.release_buying_power(incoming.broker_id, savings)

            # Remove fully filled resting order
            if resting.is_filled:
                level.pop_front()
                book.order_map.pop(resting.order_id, None)

            # Update market data
            book.record_trade(exec_price, fill_qty)

            # Update reference price for risk bands
            self.risk_engine.set_reference_price(incoming.symbol, exec_price)

            trades.append(trade)
            self.all_trades.append(trade)

            self.execution_log.append({
                "type": "TRADE",
                "trade": trade.to_dict()
            })

        return trades

    def _process_cancel(self, event: dict) -> List[dict]:
        """Cancel an existing order"""
        self.total_cancels_processed += 1
        order_id = event.get("order_id_to_cancel", "")
        broker_id = event.get("broker_id", "")

        order = self.all_orders.get(order_id)
        if not order:
            return [{"type": "CANCEL_REJECTED", "order_id": order_id,
                      "reason": "ORDER_NOT_FOUND"}]

        if order.broker_id != broker_id:
            return [{"type": "CANCEL_REJECTED", "order_id": order_id,
                      "reason": "NOT_YOUR_ORDER"}]

        if order.status in (OrderStatus.FILLED, OrderStatus.CANCELLED):
            return [{"type": "CANCEL_REJECTED", "order_id": order_id,
                      "reason": f"ORDER_ALREADY_{order.status.value}"}]

        book = self.order_books.get(order.symbol)
        if book:
            book.remove_order(order_id)

        # Release buying power
        if order.side == Side.BUY:
            self.risk_engine.release_buying_power(
                order.broker_id, order.price * order.remaining_quantity
            )

        order.status = OrderStatus.CANCELLED

        self.execution_log.append({
            "type": "CANCEL",
            "order": order.to_dict()
        })

        return [{"type": "CANCELLED", "order_id": order_id,
                  "remaining": order.remaining_quantity}]

    def _process_modify(self, event: dict) -> List[dict]:
        """Modify = Cancel + New (atomic)"""
        cancel_result = self._process_cancel({
            "order_id_to_cancel": event.get("order_id_to_modify", ""),
            "broker_id": event.get("broker_id", "")
        })

        if cancel_result and cancel_result[0].get("type") == "CANCELLED":
            new_event = dict(event)
            new_event["type"] = "NEW_ORDER"
            new_event["order_id"] = event.get("new_order_id", event.get("order_id", ""))
            new_results = self._process_new_order(new_event)
            return cancel_result + new_results

        return cancel_result

    # ────────────────────────────────────
    # STATS & OBSERVABILITY
    # ────────────────────────────────────

    def get_stats(self) -> dict:
        latencies = self.processing_times_ns[-10000:] if self.processing_times_ns else []
        latency_stats = {}
        if latencies:
            sorted_lat = sorted(latencies)
            latency_stats = {
                "avg_ns": int(statistics.mean(sorted_lat)),
                "p50_ns": sorted_lat[len(sorted_lat) // 2],
                "p95_ns": sorted_lat[int(len(sorted_lat) * 0.95)],
                "p99_ns": sorted_lat[int(len(sorted_lat) * 0.99)],
                "max_ns": sorted_lat[-1],
                "avg_us": round(statistics.mean(sorted_lat) / 1000, 2),
                "p99_us": round(sorted_lat[int(len(sorted_lat) * 0.99)] / 1000, 2)
            }

        book_stats = {}
        for symbol, book in self.order_books.items():
            book_stats[symbol] = {
                "total_orders": book.total_orders,
                "best_bid": book.best_bid / 100 if book.best_bid else 0,
                "best_ask": book.best_ask / 100 if book.best_ask else 0,
                "spread": book.spread / 100 if book.spread else 0,
                "volume": book.total_volume,
                "trades": book.trade_count
            }

        return {
            "orders_processed": self.total_orders_processed,
            "cancels_processed": self.total_cancels_processed,
            "total_trades": len(self.all_trades),
            "last_sequence": self.last_processed_seq,
            "latency": latency_stats,
            "risk": self.risk_engine.get_stats(),
            "books": book_stats
        }

class Exchange:
    """
    The complete exchange system.
    Ties together: sequencer, matching engine, risk, market data.
    """

    def __init__(self, name: str = "EXCHANGECORE"):
        self.name = name
        self.sequencer = Sequencer()
        self.matching_engine = MatchingEngine()
        self.market_data = MarketDataPublisher()

        self.is_open = False
        self.order_id_counter = 0
        self._lock = threading.Lock()

    def open_market(self, symbols: List[Tuple[str, int]]):
        """Open market for trading. symbols: list of (symbol, ref_price_cents)"""
        self.is_open = True
        for symbol, ref_price in symbols:
            self.matching_engine.get_or_create_book(symbol)
            self.matching_engine.risk_engine.set_reference_price(symbol, ref_price)

    def close_market(self):
        self.is_open = False

    def _next_order_id(self) -> str:
        with self._lock:
            self.order_id_counter += 1
            return f"ORD{self.order_id_counter:08d}"

    def submit_order(self, symbol: str, side: str, order_type: str,
                     price: float, quantity: int, broker_id: str) -> dict:
        """
        Submit a new order to the exchange.
        
        Flow: validate → sequence → match → respond
        """
        if not self.is_open:
            return {"success": False, "error": "MARKET_CLOSED"}

        order_id = self._next_order_id()
        price_cents = int(round(price * 100))

        event = {
            "type": "NEW_ORDER",
            "order_id": order_id,
            "symbol": symbol,
            "side": side,
            "order_type": order_type,
            "price_cents": price_cents,
            "quantity": quantity,
            "broker_id": broker_id,
            "submitted_at_ns": time.time_ns()
        }

        # Sequence
        seq_num = self.sequencer.sequence(event)

        # Process
        results = self.matching_engine.process_event(event)

        # Publish market data
        book = self.matching_engine.order_books.get(symbol)
        if book:
            self.market_data.publish_book_update(symbol, book)
            for r in results:
                if r.get("type") == "FILL":
                    trade_data = r["trade"]
                    # Find the trade object
                    for t in self.matching_engine.all_trades:
                        if t.trade_id == trade_data["trade_id"]:
                            self.market_data.publish_trade(t, book)
                            break

        return {
            "success": True,
            "order_id": order_id,
            "sequence": seq_num,
            "results": results
        }

    def cancel_order(self, order_id: str, broker_id: str) -> dict:
        event = {
            "type": "CANCEL_ORDER",
            "order_id_to_cancel": order_id,
            "broker_id": broker_id
        }

        seq_num = self.sequencer.sequence(event)
        results = self.matching_engine.process_event(event)

        return {
            "success": True,
            "sequence": seq_num,
            "results": results
        }

    def get_book_depth(self, symbol: str, levels: int = 5) -> dict:
        book = self.matching_engine.order_books.get(symbol)
        if book:
            return book.get_depth(levels)
        return {"bids": [], "asks": []}

    def get_market_data(self, symbol: str) -> dict:
        return self.market_data.get_snapshot(symbol) or {}

    def get_exchange_stats(self) -> dict:
        engine_stats = self.matching_engine.get_stats()
        return {
            "exchange": self.name,
            "is_open": self.is_open,
            "sequencer_last": self.sequencer.last_sequence,
            "sequencer_log_size": self.sequencer.log_size,
            **engine_stats
        }

class ReplayEngine:
    """
    Replays the sequenced log to build identical state.
    Used for: backup, disaster recovery, auditing.
    
    GUARANTEE: Given same log → produces EXACT same state.
    """

    def __init__(self):
        self.matching_engine = MatchingEngine()
        self.events_replayed = 0
        self.replay_start_time = 0
        self.replay_end_time = 0

    def replay(self, log: List[dict]) -> dict:
        """Replay entire log, return summary"""
        self.replay_start_time = time.time_ns()
        self.events_replayed = 0

        for event in log:
            self.matching_engine.process_event(event)
            self.events_replayed += 1

        self.replay_end_time = time.time_ns()
        elapsed_ms = (self.replay_end_time - self.replay_start_time) / 1_000_000

        return {
            "events_replayed": self.events_replayed,
            "elapsed_ms": round(elapsed_ms, 2),
            "rate_per_sec": int(self.events_replayed / max(elapsed_ms / 1000, 0.001)),
            "final_sequence": self.matching_engine.last_processed_seq,
            "total_trades": len(self.matching_engine.all_trades)
        }

    def verify_against(self, primary_engine: MatchingEngine) -> dict:
        """Verify replay state matches primary state"""
        mismatches = []

        # Compare trade counts
        if len(self.matching_engine.all_trades) != len(primary_engine.all_trades):
            mismatches.append(
                f"Trade count: replay={len(self.matching_engine.all_trades)} "
                f"vs primary={len(primary_engine.all_trades)}"
            )

        # Compare each trade
        min_trades = min(len(self.matching_engine.all_trades),
                         len(primary_engine.all_trades))
        for i in range(min_trades):
            rt = self.matching_engine.all_trades[i]
            pt = primary_engine.all_trades[i]
            if (rt.trade_id != pt.trade_id or rt.price != pt.price
                    or rt.quantity != pt.quantity):
                mismatches.append(
                    f"Trade #{i}: replay={rt.to_dict()} vs primary={pt.to_dict()}"
                )

        # Compare order book state
        for symbol in primary_engine.order_books:
            p_book = primary_engine.order_books[symbol]
            r_book = self.matching_engine.order_books.get(symbol)

            if not r_book:
                mismatches.append(f"Missing book for {symbol} in replay")
                continue

            if p_book.best_bid != r_book.best_bid:
                mismatches.append(
                    f"{symbol} best_bid: replay={r_book.best_bid} "
                    f"vs primary={p_book.best_bid}"
                )
            if p_book.best_ask != r_book.best_ask:
                mismatches.append(
                    f"{symbol} best_ask: replay={r_book.best_ask} "
                    f"vs primary={p_book.best_ask}"
                )

        return {
            "identical": len(mismatches) == 0,
            "mismatches": mismatches,
            "trades_compared": min_trades
        }

class OrderGenerator:
    """Generates realistic order flow for simulation"""

    SYMBOLS = [
        ("AAPL", 15000), ("GOOG", 28000), ("MSFT", 42000),
        ("AMZN", 18500), ("META", 51000), ("TSLA", 24500),
        ("NVDA", 88000), ("JPM", 19800), ("V", 27500), ("JNJ", 16000)
    ]

    BROKERS = [f"BROKER_{chr(65+i)}" for i in range(10)]

    @classmethod
    def generate_order(cls, tick: int = 0) -> dict:
        symbol, ref_price = random.choice(cls.SYMBOLS)
        broker = random.choice(cls.BROKERS)
        side = random.choice(["BUY", "SELL"])

        # Price: cluster around reference with some spread
        spread = random.gauss(0, ref_price * 0.002)
        if side == "BUY":
            price_cents = int(ref_price + spread - abs(random.gauss(0, 5)))
        else:
            price_cents = int(ref_price + spread + abs(random.gauss(0, 5)))

        price_cents = max(100, price_cents)
        price = price_cents / 100

        quantity = random.choice([10, 25, 50, 100, 200, 500, 1000])

        # Order type distribution
        order_type_roll = random.random()
        if order_type_roll < 0.70:
            order_type = "LIMIT"
        elif order_type_roll < 0.85:
            order_type = "IOC"
        elif order_type_roll < 0.95:
            order_type = "MARKET"
        else:
            order_type = "FOK"

        return {
            "symbol": symbol,
            "side": side,
            "order_type": order_type,
            "price": price,
            "quantity": quantity,
            "broker_id": broker
        }
