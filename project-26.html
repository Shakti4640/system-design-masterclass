THIS is the Leader Election problem

🔍 HOW TO APPROACH IT

Think of it like electing a PRESIDENT of a country:

  WITHOUT election process:
    → King dies
    → Three princes each declare themselves king
    → Civil war
    → Country collapses

  WITH election process:
    → King dies (leader failure detected)
    → Succession rules kick in (election algorithm)
    → Candidates campaign (nodes propose themselves)
    → Votes are cast (nodes agree on one)
    → ONE winner inaugurated (new leader confirmed)
    → Old king returns → told "you've been replaced" (fencing)
    → Peaceful transition

MAP TO DISTRIBUTED SYSTEMS:

  King                    = Leader node
  Princes                 = Candidate nodes
  Succession rules        = Election algorithm
  Votes                   = Consensus messages
  Inauguration            = Leader announcement
  Crown jewels            = Fencing token (proof of leadership)
  Royal decree validity   = Epoch/term number

KEY COMPONENTS:

  1. FAILURE DETECTOR
     → Heartbeat-based (Project 15)
     → Leader sends heartbeat every N seconds
     → Followers start election after timeout

  2. ELECTION ALGORITHM
     → Determines which node becomes leader
     → Must produce exactly ONE leader
     → Must handle simultaneous elections

  3. FENCING TOKEN
     → Monotonically increasing number
     → New leader gets token = old_token + 1
     → Old leader's token is LOWER → rejected by followers
     → Prevents split-brain from stale leaders

  4. EPOCH / TERM NUMBER
     → Each election increments the epoch
     → All messages carry the epoch
     → Messages from old epochs are IGNORED
     → "I was elected in epoch 5, your epoch 3 is outdated"

OPTION A: Fixed Primary (No Election)

  → Highest node ID always wins

  → PROBLEM: Dependency on external service

✅ WHAT SHOULD DO

1. ALWAYS use fencing tokens
   → New leader gets incrementing token
   → All operations carry the token
   → Stale leader's operations rejected
   → Prevents split-brain damage

2. ALWAYS require majority (quorum) for election
   → 5 nodes: need 3 votes to become leader
   → Prevents two leaders in split network
   → Partition with 2 nodes: cannot elect (no majority)
   → Partition with 3 nodes: CAN elect (has majority)

3. ALWAYS implement leader heartbeat
   → Leader sends periodic "I'm alive" messages
   → Followers reset their election timers
   → If no heartbeat within timeout → start election
   → Timeout must be > 2x heartbeat interval

4. ALWAYS use randomized election timeouts
   → If all followers start election at same time → chaos
   → Each follower waits random time before starting
   → Reduces collision probability dramatically
   → Raft uses 150-300ms random timeout

5. ALWAYS handle the returning old leader
   → Old leader crashes, recovers, thinks it's still leader
   → Must discover it's been replaced
   → Epoch number: "I'm epoch 3, cluster is epoch 5 → I step down"
   → Fencing token: "My token is 3, current is 5 → I'm stale"

6. ALWAYS log all election events
   → "Node 3 started election for epoch 5"
   → "Node 2 voted for Node 3 in epoch 5"
   → "Node 3 became leader with token 5"
   → Critical for debugging election issues

❌ WHAT SHOULD NOT DO

1. NEVER rely on timestamps for ordering
   → Server clocks are NOT synchronized perfectly
   → "I became leader at 14:00:00.001" vs "14:00:00.002"
   → Clock skew can be seconds or more
   → USE: Epoch numbers (logical clocks), not wall clocks

2. NEVER allow leader election without quorum
   → Network splits into [A,B] and [C,D,E]
   → Both partitions elect a leader → SPLIT BRAIN
   → Quorum: only partition with majority can elect
   → [A,B] = 2/5 → cannot elect
   → [C,D,E] = 3/5 → CAN elect

3. NEVER assume leader failure is instant
   → Leader might be slow, not dead
   → Network might be congested, not partitioned
   → Use multiple missed heartbeats before declaring dead
   → "Leader missed 3 consecutive heartbeats → probably dead"

4. NEVER let old leader continue operating after replacement
   → Old leader recovers from GC pause
   → Doesn't know it's been replaced
   → Sends commands with old fencing token
   → Followers MUST reject old-token commands
   → This is WHY fencing tokens exist

5. NEVER use even number of nodes for consensus
   → 4 nodes: quorum = 3 (can only lose 1)
   → 5 nodes: quorum = 3 (can lose 2)
   → 4 nodes gives WORSE fault tolerance than 3!
   → Always use odd numbers: 3, 5, 7

6. NEVER make election timeout too short
   → Short timeout → frequent false elections
   → Leader was just slow, not dead
   → Election disrupts normal operation
   → Minimum: 3x heartbeat interval
   → Typical: heartbeat=1s, timeout=3-5s

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ALGORITHM 1: BULLY ALGORITHM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  RULE: Highest node ID ALWAYS wins

  PROCESS:
    1. Node detects leader is dead (heartbeat timeout)
    2. Node sends ELECTION message to all HIGHER-ID nodes
    3. If any higher node responds → it takes over election
    4. If NO higher node responds → I become leader
    5. New leader sends COORDINATOR message to all nodes

  EXAMPLE (5 nodes, IDs 1-5, Node 5 is leader):

    Node 5 (leader) crashes
    
    Node 2 detects failure (heartbeat timeout)
    Node 2 sends ELECTION to Node 3, 4, 5
    
    Node 3 receives ELECTION from Node 2
      → Node 3 has higher ID → responds "I'll handle this"
      → Node 3 sends ELECTION to Node 4, 5
    
    Node 4 receives ELECTION from Node 3
      → Node 4 has higher ID → responds "I'll handle this"
      → Node 4 sends ELECTION to Node 5
    
    Node 5 is dead → no response
    
    Node 4 waits timeout → no higher node responded
    Node 4 declares: "I AM THE NEW LEADER"
    Node 4 sends COORDINATOR to Nodes 1, 2, 3
    
    All nodes update: leader = Node 4

  TIMELINE:
    T=0:    Node 5 crashes
    T=3s:   Node 2 detects (heartbeat timeout)
    T=3.1s: Node 2 sends ELECTION
    T=3.2s: Node 3 bullies Node 2
    T=3.3s: Node 4 bullies Node 3
    T=6.3s: Node 4 timeout (no response from 5)
    T=6.4s: Node 4 becomes leader
    
    Total election time: ~3.4 seconds

  WEAKNESS:
    → Highest-ID node ALWAYS wins
    → Even if it's the slowest/weakest
    → No consideration of node capability

ALGORITHM 2: RING ALGORITHM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Nodes form a logical ring:
    Node 1 → Node 2 → Node 3 → Node 4 → Node 5 → Node 1

  PROCESS:
    1. Node detects leader failure
    2. Creates ELECTION message containing its own ID
    3. Passes message to next node in ring
    4. Each node adds its own ID to the message
    5. Message circulates back to initiator
    6. Initiator picks highest ID from message
    7. Sends COORDINATOR message around the ring

  EXAMPLE:
    Node 3 detects leader failure
    
    Node 3 creates: ELECTION [3]
    Sends to Node 4
    
    Node 4 adds itself: ELECTION [3, 4]
    Sends to Node 5 (dead, skip to Node 1)
    
    Node 1 adds itself: ELECTION [3, 4, 1]
    Sends to Node 2
    
    Node 2 adds itself: ELECTION [3, 4, 1, 2]
    Sends to Node 3
    
    Node 3 receives its own message back!
    Candidates: [3, 4, 1, 2]
    Winner: max(3, 4, 1, 2) = Node 4
    
    Node 3 sends: COORDINATOR [leader=4]
    Message circulates ring: 3→4→1→2→3
    All nodes know: leader = Node 4

ALGORITHM 3: CONSENSUS-BASED (Simplified Raft-Style)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  THE GOLD STANDARD for production systems

  KEY CONCEPTS:
    → TERM (epoch): Monotonically increasing election number
    → CANDIDATE: Node that wants to be leader
    → VOTE: Each node can vote ONCE per term
    → QUORUM: Majority of nodes must agree (⌊N/2⌋ + 1)

  PROCESS:
    1. Follower's election timeout expires (no heartbeat from leader)
    2. Follower becomes CANDIDATE
    3. Increments current term: term = term + 1
    4. Votes for itself
    5. Sends REQUEST_VOTE to all other nodes
    6. Other nodes vote YES if:
       → Candidate's term > their current term
       → They haven't voted in this term yet
    7. If candidate gets majority votes → becomes LEADER
    8. If another node has higher term → candidate steps down
    9. If timeout without result → increment term, try again

  EXAMPLE (5 nodes, Node 3 was leader, term=4):

    Node 3 crashes
    
    T=0: Nodes 1,2,4,5 have election timeouts running
    
    T=2.1s: Node 4's timeout fires first (randomized!)
      Node 4 → CANDIDATE, term=5
      Node 4 votes for itself (1 vote)
      Node 4 sends REQUEST_VOTE(term=5) to all
    
    T=2.2s: Nodes receive REQUEST_VOTE
      Node 1: term 5 > my term 4, haven't voted → VOTE YES
      Node 2: term 5 > my term 4, haven't voted → VOTE YES
      Node 5: term 5 > my term 4, haven't voted → VOTE YES
      Node 3: dead, no response
    
    T=2.3s: Node 4 has 4 votes (self + 1,2,5)
      4 ≥ quorum(3) → NODE 4 IS LEADER (term=5)
      Node 4 sends HEARTBEAT(term=5, leader=4) to all
    
    T=2.4s: All nodes acknowledge Node 4 as leader
    
    Total election time: ~0.3 seconds!

  SPLIT VOTE SCENARIO:
    T=2.0s: Node 4 AND Node 2 both timeout simultaneously
    
    Node 4: CANDIDATE, term=5, votes for self
    Node 2: CANDIDATE, term=5, votes for self
    
    Node 1: receives REQUEST_VOTE from Node 4 first → votes Node 4
    Node 5: receives REQUEST_VOTE from Node 2 first → votes Node 2
    
    Node 4 has: self + Node 1 = 2 votes (not majority)
    Node 2 has: self + Node 5 = 2 votes (not majority)
    
    Neither wins → both timeout → try again with term=6
    → Randomized timeout makes one start first next time
    → Eventually one gets majority

  WHY THIS IS SAFE:
    → Each node votes ONCE per term → can't vote for two candidates
    → Quorum required → at most ONE candidate can get majority
    → Higher terms override lower terms → stale leaders step down

COMPONENT: FENCING TOKENS — PREVENTING SPLIT BRAIN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  THE PROBLEM:
    → Leader A holds lock on resource
    → Leader A pauses (GC, network delay)
    → Cluster elects Leader B
    → Leader A resumes, thinks it's still leader
    → Both A and B operate on the resource → corruption

  THE SOLUTION:
    → Each leader gets a FENCING TOKEN (incrementing number)
    → Leader A: token=5
    → Leader B (after election): token=6
    → All resources check: is this token ≥ last seen token?
    → Leader A sends request with token=5
    → Resource: "Last token I saw was 6, token 5 is stale → REJECTED"
    → Leader B sends request with token=6
    → Resource: "6 ≥ 6 → ACCEPTED"

  IMPLEMENTATION:
    ┌──────────────┐     token=5      ┌──────────────┐
    │ Leader A     │────────────────→ │   Resource   │
    │ (old, stale) │                  │              │
    │ token=5      │     REJECTED!    │ last_token=6 │
    │              │←────────────────│ "5 < 6"      │
    └──────────────┘                  └──────────────┘
    
    ┌──────────────┐     token=6      ┌──────────────┐
    │ Leader B     │────────────────→ │   Resource   │
    │ (current)    │                  │              │
    │ token=6      │     ACCEPTED!    │ last_token=6 │
    │              │←────────────────│ "6 ≥ 6"      │
    └──────────────┘                  └──────────────┘

COMPONENT: LEASE-BASED LEADERSHIP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Alternative to heartbeat-based detection:

  CONCEPT:
    → Leader holds a LEASE (time-limited lock)
    → Lease duration: 15 seconds
    → Leader must RENEW before expiry
    → If leader fails to renew → lease expires → new election

  ADVANTAGE:
    → Bounded time for failure detection
    → Leader KNOWS when its lease expires
    → Leader can stop operating before lease expires
    → No ambiguity about leadership duration

  FLOW:
    T=0:   Leader acquires lease (expires T=15)
    T=10:  Leader renews lease (expires T=25)
    T=20:  Leader renews lease (expires T=35)
    T=30:  Leader crashes
    T=35:  Lease expires → followers start election
    T=36:  New leader elected with new lease
    
    Maximum downtime: lease_duration (15 seconds)

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: Split Brain (Two Leaders)
  → Network partition: [A,B] and [C,D,E]
  → Both partitions elect a leader
  → Two leaders make conflicting decisions
  → FIX: Quorum requirement (majority)
  → [A,B] = 2/5 → cannot elect
  → [C,D,E] = 3/5 → elects leader
  → Only ONE partition can have a leader

PITFALL 2: Zombie Leader (Stale Leader Returns)
  → Leader pauses (long GC, network delay)
  → Cluster elects new leader
  → Old leader resumes, thinks it's still leading
  → FIX: Fencing tokens on all operations
  → FIX: Epoch/term numbers in all messages
  → FIX: Leader checks lease before operating

PITFALL 3: Election Storm (Repeated Failed Elections)
  → All nodes timeout simultaneously
  → All become candidates
  → Votes split every time
  → Election never completes
  → FIX: Randomized election timeouts
  → FIX: Exponential backoff on failed elections

PITFALL 4: False Leader Detection
  → Leader is alive but network is slow
  → Followers think leader is dead
  → Unnecessary election → disruption
  → FIX: Multiple missed heartbeats before declaring dead
  → FIX: Longer timeout (but trades off detection speed)

PITFALL 5: Leader Overload
  → Leader handles coordination AND heavy work
  → Leader slows down → heartbeats delayed
  → Followers think leader is dead → election
  → New leader also overloaded → same problem
  → FIX: Leader does coordination only, not heavy work
  → FIX: Separate heartbeat thread from work thread

PITFALL 6: Network Asymmetry
  → Leader can send TO followers but not receive FROM them
  → Followers think leader is dead (no heartbeat ACKs)
  → Leader thinks everything is fine (sends succeed)
  → FIX: Bidirectional heartbeat
  → FIX: Leader must receive ACKs to confirm connectivity

📈 SCALABILITY CONSIDERATIONS

ELECTION OVERHEAD:
  → Bully: O(n²) messages in worst case
  → Ring: O(n) messages (one full circulation)
  → Consensus: O(n) messages per election round

CLUSTER SIZE:
  → 3 nodes: Can lose 1 (quorum = 2)
  → 5 nodes: Can lose 2 (quorum = 3)
  → 7 nodes: Can lose 3 (quorum = 4)
  → Beyond 7: Diminishing returns, slower elections
  → Typical production: 3 or 5 nodes

class SimulatedNetwork:
    """
    Simulates network communication between nodes.
    Supports message delay, partitions, and node failure.
    """

    def __init__(self):
        self.nodes = {}
        self.partitions = []  # List of sets of isolated node groups
        self.message_delay_ms = (5, 50)  # Min/max delay
        self.lock = threading.Lock()
        self.message_log = []

    def register_node(self, node_id, node):
        with self.lock:
            self.nodes[node_id] = node

    def send(self, from_id, to_id, message):
        """Send a message from one node to another."""
        with self.lock:
            # Check if sender/receiver exist
            if to_id not in self.nodes:
                return False

            # Check if nodes are partitioned
            if self._is_partitioned(from_id, to_id):
                return False

            # Check if target node is alive
            target = self.nodes[to_id]
            if not target.is_alive:
                return False

            # Log message
            self.message_log.append({
                "from": from_id,
                "to": to_id,
                "type": message.get("type", "unknown"),
                "term": message.get("term", "?"),
                "timestamp": time.time()
            })

        # Simulate network delay
        delay = random.uniform(
            self.message_delay_ms[0] / 1000,
            self.message_delay_ms[1] / 1000
        )
        time.sleep(delay)

        # Deliver message
        target = self.nodes.get(to_id)
        if target and target.is_alive:
            target.receive_message(from_id, message)
            return True
        return False

    def broadcast(self, from_id, message, exclude=None):
        """Send message to all nodes except excluded."""
        exclude = exclude or set()
        exclude.add(from_id)
        results = {}
        for node_id in list(self.nodes.keys()):
            if node_id not in exclude:
                results[node_id] = self.send(from_id, node_id, message)
        return results

    def create_partition(self, group_a, group_b):
        """Create a network partition between two groups."""
        with self.lock:
            self.partitions.append((set(group_a), set(group_b)))
            print(f"  🔌 PARTITION created: {group_a} || {group_b}")

    def heal_partition(self):
        """Heal all network partitions."""
        with self.lock:
            self.partitions.clear()
            print(f"  🔗 All partitions HEALED")

    def _is_partitioned(self, node_a, node_b):
        for group_a, group_b in self.partitions:
            if ((node_a in group_a and node_b in group_b) or
                    (node_a in group_b and node_b in group_a)):
                return True
        return False

    def get_message_count(self):
        with self.lock:
            return len(self.message_log)

class FencingTokenManager:
    """
    Manages monotonically increasing fencing tokens.
    Each new leader gets a higher token than the previous.
    """

    def __init__(self):
        self.current_token = 0
        self.lock = threading.Lock()

    def next_token(self):
        with self.lock:
            self.current_token += 1
            return self.current_token

    def validate_token(self, token):
        """Check if token is current (not stale)."""
        with self.lock:
            return token >= self.current_token

class ConsensusNode:
    """
    A node that participates in consensus-based leader election.
    Simplified Raft: leader election only (no log replication).
    Full Raft → Project 45.
    """

    def __init__(self, node_id, all_node_ids, network,
                 fencing_manager):
        self.node_id = node_id
        self.all_node_ids = all_node_ids
        self.network = network
        self.fencing_manager = fencing_manager

        # Persistent state
        self.current_term = 0
        self.voted_for = None  # Who I voted for in current term
        self.role = NodeRole.FOLLOWER

        # Leader state
        self.leader_id = None
        self.fencing_token = None

        # Timing
        self.heartbeat_interval = 1.0  # seconds
        self.election_timeout_base = (3.0, 5.0)  # Random range
        self.last_heartbeat_time = time.time()

        # Election state
        self.votes_received = set()

        # Node health
        self.is_alive = True

        # Event log
        self.event_log = []

        # Threading
        self.lock = threading.RLock()
        self._running = False

        # Message queue
        self.message_queue = []
        self.msg_lock = threading.Lock()

        # Register with network
        self.network.register_node(node_id, self)

    # ============================================================
    # START / STOP
    # ============================================================

    def start(self):
        """Start the node."""
        self._running = True
        self.is_alive = True
        self.last_heartbeat_time = time.time()

        # Start election timer thread
        self._election_thread = threading.Thread(
            target=self._election_timer_loop, daemon=True
        )
        self._election_thread.start()

        # Start message processor
        self._msg_thread = threading.Thread(
            target=self._message_processor_loop, daemon=True
        )
        self._msg_thread.start()

        self._log_event("STARTED", f"Term={self.current_term}")

    def stop(self):
        """Stop the node (simulate crash)."""
        self._running = False
        self.is_alive = False
        self._log_event("STOPPED", "Node crashed or shut down")

    def recover(self):
        """Recover after crash (keeps persistent state)."""
        self.role = NodeRole.FOLLOWER
        self.leader_id = None
        self.voted_for = None
        self.votes_received = set()
        self.start()
        self._log_event("RECOVERED",
                         f"Rejoined as FOLLOWER, term={self.current_term}")

    # ============================================================
    # MESSAGE HANDLING
    # ============================================================

    def receive_message(self, from_id, message):
        """Queue incoming message for processing."""
        with self.msg_lock:
            self.message_queue.append((from_id, message))

    def _message_processor_loop(self):
        """Process messages from queue."""
        while self._running and self.is_alive:
            msg = None
            with self.msg_lock:
                if self.message_queue:
                    msg = self.message_queue.pop(0)

            if msg:
                from_id, message = msg
                self._handle_message(from_id, message)
            else:
                time.sleep(0.01)

    def _handle_message(self, from_id, message):
        """Handle a single message."""
        msg_type = message.get("type")
        msg_term = message.get("term", 0)

        with self.lock:
            # If message has higher term → step down to follower
            if msg_term > self.current_term:
                self._step_down(msg_term)

            if msg_type == MessageType.REQUEST_VOTE.value:
                self._handle_vote_request(from_id, message)

            elif msg_type == MessageType.VOTE_GRANTED.value:
                self._handle_vote_granted(from_id, message)

            elif msg_type == MessageType.HEARTBEAT.value:
                self._handle_heartbeat(from_id, message)

    def _handle_vote_request(self, from_id, message):
        """Handle a vote request from a candidate."""
        candidate_term = message["term"]
        candidate_id = message["candidate_id"]

        grant = False

        if candidate_term < self.current_term:
            grant = False  # Stale term
        elif (self.voted_for is None or
              self.voted_for == candidate_id):
            grant = True
            self.voted_for = candidate_id
            self.current_term = candidate_term
            self.last_heartbeat_time = time.time()  # Reset timer

        response = {
            "type": (MessageType.VOTE_GRANTED.value if grant
                     else MessageType.VOTE_DENIED.value),
            "term": self.current_term,
            "voter_id": self.node_id,
            "granted": grant
        }

        if grant:
            self._log_event("VOTED",
                            f"Voted for {candidate_id} in term "
                            f"{candidate_term}")

        # Send response (outside lock to prevent deadlock)
        threading.Thread(
            target=self.network.send,
            args=(self.node_id, from_id, response),
            daemon=True
        ).start()

    def _handle_vote_granted(self, from_id, message):
        """Handle receiving a vote."""
        if self.role != NodeRole.CANDIDATE:
            return
        if message.get("term") != self.current_term:
            return

        self.votes_received.add(from_id)
        total_votes = len(self.votes_received)
        quorum = len(self.all_node_ids) // 2 + 1

        self._log_event("VOTE_RECEIVED",
                         f"From {from_id}, total={total_votes}, "
                         f"need={quorum}")

        if total_votes >= quorum:
            self._become_leader()

    def _handle_heartbeat(self, from_id, message):
        """Handle heartbeat from leader."""
        if message.get("term") >= self.current_term:
            self.current_term = message["term"]
            self.leader_id = from_id
            self.role = NodeRole.FOLLOWER
            self.voted_for = None
            self.last_heartbeat_time = time.time()
            self.fencing_token = message.get("fencing_token")

    # ============================================================
    # ELECTION LOGIC
    # ============================================================

    def _election_timer_loop(self):
        """Background loop that triggers elections on timeout."""
        while self._running and self.is_alive:
            time.sleep(0.1)

            with self.lock:
                if self.role == NodeRole.LEADER:
                    # Leader sends heartbeats
                    if (time.time() - self.last_heartbeat_time >
                            self.heartbeat_interval):
                        self._send_heartbeats()
                        self.last_heartbeat_time = time.time()
                else:
                    # Follower/Candidate checks election timeout
                    timeout = random.uniform(
                        *self.election_timeout_base
                    )
                    elapsed = time.time() - self.last_heartbeat_time
                    if elapsed > timeout:
                        self._start_election()

    def _start_election(self):
        """Start a new election."""
        self.current_term += 1
        self.role = NodeRole.CANDIDATE
        self.voted_for = self.node_id
        self.votes_received = {self.node_id}  # Vote for self
        self.last_heartbeat_time = time.time()

        self._log_event("ELECTION_STARTED",
                         f"Term={self.current_term}")

        # Request votes from all other nodes
        message = {
            "type": MessageType.REQUEST_VOTE.value,
            "term": self.current_term,
            "candidate_id": self.node_id
        }

        # Send in background threads to avoid blocking
        for node_id in self.all_node_ids:
            if node_id != self.node_id:
                threading.Thread(
                    target=self.network.send,
                    args=(self.node_id, node_id, message),
                    daemon=True
                ).start()

    def _become_leader(self):
        """Transition to leader role."""
        if self.role == NodeRole.LEADER:
            return  # Already leader

        self.role = NodeRole.LEADER
        self.leader_id = self.node_id
        self.fencing_token = self.fencing_manager.next_token()

        self._log_event("BECAME_LEADER",
                         f"Term={self.current_term}, "
                         f"token={self.fencing_token}")

        # Immediately send heartbeats
        self._send_heartbeats()

    def _send_heartbeats(self):
        """Send heartbeat to all followers."""
        message = {
            "type": MessageType.HEARTBEAT.value,
            "term": self.current_term,
            "leader_id": self.node_id,
            "fencing_token": self.fencing_token
        }

        for node_id in self.all_node_ids:
            if node_id != self.node_id:
                threading.Thread(
                    target=self.network.send,
                    args=(self.node_id, node_id, message),
                    daemon=True
                ).start()

    def _step_down(self, new_term):
        """Step down to follower when higher term discovered."""
        old_role = self.role
        self.current_term = new_term
        self.role = NodeRole.FOLLOWER
        self.voted_for = None
        self.votes_received = set()

        if old_role != NodeRole.FOLLOWER:
            self._log_event("STEPPED_DOWN",
                             f"From {old_role.value} to FOLLOWER, "
                             f"term={new_term}")

    # ============================================================
    # STATUS AND LOGGING
    # ============================================================

    def get_status(self):
        """Get current node status."""
        with self.lock:
            return {
                "node_id": self.node_id,
                "role": self.role.value,
                "term": self.current_term,
                "leader": self.leader_id,
                "fencing_token": self.fencing_token,
                "voted_for": self.voted_for,
                "is_alive": self.is_alive
            }

    def _log_event(self, event_type, details=""):
        timestamp = datetime.now(timezone.utc).isoformat()[:19]
        entry = {
            "timestamp": timestamp,
            "node": self.node_id,
            "event": event_type,
            "term": self.current_term,
            "role": self.role.value,
            "details": details
        }
        self.event_log.append(entry)

        # Print important events
        icons = {
            "BECAME_LEADER": "👑",
            "ELECTION_STARTED": "🗳️ ",
            "VOTED": "✋",
            "STEPPED_DOWN": "⬇️ ",
            "STOPPED": "💀",
            "RECOVERED": "🔄",
            "STARTED": "🟢"
        }
        icon = icons.get(event_type, "  ")
        print(f"    {icon} [{self.node_id}] {event_type}: {details}")

class BullyNode:
    """
    Node using the Bully election algorithm.
    Rule: Highest node ID always wins the election.
    """

    def __init__(self, node_id, all_node_ids, network):
        self.node_id = node_id
        self.all_node_ids = sorted(all_node_ids)
        self.network = network
        self.leader_id = max(all_node_ids)  # Initially assume highest

        self.is_alive = True
        self.election_in_progress = False
        self.received_answer = False
        self.lock = threading.Lock()
        self.event_log = []
        self.role = NodeRole.FOLLOWER
        self.current_term = 0
        self.fencing_token = None

        self.message_queue = []
        self.msg_lock = threading.Lock()
        self._running = False

        self.network.register_node(node_id, self)

    def start(self):
        self._running = True
        self.is_alive = True
        threading.Thread(
            target=self._message_processor_loop, daemon=True
        ).start()

    def stop(self):
        self._running = False
        self.is_alive = False

    def receive_message(self, from_id, message):
        with self.msg_lock:
            self.message_queue.append((from_id, message))

    def _message_processor_loop(self):
        while self._running and self.is_alive:
            msg = None
            with self.msg_lock:
                if self.message_queue:
                    msg = self.message_queue.pop(0)
            if msg:
                from_id, message = msg
                self._handle_message(from_id, message)
            else:
                time.sleep(0.01)

    def _handle_message(self, from_id, message):
        msg_type = message.get("type")

        if msg_type == MessageType.ELECTION.value:
            # Someone with lower ID started election → bully them
            if self.node_id > from_id:
                self.network.send(self.node_id, from_id, {
                    "type": MessageType.ANSWER.value,
                    "node_id": self.node_id
                })
                # Start my own election
                self.start_election()

        elif msg_type == MessageType.ANSWER.value:
            # Higher node responded → I back off
            with self.lock:
                self.received_answer = True

        elif msg_type == MessageType.COORDINATOR.value:
            # New leader announced
            with self.lock:
                self.leader_id = message["leader_id"]
                self.fencing_token = message.get("fencing_token")
                self.role = NodeRole.FOLLOWER
                self.current_term = message.get("term", 0)
                print(f"    📢 [{self.node_id}] Acknowledged "
                      f"leader: {self.leader_id}")

    def start_election(self):
        """Initiate a bully election."""
        with self.lock:
            if self.election_in_progress:
                return
            self.election_in_progress = True
            self.received_answer = False

        print(f"    🗳️  [{self.node_id}] Starting BULLY election")

        # Send ELECTION to all higher-ID nodes
        higher_nodes = [n for n in self.all_node_ids
                        if n > self.node_id]

        if not higher_nodes:
            # I am the highest → I become leader
            self._declare_leader()
            return

        for node_id in higher_nodes:
            self.network.send(self.node_id, node_id, {
                "type": MessageType.ELECTION.value,
                "node_id": self.node_id
            })

        # Wait for answers
        time.sleep(2.0)  # Timeout

        with self.lock:
            if not self.received_answer:
                # No higher node responded → I become leader
                self._declare_leader()
            self.election_in_progress = False

    def _declare_leader(self):
        """Declare self as the new leader."""
        with self.lock:
            self.current_term += 1
            self.role = NodeRole.LEADER
            self.leader_id = self.node_id
            self.fencing_token = self.current_term

        print(f"    👑 [{self.node_id}] BULLY winner! "
              f"I am the leader (term={self.current_term})")

        # Announce to all nodes
        for node_id in self.all_node_ids:
            if node_id != self.node_id:
                self.network.send(self.node_id, node_id, {
                    "type": MessageType.COORDINATOR.value,
                    "leader_id": self.node_id,
                    "term": self.current_term,
                    "fencing_token": self.fencing_token
                })

        with self.lock:
            self.election_in_progress = False

    def get_status(self):
        with self.lock:
            return {
                "node_id": self.node_id,
                "role": self.role.value,
                "term": self.current_term,
                "leader": self.leader_id,
                "fencing_token": self.fencing_token,
                "is_alive": self.is_alive
            }

    class LeaseBasedLeader:
        """
        Simplified lease-based leader election.
        Leader holds a time-limited lease.
        Must renew before expiry or lose leadership.
        """

        def __init__(self):
            self.leader_id = None
            self.lease_expiry = 0
            self.lease_duration = 5  # seconds
            self.fencing_token = 0
            self.lock = threading.Lock()
            self.history = []

        def try_acquire(self, node_id):
            """Try to become leader by acquiring the lease."""
            with self.lock:
                now = time.time()

                # Lease expired or no leader
                if now > self.lease_expiry or self.leader_id is None:
                    self.leader_id = node_id
                    self.lease_expiry = now + self.lease_duration
                    self.fencing_token += 1
                    self.history.append({
                        "event": "ACQUIRED",
                        "node": node_id,
                        "token": self.fencing_token,
                        "time": datetime.now(timezone.utc).isoformat()[:19]
                    })
                    return {
                        "acquired": True,
                        "token": self.fencing_token,
                        "expires_in": self.lease_duration
                    }

                # Lease still held by someone else
                return {
                    "acquired": False,
                    "current_leader": self.leader_id,
                    "expires_in": round(self.lease_expiry - now, 1)
                }

        def renew(self, node_id, token):
            """Renew the lease (only if you're the current leader)."""
            with self.lock:
                now = time.time()

                if (self.leader_id == node_id and
                        token == self.fencing_token and
                        now < self.lease_expiry):
                    self.lease_expiry = now + self.lease_duration
                    self.history.append({
                        "event": "RENEWED",
                        "node": node_id,
                        "token": token,
                        "time": datetime.now(timezone.utc).isoformat()[:19]
                    })
                    return {"renewed": True, "expires_in": self.lease_duration}

                return {
                    "renewed": False,
                    "reason": "Not the current leader or lease expired"
                }

        def get_leader(self):
            with self.lock:
                now = time.time()
                if now > self.lease_expiry:
                    return {"leader": None, "status": "NO_LEADER"}
                return {
                    "leader": self.leader_id,
                    "token": self.fencing_token,
                    "expires_in": round(self.lease_expiry - now, 1)
                }

    class DistributedTaskScheduler:
        """
        Only the leader node runs scheduled tasks.
        Prevents duplicate execution across cluster.
        Uses fencing tokens for safety.
        """

        def __init__(self):
            self.executed_tasks = []
            self.lock = threading.Lock()

        def execute_if_leader(self, node_id, role, fencing_token,
                              task_name):
            """Execute task only if this node is the leader."""
            with self.lock:
                if role != NodeRole.LEADER:
                    return {
                        "executed": False,
                        "reason": f"{node_id} is {role.value}, not LEADER"
                    }

                # Fencing check: only latest leader can execute
                if self.executed_tasks:
                    last_token = self.executed_tasks[-1].get("token", 0)
                    if fencing_token < last_token:
                        return {
                            "executed": False,
                            "reason": f"Stale token {fencing_token} "
                                      f"< {last_token}"
                        }

                execution = {
                    "task": task_name,
                    "node": node_id,
                    "token": fencing_token,
                    "time": datetime.now(timezone.utc).isoformat()[:19]
                }
                self.executed_tasks.append(execution)
                return {"executed": True, "task": execution}
