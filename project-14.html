🎯 HYPOTHETICAL PROBLEM

Your pizza shop chain now has a MASSIVE caching layer (Project 5)
and multiple database replicas (Project 13):

  → 50 million orders in the database
  → 10 million cached items across your cache servers
  → 5 cache servers handling 100,000 lookups/second

The cache uses a simple distribution strategy:

  server_index = hash(key) % number_of_servers
  
  hash("order_1001") % 5 = 3  → Cache Server 3
  hash("order_1002") % 5 = 1  → Cache Server 1
  hash("order_1003") % 5 = 0  → Cache Server 0

This works PERFECTLY... until it doesn't.


SCENARIO A: Adding a Server (Scale Up)
  → Traffic grows, you add a 6th cache server
  → Now: hash(key) % 6 instead of hash(key) % 5
  
  hash("order_1001") % 5 = 3  →  hash("order_1001") % 6 = 1  ← MOVED
  hash("order_1002") % 5 = 1  →  hash("order_1002") % 6 = 4  ← MOVED
  hash("order_1003") % 5 = 0  →  hash("order_1003") % 6 = 0  ← stayed
  
  → ~80% of keys now map to DIFFERENT servers!
  → 80% of your cache is INVALIDATED instantly
  → 8 million cache misses hit the database simultaneously
  → Database: 5,000 queries/sec → 80,000 queries/sec (OVERLOAD)
  → This is called a "CACHE STAMPEDE" or "THUNDERING HERD"
  → Your database crashes
  → Adding a server CAUSED a catastrophic failure


SCENARIO B: Losing a Server (Crash)
  → Cache Server 2 crashes
  → Now: hash(key) % 4 instead of hash(key) % 5
  → SAME problem: ~80% of keys remap
  → Already under stress (lost a server) + cache stampede
  → Cascading failure: remaining servers overwhelmed
  → More servers crash → more remapping → total collapse
  → This is a "CASCADING FAILURE"


SCENARIO C: Uneven Distribution
  → 5 servers, but hash function clusters data unevenly
  → Server 0: 30% of data (overloaded)
  → Server 1: 25% of data
  → Server 2: 5% of data (wasted capacity)
  → Server 3: 35% of data (most overloaded)
  → Server 4: 5% of data (wasted capacity)
  → "Hot spots" cause some servers to fail while others idle


SCENARIO D: Database Sharding
  → 50 million orders across 5 database shards (Project 18 preview)
  → Shard assignment: hash(user_id) % 5
  → Need to add a 6th shard (data growing)
  → hash(user_id) % 5 → hash(user_id) % 6
  → Billions of rows need to PHYSICALLY MOVE between servers
  → Migration takes DAYS with the system partially down
  → During migration: some queries go to wrong shard → errors


THE CORE QUESTION:
  → How do you distribute data across N servers
  → Such that adding/removing a server
  → Only affects 1/N of the data (not 80%)?
  → And the distribution stays roughly even?

THE ANALOGY:
  → Imagine a ROUND TABLE with seats numbered 0-359 (like a clock)
  → 5 people sit at positions: 72, 144, 216, 288, 360(=0)
  → Each person is "responsible" for the arc BEFORE them
  → Person at 72 handles seats 1-72
  → Person at 144 handles seats 73-144
  → etc.

  → A plate (data) arrives with a number: 100
  → Walk clockwise from 100 → first person you meet: position 144
  → That person holds the plate

  → Now a 6th person sits at position 108
  → Only plates between 73-108 move from person-144 to person-108
  → Everyone else keeps their plates!
  → Only a SMALL fraction of data moves

  → This is CONSISTENT HASHING

🔍 HOW TO APPROACH IT

Step 1: Understand WHY modulo hashing fails
         → hash(key) % N changes for most keys when N changes
         → Adding 1 server: ~(N-1)/N keys remap (~80% for 5 servers)
         → This is catastrophic for caches and databases

Step 2: Understand the HASH RING concept
         → Map the hash space to a circle (ring): 0 to 2^32-1
         → Place servers at positions on the ring (hash of server name)
         → Place data at positions on the ring (hash of key)
         → Data belongs to the NEXT server clockwise on the ring

Step 3: Understand what happens when a server is added/removed
         → Only data between the new server and its predecessor moves
         → All other data stays in place
         → ~1/N of data moves (not 80%)

Step 4: Understand VIRTUAL NODES
         → 5 physical servers → place each at MULTIPLE positions
         → Server A at positions: 10, 120, 230, 340 (4 virtual nodes)
         → More virtual nodes = more even distribution
         → Standard: 100-200 virtual nodes per physical server

Step 5: Implement and measure
         → Build the hash ring
         → Test: add/remove servers → measure data movement
         → Test: distribution evenness with/without virtual nodes

📌 WHAT COULD BE DONE

Option A: Simple Modulo Hashing (hash % N)
  → Simplest possible approach
  → Works if N NEVER changes
  → Catastrophic when N changes (80%+ remapping)
  → Only for fixed-size, never-changing clusters

Option B: Consistent Hashing with Physical Nodes Only
  → Hash ring with one point per server
  → Only 1/N data moves on changes
  → But: very uneven distribution
  → Not practical for production

Option C: Consistent Hashing with Virtual Nodes
  → Hash ring with 100-200 virtual nodes per server
  → Only ~1/N data moves on changes
  → Even distribution across servers
  → THIS IS WHAT WE BUILD (industry standard)

Option D: Rendezvous Hashing (Highest Random Weight)
  → For each key: compute score for EVERY server
  → Assign key to server with highest score
  → Adding/removing server: only affected keys move
  → Simpler to implement, but O(N) per lookup vs O(log N)
  → Good for small N (few servers)

Option E: Jump Consistent Hashing
  → Google's algorithm for sequential server IDs
  → O(ln N) per lookup, zero memory overhead
  → But: only works when adding servers (not arbitrary removal)
  → Specialized use case

1. ALWAYS use virtual nodes in production

   → With 150 virtual nodes each: imbalance < 5%

2. ALWAYS use a good hash function

   → Return error or fallback, don't crash

6. ALWAYS plan for data migration when adding servers

7. ALWAYS monitor distribution evenness

1. NEVER use modulo hashing for distributed systems
   → hash(key) % N → catastrophic on any cluster change
   → The entire purpose of consistent hashing is to avoid this

   → PYTHONHASHSEED randomization (security feature)

   → Actually MOVING the data takes time
   → During migration: old server still serves, new server warms up

5. NEVER ignore hot keys

LOOKUP ALGORITHM:
═════════════════

  Input: key (e.g., "order_1001")
  
  Step 1: Compute hash of key
    hash_pos = md5("order_1001") → 28,445 (mapped to ring space)
  
  Step 2: Binary search for first ring entry >= hash_pos
    → ring[4] = (25110, "D") → 25110 < 28445 → too small
    → ring[5] = (31445, "C") → 31445 >= 28445 → FOUND!
  
  Step 3: Return server "C"
  
  Edge case: hash_pos > last entry → wrap around to first entry
    → This is the "circular" part of the ring
  
  Time complexity: O(log V) where V = total virtual nodes
  Space complexity: O(V)
  
  For 5 servers × 150 vnodes: O(log 750) ≈ 10 comparisons


ADD SERVER ALGORITHM:
═════════════════════

  Adding Server E with 150 virtual nodes:
  
  Step 1: Generate 150 hash positions
    hash("E-0") → pos 3400
    hash("E-1") → pos 15600
    hash("E-2") → pos 28100
    ...
    hash("E-149") → pos 4294960000
  
  Step 2: Insert each into sorted ring array
    Binary search + insert for each position
    O(V × log V) total
  
  Step 3: Identify data that needs to move
    For each new virtual node position P:
      → Find predecessor position PRED on ring
      → All keys with hash in (PRED, P] now belong to Server E
      → These keys were previously on the NEXT server after P
      → THAT server sends these keys to Server E
  
  Data movement: ~1/(N+1) of total keys
    5 servers → add 1 → ~1/6 ≈ 16.7% of keys move
    NOT 80% like modulo hashing!


REMOVE SERVER ALGORITHM:
════════════════════════

  Removing Server B:
  
  Step 1: Find all virtual node positions for Server B
    ring entries where server == "B"
  
  Step 2: For each position P being removed:
    → Find NEXT server position NEXT on ring (clockwise)
    → All keys in the arc before P now belong to server at NEXT
    → Server B sends those keys to server at NEXT
  
  Step 3: Remove all Server B entries from ring
  
  Data movement: ~1/N of total keys (Server B's share)
    5 servers → remove 1 → ~1/5 ≈ 20% of keys move


WHY VIRTUAL NODES IMPROVE DISTRIBUTION:
════════════════════════════════════════

  WITHOUT virtual nodes (5 points on ring):

# ============================================================

def consistent_hash(key: str, ring_size: int = 2**32) -> int:
    """
    Deterministic hash function using MD5
    Returns integer in range [0, ring_size)
    
    NEVER use Python's built-in hash() — it's randomized per run
    """
    md5_digest = hashlib.md5(key.encode('utf-8')).hexdigest()
    return int(md5_digest, 16) % ring_size

        self.nodes = {}           # node_id → {"weight": int, "vnodes": [positions]}
        
        # Statistics
        self.total_lookups = 0
    
    def add_node(self, node_id: str, weight: int = 1) -> dict:
        """
        Add a node to the ring with virtual nodes
        Weight controls how many virtual nodes (proportional to capacity)
        Returns: which existing nodes lose data to the new node
        """
        if node_id in self.nodes:
            return {"error": f"Node {node_id} already exists"}
        
        num_vnodes = self.virtual_nodes * weight
        positions = []
        affected_nodes = defaultdict(int)
        
        for i in range(num_vnodes):
            # Generate virtual node key
            vnode_key = f"{node_id}:vn{i}"
            pos = consistent_hash(vnode_key, self.ring_size)
            
            # Check which node currently owns this position
            # (to track data movement)
            if self.ring_positions:
                current_owner = self._find_node_for_position(pos)
                if current_owner and current_owner != node_id:
                    affected_nodes[current_owner] += 1
            
            # Insert into ring
            bisect.insort(self.ring_positions, pos)
            self.ring_map[pos] = node_id
            positions.append(pos)
        
        self.nodes[node_id] = {
            "weight": weight,
            "vnodes": positions,
            "added_at": time.time()
        }
        
        return {
            "node_id": node_id,
            "virtual_nodes_added": num_vnodes,
            "affected_nodes": dict(affected_nodes)
        }
    
    def remove_node(self, node_id: str) -> dict:
        """
        Remove a node and all its virtual nodes from the ring
        Returns: which nodes inherit the removed node's data
        """
        if node_id not in self.nodes:
            return {"error": f"Node {node_id} not found"}
        
        inheriting_nodes = defaultdict(int)
        positions_to_remove = self.nodes[node_id]["vnodes"]
        
        # For each virtual node being removed, find who inherits its data
        for pos in positions_to_remove:
            # Remove from ring temporarily to find next node
            idx = bisect.bisect_left(self.ring_positions, pos)
            if idx < len(self.ring_positions) and self.ring_positions[idx] == pos:
                # Temporarily remove to find next
                self.ring_positions.pop(idx)
                del self.ring_map[pos]
                
                # Find who inherits this arc
                if self.ring_positions:
                    inheritor = self._find_node_for_position(pos)
                    if inheritor:
                        inheriting_nodes[inheritor] += 1
        
        vnodes_removed = len(positions_to_remove)
        del self.nodes[node_id]
        
        return {
            "node_id": node_id,
            "virtual_nodes_removed": vnodes_removed,
            "inheriting_nodes": dict(inheriting_nodes)
        }
    
    def get_node(self, key: str) -> Optional[str]:
        """
        Find which node owns the given key
        O(log V) where V = total virtual nodes
        """
        if not self.ring_positions:
            return None
        
        self.total_lookups += 1
        
        pos = consistent_hash(key, self.ring_size)
        return self._find_node_for_position(pos)
    
    def _find_node_for_position(self, pos: int) -> Optional[str]:
        """Binary search for the next node clockwise from position"""
        if not self.ring_positions:
            return None
        
        # Find first position >= pos
        idx = bisect.bisect_left(self.ring_positions, pos)
        
        # Wrap around if past the end
        if idx >= len(self.ring_positions):
            idx = 0
        
        ring_pos = self.ring_positions[idx]
        return self.ring_map[ring_pos]
    
    def get_nodes(self, key: str, count: int = 3) -> list:
        """
        Find N DISTINCT nodes for a key (for replication)
        Walks clockwise, skipping duplicate physical nodes
        """
        if not self.ring_positions:
            return []
        
        pos = consistent_hash(key, self.ring_size)
        idx = bisect.bisect_left(self.ring_positions, pos)
        
        result = []
        seen = set()
        
        for i in range(len(self.ring_positions)):
            actual_idx = (idx + i) % len(self.ring_positions)
            ring_pos = self.ring_positions[actual_idx]
            node_id = self.ring_map[ring_pos]
            
            if node_id not in seen:
                result.append(node_id)
                seen.add(node_id)
                
                if len(result) >= count:
                    break
        
        return result
    
    def get_distribution(self, num_keys: int = 100000) -> dict:
        """
        Analyze distribution by hashing many random keys
        Returns: keys per node, standard deviation, max/min ratio
        """
        if not self.ring_positions:
            return {"error": "Ring is empty"}
        
        counts = defaultdict(int)
        
        for i in range(num_keys):
            key = f"test_key_{i}"
            node = self.get_node(key)
            if node:
                counts[node] += 1
        
        values = list(counts.values())
        mean = num_keys / len(counts)
        variance = sum((v - mean) ** 2 for v in values) / len(values)
        std_dev = math.sqrt(variance)
        
        return {
            "num_keys": num_keys,
            "num_nodes": len(counts),
            "distribution": dict(counts),
            "mean_keys_per_node": round(mean, 1),
            "std_dev": round(std_dev, 1),
            "std_dev_pct": round(std_dev / mean * 100, 2),
            "min_keys": min(values),
            "max_keys": max(values),
            "max_min_ratio": round(max(values) / max(min(values), 1), 2),
            "imbalance_pct": round(
                (max(values) - min(values)) / mean * 100, 1)
        }
    
    def get_ring_info(self) -> dict:
        return {
            "total_nodes": len(self.nodes),
            "total_virtual_nodes": len(self.ring_positions),
            "vnodes_per_node": self.virtual_nodes,
            "ring_size": self.ring_size,
            "nodes": {
                nid: {
                    "weight": info["weight"],
                    "vnode_count": len(info["vnodes"])
                }
                for nid, info in self.nodes.items()
            }
        }

class RendezvousHashRouter:
    """
    Highest Random Weight (HRW) hashing
    For each key: score every server, pick highest score
    """
    
    def __init__(self):
        self.servers = []
    
    def add_server(self, server_id: str):
        self.servers.append(server_id)
    
    def remove_server(self, server_id: str):
        if server_id in self.servers:
            self.servers.remove(server_id)
    
    def get_server(self, key: str) -> Optional[str]:
        if not self.servers:
            return None
        
        best_server = None
        best_score = -1
        
        for server in self.servers:
            combined = f"{key}:{server}"
            score = consistent_hash(combined)
            if score > best_score:
                best_score = score
                best_server = server
        
        return best_server
    
    def get_distribution(self, num_keys: int = 100000) -> dict:
        counts = defaultdict(int)
        for i in range(num_keys):
            server = self.get_server(f"test_key_{i}")
            if server:
                counts[server] += 1
        
        values = list(counts.values())
        mean = num_keys / max(len(counts), 1)
        
        return {
            "distribution": dict(counts),
            "mean": round(mean, 1),
            "max_min_ratio": round(
                max(values) / max(min(values), 1), 2) if values else 0
        }

import hashlib
import bisect
import time
import random
from collections import defaultdict


class CacheNode:
    """Single cache server node"""
    
    def __init__(self, node_id: str, capacity_mb: int = 1024):
        self.node_id = node_id
        self.capacity_mb = capacity_mb
        self.data = {}
        self.alive = True
        
        self.stats = {
            "hits": 0,
            "misses": 0,
            "sets": 0,
            "evictions": 0
        }
    
    def get(self, key: str):
        if not self.alive:
            return None
        if key in self.data:
            self.stats["hits"] += 1
            return self.data[key]
        self.stats["misses"] += 1
        return None
    
    def set(self, key: str, value):
        if not self.alive:
            return False
        self.data[key] = value
        self.stats["sets"] += 1
        return True
    
    def delete(self, key: str):
        return self.data.pop(key, None) is not None
    
    def get_hit_rate(self) -> float:
        total = self.stats["hits"] + self.stats["misses"]
        if total == 0:
            return 0
        return self.stats["hits"] / total * 100


class DistributedCache:
    """
    Distributed cache using consistent hashing
    Simulates a real cache cluster like Memcached or Redis Cluster
    """
    
    def __init__(self, virtual_nodes: int = 150):
        self.virtual_nodes = virtual_nodes
        self.ring_positions = []
        self.ring_map = {}
        self.nodes = {}  # node_id → CacheNode
        self.node_vnodes = {}  # node_id → [positions]
        
        self.global_stats = {
            "total_gets": 0,
            "total_hits": 0,
            "total_misses": 0,
            "total_sets": 0,
            "migrations": 0,
            "migration_keys": 0
        }
    
    def _hash(self, key: str) -> int:
        return int(hashlib.md5(key.encode()).hexdigest(), 16) % (2**32)
    
    def _find_node(self, key: str) -> str:
        if not self.ring_positions:
            return None
        pos = self._hash(key)
        idx = bisect.bisect_left(self.ring_positions, pos)
        if idx >= len(self.ring_positions):
            idx = 0
        return self.ring_map[self.ring_positions[idx]]
    
    def add_node(self, node_id: str, capacity_mb: int = 1024) -> dict:
        """Add a cache node and migrate data that belongs to it"""
        node = CacheNode(node_id, capacity_mb)
        self.nodes[node_id] = node
        
        positions = []
        for i in range(self.virtual_nodes):
            vnode_key = f"{node_id}:vn{i}"
            pos = self._hash(vnode_key)
            bisect.insort(self.ring_positions, pos)
            self.ring_map[pos] = node_id
            positions.append(pos)
        
        self.node_vnodes[node_id] = positions
        
        # Migrate data from existing nodes that now belongs to new node
        migrated = 0
        for other_id, other_node in self.nodes.items():
            if other_id == node_id:
                continue
            
            keys_to_move = []
            for key in list(other_node.data.keys()):
                owner = self._find_node(key)
                if owner == node_id:
                    keys_to_move.append(key)
            
            for key in keys_to_move:
                value = other_node.data.pop(key)
                node.data[key] = value
                migrated += 1
        
        self.global_stats["migrations"] += 1
        self.global_stats["migration_keys"] += migrated
        
        return {
            "node_id": node_id,
            "keys_migrated_in": migrated,
            "total_nodes": len(self.nodes)
        }
    
    def remove_node(self, node_id: str) -> dict:
        """Remove a node and redistribute its data"""
        if node_id not in self.nodes:
            return {"error": "Node not found"}
        
        node = self.nodes[node_id]
        orphaned_keys = dict(node.data)
        
        # Remove from ring
        for pos in self.node_vnodes.get(node_id, []):
            idx = bisect.bisect_left(self.ring_positions, pos)
            if idx < len(self.ring_positions) and self.ring_positions[idx] == pos:
                self.ring_positions.pop(idx)
                del self.ring_map[pos]
        
        del self.nodes[node_id]
        del self.node_vnodes[node_id]
        
        # Redistribute orphaned data
        redistributed = 0
        lost = 0
        for key, value in orphaned_keys.items():
            new_owner = self._find_node(key)
            if new_owner and new_owner in self.nodes:
                self.nodes[new_owner].data[key] = value
                redistributed += 1
            else:
                lost += 1
        
        self.global_stats["migrations"] += 1
        self.global_stats["migration_keys"] += redistributed
        
        return {
            "node_id": node_id,
            "keys_redistributed": redistributed,
            "keys_lost": lost,
            "total_nodes": len(self.nodes)
        }
    
    def get(self, key: str):
        """Get a value from the cache"""
        self.global_stats["total_gets"] += 1
        
        node_id = self._find_node(key)
        if not node_id or node_id not in self.nodes:
            self.global_stats["total_misses"] += 1
            return None
        
        node = self.nodes[node_id]
        result = node.get(key)
        
        if result is not None:
            self.global_stats["total_hits"] += 1
        else:
            self.global_stats["total_misses"] += 1
        
        return result
    
    def set(self, key: str, value):
        """Set a value in the cache"""
        self.global_stats["total_sets"] += 1
        
        node_id = self._find_node(key)
        if not node_id or node_id not in self.nodes:
            return False
        
        return self.nodes[node_id].set(key, value)
    
    def get_cluster_stats(self) -> dict:
        total_keys = sum(len(n.data) for n in self.nodes.values())
        hit_rate = 0
        if self.global_stats["total_gets"] > 0:
            hit_rate = self.global_stats["total_hits"] / self.global_stats["total_gets"] * 100
        
        node_stats = {}
        for nid, node in self.nodes.items():
            node_stats[nid] = {
                "keys": len(node.data),
                "hit_rate": round(node.get_hit_rate(), 1),
                "alive": node.alive
            }
        
        return {
            "total_nodes": len(self.nodes),
            "total_keys": total_keys,
            "global_hit_rate": round(hit_rate, 1),
            "global_stats": self.global_stats,
            "nodes": node_stats
        }

import hashlib
import math
import bisect
from collections import defaultdict


def hash_to_ring(key: str, ring_size: int = 360) -> int:
    """Hash to a position on a 360-degree ring (for visualization)"""
    h = int(hashlib.md5(key.encode()).hexdigest(), 16)
    return h % ring_size


class VisualHashRing:
    """Hash ring with ASCII circle visualization"""
    
    def __init__(self, virtual_nodes: int = 8):
        """Using fewer vnodes for readable visualization"""
        self.virtual_nodes = virtual_nodes
        self.ring_size = 360  # Degrees for visual
        self.positions = []   # Sorted positions
        self.pos_map = {}     # position → node_id
        self.nodes = {}       # node_id → [positions]
        
        # Visual symbols for each server
        self.symbols = {}
        self.available_symbols = list("ABCDEFGHIJKLMNOP")
        self.symbol_idx = 0
    
    def add_node(self, node_id: str):
        if node_id not in self.symbols:
            self.symbols[node_id] = self.available_symbols[self.symbol_idx % len(self.available_symbols)]
            self.symbol_idx += 1
        
        positions = []
        for i in range(self.virtual_nodes):
            pos = hash_to_ring(f"{node_id}:vn{i}", self.ring_size)
            bisect.insort(self.positions, pos)
            self.pos_map[pos] = node_id
            positions.append(pos)
        
        self.nodes[node_id] = positions
    
    def remove_node(self, node_id: str):
        if node_id not in self.nodes:
            return
        for pos in self.nodes[node_id]:
            if pos in self.pos_map:
                idx = bisect.bisect_left(self.positions, pos)
                if idx < len(self.positions) and self.positions[idx] == pos:
                    self.positions.pop(idx)
                del self.pos_map[pos]
        del self.nodes[node_id]
    
    def find_node(self, key: str) -> str:
        if not self.positions:
            return None
        pos = hash_to_ring(key, self.ring_size)
        idx = bisect.bisect_left(self.positions, pos)
        if idx >= len(self.positions):
            idx = 0
        return self.pos_map[self.positions[idx]]
    
    def render_ring(self, keys: list = None, radius: int = 10):
        """Render the hash ring as ASCII art"""
        # Create a 2D grid
        width = radius * 2 + 5
        height = radius * 2 + 5
        center_x = width // 2
        center_y = height // 2
        
        grid = [[' ' for _ in range(width)] for _ in range(height)]
        
        # Draw circle outline
        for angle in range(360):
            rad = math.radians(angle)
            x = int(center_x + radius * math.cos(rad))
            y = int(center_y + radius * math.sin(rad))
            if 0 <= x < width and 0 <= y < height:
                if grid[y][x] == ' ':
                    grid[y][x] = '·'
        
        # Place server positions on ring
        for pos, node_id in self.pos_map.items():
            rad = math.radians(pos)
            x = int(center_x + radius * math.cos(rad))
            y = int(center_y + radius * math.sin(rad))
            if 0 <= x < width and 0 <= y < height:
                grid[y][x] = self.symbols[node_id]
        
        # Place keys on ring
        key_positions = {}
        if keys:
            for key in keys:
                pos = hash_to_ring(key, self.ring_size)
                owner = self.find_node(key)
                rad = math.radians(pos)
                x = int(center_x + (radius - 2) * math.cos(rad))
                y = int(center_y + (radius - 2) * math.sin(rad))
                if 0 <= x < width and 0 <= y < height:
                    grid[y][x] = '★'
                key_positions[key] = {"pos": pos, "owner": owner}
        
        # Center label
        grid[center_y][center_x - 2:center_x + 3] = list("RING ")
        
        # Print grid
        for row in grid:
            print(f"    {''.join(row)}")
        
        return key_positions
    
    def render_linear(self, keys: list = None):
        """Render ring as linear bar (easier to read)"""
        bar_width = 72
        
        # Create bar
        bar = list('─' * bar_width)
        
        # Place servers
        for pos, node_id in self.pos_map.items():
            bar_pos = int(pos / self.ring_size * bar_width)
            bar_pos = min(bar_pos, bar_width - 1)
            bar[bar_pos] = self.symbols[node_id]
        
        # Print bar
        print(f"\n    0°{''.join(bar)}360°")
        
        # Place keys below
        if keys:
            key_bar = list(' ' * (bar_width + 4))
            for key in keys:
                pos = hash_to_ring(key, self.ring_size)
                bar_pos = int(pos / self.ring_size * bar_width) + 2
                bar_pos = min(bar_pos, bar_width + 1)
                key_bar[bar_pos] = '↑'
            print(f"    {''.join(key_bar)}")
        
        # Legend
        print(f"\n    Legend:")
        for node_id in sorted(self.nodes.keys()):
            symbol = self.symbols[node_id]
            positions = sorted(self.nodes[node_id])
            pos_str = ', '.join(f"{p}°" for p in positions[:4])
            if len(positions) > 4:
                pos_str += f" ... ({len(positions)} vnodes)"
            print(f"      {symbol} = {node_id} at {pos_str}")
    
    def show_ownership(self, num_keys: int = 1000):
        """Show which server owns what percentage"""
        counts = defaultdict(int)
        for i in range(num_keys):
            node = self.find_node(f"key_{i}")
            if node:
                counts[node] += 1
        
        print(f"\n    Ownership distribution ({num_keys:,} keys):")
        for node_id in sorted(counts.keys()):
            count = counts[node_id]
            pct = count / num_keys * 100
            bar = "█" * int(pct)
            symbol = self.symbols[node_id]
            print(f"      [{symbol}] {node_id:15s}: {pct:>5.1f}% {bar}")

✅ Weighted nodes: more vnodes = more data (proportional to capacity)
✅ Replication: key stored on next N distinct servers clockwise
✅ Rendezvous hashing: alternative for small clusters (O(N) lookup)
✅ Use deterministic hash (MD5/SHA), never Python's built-in hash()
✅ Real-world: DynamoDB, Cassandra, Memcached, Redis Cluster all use this

THE PROBLEM THAT REMAINS:
  → Cache server silently stops responding
    → Keys routed to it → all cache misses → database overloaded
    → Nobody knows until customers complain

  → Database replica stops replicating (network issue)
    → Reads go to stale replica → wrong data served
    → Nobody knows until inconsistencies are reported

  → Message queue consumer crashes
    → Messages pile up → queue grows → memory exhaustion
    → Nobody knows until the queue server itself crashes

  → Load balancer still sends traffic to a dead server
    → 20% of requests fail → customers see errors
    → Nobody knows for 30 minutes

  → ALL of these are "silent failures"
    → The component doesn't announce it's broken
    → Other components keep interacting with it
    → Cascading effects before anyone notices
