     → Deploying a new API version: coordinate 12 services

     → No centralized view of ALL API traffic

🔍 HOW TO APPROACH IT

THE API GATEWAY HANDLES 8 RESPONSIBILITIES:

Responsibility 1: REQUEST ROUTING
  → /api/users/*     → route to User Service
  → /api/urls/*      → route to URL Shortener Service
  → /api/paste/*     → route to Pastebin Service
  → /api/payments/*  → route to Payment Service
  → Path-based, header-based, or method-based routing

Responsibility 2: AUTHENTICATION & AUTHORIZATION
  → Client sends JWT token in Authorization header
  → Gateway validates the token ONCE
  → Passes authenticated user info to backend services
  → Backend services TRUST the gateway (no re-validation)
  → Centralizes ALL auth logic in one place

Responsibility 3: RATE LIMITING (Project 9 applied)
  → Per-user rate limits: 100 req/min for free tier
  → Per-endpoint rate limits: 10 creates/min for URL shortener
  → Global rate limits: 10,000 req/sec total
  → Applied BEFORE routing (saves backend resources)

Responsibility 4: REQUEST/RESPONSE TRANSFORMATION
  → Client sends v2 format → gateway transforms to v1 for old backend
  → Backend returns XML → gateway transforms to JSON for client
  → Add/remove headers, reshape payloads
  → API versioning without changing backends

Responsibility 5: LOAD BALANCING (Project 6 applied)
  → Each backend service has multiple instances
  → Gateway load-balances across instances
  → Health-aware routing (Project 15)
  → Remove unhealthy instances automatically

Responsibility 6: CACHING
  → Cache GET responses at the gateway level
  → Identical requests from different clients → same cached response
  → Reduces load on backend services
  → Short TTL for dynamic data, long TTL for static

Responsibility 7: LOGGING & MONITORING
  → Log EVERY request: who, what, when, how long
  → Centralized access logs across ALL services
  → Request tracing: assign correlation ID (Project 22 preview)
  → Metrics: latency, error rate, throughput per route

Responsibility 8: CIRCUIT BREAKING (Project 21 preview)
  → If backend service is failing → stop sending traffic
  → Return cached response or error immediately
  → Prevents cascade failures
  → Auto-retry when backend recovers

  Just route by URL path to backend services.
  No auth, no rate limiting, no transformation.

  location /api/users/ {
      proxy_pass http://user-service:8080/;
  }
  location /api/urls/ {
      proxy_pass http://url-service:8081/;

    → Simplest possible solution

    → Battle-tested, production-ready
  
  Cons:
    → No auth logic (each service still handles it)
    → No rate limiting (need separate solution)
    → No request transformation
    → Limited programmability

  BEST FOR: Simple routing with minimal logic

✅ WHAT SHOULD DO

1. IMPLEMENT middleware as a pipeline (chain of responsibility)
   → Each middleware is independent
   → Each can short-circuit (return error, skip rest)
   → Easy to add/remove/reorder middleware
   → Same pattern as Express.js, Django, Flask middleware

2. CENTRALIZE authentication at the gateway
   → Validate JWT/API key ONCE at the gateway
   → Pass user info to backends via internal headers
   → X-User-Id: 42, X-User-Role: admin
   → Backends TRUST these headers (gateway is trusted)
   → NEVER validate auth again in backend services

3. ASSIGN correlation IDs to every request
   → Generate unique ID at gateway entry
   → Pass to all backend calls: X-Correlation-Id: abc-123
   → Log with this ID everywhere
   → When debugging: search ALL logs by correlation ID
   → This is the FOUNDATION of distributed tracing (Project 22)

4. IMPLEMENT per-route configuration
   → Different rate limits per route
   → Different auth requirements per route
   → Different caching policies per route
   → /api/urls/* → rate limit 100/min, cache 60s
   → /api/payments/* → rate limit 10/min, no cache, strict auth

5. STRIP internal paths before forwarding
   → Client calls: /api/urls/shorten
   → Gateway routes to URL service
   → Forward as: /shorten (strip the /api/urls prefix)
   → Backend service doesn't know it's behind a gateway

6. ADD standard response headers
   → X-Request-Id: correlation ID
   → X-Response-Time: 45ms
   → X-Rate-Limit-Remaining: 95
   → X-Gateway-Version: 2.1.0
   → Consistent across ALL backend responses

7. IMPLEMENT graceful degradation
   → Backend down → return cached response if available
   → Backend slow → timeout and return 504
   → All backends down → return minimal error page
   → Gateway itself must NEVER crash

❌ WHAT SHOULD NOT DO

1. NEVER put business logic in the gateway
   → Gateway routes, authenticates, rate-limits
   → Gateway does NOT process orders, create users, etc.
   → Business logic belongs in backend services
   → Gateway is a TRAFFIC COP, not a FACTORY

2. NEVER make the gateway a single point of failure
   → Run MULTIPLE gateway instances
   → Load balancer in front of gateways (yes, LB → Gateway → LB → Service)
   → Each gateway instance is stateless (Project 8)
   → Shared state (rate limits, cache) in external store (Redis)

3. NEVER forward client auth tokens to backends
   → Client sends JWT to gateway
   → Gateway validates and EXTRACTS user info
   → Gateway sends user info via INTERNAL headers
   → Backend NEVER sees the raw JWT
   → Why: if backend is compromised, attacker can't reuse tokens

4. NEVER cache POST/PUT/DELETE responses
   → Only cache GET (safe, idempotent)
   → POST creates data → caching would serve stale creates
   → DELETE removes data → caching would pretend it still exists

5. NEVER ignore gateway latency overhead
   → Gateway adds 1-5ms per request
   → For a 10ms backend response, that's 10-50% overhead
   → Optimize gateway code aggressively
   → Don't do heavy computation in gateway middleware

6. NEVER expose internal service topology to clients
   → Client should NOT know backend service names or ports
   → Error messages should NOT reveal "user-service:8080 unreachable"
   → Return generic: "Service temporarily unavailable"

7. NEVER skip rate limiting on internal endpoints
   → /api/admin/* still needs rate limiting
   → Internal tools can accidentally DDoS your own services
   → Rate limit EVERYTHING, adjust limits per route

🚫 WHEN NOT TO USE

1. DON'T use a gateway for service-to-service communication
   → Gateway is for EXTERNAL → INTERNAL traffic
   → Service A calling Service B should go DIRECTLY
   → Adding gateway hop for internal calls = unnecessary latency
   → For internal: use service discovery (Project 20)

2. DON'T use a gateway when you have ONE monolithic backend
   → Gateway adds complexity for zero benefit
   → Just put auth and rate limiting in your monolith
   → Gateway makes sense when you have MULTIPLE services

3. DON'T use a gateway for WebSocket/streaming heavy workloads
   → Gateways add latency to every message
   → WebSocket connections are long-lived → gateway holds resources
   → For WebSocket: direct connection with auth at handshake
   → Or use a specialized WebSocket gateway

4. DON'T use a gateway as a service mesh
   → Gateway handles NORTH-SOUTH traffic (external → internal)
   → Service mesh handles EAST-WEST traffic (service → service)
   → They're complementary, not interchangeable

⚙️ HOW IT ONLY WORKS (INTERNAL MECHANICS)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
MECHANIC 1: MIDDLEWARE PIPELINE (Chain Pattern)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  CONCEPT: Each middleware is a function that:
    → Receives the request
    → Can MODIFY the request
    → Can SHORT-CIRCUIT (return response immediately)
    → Can PASS to the next middleware
    → Can MODIFY the response on the way back

  EXECUTION ORDER:

  Request → M1 → M2 → M3 → M4 → Backend
                                     │
  Response ← M1 ← M2 ← M3 ← M4 ← ─┘

  If M2 short-circuits (e.g., rate limit exceeded):

  Request → M1 → M2 → ✋ STOP
                  │
  Response ← M1 ← M2 (returns 429)

  M3 and M4 never execute.
  Backend never receives the request.
  This SAVES backend resources.

  IMPLEMENTATION:

  class Middleware:
      def process_request(self, request, context):
          # Return None to continue, or Response to short-circuit
          return None
      
      def process_response(self, response, context):
          # Modify response on the way back
          return response

  class Pipeline:
      def __init__(self, middlewares):
          self.middlewares = middlewares
      
      def execute(self, request):
          context = {}
          
          # Forward pass (request)
          for mw in self.middlewares:
              result = mw.process_request(request, context)
              if result is not None:
                  # Short circuit — run response pass on executed MWs only
                  return result
          
          # Forward to backend
          response = forward_to_backend(request)
          
          # Backward pass (response)
          for mw in reversed(self.middlewares):
              response = mw.process_response(response, context)
          
          return response

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: Gateway Becomes Bottleneck
  → ALL traffic goes through gateway
  → Gateway CPU maxes out at 50K req/sec
  → Adding more backends doesn't help (gateway is the limit)
  → FIX: Run multiple gateway instances behind a load balancer
         Keep gateway logic LIGHTWEIGHT
         Offload heavy work to backends

PITFALL 2: Single Point of Failure
  → One gateway instance → it dies → entire system down
  → FIX: Multiple instances (minimum 3)
         Health checks on gateway itself (meta-monitoring)
         Stateless design → any instance handles any request
         Rate limit state in Redis (shared, not in-process)

PITFALL 3: Configuration Drift
  → Routes configured manually → someone forgets one service
  → New service deployed → not added to gateway routes
  → FIX: Dynamic route discovery from service registry (Project 20)
         Or configuration-as-code with automated deployment
         Health checks verify route validity

PITFALL 4: Auth Token Leaking to Backends
  → Gateway validates JWT but forwards it to backend
  → Backend has a vulnerability → attacker extracts JWT
  → Can now impersonate any user across ALL services
  → FIX: Strip Authorization header at gateway
         Only forward extracted claims via internal headers

PITFALL 5: Cascading Timeouts
  → Client timeout: 30 seconds
  → Gateway timeout: 60 seconds (LONGER than client)
  → Client gives up at 30s, gateway still waiting
  → Gateway holds resources for dead requests
  → FIX: Gateway timeout < client timeout
         Typically: gateway timeout = 10-15 seconds

PITFALL 6: Cache Poisoning
  → Attacker sends request with crafted headers
  → Gateway caches the poisoned response
  → All subsequent users get the poisoned response
  → FIX: Sanitize cache keys
         Only cache based on known, validated parameters
         Never include raw user input in cache keys

PITFALL 7: Rate Limiter Memory Leak
  → Storing rate limit counters per IP in memory
  → 1 million unique IPs → 1 million counters
  → Memory grows unbounded
  → FIX: Use Redis with TTL for rate limit counters
         Or fixed-size sliding window with cleanup

class RateLimiterMiddleware(Middleware):
    """
    Token bucket rate limiter per client IP.
    From Project 9: now centralized at the gateway.
    Applied BEFORE auth — even unauthenticated abuse is blocked.
    """
    
    def __init__(self, default_limit=100, window_sec=60):
        super().__init__("rate_limiter")
        self.default_limit = default_limit
        self.window_sec = window_sec
        self.buckets = {}  # ip → {"tokens": [...timestamps]}
        self.lock = threading.Lock()
        
        # Per-route overrides
        self.route_limits = {}
    
    def set_route_limit(self, path_prefix, limit_per_minute):
        """Set custom rate limit for specific routes"""
        self.route_limits[path_prefix] = limit_per_minute
    
    def _get_limit(self, path):
        """Get rate limit for a given path"""
        for prefix, limit in sorted(self.route_limits.items(),
                                     key=lambda x: len(x[0]), reverse=True):
            if path.startswith(prefix):
                return limit
        return self.default_limit
    
    def _is_allowed(self, ip, path):
        """Check if request is within rate limit"""
        limit = self._get_limit(path)
        now = time.time()
        
        with self.lock:
            key = f"{ip}:{path.split('/')[1] if '/' in path else 'root'}"
            
            if key not in self.buckets:
                self.buckets[key] = []
            
            # Remove timestamps outside window
            self.buckets[key] = [
                t for t in self.buckets[key]
                if now - t < self.window_sec
            ]
            
            if len(self.buckets[key]) >= limit:
                remaining = 0
                reset_at = self.buckets[key][0] + self.window_sec
                return False, remaining, reset_at
            
            self.buckets[key].append(now)
            remaining = limit - len(self.buckets[key])
            return True, remaining, now + self.window_sec
    
    def process_request(self, request, context):
        allowed, remaining, reset_at = self._is_allowed(
            request.client_ip, request.path
        )
        
        context["rate_limit_remaining"] = remaining
        
        if not allowed:
            return GatewayResponse(
                status_code=429,
                body={
                    "error": "Rate limit exceeded",
                    "message": "Too many requests. Please slow down.",
                    "retry_after_seconds": round(reset_at - time.time(), 1)
                },
                headers={
                    "Retry-After": str(int(reset_at - time.time())),
                    "X-Rate-Limit-Remaining": "0"
                }
            )
        
        return None  # Continue pipeline
    
    def process_response(self, request, response, context):
        remaining = context.get("rate_limit_remaining", "?")
        response.headers["X-Rate-Limit-Remaining"] = str(remaining)
        return response

class AuthMiddleware(Middleware):
    """
    JWT authentication at the gateway.
    
    From Project 8: Session/auth management.
    Now centralized: backends NEVER validate auth themselves.
    
    Supports:
      → JWT Bearer token
      → API Key (for service-to-service)
      → Public routes (no auth required)
    """
    
    def __init__(self, jwt_secret="gateway_secret_key_change_in_production"):
        super().__init__("authenticator")
        self.jwt_secret = jwt_secret
        
        # Routes that don't require authentication
        self.public_routes = set()
        
        # API keys for service-to-service
        self.api_keys = {}
    
    def add_public_route(self, path_prefix):
        """Mark a route as publicly accessible"""
        self.public_routes.add(path_prefix)
    
    def add_api_key(self, key, service_name, permissions=None):
        """Register an API key"""
        self.api_keys[key] = {
            "service": service_name,
            "permissions": permissions or ["read"]
        }
    
    def _is_public_route(self, path):
        """Check if route requires authentication"""
        for prefix in self.public_routes:
            if path.startswith(prefix):
                return True
        return False
    
    def _validate_jwt(self, token):
        """
        Validate JWT token and extract claims.
        
        JWT Structure: header.payload.signature
        Each part is Base64URL encoded.
        
        Simplified implementation for learning.
        Production: use PyJWT library.
        """
        try:
            parts = token.split('.')
            if len(parts) != 3:
                return None, "Invalid token format"
            
            # Decode payload (middle part)
            payload_b64 = parts[1]
            # Add padding if needed
            padding = 4 - len(payload_b64) % 4
            if padding != 4:
                payload_b64 += '=' * padding
            
            payload_json = base64.urlsafe_b64decode(payload_b64)
            payload = json.loads(payload_json)
            
            # Verify signature
            message = f"{parts[0]}.{parts[1]}"
            expected_sig = hmac.new(
                self.jwt_secret.encode('utf-8'),
                message.encode('utf-8'),
                hashlib.sha256
            ).digest()
            expected_sig_b64 = base64.urlsafe_b64encode(expected_sig).rstrip(b'=').decode()
            
            if not hmac.compare_digest(expected_sig_b64, parts[2]):
                return None, "Invalid signature"
            
            # Check expiration
            if "exp" in payload:
                if time.time() > payload["exp"]:
                    return None, "Token expired"
            
            return payload, None
            
        except Exception as e:
            return None, f"Token validation failed: {str(e)}"
    
    def _validate_api_key(self, key):
        """Validate API key"""
        if key in self.api_keys:
            return self.api_keys[key], None
        return None, "Invalid API key"
    
    def create_test_jwt(self, user_id, role="user", plan="free",
                        expires_hours=24):
        """Helper: create a JWT for testing"""
        header = base64.urlsafe_b64encode(
            json.dumps({"alg": "HS256", "typ": "JWT"}).encode()
        ).rstrip(b'=').decode()
        
        payload_data = {
            "user_id": user_id,
            "role": role,
            "plan": plan,
            "exp": int(time.time() + expires_hours * 3600),
            "iat": int(time.time())
        }
        payload = base64.urlsafe_b64encode(
            json.dumps(payload_data).encode()
        ).rstrip(b'=').decode()
        
        message = f"{header}.{payload}"
        signature = hmac.new(
            self.jwt_secret.encode('utf-8'),
            message.encode('utf-8'),
            hashlib.sha256
        ).digest()
        sig_b64 = base64.urlsafe_b64encode(signature).rstrip(b'=').decode()
        
        return f"{header}.{payload}.{sig_b64}"
    
    def process_request(self, request, context):
        # Public routes skip auth
        if self._is_public_route(request.path):
            request.authenticated = False
            context["auth_skipped"] = True
            return None
        
        # Check for Bearer token
        auth_header = request.headers.get("Authorization", "")
        api_key = request.headers.get("X-API-Key", "")
        
        if auth_header.startswith("Bearer "):
            token = auth_header[7:]
            payload, error = self._validate_jwt(token)
            
            if error:
                return GatewayResponse(
                    status_code=401,
                    body={"error": "Unauthorized", "detail": error}
                )
            
            # Extract user info and attach to request
            request.user_id = payload.get("user_id")
            request.user_role = payload.get("role", "user")
            request.user_plan = payload.get("plan", "free")
            request.authenticated = True
            context["auth_method"] = "jwt"
            
        elif api_key:
            key_info, error = self._validate_api_key(api_key)
            
            if error:
                return GatewayResponse(
                    status_code=401,
                    body={"error": "Unauthorized", "detail": error}
                )
            
            request.user_id = f"service:{key_info['service']}"
            request.user_role = "service"
            request.authenticated = True
            context["auth_method"] = "api_key"
            
        else:
            return GatewayResponse(
                status_code=401,
                body={
                    "error": "Unauthorized",
                    "detail": "Provide Bearer token or X-API-Key header"
                }
            )
        
        return None  # Continue pipeline

class LoggerMiddleware(Middleware):
    """
    Logs every request and response.
    Assigns correlation ID for distributed tracing (Project 22 preview).
    """
    
    def __init__(self, max_log_size=5000):
        super().__init__("logger")
        self.access_log = []
        self.lock = threading.Lock()
        self.max_log_size = max_log_size
    
    def process_request(self, request, context):
        context["log_start_time"] = time.time()
        context["correlation_id"] = request.correlation_id
        return None
    
    def process_response(self, request, response, context):
        elapsed_ms = round(
            (time.time() - context.get("log_start_time", request.start_time)) * 1000, 1
        )
        
        log_entry = {
            "timestamp": datetime.now().isoformat(),
            "correlation_id": request.correlation_id,
            "method": request.method,
            "path": request.path,
            "client_ip": request.client_ip,
            "user_id": request.user_id,
            "status": response.status_code,
            "latency_ms": elapsed_ms,
            "cached": response.from_cache,
            "backend": request.selected_instance,
            "service": request.target_service
        }
        
        with self.lock:
            self.access_log.append(log_entry)
            if len(self.access_log) > self.max_log_size:
                self.access_log = self.access_log[-self.max_log_size:]
        
        # Add tracing headers to response
        response.headers["X-Request-Id"] = request.correlation_id
        response.headers["X-Response-Time"] = f"{elapsed_ms}ms"
        response.total_latency_ms = elapsed_ms
        
        return response
    
    def get_recent_logs(self, limit=50):
        with self.lock:
            return list(reversed(self.access_log[-limit:]))
    
    def get_metrics(self):
        with self.lock:
            if not self.access_log:
                return {"total_requests": 0}
            
            total = len(self.access_log)
            errors = sum(1 for l in self.access_log if l["status"] >= 400)
            cached = sum(1 for l in self.access_log if l.get("cached"))
            latencies = [l["latency_ms"] for l in self.access_log]
            latencies.sort()
            
            # Per-service breakdown
            by_service = defaultdict(lambda: {"count": 0, "errors": 0})
            for l in self.access_log:
                svc = l.get("service") or "unknown"
                by_service[svc]["count"] += 1
                if l["status"] >= 400:
                    by_service[svc]["errors"] += 1
            
            return {
                "total_requests": total,
                "error_count": errors,
                "error_rate_pct": round(errors / total * 100, 2),
                "cache_hit_count": cached,
                "cache_hit_rate_pct": round(cached / total * 100, 2),
                "latency_p50_ms": latencies[int(total * 0.5)] if total else 0,
                "latency_p95_ms": latencies[int(total * 0.95)] if total else 0,
                "latency_p99_ms": latencies[int(total * 0.99)] if total else 0,
                "by_service": dict(by_service)
            }

class CacheMiddleware(Middleware):
    """
    Gateway-level response caching.
    From Project 5: Cache-aside pattern, now at the gateway.
    Only caches GET requests for configured routes.
    """
    
    def __init__(self, capacity=5000, default_ttl=60):
        super().__init__("cache")
        self.capacity = capacity
        self.default_ttl = default_ttl
        self.cache = OrderedDict()
        self.lock = threading.Lock()
        
        # Per-route cache config
        self.route_config = {}
        
        # Metrics
        self.hits = 0
        self.misses = 0
    
    def configure_route(self, path_prefix, ttl_seconds, vary_by_user=False):
        """Configure caching for a route"""
        self.route_config[path_prefix] = {
            "ttl": ttl_seconds,
            "vary_by_user": vary_by_user
        }
    
    def _get_route_config(self, path):
        for prefix in sorted(self.route_config.keys(),
                             key=len, reverse=True):
            if path.startswith(prefix):
                return self.route_config[prefix]
        return None
    
    def _build_cache_key(self, request, config):
        """Build cache key from request"""
        parts = [request.method, request.path]
        
        # Include sorted query params
        if request.query_params:
            sorted_params = sorted(request.query_params.items())
            parts.append(str(sorted_params))
        
        # Include user if vary_by_user
        if config.get("vary_by_user") and request.user_id:
            parts.append(f"user:{request.user_id}")
        
        key_str = "|".join(parts)
        return hashlib.md5(key_str.encode()).hexdigest()
    
    def process_request(self, request, context):
        # Only cache GET requests
        if request.method != "GET":
            context["cache_skip"] = True
            return None
        
        config = self._get_route_config(request.path)
        if not config:
            context["cache_skip"] = True
            return None
        
        cache_key = self._build_cache_key(request, config)
        context["cache_key"] = cache_key
        context["cache_config"] = config
        
        with self.lock:
            if cache_key in self.cache:
                entry = self.cache[cache_key]
                if time.time() < entry["expires_at"]:
                    self.cache.move_to_end(cache_key)
                    self.hits += 1
                    
                    response = GatewayResponse(
                        status_code=entry["status_code"],
                        body=entry["body"],
                        headers=dict(entry.get("headers", {}))
                    )
                    response.from_cache = True
                    response.headers["X-Cache"] = "HIT"
                    return response
                else:
                    del self.cache[cache_key]
        
        self.misses += 1
        return None
    
    def process_response(self, request, response, context):
        if context.get("cache_skip"):
            response.headers["X-Cache"] = "SKIP"
            return response
        
        if response.from_cache:
            return response
        
        response.headers["X-Cache"] = "MISS"
        
        # Only cache successful responses
        if response.status_code != 200:
            return response
        
        cache_key = context.get("cache_key")
        config = context.get("cache_config")
        
        if cache_key and config:
            ttl = config.get("ttl", self.default_ttl)
            
            with self.lock:
                self.cache[cache_key] = {
                    "status_code": response.status_code,
                    "body": response.body,
                    "headers": dict(response.headers),
                    "expires_at": time.time() + ttl,
                    "cached_at": datetime.now().isoformat()
                }
                
                while len(self.cache) > self.capacity:
                    self.cache.popitem(last=False)
        
        return response
    
    def get_stats(self):
        total = self.hits + self.misses
        return {
            "size": len(self.cache),
            "capacity": self.capacity,
            "hits": self.hits,
            "misses": self.misses,
            "hit_rate_pct": round(
                self.hits / total * 100, 2
            ) if total > 0 else 0
        }

class RouteResolverMiddleware(Middleware):
    def __init__(self):
        super().__init__("router")
        self.routes = []
        self.service_instances = {}
        self.round_robin = defaultdict(int)
        self.lock = threading.Lock()

    def add_route(self, path_prefix, service_name, strip_prefix=True,
                  methods=None, require_auth=True):
        self.routes.append({
            "path_prefix": path_prefix, "service_name": service_name,
            "strip_prefix": strip_prefix, "methods": methods,
            "require_auth": require_auth
        })
        self.routes.sort(key=lambda r: len(r["path_prefix"]), reverse=True)

    def register_instances(self, service_name, instances):
        self.service_instances[service_name] = list(instances)

    def _match_route(self, method, path):
        for route in self.routes:
            if path.startswith(route["path_prefix"]):
                if route["methods"] and method not in route["methods"]:
                    continue
                return route
        return None

    def _select_instance(self, service_name):
        instances = self.service_instances.get(service_name, [])
        if not instances:
            return None
        with self.lock:
            idx = self.round_robin[service_name] % len(instances)
            self.round_robin[service_name] += 1
        return instances[idx]

    def process_request(self, request, context):
        route = self._match_route(request.method, request.path)
        if not route:
            return GatewayResponse(
                status_code=404,
                body={"error": "Not Found",
                      "detail": f"No route for {request.method} {request.path}",
                      "routes": [r["path_prefix"] for r in self.routes]}
            )

        service_name = route["service_name"]

        if route["strip_prefix"]:
            target_path = request.path[len(route["path_prefix"]):]
            if not target_path.startswith("/"):
                target_path = "/" + target_path
        else:
            target_path = request.path

        instance = self._select_instance(service_name)
        if not instance:
            return GatewayResponse(
                status_code=503,
                body={"error": "Service Unavailable",
                      "detail": f"No healthy instances for '{service_name}'"}
            )

        request.target_service = service_name
        request.target_path = target_path
        request.selected_instance = instance
        context["route"] = route
        return None

class CircuitBreakerMiddleware(Middleware):
    """
    Simple circuit breaker per backend service.
    States: CLOSED (normal) → OPEN (failing) → HALF_OPEN (testing)
    Full implementation in Project 21.
    """

    CLOSED = "closed"
    OPEN = "open"
    HALF_OPEN = "half_open"

    def __init__(self, failure_threshold=5, recovery_timeout=30):
        super().__init__("circuit_breaker")
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.circuits = {}  # service_name → circuit state
        self.lock = threading.Lock()

    def _get_circuit(self, service_name):
        if service_name not in self.circuits:
            self.circuits[service_name] = {
                "state": self.CLOSED,
                "failure_count": 0,
                "last_failure_time": 0,
                "success_count": 0
            }
        return self.circuits[service_name]

    def process_request(self, request, context):
        if not request.target_service:
            return None

        with self.lock:
            circuit = self._get_circuit(request.target_service)

            if circuit["state"] == self.OPEN:
                if time.time() - circuit["last_failure_time"] > self.recovery_timeout:
                    circuit["state"] = self.HALF_OPEN
                    context["circuit_state"] = self.HALF_OPEN
                    return None  # Allow one test request
                else:
                    return GatewayResponse(
                        status_code=503,
                        body={
                            "error": "Service Unavailable",
                            "detail": f"Circuit breaker OPEN for {request.target_service}",
                            "retry_after": round(
                                self.recovery_timeout -
                                (time.time() - circuit["last_failure_time"]), 1
                            )
                        },
                        headers={"Retry-After": str(self.recovery_timeout)}
                    )

            context["circuit_state"] = circuit["state"]
            return None

    def record_success(self, service_name):
        with self.lock:
            circuit = self._get_circuit(service_name)
            circuit["failure_count"] = 0
            circuit["success_count"] += 1
            if circuit["state"] == self.HALF_OPEN:
                circuit["state"] = self.CLOSED

    def record_failure(self, service_name):
        with self.lock:
            circuit = self._get_circuit(service_name)
            circuit["failure_count"] += 1
            circuit["last_failure_time"] = time.time()

            if circuit["state"] == self.HALF_OPEN:
                circuit["state"] = self.OPEN
            elif circuit["failure_count"] >= self.failure_threshold:
                circuit["state"] = self.OPEN

    def get_all_states(self):
        with self.lock:
            return {
                name: {
                    "state": c["state"],
                    "failures": c["failure_count"],
                    "successes": c["success_count"]
                }
                for name, c in self.circuits.items()
            }

class TransformMiddleware(Middleware):
    """
    Transforms requests before forwarding and responses before returning.
    → Adds internal headers (user info from auth)
    → Strips sensitive headers
    → Adds standard response headers
    """

    def __init__(self, gateway_version="1.0.0"):
        super().__init__("transformer")
        self.gateway_version = gateway_version

    def process_request(self, request, context):
        # Add authenticated user info as internal headers
        if request.authenticated:
            request.headers["X-User-Id"] = str(request.user_id)
            request.headers["X-User-Role"] = str(request.user_role)
            request.headers["X-User-Plan"] = str(request.user_plan or "free")
            request.headers["X-Auth-Verified"] = "true"

        # Add correlation ID for distributed tracing
        request.headers["X-Correlation-Id"] = request.correlation_id

        # Strip client auth header (backend shouldn't see raw tokens)
        request.headers.pop("Authorization", None)
        request.headers.pop("X-API-Key", None)

        return None

    def process_response(self, request, response, context):
        response.headers["X-Gateway-Version"] = self.gateway_version
        response.headers["X-Powered-By"] = "APIGateway"
        response.headers["Access-Control-Allow-Origin"] = "*"
        response.headers["Access-Control-Allow-Methods"] = "GET, POST, PUT, DELETE, OPTIONS"
        response.headers["Access-Control-Allow-Headers"] = "Authorization, Content-Type, X-API-Key"
        return response

class APIGateway:
    """
    The complete API Gateway.
    Orchestrates the middleware pipeline and forwards to backends.
    """

    def __init__(self):
        # Initialize all middleware
        self.rate_limiter = RateLimiterMiddleware(default_limit=100, window_sec=60)
        self.auth = AuthMiddleware()
        self.logger = LoggerMiddleware()
        self.cache = CacheMiddleware()
        self.router = RouteResolverMiddleware()
        self.circuit_breaker = CircuitBreakerMiddleware(failure_threshold=5,
                                                         recovery_timeout=30)
        self.transformer = TransformMiddleware()

        # The middleware pipeline — ORDER MATTERS
        self.pipeline = [
            self.rate_limiter,     # 1. Block abusers first
            self.auth,             # 2. Verify identity
            self.logger,           # 3. Start logging
            self.cache,            # 4. Return cached if available
            self.router,           # 5. Resolve backend route
            self.transformer,      # 6. Transform request
            self.circuit_breaker,  # 7. Check circuit state
        ]

        # Track which middleware executed (for proper response pass)
        self.backend_timeout = 10  # seconds

    def process(self, request):
        """
        Run the full middleware pipeline.

        Forward pass: each middleware processes the request
        If any middleware returns a response → short circuit
        Otherwise → forward to backend
        Backward pass: each middleware processes the response (reverse order)
        """
        context = {}
        executed_middleware = []

        # ── FORWARD PASS ──
        for mw in self.pipeline:
            executed_middleware.append(mw)
            result = mw.process_request(request, context)

            if result is not None:
                # Short circuit — run response pass on executed middleware only
                response = result
                for rmw in reversed(executed_middleware):
                    response = rmw.process_response(request, response, context)
                return response

        # ── FORWARD TO BACKEND ──
        response = self._forward_to_backend(request)

        # Record success/failure for circuit breaker
        if request.target_service:
            if response.status_code >= 500:
                self.circuit_breaker.record_failure(request.target_service)
            else:
                self.circuit_breaker.record_success(request.target_service)

        # ── BACKWARD PASS ──
        for mw in reversed(executed_middleware):
            response = mw.process_response(request, response, context)

        return response

    def _forward_to_backend(self, request):
        """Forward the request to the selected backend instance"""
        instance = request.selected_instance
        target_path = request.target_path or request.path

        if not instance:
            return GatewayResponse(
                status_code=502,
                body={"error": "Bad Gateway", "detail": "No backend instance selected"}
            )

        # Build backend URL
        backend_url = f"http://{instance}{target_path}"

        # Build query string
        if request.query_params:
            qs = "&".join(f"{k}={v}" for k, v in request.query_params.items())
            backend_url += f"?{qs}"

        try:
            # Prepare request body
            data = None
            if request.body:
                data = json.dumps(request.body).encode('utf-8')

            req = urllib.request.Request(
                backend_url, data=data, method=request.method
            )

            # Forward relevant headers
            for key, value in request.headers.items():
                if key.lower() not in ('host', 'content-length'):
                    req.add_header(key, str(value))

            if data:
                req.add_header('Content-Type', 'application/json')

            # Execute with timeout
            start = time.time()
            resp = urllib.request.urlopen(req, timeout=self.backend_timeout)
            backend_latency = round((time.time() - start) * 1000, 1)

            # Parse response
            resp_body = resp.read().decode('utf-8')
            try:
                resp_data = json.loads(resp_body)
            except json.JSONDecodeError:
                resp_data = {"raw": resp_body}

            response = GatewayResponse(
                status_code=resp.status,
                body=resp_data,
                headers={k: v for k, v in resp.headers.items()}
            )
            response.backend_latency_ms = backend_latency
            response.headers["X-Backend-Instance"] = instance
            response.headers["X-Backend-Latency"] = f"{backend_latency}ms"

            return response

        except urllib.error.HTTPError as e:
            backend_latency = round((time.time() - start) * 1000, 1)
            try:
                error_body = json.loads(e.read().decode('utf-8'))
            except Exception:
                error_body = {"error": str(e)}

            response = GatewayResponse(
                status_code=e.code, body=error_body
            )
            response.backend_latency_ms = backend_latency
            return response

        except urllib.error.URLError as e:
            return GatewayResponse(
                status_code=502,
                body={"error": "Bad Gateway",
                      "detail": f"Backend unreachable: {str(e.reason)}"}
            )

        except Exception as e:
            return GatewayResponse(
                status_code=502,
                body={"error": "Bad Gateway", "detail": str(e)}
            )

    # ================================================================

    # ================================================================

    def get_health(self):
        return {
            "status": "healthy",
            "service": "api-gateway",
            "timestamp": datetime.now().isoformat(),
            "circuits": self.circuit_breaker.get_all_states(),
            "cache": self.cache.get_stats()
        }

    def get_metrics(self):
        return {
            "gateway_metrics": self.logger.get_metrics(),
            "cache_stats": self.cache.get_stats(),
            "circuit_states": self.circuit_breaker.get_all_states()
        }

                "auth_required": r["require_auth"],

            }
            for r in self.router.routes
        ]

class GatewayHTTPHandler(BaseHTTPRequestHandler):

    def _build_gateway_request(self):
        parsed = urlparse(self.path)
        query = {}
        for k, v in parse_qs(parsed.query).items():
            query[k] = v[0]

        headers = {k: v for k, v in self.headers.items()}

        body = None
        content_length = int(self.headers.get('Content-Length', 0))
        if content_length > 0:
            raw = self.rfile.read(content_length).decode('utf-8')
            try:
                body = json.loads(raw)
            except json.JSONDecodeError:
                body = {"raw": raw}

        forwarded = self.headers.get('X-Forwarded-For')
        client_ip = forwarded.split(',')[0].strip() if forwarded else self.client_address[0]

        return GatewayRequest(
            method=self.command,
            path=parsed.path,
            headers=headers,
            body=body,
            query_params=query,
            client_ip=client_ip
        )

    def _send_gateway_response(self, response):
        self.send_response(response.status_code)
        self.send_header('Content-Type', 'application/json')
        for k, v in response.headers.items():
            if k.lower() not in ('content-type', 'content-length',
                                  'transfer-encoding'):
                self.send_header(k, str(v))
        body = json.dumps(response.body, indent=2)
        self.send_header('Content-Length', str(len(body)))
        self.end_headers()
        self.wfile.write(body.encode('utf-8'))

    def _handle_request(self):
        path = urlparse(self.path).path

        # Gateway management endpoints (not routed to backends)
        if path == '/gateway/health':
            self._send_gateway_response(GatewayResponse(200, gateway.get_health()))
            return
        if path == '/gateway/metrics':
            self._send_gateway_response(GatewayResponse(200, gateway.get_metrics()))
            return
        if path == '/gateway/routes':
            self._send_gateway_response(GatewayResponse(200, {"routes": gateway.get_routes()}))
            return
        if path == '/gateway/logs':
            logs = gateway.logger.get_recent_logs(limit=30)
            self._send_gateway_response(GatewayResponse(200, {"logs": logs}))
            return

        # Process through the full pipeline
        gw_request = self._build_gateway_request()
        gw_response = gateway.process(gw_request)
        self._send_gateway_response(gw_response)

    def do_GET(self):
        self._handle_request()

    def do_POST(self):
        self._handle_request()

    def do_PUT(self):
        self._handle_request()

    def do_DELETE(self):
        self._handle_request()

    def do_OPTIONS(self):
        self.send_response(200)
        self.send_header('Access-Control-Allow-Origin', '*')
        self.send_header('Access-Control-Allow-Methods',
                         'GET, POST, PUT, DELETE, OPTIONS')
        self.send_header('Access-Control-Allow-Headers',
                         'Authorization, Content-Type, X-API-Key')
        self.end_headers()

    def log_message(self, format, *args):
        pass  # Suppress default logging (we have our own)

        r = urllib.request.Request(url, data=data, method=method)
        if data:
            r.add_header('Content-Type', 'application/json')
        if headers:
            for k, v in headers.items():
                r.add_header(k, v)
        try:
            resp = urllib.request.urlopen(r, timeout=5)
            result = json.loads(resp.read().decode())
            resp_headers = {k: v for k, v in resp.headers.items()
                           if k.lower().startswith('x-')}
            return {"status": resp.status, "body": result,
                    "gw_headers": resp_headers}
        except urllib.error.HTTPError as e:
            try:
                result = json.loads(e.read().decode())
            except Exception:
                result = {"error": str(e)}
            return {"status": e.code, "body": result, "gw_headers": {}}
        except Exception as e:
            return {"status": 0, "body": {"error": str(e)}, "gw_headers": {}}

  → New instance spins up → gateway doesn't know about it
  → Instance dies → gateway still sends traffic to it
  → We deploy 10 new instances → must manually update gateway config
  → In containers (Docker/K8s): IP addresses change on EVERY restart
