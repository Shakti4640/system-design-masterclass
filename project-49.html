📌 WHAT COULD BE DONE

MODEL SERVING PATTERN OPTIONS:

  Option A: Embedded Model (In-Process)
    → Model loaded directly into application process
    → Prediction = function call (no network hop)
    → Pro: Lowest latency (<1ms)
    → Con: Application restart to update model
    → Con: Model size limited by application memory
    → Used for: Small models, edge devices

  Option B: Model-as-a-Service (Dedicated Server)
    → Separate server process for each model
    → Application calls model server via RPC/HTTP
    → Pro: Independent scaling, updates without app restart
    → Con: Network hop adds 1-5ms
    → Used for: Most production ML systems
    → THIS IS OUR PRIMARY APPROACH

  Option C: Serverless Inference
    → Model loaded on-demand per request
    → Cold start: 1-30 seconds (loading model)
    → Pro: Pay only for inference time
    → Con: Cold start latency is terrible
    → Used for: Low-traffic, batch-oriented models

  Option D: Edge Inference
    → Model runs on client device (phone, browser)
    → No server call needed
    → Pro: Zero network latency, works offline
    → Con: Limited model size, hard to update
    → Used for: On-device ML (keyboard prediction, face detection)

BATCHING STRATEGIES:

  Option A: No Batching (Single Request)
    → Each request processed individually
    → GPU utilization: ~5-20% (GPU is idle between requests)
    → Latency: lowest for individual requests
    → Throughput: very low

  Option B: Static Batching
    → Wait for N requests, process together
    → GPU utilization: high
    → Latency: higher (waiting to fill batch)
    → Throughput: high

  Option C: Dynamic Batching (Adaptive)
    → Collect requests for up to T milliseconds OR until N requests
    → Whichever comes first → process batch
    → Balance latency and throughput dynamically
    → Used by: TensorFlow Serving, Triton Inference Server
    → THIS IS WHAT WE IMPLEMENT

  Option D: Continuous Batching
    → For autoregressive models (LLMs)
    → Don't wait for entire batch to finish
    → New requests join the batch mid-inference
    → Used by: vLLM, TGI for large language models

✅ WHAT SHOULD DO

1. ALWAYS SEPARATE MODEL ARTIFACTS FROM SERVING CODE
   → Model = data (weights, parameters)
   → Server = code (loading, inference, batching)
   → Update model without redeploying server
   → Store models in object storage (S3/GCS)
   → Server pulls model on startup or on-demand

2. ALWAYS VERSION EVERYTHING
   → Model version: v1, v2, v3
   → Feature schema version: what features the model expects
   → API version: prediction request/response format
   → Training data version: which dataset produced this model
   → Without versioning: "which model made this prediction?" → unanswerable

3. ALWAYS USE FEATURE STORE FOR CONSISTENCY
   → Training: features computed from historical data
   → Serving: features computed in real-time
   → If computation differs → "training-serving skew"
   → Feature store: ONE definition, used in both contexts
   → Most common source of ML bugs in production

4. ALWAYS CANARY BEFORE FULL DEPLOYMENT
   → Never go from 0% to 100% traffic at once
   → 0% → 1% → 5% → 25% → 50% → 100%
   → At each step: compare metrics against baseline
   → Auto-rollback if metrics degrade beyond threshold
   → This caught 30% of bad models at Google (per their papers)

5. ALWAYS HAVE A FALLBACK MODEL
   → Primary model fails (OOM, bug, timeout)
   → Fallback: simpler model that always works
   → Fallback can be: previous version, rule-based, default score
   → Circuit breaker pattern (Project 21): if primary fails N times → fallback

6. ALWAYS LOG PREDICTIONS WITH CONTEXT
   → Log: request_id, model_version, features_used, prediction, latency
   → This enables: debugging, auditing, retraining, A/B analysis
   → Delayed labels (user clicked/didn't click) join later
   → Without logging: cannot measure model quality in production

❌ WHAT SHOULD NOT DO

1. NEVER SERVE A MODEL WITHOUT OFFLINE VALIDATION
   → "It works on my laptop" is not sufficient
   → Run on golden dataset: known inputs → expected outputs
   → Compare metrics (AUC, RMSE, precision, recall) against threshold
   → If metrics below threshold → block deployment automatically

2. NEVER COMPUTE FEATURES DIFFERENTLY IN TRAINING AND SERVING
   → Training: "average user clicks in last 30 days"
     computed from batch SQL query → 47.3
   → Serving: "average user clicks in last 30 days"
     computed from streaming counter → 46.8
   → Difference of 0.5 → model predictions are WRONG
   → Feature store solves this: ONE computation, TWO access patterns

3. NEVER DEPLOY GPU MODELS WITHOUT BATCHING
   → GPU inference: 2ms for 1 request, 3ms for 32 requests
   → Without batching: 1 request at a time → GPU 95% idle
   → With batching: 32 requests at once → 10× throughput, minimal latency cost
   → GPU costs $2-10/hour → wasting GPU = burning money

4. NEVER IGNORE MODEL DRIFT
   → Model trained on January data
   → World changes in March (new product, pandemic, trend)
   → Model's predictions become increasingly wrong
   → But no crash, no error → SILENT FAILURE
   → Must monitor: prediction distribution, feature distribution, accuracy

5. NEVER HARDCODE MODEL PATHS OR VERSIONS
   → "model = load('/models/v2/model.pt')" → disaster
   → Use model registry: "model = registry.get('recommendation', stage='production')"
   → Deployment changes which version maps to 'production'
   → No code changes needed for model updates

6. NEVER SKIP SHADOW DEPLOYMENT FOR HIGH-STAKES MODELS
   → Fraud detection model: wrong prediction = millions lost
   → Shadow mode: new model receives traffic, predictions logged but NOT used
   → Compare shadow predictions with production model
   → Only promote after weeks of shadow validation

MECHANIC 2: FEATURE STORE — TRAINING-SERVING CONSISTENCY
═══════════════════════════════════════════════════════════

  THE PROBLEM (Training-Serving Skew):
  
    Training time:
      feature = SQL("SELECT avg(clicks) FROM events 
                     WHERE user_id=123 AND date > '2025-01-01'")
      → Result: 47.3
    
    Serving time:
      feature = redis.get("user_123_avg_clicks")
      → Result: 46.8  (different computation path!)
    
    Model trained on 47.3, served 46.8 → predictions are WRONG
    This is the #1 cause of ML bugs in production

  THE SOLUTION: Feature Store
  
    Feature Definition (written ONCE):
      name: "user_avg_clicks_30d"
      entity: "user_id"
      computation: "AVG(clicks) WHERE date > NOW() - 30 days"
      refresh: "hourly"
    
    Offline Store (for training):
      → Batch computation runs on data warehouse
      → Stores historical feature values
      → Training pipeline reads features from here
    
    Online Store (for serving):
      → Same computation, materialized into Redis/cache
      → Serving pipeline reads features from here
      → Updated by streaming pipeline (Project 43 pattern)
    
    SAME definition → SAME values → NO skew ✅

  Feature Lookup at Serving Time:
    Request: predict(user_id=123, item_id=456)
    
    Feature Store returns:
    {
      "user_avg_clicks_30d": 47.3,
      "user_signup_days": 365,
      "user_segment": "power_user",
      "item_category": "electronics",
      "item_price": 29.99,
      "item_avg_rating": 4.2
    }
    
    These features → model input vector → prediction

MECHANIC 3: DYNAMIC BATCHING — GPU EFFICIENCY
═══════════════════════════════════════════════════════════

  GPU Processing Model:
    Single request:  GPU inference = 2ms (GPU 5% utilized)
    Batch of 32:     GPU inference = 3ms (GPU 90% utilized)
    Batch of 128:    GPU inference = 8ms (GPU 95% utilized)
    
    Key insight: GPU parallelism means batching is nearly free!
    32 requests in 3ms vs 32 × 2ms = 64ms sequentially

  Dynamic Batching Algorithm:
    
    Config:
      max_batch_size = 32
      max_wait_time_ms = 5
    
    Loop:
      batch = []
      deadline = now() + max_wait_time_ms
      
      while len(batch) < max_batch_size AND now() < deadline:
        if request_queue.has_next():
          batch.append(request_queue.pop())
        else:
          sleep(0.1ms)  # Brief wait
      
      if batch:
        predictions = model.predict_batch(batch)
        for request, prediction in zip(batch, predictions):
          request.respond(prediction)
    
    Behavior:
      Low traffic (10 req/sec):
        → Batch fills slowly, hits timeout
        → Batch size: 1-3 (most requests served individually)
        → Latency: ~5ms (mostly waiting for timeout)
      
      High traffic (1000 req/sec):
        → Batch fills instantly
        → Batch size: 32 (full batch every 32ms)
        → Latency: ~3ms (GPU inference dominates)
      
      Adaptive: latency-optimal at low traffic,
                throughput-optimal at high traffic

MECHANIC 4: A/B TESTING — TRAFFIC ROUTING
═══════════════════════════════════════════════════════════

  Experiment Configuration:
    experiment: "rec_model_v3_test"
    control:    {model: "rec_v2", traffic: 95%}
    treatment:  {model: "rec_v3", traffic: 5%}
    metric:     "click_through_rate"
    duration:   7 days
    min_sample: 100,000 predictions per variant

  Routing Decision:
    For each request:
      bucket = hash(user_id + experiment_name) % 100
      if bucket < 5:
        route to rec_v3 (treatment)
      else:
        route to rec_v2 (control)
    
    WHY hash(user_id)?
      → Same user always gets same variant (consistency)
      → Different users randomly assigned (randomization)
      → No session-level switching (confusing UX)

  Statistical Analysis (after experiment):
    Control (rec_v2):   CTR = 2.10% ± 0.05%  (n=950,000)
    Treatment (rec_v3): CTR = 2.38% ± 0.12%  (n=50,000)
    
    Lift = +0.28% (13.3% relative improvement)
    p-value = 0.003 (statistically significant at p<0.05)
    
    → PROMOTE rec_v3 to production ✅

MECHANIC 5: MODEL DRIFT DETECTION
═══════════════════════════════════════════════════════════

  Types of Drift:

  1. DATA DRIFT (input distribution changes)
     → Training: avg user age = 32
     → Production (6 months later): avg user age = 28
     → Model never saw many 28-year-olds → predictions unreliable
     
     Detection: KL-divergence or PSI between training and serving
     distributions for each feature
     
     Alert threshold: PSI > 0.2 → significant drift

  2. CONCEPT DRIFT (relationship between input and output changes)
     → Training: "expensive items" = items > $100
     → After inflation: "expensive items" = items > $150
     → Model's price sensitivity is miscalibrated
     
     Detection: Track prediction accuracy over time
     If accuracy drops consistently → concept drift
     
  3. PREDICTION DRIFT (output distribution changes)
     → Training: avg prediction score = 0.35
     → Production: avg prediction score = 0.52
     → Model is systematically over-predicting
     
     Detection: Compare prediction distribution weekly
     
  Monitoring Pipeline:
    Every prediction → logged with features + prediction
    Hourly batch job → compute distribution statistics
    Compare with training distribution → flag drift
    Alert → data scientist → investigate → retrain if needed

🚨 COMMON PITFALLS AND FAILURE MODES

PITFALL 1: TRAINING-SERVING SKEW
  Scenario:
    → Training: feature "user_age" computed as float (28.0)
    → Serving: feature "user_age" computed as int (28)
    → Model expects float → receives int → silent type coercion
    → Predictions slightly off for every single request
    → Revenue loss: $2M before anyone notices
  
  Fix:
    → Feature store with strict schema validation
    → Integration tests: training features == serving features
    → Log feature values and compare distributions

PITFALL 2: MODEL LOADING OOM
  Scenario:
    → New model is 8GB (transformer)
    → Server has 16GB RAM, 4GB already used
    → Load model → 8GB → leaves 4GB for OS + requests
    → Under load: memory grows → OOM kill → server dies
    → All predictions fail for that server
  
  Fix:

class ModelRegistry:
    """
    Central registry for all ML models.
    Manages versions, stages, and lifecycle transitions.
    
    Similar to Docker Registry (Project 20) but for ML models.
    """

    def __init__(self):
        self.models: Dict[str, Dict[int, ModelVersion]] = defaultdict(dict)
        self.stage_index: Dict[str, Dict[ModelStage, List[int]]] = defaultdict(
            lambda: defaultdict(list)
        )
        self.transition_log: List[dict] = []
        self._lock = threading.Lock()

    def register_model(self, model_version: ModelVersion) -> dict:
        with self._lock:
            name = model_version.model_name
            version = model_version.version

            if version in self.models[name]:
                return {"success": False, "error": f"Version {version} already exists"}

            self.models[name][version] = model_version
            self.stage_index[name][model_version.stage].append(version)

            self.transition_log.append({
                "action": "REGISTER",
                "model": name,
                "version": version,
                "stage": model_version.stage.value,
                "timestamp": time.time()
            })

            return {"success": True, "model": name, "version": version}

    def transition_stage(self, model_name: str, version: int,
                         new_stage: ModelStage, reason: str = "") -> dict:
        with self._lock:
            if model_name not in self.models or version not in self.models[model_name]:
                return {"success": False, "error": "Model version not found"}

            mv = self.models[model_name][version]
            old_stage = mv.stage

            # Remove from old stage index
            if version in self.stage_index[model_name][old_stage]:
                self.stage_index[model_name][old_stage].remove(version)

            # If promoting to production, demote current production
            if new_stage == ModelStage.PRODUCTION:
                for v in list(self.stage_index[model_name][ModelStage.PRODUCTION]):
                    old_prod = self.models[model_name][v]
                    old_prod.stage = ModelStage.ARCHIVED
                    self.stage_index[model_name][ModelStage.PRODUCTION].remove(v)
                    self.stage_index[model_name][ModelStage.ARCHIVED].append(v)

            mv.stage = new_stage
            self.stage_index[model_name][new_stage].append(version)

            self.transition_log.append({
                "action": "TRANSITION",
                "model": model_name,
                "version": version,
                "from_stage": old_stage.value,
                "to_stage": new_stage.value,
                "reason": reason,
                "timestamp": time.time()
            })

            return {
                "success": True,
                "model": model_name,
                "version": version,
                "old_stage": old_stage.value,
                "new_stage": new_stage.value
            }

    def get_production_version(self, model_name: str) -> Optional[ModelVersion]:
        with self._lock:
            versions = self.stage_index.get(model_name, {}).get(ModelStage.PRODUCTION, [])
            if versions:
                return self.models[model_name][versions[-1]]
            return None

    def get_canary_version(self, model_name: str) -> Optional[ModelVersion]:
        with self._lock:
            versions = self.stage_index.get(model_name, {}).get(ModelStage.CANARY, [])
            if versions:
                return self.models[model_name][versions[-1]]
            return None

    def get_shadow_version(self, model_name: str) -> Optional[ModelVersion]:
        with self._lock:
            versions = self.stage_index.get(model_name, {}).get(ModelStage.SHADOW, [])
            if versions:
                return self.models[model_name][versions[-1]]
            return None

    def get_version(self, model_name: str, version: int) -> Optional[ModelVersion]:
        return self.models.get(model_name, {}).get(version)

    def list_models(self) -> List[dict]:
        with self._lock:
            result = []
            for name, versions in self.models.items():
                for v, mv in sorted(versions.items()):
                    result.append(mv.to_dict())
            return result

    def get_model_summary(self) -> dict:
        with self._lock:
            summary = {
                "total_models": len(self.models),
                "total_versions": sum(len(v) for v in self.models.values()),
                "by_stage": {}
            }
            for stage in ModelStage:
                count = sum(
                    len(stages.get(stage, []))
                    for stages in self.stage_index.values()
                )
                if count > 0:
                    summary["by_stage"][stage.value] = count
            return summary

class FeatureStore:
    """
    Provides consistent features for both training and serving.
    
    Online store: low-latency lookup (simulated Redis, Project 5/23)
    Offline store: batch features (simulated data warehouse)
    
    KEY GUARANTEE: Same feature definition → same value
    regardless of whether accessed from training or serving.
    """

    def __init__(self):
        # Online store (serving path, low-latency)
        self.online_store: Dict[str, Dict[str, Any]] = {}

        # Feature definitions
        self.feature_definitions: Dict[str, dict] = {}

        # Access tracking
        self.online_lookups = 0
        self.cache_hits = 0
        self._lock = threading.Lock()

    def register_feature(self, name: str, entity_type: str,
                         computation: str, dtype: str):
        self.feature_definitions[name] = {
            "name": name,
            "entity_type": entity_type,
            "computation": computation,
            "dtype": dtype,
            "registered_at": time.time()
        }

    def materialize_features(self, entity_id: str, features: Dict[str, Any]):
        """Write features to online store (called by feature pipeline)"""
        with self._lock:
            self.online_store[entity_id] = features

    def get_features(self, entity_id: str,
                     feature_names: List[str]) -> Dict[str, Any]:
        """Read features from online store (called during serving)"""
        with self._lock:
            self.online_lookups += 1
            stored = self.online_store.get(entity_id, {})

            result = {}
            for name in feature_names:
                if name in stored:
                    result[name] = stored[name]
                    self.cache_hits += 1
                else:
                    # Return default value if feature missing
                    defn = self.feature_definitions.get(name, {})
                    result[name] = self._default_for_dtype(defn.get("dtype", "float"))

            return result

    def _default_for_dtype(self, dtype: str) -> Any:
        defaults = {"float": 0.0, "int": 0, "string": "", "vector": []}
        return defaults.get(dtype, 0.0)

    def get_stats(self) -> dict:
        with self._lock:
            return {
                "entities_stored": len(self.online_store),
                "features_defined": len(self.feature_definitions),
                "online_lookups": self.online_lookups,
                "cache_hit_rate": round(
                    self.cache_hits / max(self.online_lookups, 1) * 100, 1
                )
            }

class SimulatedModel:
    """
    Simulates an ML model for prediction.
    
    In production: actual PyTorch/TF model loaded into memory.
    Here: parameterized function that simulates prediction behavior.
    """

    def __init__(self, model_version: ModelVersion, quality: float = 0.5,
                 latency_ms: float = 5.0, noise: float = 0.1):
        self.model_version = model_version
        self.quality = quality  # 0.0 = random, 1.0 = perfect
        self.base_latency_ms = latency_ms
        self.noise = noise
        self.is_loaded = True
        self.total_inferences = 0
        self.total_latency_ms = 0.0

    def predict(self, features: Dict[str, Any]) -> Tuple[float, float]:
        """
        Simulate model inference.
        Returns (prediction_score, confidence).
        """
        self.total_inferences += 1

        # Simulate inference latency
        latency = self.base_latency_ms + random.gauss(0, self.base_latency_ms * 0.1)
        latency = max(0.5, latency)
        self.total_latency_ms += latency

        # Simulate prediction based on features
        feature_signal = 0.0
        for name, value in features.items():
            if isinstance(value, (int, float)):
                feature_signal += value * 0.01

        # Quality determines how much signal vs noise
        signal = math.tanh(feature_signal) * self.quality
        noise_val = random.gauss(0, self.noise) * (1 - self.quality)
        raw_score = 0.5 + signal + noise_val

        # Clamp to [0, 1]
        prediction = max(0.0, min(1.0, raw_score))
        confidence = self.quality * 0.8 + random.uniform(0, 0.2)

        # Simulate the latency
        time.sleep(latency / 10000)  # Scaled down for simulation

        return prediction, confidence, latency

    def predict_batch(self, feature_list: List[Dict[str, Any]]) -> List[Tuple]:
        """
        Batch prediction — more efficient than individual predictions.
        GPU: batch of 32 takes ~1.5× time of single inference.
        """
        batch_size = len(feature_list)
        # Batch efficiency: sublinear latency growth
        batch_latency = self.base_latency_ms * (1.0 + 0.02 * batch_size)
        time.sleep(batch_latency / 10000)

        results = []
        for features in feature_list:
            self.total_inferences += 1
            feature_signal = sum(
                v * 0.01 for v in features.values()
                if isinstance(v, (int, float))
            )
            signal = math.tanh(feature_signal) * self.quality
            noise_val = random.gauss(0, self.noise) * (1 - self.quality)
            prediction = max(0.0, min(1.0, 0.5 + signal + noise_val))
            confidence = self.quality * 0.8 + random.uniform(0, 0.2)
            per_item_latency = batch_latency / batch_size

            self.total_latency_ms += per_item_latency
            results.append((prediction, confidence, per_item_latency))

        return results

    def get_stats(self) -> dict:
        avg_lat = (self.total_latency_ms / self.total_inferences
                   if self.total_inferences > 0 else 0)
        return {
            "model": self.model_version.model_name,
            "version": self.model_version.version,
            "total_inferences": self.total_inferences,
            "avg_latency_ms": round(avg_lat, 2),
            "quality": self.quality
        }

class ModelServer:
    """
    Serves predictions for loaded models.
    Implements dynamic batching for GPU efficiency.
    """

    def __init__(self, server_id: str, max_batch_size: int = 32,
                 max_wait_ms: float = 5.0):
        self.server_id = server_id
        self.max_batch_size = max_batch_size
        self.max_wait_ms = max_wait_ms

        self.loaded_models: Dict[str, SimulatedModel] = {}
        self.request_queue: deque = deque()
        self.total_requests = 0
        self.total_batches = 0
        self.batch_sizes: List[int] = []
        self.latencies: List[float] = []
        self._lock = threading.Lock()

    def load_model(self, model_version: ModelVersion, quality: float = 0.5,
                   latency_ms: float = 5.0):
        model_key = f"{model_version.model_name}_v{model_version.version}"
        self.loaded_models[model_key] = SimulatedModel(
            model_version, quality=quality, latency_ms=latency_ms
        )
        model_version.is_loaded = True

    def unload_model(self, model_name: str, version: int):
        model_key = f"{model_name}_v{version}"
        self.loaded_models.pop(model_key, None)

    def predict(self, model_name: str, version: int,
                features: Dict[str, Any]) -> Optional[Tuple]:
        model_key = f"{model_name}_v{version}"
        model = self.loaded_models.get(model_key)

        if not model:
            return None

        with self._lock:
            self.total_requests += 1

        prediction, confidence, latency = model.predict(features)

        with self._lock:
            self.latencies.append(latency)

        return prediction, confidence, latency

    def predict_batch_sync(self, model_name: str, version: int,
                           feature_list: List[Dict[str, Any]]) -> List[Tuple]:
        model_key = f"{model_name}_v{version}"
        model = self.loaded_models.get(model_key)

        if not model:
            return [(0.5, 0.0, 0.0)] * len(feature_list)

        with self._lock:
            self.total_requests += len(feature_list)
            self.total_batches += 1
            self.batch_sizes.append(len(feature_list))

        results = model.predict_batch(feature_list)

        with self._lock:
            for _, _, lat in results:
                self.latencies.append(lat)

        return results

    def get_stats(self) -> dict:
        with self._lock:
            lat_stats = {}
            if self.latencies:
                sorted_lat = sorted(self.latencies[-1000:])
                lat_stats = {
                    "avg_ms": round(statistics.mean(sorted_lat), 2),
                    "p50_ms": round(sorted_lat[len(sorted_lat) // 2], 2),
                    "p99_ms": round(sorted_lat[int(len(sorted_lat) * 0.99)], 2)
                }

            avg_batch = (statistics.mean(self.batch_sizes)
                         if self.batch_sizes else 0)

            return {
                "server_id": self.server_id,
                "loaded_models": len(self.loaded_models),
                "total_requests": self.total_requests,
                "total_batches": self.total_batches,
                "avg_batch_size": round(avg_batch, 1),
                "latency": lat_stats
            }

class TrafficRouter:
    """
    Routes prediction requests to appropriate model version.
    
    Supports:
      - Production routing (100% to production model)
      - A/B testing (split traffic between control and treatment)
      - Canary (small % to new version)
      - Shadow (new version receives traffic but results unused)
    
    Uses consistent hashing on entity_id for stable assignment
    (same user always gets same variant — Project 14 concept).
    """

    def __init__(self, registry: ModelRegistry):
        self.registry = registry
        self.experiments: Dict[str, Experiment] = {}
        self.routing_decisions: List[dict] = []
        self._lock = threading.Lock()

    def create_experiment(self, experiment: Experiment):
        with self._lock:
            self.experiments[experiment.name] = experiment

    def stop_experiment(self, experiment_name: str):
        with self._lock:
            if experiment_name in self.experiments:
                self.experiments[experiment_name].is_active = False

    def route(self, model_name: str, entity_id: str) -> dict:
        """
        Determine which model version to use for this request.
        Returns routing decision with version and experiment info.
        """
        with self._lock:
            # Check for active experiments on this model
            active_experiment = None
            for exp in self.experiments.values():
                if exp.model_name == model_name and exp.is_active:
                    active_experiment = exp
                    break

            if active_experiment:
                # Consistent hash for stable assignment
                hash_input = f"{entity_id}:{active_experiment.name}"
                hash_val = int(hashlib.md5(hash_input.encode()).hexdigest(), 16)
                bucket = hash_val % 100

                is_treatment = bucket < active_experiment.treatment_traffic_pct

                if active_experiment.experiment_type == "shadow":
                    # Shadow: always use control, but also run treatment
                    decision = {
                        "version": active_experiment.control_version,
                        "experiment": active_experiment.name,
                        "variant": "control",
                        "shadow_version": active_experiment.treatment_version,
                        "is_shadow": True
                    }
                elif is_treatment:
                    decision = {
                        "version": active_experiment.treatment_version,
                        "experiment": active_experiment.name,
                        "variant": "treatment"
                    }
                else:
                    decision = {
                        "version": active_experiment.control_version,
                        "experiment": active_experiment.name,
                        "variant": "control"
                    }

                self.routing_decisions.append(decision)
                return decision

            # No experiment: use production version
            prod = self.registry.get_production_version(model_name)
            if prod:
                return {
                    "version": prod.version,
                    "experiment": "",
                    "variant": "production"
                }

            return {"version": None, "experiment": "", "variant": "none"}

    def record_outcome(self, experiment_name: str, variant: str,
                       prediction: float, outcome: int):
        """Record outcome for experiment analysis"""
        with self._lock:
            exp = self.experiments.get(experiment_name)
            if not exp:
                return

            if variant == "control":
                exp.control_predictions.append(prediction)
                exp.control_outcomes.append(outcome)
            elif variant == "treatment":
                exp.treatment_predictions.append(prediction)
                exp.treatment_outcomes.append(outcome)

    def analyze_experiment(self, experiment_name: str) -> dict:
        """Statistical analysis of experiment results"""
        with self._lock:
            exp = self.experiments.get(experiment_name)
            if not exp:
                return {"error": "Experiment not found"}

            control_n = len(exp.control_outcomes)
            treatment_n = len(exp.treatment_outcomes)

            if control_n < 100 or treatment_n < 10:
                return {
                    "experiment": experiment_name,
                    "status": "INSUFFICIENT_DATA",
                    "control_n": control_n,
                    "treatment_n": treatment_n,
                    "min_required": 100
                }

            # Calculate conversion rates
            control_rate = sum(exp.control_outcomes) / max(control_n, 1)
            treatment_rate = sum(exp.treatment_outcomes) / max(treatment_n, 1)

            # Lift
            lift = treatment_rate - control_rate
            relative_lift = (lift / max(control_rate, 0.001)) * 100

            # Simplified statistical significance (z-test approximation)
            pooled_rate = (sum(exp.control_outcomes) + sum(exp.treatment_outcomes)) / \
                          max(control_n + treatment_n, 1)
            if pooled_rate > 0 and pooled_rate < 1:
                se = math.sqrt(pooled_rate * (1 - pooled_rate) *
                               (1 / max(control_n, 1) + 1 / max(treatment_n, 1)))
                z_score = lift / max(se, 0.0001)
                # Approximate p-value from z-score
                p_value = max(0.001, min(1.0, math.exp(-0.5 * z_score * z_score)))
            else:
                z_score = 0
                p_value = 1.0

            significant = p_value < 0.05
            recommendation = "PROMOTE" if significant and lift > 0 else \
                            "ROLLBACK" if significant and lift < 0 else "CONTINUE"

            # Average prediction scores
            control_avg_pred = statistics.mean(exp.control_predictions) if exp.control_predictions else 0
            treatment_avg_pred = statistics.mean(exp.treatment_predictions) if exp.treatment_predictions else 0

            return {
                "experiment": experiment_name,
                "type": exp.experiment_type,
                "status": "ACTIVE" if exp.is_active else "COMPLETED",
                "control": {
                    "version": exp.control_version,
                    "samples": control_n,
                    "conversion_rate": round(control_rate, 4),
                    "avg_prediction": round(control_avg_pred, 4)
                },
                "treatment": {
                    "version": exp.treatment_version,
                    "samples": treatment_n,
                    "conversion_rate": round(treatment_rate, 4),
                    "avg_prediction": round(treatment_avg_pred, 4)
                },
                "lift": round(lift, 4),
                "relative_lift_pct": round(relative_lift, 2),
                "p_value": round(p_value, 4),
                "statistically_significant": significant,
                "recommendation": recommendation
            }

class DriftDetector:
    """
    Monitors for data drift and prediction drift.
    
    Data Drift: Input feature distributions changing over time.
    Prediction Drift: Model output distribution shifting.
    Concept Drift: Relationship between inputs and outcomes changing.
    
    Uses PSI (Population Stability Index) for detection.
    PSI > 0.1: slight drift, PSI > 0.2: significant drift.
    """

    def __init__(self):
        self.reference_distributions: Dict[str, Dict[str, List[float]]] = {}
        self.current_windows: Dict[str, Dict[str, deque]] = defaultdict(
            lambda: defaultdict(lambda: deque(maxlen=5000))
        )
        self.prediction_windows: Dict[str, deque] = defaultdict(
            lambda: deque(maxlen=10000)
        )
        self.drift_alerts: List[dict] = []
        self._lock = threading.Lock()

    def set_reference(self, model_name: str, feature_name: str,
                      values: List[float]):
        """Set reference distribution (from training data)"""
        with self._lock:
            if model_name not in self.reference_distributions:
                self.reference_distributions[model_name] = {}
            self.reference_distributions[model_name][feature_name] = values

    def record_serving_features(self, model_name: str,
                                features: Dict[str, Any]):
        """Record feature values from serving for drift comparison"""
        with self._lock:
            for name, value in features.items():
                if isinstance(value, (int, float)):
                    self.current_windows[model_name][name].append(float(value))

    def record_prediction(self, model_name: str, prediction: float):
        """Record prediction value for prediction drift monitoring"""
        with self._lock:
            self.prediction_windows[model_name].append(prediction)

    def _compute_psi(self, reference: List[float],
                     current: List[float], bins: int = 10) -> float:
        """
        Population Stability Index (PSI).
        Measures distribution shift between reference and current.
        
        PSI < 0.1: no significant shift
        PSI 0.1-0.2: moderate shift
        PSI > 0.2: significant shift
        """
        if len(reference) < bins or len(current) < bins:
            return 0.0

        # Create bins from reference distribution
        min_val = min(min(reference), min(current))
        max_val = max(max(reference), max(current))

        if min_val == max_val:
            return 0.0

        bin_edges = [min_val + (max_val - min_val) * i / bins
                     for i in range(bins + 1)]

        def bin_counts(values, edges):
            counts = [0] * (len(edges) - 1)
            for v in values:
                for i in range(len(edges) - 1):
                    if edges[i] <= v < edges[i + 1] or (i == len(edges) - 2 and v == edges[i + 1]):
                        counts[i] += 1
                        break
            # Convert to proportions, avoid zero
            total = max(sum(counts), 1)
            return [max(c / total, 0.0001) for c in counts]

        ref_props = bin_counts(reference, bin_edges)
        cur_props = bin_counts(current, bin_edges)

        # PSI = sum((current - reference) × ln(current / reference))
        psi = 0.0
        for r, c in zip(ref_props, cur_props):
            psi += (c - r) * math.log(c / r)

        return round(abs(psi), 4)

    def check_drift(self, model_name: str) -> dict:
        """Check all features and predictions for drift"""
        with self._lock:
            results = {
                "model": model_name,
                "feature_drift": {},
                "prediction_drift": {},
                "alerts": []
            }

            ref_dist = self.reference_distributions.get(model_name, {})
            cur_windows = self.current_windows.get(model_name, {})

            for feature_name in ref_dist:
                if feature_name in cur_windows and len(cur_windows[feature_name]) > 100:
                    reference = ref_dist[feature_name]
                    current = list(cur_windows[feature_name])

                    psi = self._compute_psi(reference, current)

                    status = "OK"
                    if psi > 0.2:
                        status = "SIGNIFICANT_DRIFT"
                        alert = {
                            "type": "DATA_DRIFT",
                            "model": model_name,
                            "feature": feature_name,
                            "psi": psi,
                            "severity": "HIGH",
                            "timestamp": time.time()
                        }
                        self.drift_alerts.append(alert)
                        results["alerts"].append(alert)
                    elif psi > 0.1:
                        status = "MODERATE_DRIFT"

                    results["feature_drift"][feature_name] = {
                        "psi": psi,
                        "status": status,
                        "reference_mean": round(statistics.mean(reference), 3),
                        "current_mean": round(statistics.mean(current), 3),
                        "reference_std": round(statistics.stdev(reference) if len(reference) > 1 else 0, 3),
                        "current_std": round(statistics.stdev(current) if len(current) > 1 else 0, 3)
                    }

            # Prediction drift
            pred_window = self.prediction_windows.get(model_name)
            if pred_window and len(pred_window) > 200:
                preds = list(pred_window)
                first_half = preds[:len(preds) // 2]
                second_half = preds[len(preds) // 2:]

                pred_psi = self._compute_psi(first_half, second_half)
                pred_mean = statistics.mean(preds)
                pred_std = statistics.stdev(preds) if len(preds) > 1 else 0

                status = "OK"
                if pred_psi > 0.2:
                    status = "SIGNIFICANT_DRIFT"
                elif pred_psi > 0.1:
                    status = "MODERATE_DRIFT"

                results["prediction_drift"] = {
                    "psi": pred_psi,
                    "status": status,
                    "mean": round(pred_mean, 4),
                    "std": round(pred_std, 4),
                    "samples": len(preds)
                }

            return results

    def get_alert_summary(self) -> dict:
        with self._lock:
            return {
                "total_alerts": len(self.drift_alerts),
                "recent_alerts": self.drift_alerts[-10:]
            }

class DeploymentPipeline:
    """
    Automates the model deployment lifecycle.
    
    Pipeline: validate → stage → shadow → canary → promote
    Auto-rollback if metrics degrade during canary.
    """

    def __init__(self, registry: ModelRegistry, router: TrafficRouter):
        self.registry = registry
        self.router = router
        self.deployment_log: List[dict] = []

    def validate_model(self, model_name: str, version: int,
                       validation_data: List[dict]) -> dict:
        """Offline validation before deployment"""
        mv = self.registry.get_version(model_name, version)
        if not mv:
            return {"success": False, "error": "Model version not found"}

        # Check training metrics meet thresholds
        thresholds = {"auc": 0.65, "accuracy": 0.60}
        failures = []

        for metric, threshold in thresholds.items():
            value = mv.training_metrics.get(metric, 0)
            if value < threshold:
                failures.append(f"{metric}={value:.3f} < threshold {threshold}")

        if failures:
            return {
                "success": False,
                "stage": "VALIDATION",
                "failures": failures
            }

        self.deployment_log.append({
            "action": "VALIDATE",
            "model": model_name,
            "version": version,
            "result": "PASSED",
            "timestamp": time.time()
        })

        return {"success": True, "stage": "VALIDATED"}

    def deploy_shadow(self, model_name: str, version: int) -> dict:
        """Deploy model in shadow mode (receives traffic, results not used)"""
        prod = self.registry.get_production_version(model_name)
        if not prod:
            return {"success": False, "error": "No production version to shadow against"}

        self.registry.transition_stage(model_name, version, ModelStage.SHADOW)

        exp = Experiment(
            name=f"shadow_{model_name}_v{version}",
            model_name=model_name,
            control_version=prod.version,
            treatment_version=version,
            treatment_traffic_pct=100,
            experiment_type="shadow"
        )
        self.router.create_experiment(exp)

        self.deployment_log.append({
            "action": "DEPLOY_SHADOW",
            "model": model_name,
            "version": version,
            "timestamp": time.time()
        })

        return {"success": True, "stage": "SHADOW", "experiment": exp.name}

    def deploy_canary(self, model_name: str, version: int,
                      traffic_pct: float = 5.0) -> dict:
        """Deploy model as canary with specified traffic percentage"""
        prod = self.registry.get_production_version(model_name)
        if not prod:
            return {"success": False, "error": "No production version for canary"}

        self.registry.transition_stage(model_name, version, ModelStage.CANARY)

        # Stop any existing shadow experiment
        shadow_name = f"shadow_{model_name}_v{version}"
        self.router.stop_experiment(shadow_name)

        exp = Experiment(
            name=f"canary_{model_name}_v{version}",
            model_name=model_name,
            control_version=prod.version,
            treatment_version=version,
            treatment_traffic_pct=traffic_pct,
            experiment_type="canary"
        )
        self.router.create_experiment(exp)

        self.deployment_log.append({
            "action": "DEPLOY_CANARY",
            "model": model_name,
            "version": version,
            "traffic_pct": traffic_pct,
            "timestamp": time.time()
        })

        return {
            "success": True,
            "stage": "CANARY",
            "traffic_pct": traffic_pct,
            "experiment": exp.name
        }

    def promote_to_production(self, model_name: str, version: int,
                              reason: str = "") -> dict:
        """Promote model to production (100% traffic)"""
        # Stop canary experiment
        canary_name = f"canary_{model_name}_v{version}"
        self.router.stop_experiment(canary_name)

        result = self.registry.transition_stage(
            model_name, version, ModelStage.PRODUCTION,
            reason=reason or "Promoted after successful canary"
        )

        self.deployment_log.append({
            "action": "PROMOTE",
            "model": model_name,
            "version": version,
            "reason": reason,
            "timestamp": time.time()
        })

        return result

    def rollback(self, model_name: str, current_version: int,
                 rollback_version: int, reason: str = "") -> dict:
        """Emergency rollback to previous version"""
        # Stop all experiments for this model
        for exp_name, exp in list(self.router.experiments.items()):
            if exp.model_name == model_name:
                self.router.stop_experiment(exp_name)

        # Archive the bad version
        self.registry.transition_stage(
            model_name, current_version, ModelStage.ARCHIVED,
            reason=f"Rolled back: {reason}"
        )

        # Promote the rollback version
        self.registry.transition_stage(
            model_name, rollback_version, ModelStage.PRODUCTION,
            reason=f"Rollback from v{current_version}: {reason}"
        )

        self.deployment_log.append({
            "action": "ROLLBACK",
            "model": model_name,
            "from_version": current_version,
            "to_version": rollback_version,
            "reason": reason,
            "timestamp": time.time()
        })

        return {
            "success": True,
            "action": "ROLLBACK",
            "from": current_version,
            "to": rollback_version
        }

    def auto_evaluate_canary(self, experiment_name: str) -> dict:
        """Automatically evaluate canary and promote or rollback"""
        analysis = self.router.analyze_experiment(experiment_name)

        if analysis.get("status") == "INSUFFICIENT_DATA":
            return {"action": "WAIT", "reason": "Insufficient data", **analysis}

        exp = self.router.experiments.get(experiment_name)
        if not exp:
            return {"action": "ERROR", "reason": "Experiment not found"}

        if analysis["recommendation"] == "PROMOTE":
            result = self.promote_to_production(
                exp.model_name, exp.treatment_version,
                reason=f"Auto-promoted: lift={analysis['lift']:.4f}, p={analysis['p_value']:.4f}"
            )
            return {"action": "PROMOTED", **analysis, **result}

        elif analysis["recommendation"] == "ROLLBACK":
            result = self.rollback(
                exp.model_name, exp.treatment_version, exp.control_version,
                reason=f"Auto-rollback: negative lift={analysis['lift']:.4f}"
            )
            return {"action": "ROLLED_BACK", **analysis, **result}

        return {"action": "CONTINUE", **analysis}

class ModelServePlatform:
    """
    The complete ML model serving platform.
    Ties together all components.
    """

    def __init__(self):
        self.registry = ModelRegistry()
        self.feature_store = FeatureStore()
        self.router = TrafficRouter(self.registry)
        self.drift_detector = DriftDetector()
        self.pipeline = DeploymentPipeline(self.registry, self.router)

        # Model servers
        self.model_servers: Dict[str, ModelServer] = {}
        self.model_to_server: Dict[str, str] = {}

        # Prediction logging
        self.prediction_log: List[dict] = []
        self.total_predictions = 0
        self._lock = threading.Lock()

    def add_server(self, server_id: str, **kwargs):
        self.model_servers[server_id] = ModelServer(server_id, **kwargs)

    def deploy_model_to_server(self, model_version: ModelVersion,
                               server_id: str, quality: float = 0.5,
                               latency_ms: float = 5.0):
        server = self.model_servers.get(server_id)
        if server:
            server.load_model(model_version, quality, latency_ms)
            key = f"{model_version.model_name}_v{model_version.version}"
            self.model_to_server[key] = server_id

    def predict(self, request: PredictionRequest) -> PredictionResponse:
        """
        Complete prediction flow:
          1. Route to model version (A/B, canary, production)
          2. Fetch features from feature store
          3. Run inference on model server
          4. Log prediction
          5. Return response
        """
        start_time = time.time()

        with self._lock:
            self.total_predictions += 1

        # Step 1: Route to model version
        routing = self.router.route(request.model_name, request.entity_id)
        version = routing.get("version")

        if version is None:
            return PredictionResponse(
                request_id=request.request_id,
                model_name=request.model_name,
                model_version=0,
                prediction=0.5,
                confidence=0.0,
                served_from="fallback"
            )

        # Step 2: Fetch features
        feature_names = list(request.raw_features.keys()) if request.raw_features else []
        stored_features = self.feature_store.get_features(
            request.entity_id, feature_names
        )

        # Merge: request features override stored features
        features = {**stored_features, **request.raw_features}

        # Step 3: Run inference
        model_key = f"{request.model_name}_v{version}"
        server_id = self.model_to_server.get(model_key)
        server = self.model_servers.get(server_id) if server_id else None

        if server:
            result = server.predict(request.model_name, version, features)
            if result:
                prediction, confidence, latency = result
            else:
                prediction, confidence, latency = 0.5, 0.0, 0.0
        else:
            prediction, confidence, latency = 0.5, 0.0, 0.0

        total_latency = (time.time() - start_time) * 1000

        # Step 4: Record for drift detection
        self.drift_detector.record_serving_features(request.model_name, features)
        self.drift_detector.record_prediction(request.model_name, prediction)

        # Step 5: Handle shadow mode
        if routing.get("is_shadow"):
            shadow_version = routing.get("shadow_version")
            shadow_key = f"{request.model_name}_v{shadow_version}"
            shadow_server_id = self.model_to_server.get(shadow_key)
            shadow_server = self.model_servers.get(shadow_server_id) if shadow_server_id else None
            if shadow_server:
                shadow_server.predict(request.model_name, shadow_version, features)

        # Step 6: Build response
        response = PredictionResponse(
            request_id=request.request_id,
            model_name=request.model_name,
            model_version=version,
            prediction=round(prediction, 4),
            confidence=round(confidence, 4),
            latency_ms=round(total_latency, 2),
            experiment=routing.get("experiment", ""),
            variant=routing.get("variant", ""),
            features_used=features,
            served_from=server_id or "fallback"
        )

        # Step 7: Log prediction
        self.prediction_log.append({
            "request_id": request.request_id,
            "model": request.model_name,
            "version": version,
            "prediction": prediction,
            "latency_ms": total_latency,
            "experiment": routing.get("experiment", ""),
            "variant": routing.get("variant", ""),
            "timestamp": time.time()
        })

        return response

    def simulate_outcome(self, prediction: float, model_quality: float) -> int:
        """Simulate whether a prediction led to a positive outcome (click/conversion)"""
        # Higher quality model → predictions correlate more with outcomes
        base_rate = 0.1  # 10% base conversion rate
        model_influence = (prediction - 0.5) * model_quality * 0.3
        probability = max(0.01, min(0.99, base_rate + model_influence))
        return 1 if random.random() < probability else 0

    def get_platform_stats(self) -> dict:
        server_stats = {sid: s.get_stats() for sid, s in self.model_servers.items()}
        return {
            "total_predictions": self.total_predictions,
            "registry": self.registry.get_model_summary(),
            "feature_store": self.feature_store.get_stats(),
            "servers": server_stats,
            "drift_alerts": self.drift_detector.get_alert_summary(),
            "prediction_log_size": len(self.prediction_log)
        }
